OpenFlight API Reference Set 15.0

This is the starting page for the OpenFlight API Reference Set.

The reference set is comprised of two documents:

  • OpenFlight API Reference - describes the functions, types and symbols that make up the OpenFlight API itself.
  • OpenFlight Data Dictionary - describes the record and field codes that compose the OpenFlight data format.

Each of these documents include links to the other.

Below are links to each of these sections (in different formats) as described below.
