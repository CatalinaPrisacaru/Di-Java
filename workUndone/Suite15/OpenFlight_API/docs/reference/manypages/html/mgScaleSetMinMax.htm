mgScaleSetMinMax

FUNCTION NAME

mgScaleSetMinMax - sets the minimum and maximum logical values for a scale control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgScaleSetMinMax (mgguicontrol,
doublemin,
doublemax );

DESCRIPTION

mgScaleSetMinMax sets the range of logical values valid for the specified scale control. The minimum value is specified by min, the maximum by max.

The default minimum and maximum values for a scale control are 0 and 100 respectively.

If the current scale value is outside the new range, the scale value will be clamped to the new maximum or minimum value.

PARAMETERS

control
the scale control
min
the minimum value for the control
max
the maximum value for the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
