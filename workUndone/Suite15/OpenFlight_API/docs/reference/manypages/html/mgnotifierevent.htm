MEMBERS

MNOTIFY_SELECTLISTCHANGED
Select List Change notifier event. This is sent whenever the select list contents change.
Applicable to viewer tools only.
MNOTIFY_NEWTOPDATABASE
New Top Database notifier event. This is sent whenever a new database is selected to be the top (focus) database in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_DATABASECLOSED
Database Close notifier event. This is sent whenever a database is closed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTPRIMARYCOLORCHANGED
Current Primary Color Change notifier event. This is sent whenever the modeler selects a new primary modeling color.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTALTCOLORCHANGED
Current Alternate Color Change notifier event. This is sent whenever the modeler selects a new alternate modeling color.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTTEXTURECHANGED
Current Texture Change notifier event. This is sent whenever the modeler selects a new modeling texture.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTMATERIALCHANGED
Current Material Change notifier event. This is sent whenever the modeler selects a new modeling material.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTTEXTUREMAPPINGCHANGED
Current Texture Mapping Change notifier event. This is sent whenever the modeler selects a new modeling texture mapping.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTLIGHTPOINTAPPEARANCECHANGED
Current Light Point Appearance Change notifier event. This is sent whenever the modeler selects a new modeling light point appearance.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTLIGHTPOINTANIMATIONCHANGED
Current Light Point Animation Change notifier event. This is sent whenever the modeler selects a new modeling light point animation.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTSHADERCHANGED
Current Shader Change notifier event. This is sent whenever the modeler selects a new modeling shader.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTSOUNDCHANGED
Current Sound Change notifier event. This is sent whenever the modeler selects a new modeling sound.
Applicable to viewer and editor tools only.
MNOTIFY_CURRENTLIGHTSOURCECHANGED
Current Light Source Change notifier event. This is sent whenever the modeler selects a new modeling light source.
Applicable to viewer and editor tools only.
MNOTIFY_LIGHTPOINTPALETTEIZED
Light Point Paletteized notifier event. This is sent whenever an inline light point is read and converted to be a paletteized light point.
Applicable to viewer tools only.
MNOTIFY_DESKTOPINIT
Desktop Initialized notifier event. This is sent when the Creator desktop starts after all plug-ins are initialized.
Applicable to all tools.
MNOTIFY_DESKTOPEXIT
Desktop Exit notifier event. This is sent when the Creator desktop exits.
Applicable to all tools.
MNOTIFY_DATABASEOPENED
Database Opened notifier event. This is sent whenever a database is opened in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_DATABASESAVED
Database Saved notifier event. This is sent whenever a database is saved in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_NODECHANGED
Node Changed notifier event. This is sent whenever a node attribute is changed in Creator.
Applicable to viewer tools only.
MNOTIFY_NODEDELETED
Node Deleted notifier event. This is sent whenever a node is deleted in Creator.
Applicable to viewer tools only.
MNOTIFY_NODECREATED
Node Created notifier event. This is sent whenever a new node is created in Creator.
Applicable to viewer tools only.
MNOTIFY_NODEREPARENTED
Node Re-Parented notifier event. This is sent whenever a new node is re-parented in Creator.
Applicable to viewer tools only.
MNOTIFY_COLORPALETTECHANGED
Color Palette Changed notifier event. This is sent whenever a color palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_TEXTUREPALETTECHANGED
Texture Palette Changed notifier event. This is sent whenever a texture palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_MATERIALPALETTECHANGED
Material Palette Changed notifier event. This is sent whenever a material palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_TEXTUREMAPPINGPALETTECHANGED
Texture Mapping Palette Changed notifier event. This is sent whenever a texture mapping palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_LIGHTPOINTAPPEARANCEPALETTECHANGED
Light Point Appearance Palette Changed notifier event. This is sent whenever a light point appearance palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_LIGHTPOINTANIMATIONPALETTECHANGED
Light Point Animation Palette Changed notifier event. This is sent whenever a light point animation palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_SHADERPALETTECHANGED
Shader Palette Changed notifier event. This is sent whenever a shader palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_SOUNDPALETTECHANGED
Sound Palette Changed notifier event. This is sent whenever a sound palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_LIGHTSOURCEPALETTECHANGED
Light Source Palette Changed notifier event. This is sent whenever a light source palette entry is changed in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_SWITCHDISTANCECHANGED
Switch Distance Changed notifier event. This is sent whenever the switch distance is changed in Creator, usually via the More Detail or Less Detail commands.
Applicable to viewer and editor tools only.
MNOTIFY_TIMEOFDAYCHANGED
Time of Day Changed notifier event. This is sent whenever the sky color changes in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_TEXTURESELECTCHANGED
Texture Select Change notifier event. This is sent whenever the textures selected in the palette change in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_EXTENSIONCHANGED
Extension Changed notifier event. This is sent whenever the definition of an OpenFlight extension is changed.
Applicable to all tools.
MNOTIFY_DATABASEPRESAVE
Database Pre-Save notifier event. This is sent just before a database is saved in Creator.
Applicable to viewer and editor tools only.
MNOTIFY_EYEPOINTCHANGED
Eye Point Changed notifier event. This is sent whenever the current eye point changes in Creator.
Applicable to viewer and editor tools only.
