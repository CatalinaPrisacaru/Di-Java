mgIsXformEmpty

FUNCTION NAME

mgIsXformEmpty - determines whether a transformation node has no effect.

SYNOPSIS       (C and Python)

mgbool mgIsXformEmpty (mgrec*rec );

DESCRIPTION

mgIsXformEmpty determines whether a transformation rec has no effect. For example, a translate MXLL_TRANSLATE transformation that has zero delta has no effect and can be safely deleted without changing the position or orientation of the corresponding geometry.

Note: rec is assumed to be a valid transformation record. If it is not, the results of this function are undefined.

PARAMETERS

rec
the transformation node to check

RETURNS

Returns MG_TRUE if the transformation node does not have any effect. MG_FALSE otherwise.

ACCESS LEVEL

Level 1
