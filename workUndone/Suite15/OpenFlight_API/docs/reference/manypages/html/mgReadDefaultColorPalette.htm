mgReadDefaultColorPalette

FUNCTION NAME

mgReadDefaultColorPalette - loads the default color palette from a file.

SYNOPSIS       (C and Python)

mgbool mgReadDefaultColorPalette (const char*fileName );

DESCRIPTION

mgReadDefaultColorPalette loads the default color palette from a file named fileName.

The file fileName was created by mgWriteColorPalette, mgWriteDefaultColorPalette or via the Save Palette command in the color palette in Creator.

PARAMETERS

fileName
the color file name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1

SEE ALSO

mgReadColorPalette, mgWriteDefaultColorPalette
