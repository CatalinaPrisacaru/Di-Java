mgCursorIdsMatch

MACRO NAME

mgCursorIdsMatch - Checks if two cursor identifiers match.

SYNOPSIS

int mgCursorIdsMatch (id1,
id2 );

RETURNS

Returns 1 if identifiers match, 0 otherwise.

PARAMETERS

id1
A cursor identifier
id2
Another cursor identifier

ACCESS LEVEL

Level 4
