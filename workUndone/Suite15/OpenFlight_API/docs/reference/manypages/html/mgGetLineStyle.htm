mgGetLineStyle

FUNCTION NAME

mgGetLineStyle - gets an entry record from a database’s line style palette.

SYNOPSIS       (C and Python)

mgrec* mgGetLineStyle (mgrec*db,
intindex );

DESCRIPTION

mgGetLineStyle gets line style entry record defined by index from the line style palette of db. The record is returned. If the requested line style does not exist, MG_NULL is returned.

The code of the record returned will be fltLineStylePalette .

PARAMETERS

db
the database
index
the palette index of the requested line style palette entry

RETURNS

Returns the line style entry record if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
