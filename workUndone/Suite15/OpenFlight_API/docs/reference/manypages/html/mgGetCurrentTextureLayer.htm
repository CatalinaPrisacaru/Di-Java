mgGetCurrentTextureLayer

FUNCTION NAME

mgGetCurrentTextureLayer - returns the current modeling texture layer.

C SPECIFICATION

mgbool mgGetCurrentTextureLayer (mgrec*db,
int*layer );

PYTHON SPECIFICATION

mgbool, index mgGetCurrentTextureLayer (mgrec*db )

DESCRIPTION

mgGetCurrentTextureLayer returns the modeling texture layer selected by the user for the current graphics view of database db. If a specific layer is selected, then this layer number, 0-7 (0 is the base texture layer) is returned. If "Blend" is selected, a value of -1 is returned.

PARAMETERS

db
the database
layer
address of value to receive layer number

RETURNS

mgbool
MG_TRUE if current modeling texture layer (0-7 or -1 for "Blend") for db could be determined, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

ACCESS LEVEL

Level 4
