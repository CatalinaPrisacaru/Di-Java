mgFlipImage

FUNCTION NAME

mgFlipImage - flips an image about its u-axis.

SYNOPSIS       (C and Python)

int mgFlipImage (unsigned char*pixels,
intwidth,
intheight,
inttype );

DESCRIPTION

mgFlipImage flips an image about its u-axis. This is useful for converting an image with an origin in the upper-left corner to one with an origin in the lower-left corner (the orientation used by MultiGen, IRIS GL and OpenGL). mgFlipImage can also be used to perform the opposite conversion.

See mgReadImage for a description of the expected memory layout for pixels.

PARAMETERS

pixels
a pointer to the image pixels
width
the width of the image
height
the height of the image
type
the image type specifying how the pixels are packed - possible values are MIMG_INT, MIMG_INTA, MIMG_RGB and MIMG_RGBA

RETURNS

Returns MIMG_NO_ERROR if successful, otherwise applicable texture error code.

ACCESS LEVEL

Level 2
