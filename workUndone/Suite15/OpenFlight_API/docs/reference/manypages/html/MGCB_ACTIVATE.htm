MGCB_ACTIVATE

NAME

MGCB_ACTIVATE - Control event Activate

DESCRIPTION

This is the control event that is sent to a control (via the control callback function) when the user activates the control.

When this event is sent to a text control, the callData parameter passed to the callback function will be a pointer to an object of type mgtextactivatecallbackrec.

When this event is sent to a scale control, the callData parameter passed to the callback function will be a pointer to an object of type mgscaleactivatecallbackrec.

When this event is sent to a list control, the callData parameter passed to the callback function will be a pointer to an object of type mglistactivatecallbackrec.

When this event is sent to a tab ruler control, the callData parameter passed to the callback function will be a pointer to an object of type mgtabruleractivatecallbackrec.
