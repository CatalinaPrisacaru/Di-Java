mgSetCursor

FUNCTION NAME

mgSetCursor - sets the cursor shape.

C SPECIFICATION       (Not Available For Python)

mgstatus mgSetCursor (mgcursorcursor );

DESCRIPTION

mgSetCursor sets the cursor shape according to the specified cursor. To restore the cursor to the default shape (arrow), specify cursor as MG_NULL,

PARAMETERS

cursor
the cursor to use

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following example displays the predefined "wait" cursor while it does some processing.
// get the predefined cursor for "wait"
mgcursor waitCursor = mgResourceGetCursor (NULL, MCURS_WAIT);

// set the cursor to show the user the plug-in is busy
mgSetCursor (waitCursor);

// do some processing...

// when done, reset the cursor to the default
mgSetCursor (MG_NULL);

ACCESS LEVEL

Level 4
