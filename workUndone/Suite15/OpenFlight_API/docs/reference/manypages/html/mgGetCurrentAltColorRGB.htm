mgGetCurrentAltColorRGB

FUNCTION NAME

mgGetCurrentAltColorRGB - returns the RGB components of the current alternate modeling color.

C SPECIFICATION

mgbool mgGetCurrentAltColorRGB (mgrec*db,
short*red,
short*green,
short*blue );

PYTHON SPECIFICATION

mgbool, red, green, blue mgGetCurrentAltColorRGB (mgrec*db )

DESCRIPTION

mgGetCurrentAltColorRGB returns the red, green, and blue components of the current alternate modeling color selected for database db. The values returned are in the range 0..255.

PARAMETERS

db
the database
red
address of value to receive red component of current alternate modeling color
green
address of value to receive green component of current alternate modeling color
blue
address of value to receive blue component of current alternate modeling color

RETURNS

mgbool
MG_TRUE if RGB components of current alternate modeling color for db could be determined, MG_FALSE otherwise.
red
If function is successful, red contains the retrieved red component, otherwise red is undefined.
green
If function is successful, green contains the retrieved green component, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved blue component, otherwise blue is undefined.

ACCESS LEVEL

Level 4
