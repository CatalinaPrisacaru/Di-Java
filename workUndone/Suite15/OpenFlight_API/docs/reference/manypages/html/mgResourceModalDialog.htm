mgResourceModalDialog

FUNCTION NAME

mgResourceModalDialog - displays a modal dialog instance derived from a dialog template contained in a resource.

C SPECIFICATION       (Not Available For Python)

int mgResourceModalDialog (mgguigui,
mgresourceresource,
mgdialogiddialogId,
unsigned intcallbackMask,
mgdialogfuncdialogFunc,
void*userData );

DESCRIPTION

mgResourceModalDialog creates and displays a modal dialog instance using a dialog template extracted from the specified resource. The dialog template is identified in the resource by dialogId.

Specify a dialog function dialogFunc for the created dialog and select the dialog events you are interested in for your dialogFunc using the parameter callbackMask.

The dialog is created as a child of the dialog specified by gui. Note that gui can be a control, a dialog or MG_NULL. If gui is a control, it is assumed to be contained in the dialog that is to be the parent. If gui is a dialog, it is the dialog to be the parent. If gui is MG_NULL, the dialog is created as a child of the MultiGen modeling system main window.

The dialog is displayed automatically and this function is returned until the user explicitly dismisses it. When the user dismisses the dialog, the dialog is destroyed.

The user can dismiss the dialog by pressing one of two predefined push buttons or clicking the close icon on the title bar of the dialog. The dialog must include at least one of these controls so that the user can dismiss the dialog. The two predefined push buttons are expected to have the indentifiers MGID_OK and MGID_CANCEL, corresponding to "OK" and "Cancel", respectively.

PARAMETERS

gui
the dialog parent or a control in the dialog that is to be the parent
resource
the resource containing the dialog template
dialogId
the dialog template identifier
callbackMask
bitwise combination of dialog events selected for dialog function dialogfunc - choose any bitwise combination of MGCB_INIT, MGCB_SHOW, MGCB_REFRESH, MGCB_HIDE and MGCB_DESTROY
dialogFunc
dialog function to assign to dialog
userData
user defined data to pass to dialogfunc

RETURNS

Returns 1 if the user dismisses the dialog by pressing the button with the identifier MGID_OK. Returns 2 if the user dismisses the dialog by pressing the button with the identifier MGID_CANCEL or if the user clicked the close icon on the title bar of the dialog.

ACCESS LEVEL

Level 4
