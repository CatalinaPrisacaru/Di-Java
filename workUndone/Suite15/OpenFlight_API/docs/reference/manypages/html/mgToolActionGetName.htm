mgToolActionGetName

FUNCTION NAME

mgToolActionGetName - gets the name of a plug-in tool action.

C SPECIFICATION       (Not Available For Python)

mgstatus mgToolActionGetName (mgtoolactiontoolAction,
char*actionName,
intmaxLen );

DESCRIPTION

mgToolActionGetName returns the name of the plug-in tool action toolAction in the output string actionName. The output string is truncated and null terminated if it is longer than maxLen characters.

PARAMETERS

toolAction
the tool action
actionName
character buffer to hold name of tool action
maxLen
the maximum number of characters to store in actionName

RETURNS

Use MSTAT_ISOK to check return value for success. If successful, the output parameter actionName will contain the name of the plug-in tool action, otherwise it is undefined.

ACCESS LEVEL

Level 4
