MTA_MENULOCATION

NAME

MTA_MENULOCATION - Tool Attribute Menu Location

DESCRIPTION

Specify this tool attribute if you want your viewer or editor tool to be launched from a menu.

This string attribute specifies the menu where the corresponding menu label MTA_MENULABEL is placed for a tool.

You must specify both MTA_MENULOCATION and MTA_MENULABEL if you want your tool to be launched from a menu.

Possible values for this tool attribute are:
MMENU_FILE - File menu
MMENU_EDIT - Edit menu
MMENU_VIEW - View menu
MMENU_INFO - Info menu
MMENU_SELECT - Select menu
MMENU_ATTRIBUTES - Attributes menu
MMENU_LOD - LOD menu
MMENU_LOCALDOF - Local-DOF menu
MMENU_PALETTES - Palette menu
MMENU_SCRIPTS - Palette menu
MMENU_TERRAIN - Terrain menu
MMENU_ROAD - Road menu
MMENU_GEOFEATURE - GeoFeature menu
MMENU_SOUND - Sound menu
MMENU_INSTRUMENTS - Instruments menu
MMENU_BSP - BSP menu
