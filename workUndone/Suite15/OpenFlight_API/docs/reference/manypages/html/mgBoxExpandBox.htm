mgBoxExpandBox

FUNCTION NAME

mgBoxExpandBox - expands a box by another box.

SYNOPSIS       (C and Python)

mgbool mgBoxExpandBox (mgboxd*boxToExpand,
const mgboxd*expandByBox );

DESCRIPTION

mgBoxExpandBox updates a boxToExpand by another box expandByBox. If any part of expandByBox lies outside of boxToExpand, boxToExpand is expanded to contain expandByBox and the function returns MG_TRUE. If expandByBox lies completely within boxToExpand, boxToExpand is not expanded and the function returns MG_FALSE.

PARAMETERS

boxToExpand
address of box to expand
expandByBox
address of box to expand by

RETURNS

Returns MG_TRUE if expandByBox caused boxToExpand to expand (all or part of expandByBox was not contained inside boxToExpand), MG_FALSE otherwise.

ACCESS LEVEL

Level 1
