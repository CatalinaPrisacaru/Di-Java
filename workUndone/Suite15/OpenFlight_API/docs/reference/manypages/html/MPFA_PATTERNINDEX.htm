MPFA_PATTERNINDEX

NAME

MPFA_PATTERNINDEX - File Browser parameter for receiving the index of the selected pattern at the time the user closes the file browser

DESCRIPTION

Specify this as a parameter for mgPromptDialogFile followed by the address of the integer value to receive the selected pattern index.

The index returned will be '1' if the user selected the first pattern specified by the attribute MPFA_PATTERN, '2' if they selected the second, and so on. Since the default "All Files" pattern is always added after the patterns you specify with MPFA_PATTERN, the index returned will one greater than the number of patterns you specified if the user selected the "All Files" pattern. If the user cancels the selection in the file browser, the index returned is not defined.

PYTHON NOTES

This parameter is not supported in Python OpenFlight Script.
