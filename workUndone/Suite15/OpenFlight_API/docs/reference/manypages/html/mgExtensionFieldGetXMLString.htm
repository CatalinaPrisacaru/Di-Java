mgExtensionFieldGetXMLString

FUNCTION NAME

mgExtensionFieldGetXMLString - gets the XML string of an extension field on a node.

C SPECIFICATION

mgbool mgExtensionFieldGetXMLString (mgrec*rec,
const char*fieldGuid,
char**fieldValue );

PYTHON SPECIFICATION

mgbool, fieldValue mgExtensionFieldGetXMLString (mgrec*rec,
const char*fieldGuid )

DESCRIPTION

mgExtensionFieldGetXMLString gets the XML string value of an extension field identified by fieldGuid, attached to node rec.

If successful, the value of the extension field is returned in the output parameter fieldValue, otherwise it is undefined.

Note: If the data type of the extension field is not MTYPE_XML, this function will attempt to coerce the extension field value to string and return the corresponding value in fieldValue. For example, a field of type MTYPE_FLAG will set fieldValue to "0" or "1". A field of type MTYPE_INT, MTYPE_FLOAT or MTYPE_DOUBLE will simply convert the numeric value to string set fieldValue to the result.

C NOTES

The string returned in fieldValue is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

rec
the node to get extension value from
fieldGuid
the GUID of the extension field
fieldValue
the address of value to receive field value

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise. This function will fail if fieldGuid does not specify a valid extension field currently attached to rec or fieldValue is MG_NULL.
fieldValue
If function is successful, fieldValue contains the retrieved value, otherwise it will not be defined.

EXAMPLE

The following examples (in C and Python) illustrate the use of mgExtensionFieldGetXMLString.

C EXAMPLE

char* fieldValue = NULL;
mgbool status;

status = mgExtensionFieldGetXMLString (rec, "732202BF-3EFA-4451-B8A6-B817DDB7BBAE", &fieldValue)

if (MSTAT_ISOK (status)) {
   printf ("Field Value: %s", fieldValue);
   // free allocated memory when done with value
   mgFree (fieldValue);
}

PYTHON EXAMPLE

status, fieldValue = mgExtensionFieldGetXMLString (rec, "732202BF-3EFA-4451-B8A6-B817DDB7BBAE")
if (MSTAT_ISOK (status)):
   print "Field Value:",fieldValue
