mgTabRulerRangeMouseCallback

FUNCTION NAME

mgTabRulerRangeMouseCallback - modifies the mouse handling for a tab ruler control, to select ranges.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTabRulerRangeMouseCallback (mgguigui,
mgcontrolidcontrolId,
mgglmouseactionmouseAction,
void*userData,
void*callData );

DESCRIPTION

mgTabRulerRangeMouseCallback modifies the mouse handling of a tab ruler control by treating the range between tabs as part of the tab. This combined with the mgTabRulerRangeDrawCallback modifies the functionality of a tab ruler control to deal in ranges of values.

PARAMETERS

gui
control in the dialog to hide
controlId
the identifier of gui
mouseAction
the control event that triggered the callback
userData
user data passed through
callData
call data passed through

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
