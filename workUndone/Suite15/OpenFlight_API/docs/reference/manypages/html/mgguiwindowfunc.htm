mgguiwindowfunc

CALLBACK NAME

mgguiwindowfunc - GUI Window function

SYNOPSIS

int (*mgguiwindowfunc) ( mggui control,
mgcontrolid controlId,
mgguihandle guiHandle,
int uMsg,
mguint_ptr wParam,
mglong_ptr lParam,
void* defWindowProc,
void* userData );

DESCRIPTION

This is the signature for Window functions for GUI controls or dialogs.

When you subclass a control or dialog on Windows, using mgControlSubclass or mgDialogSubclass, respectively, you will provide a Window function of this form.

PARAMETERS

control
the control
controlId
the identifier of the control
guiHandle
native control handle, on Windows, this is type HWND
uMsg
the Windows message to process
wParam
Specifies additional message information. The contents of this parameter depend on the value of the uMsg parameter.
lParam
Specifies additional message information. The contents of this parameter depend on the value of the uMsg parameter.
defWindowProc
The old window procedure should you need to call it
userData
user defined data specified when window function was assigned to control

RETURNS

Return 0 is your Window function handles the message.
