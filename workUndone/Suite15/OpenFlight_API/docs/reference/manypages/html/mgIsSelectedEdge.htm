mgIsSelectedEdge

FUNCTION NAME

mgIsSelectedEdge - determines if an edge is selected.

SYNOPSIS       (C and Python)

mgbool mgIsSelectedEdge (mgrec*rec );

DESCRIPTION

mgIsSelectedEdge determines if the specified vertex node record rec is currently selected as an edge.

When a vertex node is selected, it may be selected as a vertex or as an edge. Use this function to distinguish between these two situations.

PARAMETERS

rec
the vertex node to check for selection as an edge

RETURNS

Returns MG_TRUE if rec is a vertex node and is selected as an edge, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
