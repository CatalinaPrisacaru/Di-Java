mgGetImageRowWidth

FUNCTION NAME

mgGetImageRowWidth - gets the padded width of an image row.

C SPECIFICATION

int mgGetImageRowWidth (intwidth,
intsampleSize,
inttype,
int*pad );

PYTHON SPECIFICATION

paddedWidth, padSize mgGetImageRowWidth (intwidth,
intsampleSize,
inttype )

DESCRIPTION

mgGetTextureSize is a convenience function which computes the padded width of an image row.

PARAMETERS

width
the unpadded width of the image
sampleSize
the number of bits per texel per channel
type
the image type
pad
address of value to receive pad value

RETURNS

paddedWidth
The padded width (in bytes).
padSize
The pad size (in bytes).

ACCESS LEVEL

Level 1
