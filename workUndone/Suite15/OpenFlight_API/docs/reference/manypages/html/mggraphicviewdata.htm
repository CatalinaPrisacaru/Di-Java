mggraphicviewdata

STRUCT NAME

mggraphicviewdata - Graphic View data

SYNOPSIS

typedef struct  {
   mgglcontext glContext;
   mggraphicdrawmode drawMode;
   mgbool lightingEnabled;
   mgbool textureEnabled;
   mggraphicdrawreason drawReason;
   void* userData;
   mggraphicview graphicView;
} mggraphicviewdata;

MEMBERS

glContext
mgglcontext - Gl rendering context associated to this graphics view
drawMode
mggraphicdrawmode - draw mode for the graphics view
lightingEnabled
mgbool - MG_TRUE if lighting mode is enabled
textureEnabled
mgbool - MG_TRUE if texture drawing is enabled
drawReason
mggraphicdrawreason - draw reason for the graphics view
userData
void* - user data supplied by mgRegisterNodeDrawGraphic
graphicView
mggraphicview - graphic view

DESCRIPTION

The graphic draw function for an extension node will be passed a pointer to an object of this type. This record contains a GL rendering context object and other objects that indicate the draw state of the graphics view in which the extension node is to be drawn.

SEE ALSO

mgRegisterGraphicDraw, mggraphicdrawfunc
