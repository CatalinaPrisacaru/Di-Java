mgGetCoord3f

FUNCTION NAME

mgGetCoord3f - gets values out of a fltCoord3f record.

C SPECIFICATION

mgbool mgGetCoord3f (mgrec*rec,
mgcodecode,
float*x,
float*y,
float*z );

PYTHON SPECIFICATION

mgbool, x, y, z mgGetCoord3f (mgrec*rec,
mgcodecode )

DESCRIPTION

Given a record rec, mgGetCoord3f gets the values of fltCoord3f attribute record code from rec and stores them in x, y, and z.

You do not have to provide addresses for all x, y, and z values. For example, if you only want to receive the x and y components (but not z), specify valid addresses for x and y and leave z NULL.

PARAMETERS

rec
the record from which to get the fltCoord3f attribute record
code
the fltCoord3f attribute record code
x
address of value to receive x component
y
address of value to receive y component
z
address of value to receive z component

RETURNS

mgbool
MG_TRUE if the coord was retrieved successfully, MG_FALSE otherwise.
x
If function is successful, x contains the retrieved x component, otherwise x is undefined.
y
If function is successful, y contains the retrieved y component, otherwise y is undefined.
z
If function is successful, z contains the retrieved z component, otherwise z is undefined.

ACCESS LEVEL

Level 1
