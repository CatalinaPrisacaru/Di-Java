mgEditorCancelTool

FUNCTION NAME

mgEditorCancelTool - cancels an editor tool instance.

C SPECIFICATION       (Not Available For Python)

mgstatus mgEditorCancelTool (mgeditorcontexteditorContext );

DESCRIPTION

mgEditorCancelTool terminates the editor tool instance identified by editor context, editorContext with the Editor Tool Termination Reason MTRM_CANCEL.

Normally, editor tool instances are terminated when the user presses the Done or Cancel buttons in the tool dialog associated with the editor instance. In some cases, however, the editor tool instance may allow other actions (user or processing) to cause the cancellation of the tool instance. In these cases, the tool can call mgEditorCancelTool to cancel itself.

When an editor tool is canceled in this way, the termination function is called with termination reason MTRM_CANCEL.

PARAMETERS

editorContext
the editor context that identifies the editor instance being canceled

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
