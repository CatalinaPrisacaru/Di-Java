MWALK_MASTER

NAME

MWALK_MASTER - mgWalk flag Walk Master

DESCRIPTION

Setting this mask causes mgWalk, mgWalkEx, mgFind, and mgFindEx to visit referenced nodes. If you set this mask without setting MWALK_MASTERALL, referenced nodes will only be visited one time no matter how many times they are referenced in the database. If you set this mask when using mgWalkRecList, it is only used when traversing the children of the nodes in the record list.

Set MWALK_MASTERALL to visit referenced nodes once each time they are referenced.

By default, referenced nodes are not visited.
