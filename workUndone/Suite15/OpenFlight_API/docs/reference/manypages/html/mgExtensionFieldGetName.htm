mgExtensionFieldGetName

FUNCTION NAME

mgExtensionFieldGetName - gets the name of an extension field.

SYNOPSIS       (C and Python)

char* mgExtensionFieldGetName (const char*fieldGuid );

DESCRIPTION

mgExtensionFieldGetName gets the name of an extension field identified by fieldGuid.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

fieldGuid
the GUID of the field

RETURNS

Returns field name if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
