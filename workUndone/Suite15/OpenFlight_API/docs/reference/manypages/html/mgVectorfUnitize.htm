mgVectorfUnitize

FUNCTION NAME

mgVectorfUnitize - calculates vector coordinates scaled uniformly to have a length of 1.0.

SYNOPSIS       (C and Python)

mgvectorf mgVectorfUnitize (const mgvectorf*vec );

DESCRIPTION

mgVectordUnitize calculates and returns single precision vector coordinates scaled uniformly to have a length of 1.0.

The original vector vec is not affected.

Note: For efficiency, vec is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

vec
vector to unitize

RETURNS

Returns a single precision vector record.

ACCESS LEVEL

Level 1
