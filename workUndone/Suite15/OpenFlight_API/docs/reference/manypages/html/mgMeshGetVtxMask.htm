mgMeshGetVtxMask

FUNCTION NAME

mgMeshGetVtxMask - gets the vertex mask for a mesh.

SYNOPSIS       (C and Python)

unsigned int mgMeshGetVtxMask (mgrec*rec );

DESCRIPTION

This function retrieves the vertex mask for mesh rec. The vertex mask of a mesh is a bitwise combination of Mesh Vertex Mask Bits specifying what kind of data is defined for the vertices in the mesh.

Mesh Vertex Mask Bits include:
MMESH_VTXCOORD
MMESH_VTXCOLOR
MMESH_VTXCOLORRGB
MMESH_VTXNORMAL
MMESH_VTXUV0
MMESH_VTXUV1
MMESH_VTXUV2
MMESH_VTXUV3
MMESH_VTXUV4
MMESH_VTXUV5
MMESH_VTXUV6
MMESH_VTXUV7

PARAMETERS

rec
the fltMesh node

RETURNS

Returns the vertex mask of mesh rec if successful, 0 otherwise.

ACCESS LEVEL

Level 1
