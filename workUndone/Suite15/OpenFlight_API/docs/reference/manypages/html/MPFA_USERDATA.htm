MPFA_USERDATA

NAME

MPFA_USERDATA - File Browser parameter for specifying the user data passed in to you dialog function

DESCRIPTION

Specify this as a parameter for mgPromptDialogFile followed by the user data you want to pass to your dialog function.

Typically you specify this parameter as well as MPFA_RESOURCE, MPFA_DIALOGID, MPFA_DIALOGFUNC, and MPFA_CALLBACKMASK together.

PYTHON NOTES

This parameter is not supported in Python OpenFlight Script.
