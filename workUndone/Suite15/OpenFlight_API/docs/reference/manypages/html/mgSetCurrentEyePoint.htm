mgSetCurrentEyePoint

FUNCTION NAME

mgSetCurrentEyePoint - sets the current eyepoint

SYNOPSIS       (C and Python)

mgbool mgSetCurrentEyePoint (mgrec*db,
mgrec*eyeRec );

DESCRIPTION

mgSetCurrentEyePoint sets the current eyepoint and refreshes the scene. The fltEyePoint record eyeRec must have previously been obtained by mgGetNewEyePoint or mgGetEyePoint

PARAMETERS

db
the database
eyeRec
the eyepoint record to set

RETURNS

Returns MG_TRUE if the current eyepoint for db could be set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgGetCurrentEyePoint, mgGetEyePoint
