mgExit

FUNCTION NAME

mgExit - exits the OpenFlight API software development environment.

SYNOPSIS       (C and Python)

mgbool mgExit ( void )

DESCRIPTION

mgExit is the very last function an application should call. This function performs the proper exit functions, such as closing all database and resource files, housekeeping, and releasing memory. It is a mandatory call to exit the entire programming environment. mgExit does not write or save any databases.

Note: This function is for use in stand-alone applications only and should not be called from inside a plug-in. Doing so may yield undefined results.

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
