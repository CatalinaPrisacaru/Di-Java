mgNewConstructVertex

FUNCTION NAME

mgNewConstructVertex - create a construction vertex node.

SYNOPSIS       (C and Python)

mgrec* mgNewConstructVertex (mgeditorcontexteditorContext,
const mgcoord3d*coord );

DESCRIPTION

mgNewConstructVertex creates a new construction vertex node with the specified coordinates, coord. The new construction node is created in the database associated with the specified editor context, editorContext.

If coord is MG_NULL, the coordinates assigned to the new construction vertex will be (0.0, 0.0, 0.0).

The attributes of the construction vertex node returned can be accessed like any other fltVertex node using mgGetAttList and mgSetAttList. The node returned, however, should not be explicitly attached to other geometry nodes in the database. Doing so yields undefined results.

PARAMETERS

editorContext
the editor context used to identify the database in which the construction vertex is to be created
coord
address of record containing initial coordinates of new construction vertex

RETURNS

Returns construction node created if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
