mgTabRulerRangeDrawCallback

FUNCTION NAME

mgTabRulerRangeDrawCallback - modifies the appearance of a tab ruler control to show ranges.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTabRulerRangeDrawCallback (mgguigui,
mgcontrolidcontrolId,
mgguicallbackreasoncallbackReason,
void*userData,
void*callData );

DESCRIPTION

mgTabRulerRangeDrawCallback modifies the appearance of a tab ruler control by connecting adjacent tabs together with a bracket. This combined with mgTabRulerRangeMouseCallback converts the tab ruler control to deal in ranges of values.

PARAMETERS

gui
control in the dialog to hide
controlId
the identifier of gui
callbackReason
the control event that triggered the callback
userData
user data passed through
callData
call data passed through

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
