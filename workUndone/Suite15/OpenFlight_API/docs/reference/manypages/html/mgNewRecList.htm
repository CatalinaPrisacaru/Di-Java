mgNewRecList

FUNCTION NAME

mgNewRecList - create a record list.

SYNOPSIS       (C and Python)

mgreclist mgNewRecList ( void )

DESCRIPTION

mgNewRecList creates and returns a new record list.

To add records to the list, use mgAppendRecList.

When you are done with the record list, you should dispose of it using mgFreeRecList.

PYTHON NOTES

It is not necessary to dispose of the record list returned in Python.

RETURNS

Returns a new record list if successfully created, MG_NULL otherwise.

ACCESS LEVEL

Level 1
