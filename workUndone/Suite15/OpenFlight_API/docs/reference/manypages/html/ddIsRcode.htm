ddIsRcode

FUNCTION NAME

ddIsRcode - checks if a code is a record code.

SYNOPSIS       (C and Python)

mgbool ddIsRcode (mgcodecode );

DESCRIPTION

ddIsRcode checks if the specified code corresponds to a record. A record code is one defined in the data dictionary by recordDef.

PARAMETERS

code
the code to check

RETURNS

Returns MG_TRUE if the code is a record code, otherwise MG_FALSE.

ACCESS LEVEL

Level 1
