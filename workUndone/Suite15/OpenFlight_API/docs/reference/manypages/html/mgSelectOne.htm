mgSelectOne

FUNCTION NAME

mgSelectOne - select a node record.

SYNOPSIS       (C and Python)

mgstatus mgSelectOne (mgrec*rec );

DESCRIPTION

mgSelectOne selects the specified node record rec.

If selecting the specified node would result in a recursive select list (that is either an ancestor or descendant of rec is already selected), the selection is not allowed.

If any nodes are already selected, they remain selected and the specified rec is added to the select list. To make a node the only node selected, call mgDeselectAll before mgSelectOne.

To select an "edge" in the scene, use mgSelectOneEdge passing the first fltVertex record of the edge. Calling mgSelectOne with the same fltVertex record will select only the vertex.

PARAMETERS

rec
the node to select

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
