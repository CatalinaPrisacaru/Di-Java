mgglmousedatatype

ENUM NAME

mgglmousedatatype - GL Mouse function callback data type

SYNOPSIS

typedef enum mgglmousedatatype {
   MGMCD_COMMON,
   MGMCD_BUTTONDATA,
   MGMCD_DOUBLECLICKDATA,
   MGMCD_MOTIONDATA,
   MGMCD_WHEELDATA,
} mgglmousedatatype;

MEMBERS

MGMCD_COMMON
GL Mouse Common Call Data. This indicates that the callData parameter to a GL mouse function is a pointer to an object of type mgglmousedatarec and has no additional data.
MGMCD_BUTTONDATA
GL Mouse Button Call Data. This indicates that the callData parameter to a GL mouse function is a pointer to an object of type mgglmousebuttondatarec.
MGMCD_DOUBLECLICKDATA
GL Mouse Double Click Call Data. This indicates that the callData parameter to a GL mouse function is a pointer to an object of type mgglmousedoubleclickdatarec.
MGMCD_MOTIONDATA
GL Mouse Motion Call Data. This indicates that the callData parameter to a GL mouse function is a pointer to an object of type mgglmousemotiondatarec.
MGMCD_WHEELDATA
GL Mouse Wheel Call Data. This indicates that the callData parameter to a GL mouse function is a pointer to an object of type mgglmousewheeldatarec.

DESCRIPTION

This type is used to indicate the type of call data passed to GL mouse function. When a GL mouse function is called, the callData parameter passed in will be a pointer to an object type that is dependent on the context of the mouse function.

Each member of this type corresponds to a different GL mouse function call data structure.
