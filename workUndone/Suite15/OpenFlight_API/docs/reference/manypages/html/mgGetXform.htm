mgGetXform

FUNCTION NAME

mgGetXform - gets the first transformation of a node.

SYNOPSIS       (C and Python)

mgrec* mgGetXform (mgrec*rec );

DESCRIPTION

mgGetXform returns the first transformation of the specified node rec. To get successive transformations of a node, use mgGetNext.

PARAMETERS

rec
the node to get first transformation for

RETURNS

Returns the first transformation node if found, MG_NULL otherwise

EXAMPLE

The following examples (in C and Python) traverse each transformation associated to node rec.

C EXAMPLE

mgrec* xForm;
mgxfllcode xType;
if (mgHasXform (rec)) {
   xForm = mgGetXform (rec);
   while (xForm) {
      // Do something with xForm
      xType = mgGetXformType (xForm);
      xForm = mgGetNext (xForm);
   }
}

PYTHON EXAMPLE

if (mgHasXform (rec)):
   xForm = mgGetXform (rec)
   while (xForm != None):
      # Do something with xForm
      xType = mgGetXformType (xForm)
      xForm = mgGetNext (xForm)

ACCESS LEVEL

Level 1
