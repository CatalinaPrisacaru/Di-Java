mgIsFlagOn

FUNCTION NAME

mgIsFlagOn - determines if a node is On.

SYNOPSIS       (C and Python)

mgbool mgIsFlagOn (mgrec*rec );

DESCRIPTION

mgIsFlagOn determines if the node rec is On. Nodes are turned On and Off to distinguish between different levels of detail. Nodes belonging to the current level of detail are, by default, On. Additionally, non-vertex nodes can be explicitly toggled On and Off by the modeler using the Toggle Display command. On the other hand, vertex nodes are always On.

PARAMETERS

rec
the node to check

RETURNS

Returns MG_TRUE if the node is On, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
