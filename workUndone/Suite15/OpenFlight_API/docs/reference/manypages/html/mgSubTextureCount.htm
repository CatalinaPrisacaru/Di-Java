mgSubTextureCount

FUNCTION NAME

mgSubTextureCount - gets the number of subtextures defined in a texture attribute record.

SYNOPSIS       (C and Python)

int mgSubTextureCount (mgrec*imgRec );

DESCRIPTION

mgSubTextureCount returns the number of subtextures in the specified texture attribute record imgRec (of type fltImage ).

PARAMETERS

imgRec
the texture attribute record

RETURNS

Returns the number of subtextures contained in the texture attribute record.

ACCESS LEVEL

Level 1
