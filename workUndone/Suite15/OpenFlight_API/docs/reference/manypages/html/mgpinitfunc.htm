mgpinitfunc

CALLBACK NAME

mgpinitfunc - Plug-in initialization function

SYNOPSIS

mgbool (*mgpinitfunc) ( mgplugin plugin,
int* argc,
char* argv [] );

DESCRIPTION

This is the signature for the plug-in initialization function you must declare in your source code so that your plug-in is properly recognized.

PARAMETERS

plugin
the plug-in module being initialized
argc
reserved for future enhancement
argv []
reserved for future enhancement
