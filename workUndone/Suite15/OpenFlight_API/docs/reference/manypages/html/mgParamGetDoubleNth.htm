mgParamGetDoubleNth

FUNCTION NAME

mgParamGetDoubleNth - gets the nth item of a double precision floating point parameter value in a parameter block.

C SPECIFICATION

mgbool mgParamGetDoubleNth (mgparamblockparamBlock,
const char*paramName,
unsigned intnth,
double*paramValue,
doubledefaultValue );

PYTHON SPECIFICATION

mgbool, paramValue mgParamGetDoubleNth (mgparamblockparamBlock,
const char*paramName )

DESCRIPTION

mgParamGetDoubleNth retrieves the nth item of a double precision floating point parameter value named paramName in the parameter block paramBlock. If the parameter exists and contains an nth item, its value is returned in paramValue. If the parameter does not exist or does not contain an nth item, paramValue is set to defaultValue.

The first item of an array parameter is nth equal to 1.

Note that if the parameter does not exist, it is NOT created in the parameter block.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
nth
which value to return from the array parameter
paramValue
storage location for the parameter value returned
defaultValue
default value returned if parameter does not exist

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
paramValue
If successful, paramValue will be the parameter value, otherwise paramValue will be defaultValue.

ACCESS LEVEL

Level 4
