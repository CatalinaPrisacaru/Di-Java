mgBoxExpandCoord3d

FUNCTION NAME

mgBoxExpandCoord3d - expands a box by a specific coordinate.

SYNOPSIS       (C and Python)

mgbool mgBoxExpandCoord3d (mgboxd*box,
const mgcoord3d*c );

DESCRIPTION

mgBoxExpandCoord3d updates a box by a given coordinate c. If the coordinate lies outside the box, the box is expanded to contain the coordinate and the function returns MG_TRUE. If the coordinate lies inside the box, the box is not expanded and the function returns MG_FALSE.

PARAMETERS

box
address of box to expand
c
address of coordinate to expand box by

RETURNS

Returns MG_TRUE if the coordinate c caused the box to expand (the coordinate was not contained inside the original box), MG_FALSE otherwise.

ACCESS LEVEL

Level 1
