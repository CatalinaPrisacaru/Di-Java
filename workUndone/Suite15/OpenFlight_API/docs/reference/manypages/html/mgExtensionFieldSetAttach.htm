mgExtensionFieldSetAttach

FUNCTION NAME

mgExtensionFieldSetAttach - sets the attach type of an extension field.

SYNOPSIS       (C and Python)

mgbool mgExtensionFieldSetAttach (const char*fieldGuid,
mgcodeattach );

DESCRIPTION

mgExtensionFieldSetAttach sets the attach type of an extension field identified by fieldGuid. The attach type of an extension field identifies the kind of OpenFlight node to which this extension can be associated (or attached). An attach type of fltPolygon , for example, indicates that an extension is associated (or can be attached) to polygon nodes.

PARAMETERS

fieldGuid
the GUID of the site
attach
the attach type to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
