mgHasAtt

FUNCTION NAME

mgHasAtt - determines whether an attribute record exists in its parent record.

SYNOPSIS       (C and Python)

mgbool mgHasAtt (mgrec*parentRec,
mgcodercode );

DESCRIPTION

mgHasAtt checks if an attribute record with code rcode exists in parentRec

PARAMETERS

parentRec
the parent record
rcode
the attribute record code

RETURNS

Returns MG_TRUE if parentRec has an atribute record with code rcode; MG_FALSE otherwise.

ACCESS LEVEL

Level 1
