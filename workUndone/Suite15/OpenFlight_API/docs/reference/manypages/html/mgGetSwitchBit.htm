mgGetSwitchBit

FUNCTION NAME

mgGetSwitchBit - gets the on/off status of a specific bit within a switch mask.

C SPECIFICATION

mgbool mgGetSwitchBit (mgrec*rec,
intmaskNo,
intbitNo,
mgbool*on );

PYTHON SPECIFICATION

mgbool, on mgGetSwitchBit (mgrec*rec,
intmaskNo,
intbitNo )

DESCRIPTION

Given a fltSwitch node, rec, mgGetSwitchBit checks if the bit specified by bitNo within the mask maskNo is on or off. The result is returned via on.

The value of bitNo must be between 0 and the number of children of rec minus 1 (the first child is bitNo 0). Similarly, the value of maskNo must be between 0 and the number of masks of rec minus 1 (the first mask is maskNo 0).

PARAMETERS

rec
the fltSwitch node
maskNo
switch mask number starting from 0
bitNo
bit number starting from 0
on
address of value to receive on/off status

RETURNS

mgbool
MG_TRUE if switch bit retrieved successfully, MG_FALSE otherwise. This function will fail if maskNo does not specify a valid mask or bitNo does not specify a valid bit of the mask of rec.
on
MG_TRUE if switch bit is on, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
