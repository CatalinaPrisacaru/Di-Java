mgglmouseaction

TYPE NAME

mgglmouseaction - GL mouse action

DESCRIPTION

When a GL mouse function is called, the mouseAction parameter passed in of this type will specify which action triggered the function being called.

The mouse actions that are sent to GL mouse functions are:
MGMA_BUTTON
MGMA_DOUBLECLICK
MGMA_MOTION
MGMA_WHEEL
