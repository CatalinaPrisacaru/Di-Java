mgCheckinLicense

FUNCTION NAME

mgCheckinLicense - checks in a license.

C SPECIFICATION       (Not Available For Python)

mgstatus mgCheckinLicense (mglicenselicense );

DESCRIPTION

mgCheckinLicense checks in a license created by mgNewLicense and previously checked out by mgCheckoutLicense.

PARAMETERS

license
the license to check in

RETURNS

Use MSTAT_ISOK to check return value for success. If unsuccessful, value returned will be one of:
1 - license is not a valid license or function was called from stand-alone program environment.
2 - license is not checked out.
3 - caller is not authorized to check in license.
4 - unknown error in the Creator License Manager.

ACCESS LEVEL

Level 4

SEE ALSO

mgCheckoutLicense, mgNewLicense, mgFreeLicense
