mgMatrixFormRotateY

FUNCTION NAME

mgMatrixFormRotateY - forms a rotation matrix for rotation about the Y axis.

C SPECIFICATION

mgbool mgMatrixFormRotateY (mgmatrix*matrix,
doubletheta );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixFormRotateY (doubletheta )

DESCRIPTION

mgMatrixFormRotateY gets a matrix for rotation about the Y axis. The parameter theta specifies the angle (in degrees) to rotate counter-clockwise (right hand rule).

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
theta
angle to rotate measured counter-clockwise in degrees about Y axis (right hand rule)

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix.

ACCESS LEVEL

Level 1
