mgWriteLightSourceFile

FUNCTION NAME

mgWriteLightSourceFile - writes a database’s light source palette to a file.

SYNOPSIS       (C and Python)

mgbool mgWriteLightSourceFile (mgrec*db,
const char*fileName );

DESCRIPTION

mgWriteLightSourceFile writes the light source palette for database db to a file named fileName.

The resulting file can be read into another database using mgReadLightSourceFile or via the Load Palette command in the light source palette in Creator.

PARAMETERS

db
the database
fileName
the name of the file to write

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
