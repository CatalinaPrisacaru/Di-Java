mgParamSetInteger

FUNCTION NAME

mgParamSetInteger - sets an integer parameter value in a parameter block.

SYNOPSIS       (C and Python)

mgbool mgParamSetInteger (mgparamblockparamBlock,
const char*paramName,
intparamValue );

DESCRIPTION

mgParamSetInteger sets the value of an integer parameter named paramName in the parameter block paramBlock to paramValue. If the parameter does not exist, no value is set.

If the parameter is an array type parameter, this function sets the first value in the array and clears all the other values in the parameter.

Note that if the parameter does not exist, it is NOT created in the parameter block.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
paramValue
value to set for paramName

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if paramName is not a parameter in paramBlock or if paramValue is not a valid value for the paramName parameter as defined by its constraints (if any).

ACCESS LEVEL

Level 4
