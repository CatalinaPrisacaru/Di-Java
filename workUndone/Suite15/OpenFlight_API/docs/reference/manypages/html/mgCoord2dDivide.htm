mgCoord2dDivide

FUNCTION NAME

mgCoord2dDivide - calculates the quotient of 2D coordinates and a scalar.

SYNOPSIS       (C and Python)

mgcoord2d mgCoord2dDivide (const mgcoord2d*coord,
doublescalar );

DESCRIPTION

mgCoord2dDivide calculates and returns the quotient of 2D double precision floating point coordinates and a double precision scalar, coord / scalar.

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

coord
address of coordinates to multiply by scalar
scalar
scalar to divide coord by

RETURNS

Returns a double precision 2D coordinate record.

ACCESS LEVEL

Level 1
