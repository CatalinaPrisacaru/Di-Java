mgNewPtrArray

FUNCTION NAME

mgNewPtrArray - allocate a new pointer array.

C SPECIFICATION       (Not Available For Python)

mgptrarray mgNewPtrArray ( void )

DESCRIPTION

mgNewPtrArray allocates a new pointer array and returns its handle. The array created is initially empty.

This function creates and returns a pointer array which can be used as a "dynamic" array. In this way, its length is not statically declared.

Use the function mgFreePtrArray to dispose of the pointer array when you are through using it.

RETURNS

Returns the pointer array if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
