mgRec2Db

FUNCTION NAME

mgRec2Db - gets the database node from any node of a database.

SYNOPSIS       (C and Python)

mgrec* mgRec2Db (mgrec*rec );

DESCRIPTION

mgRec2db returns the database node corresponding to the specified node rec. If the node is not associated with any database, the database node of the current database is returned.

PARAMETERS

rec
the node to check

RETURNS

Returns the database node of the database that contains the node.

C EXAMPLE

The following example prints a message if a polygon node, poly belongs to the current database.
mgrec* curDb;
mgrec* polyDb;
curDb = mgGetCurrentDb ();
polyDb = mgRec2Db (poly);
if (polyDb == curDb)
   printf ("poly belongs to current database");

ACCESS LEVEL

Level 1
