mgCoord2dLerp

FUNCTION NAME

mgCoord2dLerp - calculates the linear interpoloation between two 2D coordinates.

SYNOPSIS       (C and Python)

mgcoord2d mgCoord2dLerp (const mgcoord2d*coord1,
const mgcoord2d*coord2,
doubleamount );

DESCRIPTION

mgCoord2dLerp calculates and returns the linear interpolation between two 2D coordinates coord1 and coord2. The amount parameter is the amount to interpolate between the two coordinates where 0.0 is equal to coord1, 0.1 is very near coord1, 0.5 is half-way in between, etc. If amount is between 0.0 and 1.0, the coordinate returned will lie on the line segment between coord1 and coord2.

The original coordinates coord1 and coord2 are not affected.

Note: For efficiency, coord1 and coord2 are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

coord1
address of coordinates of first coordinate
coord2
address of coordinates of second coordinate
amount
amount to interpolate between coord1 and coord2

RETURNS

Returns a double precision 2D coordinate record.

ACCESS LEVEL

Level 1
