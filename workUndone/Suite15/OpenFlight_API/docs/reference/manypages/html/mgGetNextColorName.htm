mgGetNextColorName

FUNCTION NAME

mgGetNextColorName - gets the name of the next color for a particular color index.

C SPECIFICATION       (Not Available For Python)

char* mgGetNextColorName (mgrec*db,
intindex,
void**ptr );

DESCRIPTION

Each color palette entry can have a list of names associated with it. Given a database node, db, an index into the color palette, index, and the name list, ptr, mgGetNextColorName returns the next name in index’s color name list. If ptr points to MG_NULL, then it will be set to hold an opaque pointer to the name list, and the first name in the list will be returned. Subsequent calls to mgGetNextColorName for the same index can then be made using ptr.

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PARAMETERS

db
the database
index
the color palette index
ptr
an opaque pointer holding the name list (can point to MG_NULL). If points to MG_NULL, returns an opaque pointer to the name list

RETURNS

Returns the next name in the color name list for the given index if successful, MG_NULL otherwise.

C EXAMPLE

The following example iterates all the names associated with color index 4 in a database.
void* nameList = MG_NULL;
int colorIndex = 4;
int n = 1;
char* name;

// get first color name - make sure nameList is MG_NULL
name = mgGetNextColorName (db, colorIndex, &nameList);
while (name != MG_NULL)
{
   // at this point name holds the nth name associated with color index 4
   printf ("Name %d: %s\n", n, name);

   // remember to free the name when you're done with it
   mgFree (name);

   // and go get the next one
   name = mgGetNextColorName (db, colorIndex, &nameList);
   n++;
}

ACCESS LEVEL

Level 1
