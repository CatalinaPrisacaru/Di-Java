mgEditorSetPointFunc

FUNCTION NAME

mgEditorSetPointFunc - sets the 2D point function for an editor tool.

C SPECIFICATION       (Not Available For Python)

void mgEditorSetPointFunc (mgplugintooleditorPluginTool,
mgeditorpointfuncpointFunc );

DESCRIPTION

mgEditorSetPointFunc assigns a 2D point function to the specified editor tool editorPluginTool.

2D point functions are called to notify your editor tool when a 2D screen coordinate has been entered by the user via the mouse.

PARAMETERS

editorPluginTool
the editor plug-in tool
pointFunc
the 2D point input function

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following is a typical 2D point function.
mgEditorSetPointFunc (pluginTool, PointFunc);

static void PointFunc (mgeditorcontext editorContext,
      mgpointinputdata* pointInputData,
      void* toolData)
{
   toolrec* toolRec = (toolrec*) toolData;
   mgmousestate mouseEvent = pointInputData->mouseEvent;
   unsigned keyboardFlags  = pointInputData->keyboardFlags;
   unsigned buttonFlags    = pointInputData->buttonFlags;
   mgcoord2i* thisPoint    = pointInputData->thisPoint;
   mgcoord2i* firstPoint   = pointInputData->firstPoint;

   int leftMouseDown       = buttonFlags & MMB_LEFTMOUSE;
   int middleMouseDown     = buttonFlags & MMB_MIDDLEMOUSE;
   int rightMouseDown      = buttonFlags & MMB_RIGHTMOUSE;

   int ctrlKeyDown         = keyboardFlags & MKB_CTRLKEY;
   int altKeyDown          = keyboardFlags & MKB_ALTKEY;
   int shiftKeyDown        = keyboardFlags & MKB_SHIFTKEY;

   switch (mouseEvent)
   {
   case MMSS_START:
      break;
   case MMSS_STOP:
      break;
   case MMSS_CONTINUE:
      break;
   }
}
