MTA_REPEATABLE

NAME

MTA_REPEATABLE - Tool Attribute Repeatable

DESCRIPTION

This boolean attribute specifies whether or not your editor tool is repeatable. A repeatable tool, after being invoked by the user, is added to the Repeat History in the Edit menu. By default, editor tools are repeatable. To make your editor tool non-repeatable, set this value to MG_FALSE.
