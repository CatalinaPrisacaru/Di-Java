mgGraphicsViewSettingGetMatrix

FUNCTION NAME

mgGraphicsViewSettingGetMatrix - gets a matrix view setting value from the current graphics view.

C SPECIFICATION

mgbool mgGraphicsViewSettingGetMatrix (mgrec*db,
const mgsettingnamesettingName,
mgmatrix*settingValue,
mgmatrixdefaultValue );

PYTHON SPECIFICATION

mgbool, settingValue mgGraphicsViewSettingGetMatrix (mgrec*db,
mgsettingnamesettingName,
mgmatrixdefaultValue )

DESCRIPTION

mgGraphicsViewSettingGetMatrix retrieves an mgmatrix value from a named graphics view setting, settingName, and stores it in settingValue. If settingName does not exist, the value specified by defaultValue is returned.

Currently only a small subset of the graphics view settings are accessible. For a complete list, see mgsettingname.

Note that the prefix of the setting name, either MGVD, MGVI, or MGVM indicates whether the corresponding setting value is double, integer or mgmatrix, respectively. If the value is double, use mgGraphicsViewSettingGetDouble and mgGraphicsViewSettingSetDouble to get and set the value, respectively. If the value is integer, use mgGraphicsViewSettingGetInteger and mgGraphicsViewSettingSetInteger. If the value is mgmatrix, use mgGraphicsViewSettingGetMatrix and mgGraphicsViewSettingSetMatrix.

PARAMETERS

db
the database
settingName
the graphics view setting name
settingValue
storage location for the setting value
defaultValue
default value returned if settingName not found

RETURNS

mgbool
MG_TRUE if there is a graphics view visible in Creator, settingName represents a valid graphics view setting and the value can be represented by a mgmatrix value, MG_FALSE otherwise.
settingValue
If function is successful, settingValue contains the retrieved value, otherwise settingValue is undefined.

ACCESS LEVEL

Level 4
