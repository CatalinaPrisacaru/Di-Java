mgSubTextureDelete

FUNCTION NAME

mgSubTextureDelete - Deletes a subtexture definition with the given index.

SYNOPSIS       (C and Python)

mgbool mgSubTextureDelete (mgrec*imgRec,
intindex );

DESCRIPTION

mgSubTextureDelete deletes a subtexture definition at index index from the specified texture attribute record imgRec (of type fltImage ). Subtexture indices are contiguous and range from 0..N-1 where N is the value returned by mgSubTextureCount.

PARAMETERS

imgRec
the texture attribute record
index
the index to be deleted

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2

SEE ALSO

mgSubTextureAdd, mgSubTextureGet, mgSubTextureCount
