mgListGetSelectedItemCount

FUNCTION NAME

mgListGetSelectedItemCount - counts the number of selected items in a list control.

C SPECIFICATION       (Not Available For Python)

int mgListGetSelectedItemCount (mgguicontrol );

DESCRIPTION

mgListGetSelectedItemCount returns the number of selected items contained in the specified list control. For single select list controls there is at most 1 item selected. For multi-select list controls, there may be 0 or more items selected.

PARAMETERS

control
the list control

RETURNS

Returns the number of selected items contained in the control, 0 if none.

ACCESS LEVEL

Level 4
