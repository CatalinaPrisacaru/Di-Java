mgundocleanupreason

ENUM NAME

mgundocleanupreason - Editor Tool Undo Cleanup Reason type

SYNOPSIS

typedef enum mgundocleanupreason {
   MUCR_AFTERUNDO,
   MUCR_NOUNDO,
} mgundocleanupreason;

MEMBERS

MUCR_AFTERUNDO
Undo cleanup function called after undo function called
MUCR_NOUNDO
Undo cleanup function called without calling undo function
