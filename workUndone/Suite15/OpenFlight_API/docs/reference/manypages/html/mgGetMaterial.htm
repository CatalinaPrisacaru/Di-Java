mgGetMaterial

FUNCTION NAME

mgGetMaterial - gets an entry record from a database’s material palette.

SYNOPSIS       (C and Python)

mgrec* mgGetMaterial (mgrec*db,
intindex );

DESCRIPTION

mgGetMaterial gets the material entry record defined by index from the material palette of database node db.

The code of the record returned will be fltFMaterial .

PARAMETERS

db
the database
index
the material palette index

RETURNS

Returns the material entry record if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1

SEE ALSO

mgIndexOfMaterial, mgNewMaterial
