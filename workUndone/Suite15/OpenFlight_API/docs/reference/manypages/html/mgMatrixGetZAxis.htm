mgMatrixGetZAxis

FUNCTION NAME

mgMatrixGetZAxis - returns the local Z axis of a matrix.

SYNOPSIS       (C and Python)

mgcoord3d mgMatrixGetZAxis (const mgmatrixmatrix );

DESCRIPTION

mgMatrixGetZAxis returns the local Z axis of matrix. The Z axis is composed of the first three elements in the 3rd row of the matrix: matrix[8], matrix[9] and matrix[10].

The original matrix matrix is not affected.

PARAMETERS

matrix
matrix to get the local Z axis of

RETURNS

Returns the local Z axis of matrix.

ACCESS LEVEL

Level 1
