mgRegisterImageImporter

FUNCTION NAME

mgRegisterImageImporter - registers an image importer tool.

C SPECIFICATION       (Not Available For Python)

mgplugintool mgRegisterImageImporter (mgpluginplugin,
const char*toolName,
intimportFlags,
mgimageopenfuncopenFunc,
mgimageclosefunccloseFunc,
mgimagegetinfofuncgetInfoFunc,
mgimagegettexelsfuncgetTexelsFunc,
void*userData,
... );

DESCRIPTION

mgRegisterImageImporter registers an image importer tool for the specified plug-in module plugin with the specified tool attributes.

The required tool attributes include the name of the tool toolName, a start function for the tool, startFunc and user defined callback data, userData (may be MG_NULL) that is passed to the start function when it is called.

Additional required tool attributes include MTA_FILTER and MTA_FILETYPE. Additional optional tool attributes include MTA_VERSION and MTA_HELPCONTEXT. These additional tool attributes are passed using variable argument style and must be terminated with MG_NULL.

PARAMETERS

plugin
the plug-in module registering the tool
toolName
the name of the tool
importFlags
image importer capability flags (reserved for future enhancement)
openFunc
open callback function for image importer
closeFunc
close callback function for image importer
getInfoFunc
get info callback function for image importer
getTexelsFunc
get texels callback function for image importer
userData
user defined data that will be passed to your import action functions openFunc, closeFunc, getInfoFunc and getTexelsFunc when they are called
...
additional viewer tool specific attributes in variable argument style

RETURNS

Pointer to plug-in tool registered if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
