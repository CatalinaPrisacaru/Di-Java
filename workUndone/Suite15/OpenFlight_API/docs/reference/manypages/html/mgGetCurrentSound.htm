mgGetCurrentSound

FUNCTION NAME

mgGetCurrentSound - returns index of current modeling sound.

SYNOPSIS       (C and Python)

mgbool mgGetCurrentSound (mgrec*db,
int*index );

DESCRIPTION

mgGetCurrentSound returns the index of the current modeling sound selected for database db.

PARAMETERS

db
the database
index
address of value to receive index of current modeling sound

RETURNS

Returns MG_TRUE if current modeling sound for db could be determined, MG_FALSE otherwise. If successful, the corresponding index is returned in the output parameter index, otherwise it is undefined.

ACCESS LEVEL

Level 4
