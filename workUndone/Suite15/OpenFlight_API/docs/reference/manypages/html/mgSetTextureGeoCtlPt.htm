mgSetTextureGeoCtlPt

FUNCTION NAME

mgSetTextureGeoCtlPt - sets the control points used to geo reference an image for a georeference info object.

C SPECIFICATION       (Not Available For Python)

mgbool mgSetTextureGeoCtlPt (mgimagegeoinfogeoInfo,
intindex,
doubleimageX,
doubleimageY,
doubleprojX,
doubleprojY );

DESCRIPTION

mgSetTextureGeoCtlPt sets a specified control point, index used to geo reference the image for the specified georeference info object, geoInfo to the specified values, imageX, imageY, projX and projY. The values imageX and imageY specify the location of the control point in image coordinates relative to the origin specified by mgSetTextureGeoImageOrigin. The values projX and projY specify the location of the control point in the projected coordinate system specified by mgSetTextureGeoProjection. If the projection was specified as MIMG_GEOPROJ_GEODETIC, these coordinates will be in degrees Longitude and Latitude respectively. If the projection was specified as MIMG_GEOPROJ_UTM, these coordinates will be in meters of Northing and Easting respectively.

When the image get geo info function, mgimagegetgeoinfofunc is called for an image importer plug-in tool, the get geo info function uses this function to set a control point for the georeference info object.

The default valuea are 0.0, 0.0, 0.0 and 0.0.

PARAMETERS

geoInfo
the georeference info object
index
the index of control point to set
imageX
the image x of the control point
imageY
the image y of control point
projX
the projected x of control point
projY
the projected y of control point

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
