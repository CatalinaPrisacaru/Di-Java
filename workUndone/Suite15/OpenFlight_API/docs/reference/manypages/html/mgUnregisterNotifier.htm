mgUnregisterNotifier

FUNCTION NAME

mgUnregisterNotifier - unregisters a notifier.

C SPECIFICATION       (Not Available For Python)

void mgUnregisterNotifier (mgnotifiernotifier );

DESCRIPTION

mgUnregisterNotifier unregisters the specified notifier. Unregistering a notifier disables and destroys it.

PARAMETERS

notifier
the notifier to unregister

ACCESS LEVEL

Level 4
