mgglmousemotiondatarec

STRUCT NAME

mgglmousemotiondatarec - callback structure for the GL mouse motion action sent to GL controls

SYNOPSIS

typedef struct  {
   mgglmousedatatype callDataType;
   unsigned int keyboardFlags;
   mgcoord2i* thisPoint;
} mgglmousemotiondatarec;

MEMBERS

callDataType
mgglmousedatatype - call data type that indicates the type of call data this object really is - will always be MGMCD_MOTIONDATA
keyboardFlags
unsigned int - the state of the Alt, Ctrl, and Shift keys - the value of this field will be a bitwise combination of MKB_ALTKEY, MKB_CTRLKEY and MKB_SHIFTKEY
thisPoint
mgcoord2i* - address of 2D coordinate record for motion reported

DESCRIPTION

When the MGMA_MOTION mouse action is reported to a GL control via its mgglmousefunc, the calldata parameter passed in will point to an object of this type.

The coordinates of mouse button input are reported relative to the lower left corner of the GL control in which the input was received. That is, if the point reported is at the lower left corner of the control, it will be reported as (0, 0).
