mgAppendRecList

FUNCTION NAME

mgAppendRecList - appends a node onto a record list.

SYNOPSIS       (C and Python)

mgbool mgAppendRecList (mgreclistrecList,
mgrec*node,
mgmatrixmatrix );

DESCRIPTION

mgAppendRecList appends node onto the end of the specified record list recList.

Nodes in a record list can have an optional matrix associated with them. Set matrix to a valid matrix to do this, otherwise set matrix to MG_NULL.

You can retrieve nodes from a record list using mgGetNextRecInList or mgGetNthRecInList. You must call mgResetRecList after appending the last node to the list before retrieving nodes from it.

PARAMETERS

recList
the record list to append to
node
the record to append onto list
matrix
the optional matrix to associate to this record

RETURNS

Returns MG_TRUE if the item was appended successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
