mgtextactivatecallbackrec

STRUCT NAME

mgtextactivatecallbackrec - callback structure for the activate control event sent to text controls

SYNOPSIS

typedef struct  {
   mgguicalldatatype callDataType;
   mgmousestate mouseState;
   mgtextactivatereason activateReason;
} mgtextactivatecallbackrec;

MEMBERS

callDataType
mgguicalldatatype - call data type that indicates the type of call data this object really is - will always be MGCD_TEXTACTIVATE
mouseState
mgmousestate - state of mouse when event was triggered - possible values are:
MMSS_START - The mouse state will be set to this value when the user first presses (and then holds) the left mouse button on either arrow button of the spin buddy control associated to the text control. Note: If the user clicks the left mouse button on the spin buddy control (without holding it down), the event will be treated like a single event and MMSS_NONE will be used.
MMSS_CONTINUE - The mouse state is set to this value as long as the user continues to hold the left mouse button down on either arrow of an associated spin buddy control.
MMSS_STOP - The mouse state is set to this value when the user releases the left mouse button after holding down either arrow of an associated spin buddy control.
MMSS_NONE - The mouse state will be set to this value under either of the following conditons:
When the user enters a numeric value into the text control. -- or --
When the user clicks (but does not hold) the left mouse button on either arrow of the spin buddy control associated to the text control.
activateReason
mgtextactivatereason - reason the activate control event was sent to the text control - possible values are:
MTEXT_LOSTFOCUS - The control lost focus.
MTEXT_TEXTCHANGED - The user typed in the text control.

DESCRIPTION

When the MGCB_ACTIVATE control event is sent to a text control via its mgguifunc, the calldata parameter passed in will point to an object of this type.
