mgSetTextureGeoImageOrigin

FUNCTION NAME

mgSetTextureGeoImageOrigin - sets the image origin attribute for a georeference info object.

C SPECIFICATION       (Not Available For Python)

mgbool mgSetTextureGeoImageOrigin (mgimagegeoinfogeoInfo,
intimageOrigin );

DESCRIPTION

mgSetTextureGeoImageOrigin sets the image origin attribute for the specified georeference info object geoInfo to the specified value imageOrigin. The image origin specifies the origin and the positive y direction used by image geographic control points.

When the image get geo info function, mgimagegetgeoinfofunc is called for an image importer plug-in tool, the get geo info function uses this function to set the image origin attribute for the georeference info object.

Valid values for the image origin attribute are MIMG_GEOIMAGEORIGIN_LL (the image origin is in the lower left corner and y increases toward the top), and MIMG_GEOIMAGEORIGIN_UL (the image origin is in the upper left corner of the image and increases toward the bottom).

The default value is MIMG_GEOIMAGEORIGIN_LL.

PARAMETERS

geoInfo
the georeference info object
imageOrigin
the hemisphere attribute value

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
