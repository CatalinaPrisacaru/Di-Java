mgSelectList

FUNCTION NAME

mgSelectList - selects all node in a record list.

SYNOPSIS       (C and Python)

mgstatus mgSelectList (mgreclistlist );

DESCRIPTION

mgSelectList selects all nodes in list. Any node that was currently selected remains selected. If you want only the nodes in list to be selected, call mgDeselectAll before calling mgSelectList.

PARAMETERS

list
the record list containg the nodes to select

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
