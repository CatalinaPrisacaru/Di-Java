mgGetTextureGeoType

FUNCTION NAME

mgGetTextureGeoType - gets geographic control type from a georeference info object.

C SPECIFICATION

mgbool mgGetTextureGeoType (mgimagegeoinfogeoInfo,
int*type );

PYTHON SPECIFICATION

mgbool, type mgGetTextureGeoType (mgimagegeoinfogeoInfo )

DESCRIPTION

mgGetTextureGeoType gets the geographic control type from the specified georeference info object geoInfo.

Valid values for the geographic control type attribute are MIMG_GEOTYPE_CTRLPT.

PARAMETERS

geoInfo
the georeference info object
type
address of integer to receive geographic control type

RETURNS

mgbool
MG_TRUE if type retrieved successfully, MG_FALSE otherwise.
type
If function is successful, type contains the retrieved projection, otherwise type is undefined.

ACCESS LEVEL

Level 1
