mgAddSwitchMask

FUNCTION NAME

mgAddSwitchMask - adds a new mask for a fltSwitch node.

SYNOPSIS       (C and Python)

int mgAddSwitchMask (mgrec*rec );

DESCRIPTION

A fltSwitch node contains at least one mask which is created automatically when the switch node is created. mgAddSwitchMask allocates a new mask, copies the bit settings of the current mask, and appends the new mask to the end of the mask table for the fltSwitch node.

PARAMETERS

rec
the fltSwitch node

RETURNS

Returns the new mask index if successful, -1 otherwise.

ACCESS LEVEL

Level 2
