mgDestroyDialog

FUNCTION NAME

mgDestroyDialog - destroys a dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgDestroyDialog (mgguidialog );

DESCRIPTION

mgDestroyDialog destroys a dialog.

If the dialog is currently displayed, it is first hidden and then destroyed.

If the destroy dialog event has been selected for the dialog function, it is sent.

PARAMETERS

dialog
dialog to destroy

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
