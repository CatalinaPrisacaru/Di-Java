mgCountAttrChild

FUNCTION NAME

mgCountAttrChild - gets the number of attribute children of a node.

SYNOPSIS       (C and Python)

int mgCountAttrChild (mgrec*rec );

DESCRIPTION

mgCountAttrChild returns the number of attribute children directly below the specified node rec. Descendants of the attribute children are not counted in the total.

PARAMETERS

rec
the node to count attribute children

RETURNS

Returns the number of attribute child nodes immediately below the node.

ACCESS LEVEL

Level 1
