mgGetNthRecInList

FUNCTION NAME

mgGetNthRecInList - returns the nth item in the record list.

C SPECIFICATION

mgrec* mgGetNthRecInList (mgreclistrecList,
mgmatrix*matrix,
intnth );

PYTHON SPECIFICATION

mgrec, mgmatrix mgGetNthRecInList (mgreclistrecList,
intnth )

DESCRIPTION

mgGetNthRecInList returns the item in the specified record list recList that is at position nth. The first item is at position 1.

This function does not affect the traversal pointer of the record list.

If this function is successful and the output parameter matrix is not MG_NULL, it is loaded with the composite matrix representing the cumulative transformations applied to this item. If you do not need this matrix, you may specify matrix as MG_NULL.

PARAMETERS

recList
the record list to traverse
matrix
the address of a matrix to receive the composite of all transformations above the returned record list item
nth
the position in the list of the item to get, 1 (one) is the first item

RETURNS

mgrec
The nth item in the record list, MG_NULL if nth item is not in the list.
mgmatrix
If an item is returned, the output parameter matrix (if specified) is loaded with the composite matrix representing the cumulative transformations applied to this item.

ACCESS LEVEL

Level 1
