mgGetActivationDb

FUNCTION NAME

mgGetActivationDb - retrieves the database from a tool activation object.

C SPECIFICATION       (Not Available For Python)

mgrec* mgGetActivationDb (mgtoolactivationtoolActivation );

DESCRIPTION

mgGetActivationDb retrieves the database associated to the specific tool activation object toolActivation.

When a plug-in tool is launched, the corresponding start function is called. The start function is passed a tool specific callback record that contains a tool activation object. Among other things, this tool activation object contains the top database or focus database for which the tool was launched.

Use this function to extract the focus database from a tool activation object.

PARAMETERS

toolActivation
the tool activation to get database node for

RETURNS

Returns the database referenced in the specified tool activation object if the tool activation is valid, MG_NULL otherwise.

ACCESS LEVEL

Level 4
