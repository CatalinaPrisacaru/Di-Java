mgMeshGetVtxColorRGBA

FUNCTION NAME

mgMeshGetVtxColorRGBA - retrieves the RGBA color values for a vertex in a mesh.

C SPECIFICATION

mgbool mgMeshGetVtxColorRGBA (mgrec*rec,
intvtxIndex,
short*red,
short*green,
short*blue,
short*alpha );

PYTHON SPECIFICATION

mgbool, red, green, blue, alpha mgMeshGetVtxColorRGBA (mgrec*rec,
intvtxIndex )

DESCRIPTION

This function retrieves the red, green, blue, and alpha color values of the vertex at index vtxIndex in the vertex pool of mesh node rec.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
red
address of value to receive red component
green
address of value to receive green component
blue
address of value to receive blue component
alpha
address of value to receive alpha component

RETURNS

mgbool
MG_TRUE if the vertex data was retrieved successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertices in the mesh do not color values.
red
If function is successful, red contains the retrieved red component, otherwise red is undefined.
green
If function is successful, green contains the retrieved green component, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved blue component, otherwise blue is undefined.
alpha
If function is successful, alpha contains the retrieved alpha component, otherwise alpha is undefined.

ACCESS LEVEL

Level 1
