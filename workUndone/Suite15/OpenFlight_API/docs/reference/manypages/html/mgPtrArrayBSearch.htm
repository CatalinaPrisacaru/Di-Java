mgPtrArrayBSearch

FUNCTION NAME

mgPtrArrayBSearch - searches a pointer array for a specific item using a binary search.

C SPECIFICATION       (Not Available For Python)

mgbool mgPtrArrayBSearch (mgptrarrayptrArray,
void*key,
mgptrarraycomparefunccompareFunc,
int*pos );

DESCRIPTION

mgPtrArrayBSearch uses a binary search to locate a specific item key in a pointer array ptrArray.

This function assumes that the pointer array ptrArray is sorted according to the comparison function compareFunc.

PARAMETERS

ptrArray
the pointer array
key
the item to search for
compareFunc
the comparison function for the items in ptrArray
pos
if function successful, contains the index of the item, if function fails, contains the index where the item should be inserted

RETURNS

Returns MG_TRUE if the item was found, otherwise MG_FALSE. If successful, the index where the item was found is returned in the output parameter, pos. If not successful, the index where the item should be inserted is returned in pos.

ACCESS LEVEL

Level 1
