mggraphicdrawfunc

CALLBACK NAME

mggraphicdrawfunc - Graphic Draw function

SYNOPSIS

mgstatus (*mggraphicdrawfunc) ( mgrec* db,
mgrec* rec,
mggraphicviewdata* viewData );

DESCRIPTION

This is the signature for graphic draw (extension node helper) functions. If your data extension declares one or more new node types, you may assign a graphic draw function for each new node type you have. If you have assigned such a function for a node type, it will be called when a node of that type is being drawn in the graphics view.

PARAMETERS

db
the database to which the node belongs
rec
the node to draw
viewData
the graphic rendering context in which the draw function is being called

RETURNS

The value returned by mggraphicdrawfunc controls how to continue drawing the graphic scene with respect to nodes below your extension node. The following are valid return values for the graphic draw function.
MSTAT_OK - continue drawing nodes below the extension node
MGD_DONTDRAWBELOW - do NOT draw nodes below the extension node
