mgGetNextLineStyle

FUNCTION NAME

mgGetNextLineStyle - gets the next entry record from a database’s line style palette.

C SPECIFICATION

mgrec* mgGetNextLineStyle (mgrec*lightSource,
int*index );

PYTHON SPECIFICATION

mgrec, index mgGetNextLineStyle (mgrec*lineStyle )

DESCRIPTION

Given a line style entry record, lineStyle, mgGetNextLineStyle returns the line style entry record following lineStyle in the light source palette. The index of the next line style in the palette is returned in index, if there is one. If there is no next line style entry record, MG_NULL is returned.

The code of the record returned will be fltLineStylePalette .

PARAMETERS

lightSource
the previous line style palette entry record
index
address of value to receive index of entry returned

RETURNS

mgrec
The next line style entry record, or MG_NULL if unsuccessful.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

C EXAMPLE

mgrec* lineStyle;
mgrec* db;
int index;
db = mgOpenDb ("anyfile.flt");
lineStyle = mgGetFirstLineStyle (db, &index);
while (lineStyle != MG_NULL) {
   // do something with lineStyle
   lineStyle = mgGetNextLineStyle (lineStyle, &index);
}

ACCESS LEVEL

Level 1
