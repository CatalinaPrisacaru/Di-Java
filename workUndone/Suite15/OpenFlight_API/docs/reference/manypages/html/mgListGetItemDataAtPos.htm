mgListGetItemDataAtPos

FUNCTION NAME

mgListGetItemDataAtPos - retrieves user defined data associated with an item in a list control.

C SPECIFICATION       (Not Available For Python)

void* mgListGetItemDataAtPos (mgguicontrol,
intitemPos );

DESCRIPTION

Use mgListGetItemDataAtPos to retrieve the user defined data associated with the item at position itemPos of list control control. The data was set using mgListSetItemDataAtPos, mgListAddItemData or mgListAppendItemData.

Position 1 indicates the first item in the control.

PARAMETERS

control
the list control
itemPos
the position of item in list to get data for

RETURNS

Returns the user defined data if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4
