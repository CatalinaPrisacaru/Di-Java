mgFindGuiById

FUNCTION NAME

mgFindGuiById - retrieves handle to a control in a dialog.

C SPECIFICATION       (Not Available For Python)

mggui mgFindGuiById (mgguigui,
mgcontrolidcontrolId );

DESCRIPTION

mgFindGuiById returns the abstract handle to a control identified by controlId contained in the dialog specified by gui.

Note that gui can be either a control or a dialog. If gui is a control, it is assumed to be contained in the dialog to be searched. If gui is a dialog, it is the dialog to be searched.

PARAMETERS

gui
the dialog that contains the control to find or a control in the dialog
controlId
the identifier of the control to find

RETURNS

Returns the control handle if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4
