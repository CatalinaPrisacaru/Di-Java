mgToolActionSetShortCut

FUNCTION NAME

mgToolActionSetShortCut - sets a plug-in tool action shortcut key sequence.

C SPECIFICATION       (Not Available For Python)

mgstatus mgToolActionSetShortCut (mgtoolactiontoolAction,
mgkeysequence*keySequence );

DESCRIPTION

mgToolActionSetShortCut sets the shortcut key sequence for the plug-in tool action toolAction to keySequence.

PARAMETERS

toolAction
the tool action
keySequence
the key sequence to set as the shortcut for this action

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
