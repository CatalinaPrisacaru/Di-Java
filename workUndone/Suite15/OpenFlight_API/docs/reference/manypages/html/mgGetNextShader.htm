mgGetNextShader

FUNCTION NAME

mgGetNextShader - gets the next shader palette entry record from the shader palette.

C SPECIFICATION

mgrec* mgGetNextShader (mgrec*shader,
int*index );

PYTHON SPECIFICATION

mgrec, index mgGetNextShader (mgrec*shader )

DESCRIPTION

Given a shader record pointer shader, mgGetNextShader gets the next shader entry record from the shader palette. The index of the next shader record is returned in index.

The code of the record returned will be fltShaderPalette .

PARAMETERS

shader
the previous shader palette entry record
index
the index of the returned shader palette

RETURNS

mgrec
The next shader entry record, or MG_NULL if unsuccessful.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

ACCESS LEVEL

Level 1
