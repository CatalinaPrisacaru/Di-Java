mgGetMorphVertex

FUNCTION NAME

mgGetMorphVertex - gets the morph vertex for a given vertex.

SYNOPSIS       (C and Python)

mgrec* mgGetMorphVertex (mgrec*rec );

DESCRIPTION

mgGetMorphVertex returns the morph vertex node corresponding to the specified vertex node rec.

PARAMETERS

rec
the vertex node

RETURNS

Returns the morph vertex of rec if it has one, MG_NULL otherwise.

ACCESS LEVEL

Level 1
