mgSetCurrentLookAt

FUNCTION NAME

mgSetCurrentLookAt - sets the current eyepoint in "look at" form.

SYNOPSIS       (C and Python)

mgbool mgSetCurrentLookAt (mgrec*db,
doubleeyex,
doubleeyey,
doubleeyez,
doublecenterx,
doublecentery,
doublecenterz,
doubleupx,
doubleupy,
doubleupz );

DESCRIPTION

mgSetCurrentLookAt sets the current eyepoint using "look at" style parameters and refreshes the scene. The "look at" parameters include 3 sets of values. The first set, eyex, eyey, and eyez is the position of the eye point. The second set, centerx, centery, centerz is the position of the reference point. The third set, upx, upy, and upz is the direction of the up vector.

This function sets the current eyepoint as derived from an eye point, a reference point indicating the center of the scene, and an up vector. The up vector must not be parallel to the line of sight from the eye to the reference point.

PARAMETERS

db
the database
eyex
the x component of the eyepoint position
eyey
the y component of the eyepoint position
eyez
the z component of the eyepoint position
centerx
the x component of the reference point
centery
the y component of the reference point
centerz
the z component of the reference point
upx
the i component of the up vector
upy
the j component of the up vector
upz
the k component of the up vector

RETURNS

Returns MG_TRUE if the current eyepoint for db could be set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
