mgParamAddInteger

FUNCTION NAME

mgParamAddInteger - adds an integer parameter to a parameter block.

C SPECIFICATION       (Not Available For Python)

mgparam mgParamAddInteger (mgparamblockparamBlock,
const char*paramName,
intdefaultValue );

DESCRIPTION

mgParamAddInteger creates a new integer parameter named paramName and adds it to the parameter block paramBlock. The parameter value is initialized with defaultValue.

You can use an integer parameter to represent numeric (the default) or enumerated values. Enumerated integer parameters represent a set (may be non-contiguous) of discrete values, while numeric integer parameters represent a range of whole numbers.

If your integer parameter is enumerated, use mgParamSetEnumerant to define each enumerant.

If your integer parameter is numeric, you can use mgParamSetIntegerMinGT, mgParamSetIntegerMinGE, mgParamSetIntegerMaxLT and mgParamSetIntegerMaxLE to define the valid range for your parameter value.

After the parameter value is defined, use mgParamGetInteger and mgParamSetInteger to query and change its value, respectively.

Note: If the named parameter already exists in paramBlock, it is replaced.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
defaultValue
default value of the parameter

RETURNS

Returns the parameter if created successfully, MG_NULL otherwise.

ACCESS LEVEL

Level 4
