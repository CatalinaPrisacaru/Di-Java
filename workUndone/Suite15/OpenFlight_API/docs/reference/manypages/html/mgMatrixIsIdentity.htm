mgMatrixIsIdentity

FUNCTION NAME

mgMatrixIsIdentity - checks to see if a matrix is the identity matrix.

SYNOPSIS       (C and Python)

mgbool mgMatrixIsIdentity (const mgmatrixmatrix );

DESCRIPTION

mgMatrixIsIdentity checks matrix to see if it is the identity matrix.

PARAMETERS

matrix
matrix to check

RETURNS

Returns MG_TRUE if matrix is the identity matrix, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
