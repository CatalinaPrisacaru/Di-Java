mgGetFirstTextureMapping

FUNCTION NAME

mgGetFirstTextureMapping - gets the first texture mapping in a database’s palette.

C SPECIFICATION

mgbool mgGetFirstTextureMapping (mgrec*db,
int*index,
char*textureMappingName );

PYTHON SPECIFICATION

mgbool, index, name mgGetFirstTextureMapping (mgrec*db )

DESCRIPTION

mgGetFirstTextureMapping gets the index and name of the first texture mapping in a database’s palette.

If you pass MG_NULL for index, the index of the texture mapping palette entry is not returned.

If you pass MG_NULL for textureMappingName, the name of the texture mapping palette entry is not returned. If you do pass a character string for textureMappingName, the buffer should be at least 20 characters.

PARAMETERS

db
the database
index
address of value to receive palette index of entry returned, -1 if none
textureMappingName
address of string to receive texture mapping name

RETURNS

mgbool
MG_TRUE if the first texture mapping palette entry is found, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.
name
If function is successful, name contains the retrieved name, otherwise name is undefined.

C EXAMPLE

This example traverses all the texture mapping entries for database db.
mgbool gotOne;
int index;
char name [200];

gotOne = mgGetFirstTextureMapping (db, &index, name);
while (gotOne == MG_TRUE) {
   // Do something with the texture mapping
   gotOne = mgGetNextTextureMapping (db, &index, name);
}

PYTHON EXAMPLE

This example traverses all the texture mapping entries for database db.
gotOne,index,name = mgGetFirstTextureMapping (db)
while (gotOne == MG_TRUE):
   // Do something with the texture mapping
   gotOne,index,name = mgGetNextTextureMapping (db)

ACCESS LEVEL

Level 1
