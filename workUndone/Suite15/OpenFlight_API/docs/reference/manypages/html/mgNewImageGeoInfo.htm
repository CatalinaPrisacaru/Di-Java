mgNewImageGeoInfo

FUNCTION NAME

mgNewImageGeoInfo - reads the georeference information from an image file.

SYNOPSIS       (C and Python)

mgimagegeoinfo mgNewImageGeoInfo (const char*textureName );

DESCRIPTION

mgNewImageGeoInfo reads the georeference information of an image on disk and returns a georeference info object containing the information read. The parameter textureName is the name of the image file.

This function is similar to mgReadImageGeoInfo except that it returns the georeference info object directly instead of returning it as an output parameter as mgReadImageGeoInfo does. Use mgReadImageGeoInfo if you need the texture error code on failure.

C NOTES

When you are through with the returned georeference info object, you must free it by calling mgFreeImageGeoInfo.

PYTHON NOTES

It is not necessary to free the returned georeference info object in Python.

PARAMETERS

textureName
the name of the image file

RETURNS

Returns georeference info object if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
