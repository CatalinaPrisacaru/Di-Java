mglined

STRUCT NAME

mglined - Double precision line record

SYNOPSIS

typedef struct mglined {
   mgcoord3d p1;
   mgcoord3d p2;
} mglined;

MEMBERS

p1
mgcoord3d - point 1 of line
p2
mgcoord3d - point 2 of line

DESCRIPTION

This record contains 2 3-D coordinates that define a line.
