mgSetPolyColorName

FUNCTION NAME

mgSetPolyColorName - sets the color name of a fltPolygon or fltMesh record.

SYNOPSIS       (C and Python)

mgbool mgSetPolyColorName (mgrec*poly,
const char*name );

DESCRIPTION

Given a fltPolygon or fltMesh record, poly, and a string, name, mgSetPolyColorName stores name as the color name for poly. If the name given is not in the color palette, mgSetPolyColorName returns MG_FALSE.

To clear the color name for a fltPolygon or fltMesh , set name to NULL or the empty string.

PARAMETERS

poly
a fltPolygon record
name
the color name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise..

ACCESS LEVEL

Level 2
