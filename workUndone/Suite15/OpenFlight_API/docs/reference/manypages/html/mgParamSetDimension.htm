mgParamSetDimension

FUNCTION NAME

mgParamSetDimension - defines a parameter in a parameter block to be an "array" type.

C SPECIFICATION       (Not Available For Python)

mgbool mgParamSetDimension (mgparamparam,
unsigned intmaxArrayDimension );

DESCRIPTION

By default, a parameter in a parameter block contains a single value. mgParamSetDimension defines a parameter so it can contain multiple values (an array of values) and sets its maximum dimension. If you want to define a parameter to contain multiple values, use this function to specify how many items the parameter can have using maxArrayDimension. If you specify 0 for maxArrayDimension, the parameter is an array that can contain an unlimited number of items.

PARAMETERS

param
the parameter to set
maxArrayDimension
the maximum number of items this parameter can have, 0 for no maximum

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

The following example creates a parameter that can contain at most 4 3D coordinates:
mgparamblock paramBlock;
mgparam param;

// create a parameter block for our array parameter
paramBlock = mgNewParamBlock();

// add the double3 parameter to hold 3D coordinates
// by default it can contain exactly one item
param = mgParamAddDouble3 (paramBlock, "Coords", 0.0, 0.0, 0.0);

// set the parameter so it can contain up to 4 items (double3 items)
mgParamSetDimension (param, 4);

ACCESS LEVEL

Level 4
