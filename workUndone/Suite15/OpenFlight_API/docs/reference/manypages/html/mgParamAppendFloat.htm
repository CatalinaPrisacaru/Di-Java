mgParamAppendFloat

FUNCTION NAME

mgParamAppendFloat - appends a value to a single precision floating point array type parameter in a parameter block.

SYNOPSIS       (C and Python)

mgbool mgParamAppendFloat (mgparamblockparamBlock,
const char*paramName,
floatparamValue );

DESCRIPTION

mgParamAppendFloat appends the value paramValue to the single precision floating point array type parameter named paramName in the parameter block paramBlock. If the parameter does not exist, no value is appended.

When setting up a single precision floating point array type parameter, use mgParamSetFloat to set the first item in the array. When you call mgParamSetFloat, the parameter array is truncated to a single item. To add the second and subsequent items to the array, use mgParamAppendFloat as needed.

Note that if the parameter does not exist, it is NOT created in the parameter block. Also, if the parameter is not an array type parameter, this function will do nothing.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
paramValue
value to append for paramName

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if paramName is not a parameter in paramBlock, if paramValue is not a valid value for the paramName parameter as defined by its constraints (if any), or if the maximum dimension of paramName would be exceeded by appending this item.

ACCESS LEVEL

Level 4
