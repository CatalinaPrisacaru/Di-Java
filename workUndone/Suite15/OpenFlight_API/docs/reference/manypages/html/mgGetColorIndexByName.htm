mgGetColorIndexByName

FUNCTION NAME

mgGetColorIndexByName - finds the index of a color palette entry from its name.

C SPECIFICATION

mgbool mgGetColorIndexByName (mgrec*db,
const char*name,
int*index );

PYTHON SPECIFICATION

mgbool, index mgGetColorIndexByName (mgrec*db,
stringname )

DESCRIPTION

Given a database node, db, and a color name name, mgGetColorIndexByName sets index to the index of the color palette entry that has name in its name list.

PARAMETERS

db
the database
name
the color name
index
address of value to receive index

RETURNS

mgbool
MG_TRUE if the named color was found, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved color index, otherwise index is undefined.

ACCESS LEVEL

Level 1
