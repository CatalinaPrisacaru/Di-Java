mgGetCurrentGridMatrix

FUNCTION NAME

mgGetCurrentGridMatrix - gets the current modeling grid matrix.

SYNOPSIS       (C and Python)

mgbool mgGetCurrentGridMatrix (mgrec*db,
mgmatrix*matrix );

DESCRIPTION

mgGetCurrentGridMatrix returns a matrix that can be used to normalize the position of the grid in the current Creator graphics view. This matrix, returned in the output parameter, matrix describes the transformation that transforms a coordinate in world space onto the grid in its current position and orientation.

The function mgMatrixInvert can be used to obtain the inverse of this grid matrix. This inverse matrix describes the transformation that moves the grid from its current position and orientation to the database origin aligned with the X and Y axes.

PARAMETERS

db
the database
matrix
address of matrix to receive grid matrix

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. If successful, the matrix will be returned in the output parameter matrix, otherwise it will be undefined.

ACCESS LEVEL

Level 4
