ddGetParentNth

FUNCTION NAME

ddGetParentNth - get the nth legal parent code of a record.

SYNOPSIS       (C and Python)

mgcode ddGetParentNth (mgcodercode,
unsigned intnth );

DESCRIPTION

ddGetParentNth returns the nth legal parent code for a record identified by rcode. The first parent code is specified by nth equal to 0.

PARAMETERS

rcode
the record code
nth
the index of parent code to return

RETURNS

Returns the nth parent code if found, 0 otherwise.

ACCESS LEVEL

Level 1
