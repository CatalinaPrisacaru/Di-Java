mgMatrixRollLeft90

FUNCTION NAME

mgMatrixRollLeft90 - rolls a matrix left 90 degrees in global space.

SYNOPSIS       (C and Python)

void mgMatrixRollLeft90 (mgmatrix*matrix );

DESCRIPTION

mgMatrixRollLeft90 rolls matrix left 90 degrees in global space.

PARAMETERS

matrix
matrix to transform
