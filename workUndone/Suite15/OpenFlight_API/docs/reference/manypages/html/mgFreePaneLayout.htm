mgFreePaneLayout

FUNCTION NAME

mgFreePaneLayout - frees a previously allocated layout.

C SPECIFICATION       (Not Available For Python)

void mgFreePaneLayout (mgpanelayoutlayout );

DESCRIPTION

mgFreePaneLayout frees a previously allocated layout. Each call to mgNewPaneLayout should be matched with a call to mgFreePaneLayout in order to prevent memory leaks.

PARAMETERS

layout
the address of the struct to be freed

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgNewPaneLayout, mgGetPaneLayout, mgSetPaneLayout
