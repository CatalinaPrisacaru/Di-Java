mgSetTextString

FUNCTION NAME

mgSetTextString - assigns a string to a fltText record.

SYNOPSIS       (C and Python)

char* mgSetTextString (mgrec*rec,
const char*name );

DESCRIPTION

Given a text record, rec, mgSetTextString allocates space and copies name into rec's string

PARAMETERS

rec
the record
name
the string to be copied from

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
