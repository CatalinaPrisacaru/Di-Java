MPRIM_TRI_FAN

NAME

MPRIM_TRI_FAN - Mesh Primitive Type Triangle Fan

DESCRIPTION

This is the mesh primitive type defining a triangle fan. A triangle fan represents a series of 3-sided polygons (triangles) defined by the vertices of the mesh primitive.

One triangle is defined for each vertex of the primitive after the first two. For any n (starting at 0), the nth triangle is defined by vertices 0, n+1 and n+2.

If a triangle fan mesh primitive has N vertices, N-2 triangles are defined.
