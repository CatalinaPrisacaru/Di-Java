mgNodeExpressionLoadFromString

FUNCTION NAME

mgNodeExpressionLoadFromString - parses expressionString to generate and return a new mgnodeexpression be right.

SYNOPSIS       (C and Python)

void mgNodeExpressionLoadFromString (const char *expressionString );

DESCRIPTION

mgNodeExpressionSetRight parses expressionString to generate and return a new mgnodeexpression. Assumes the expressionString is in infix order.

PARAMETERS

expressionString
infix expression string to parse

C EXAMPLE

The following example generates an expression that can be used to collect all polygons of material index 2 and alt color -1 in a database.
mgnodeexpression poly = mgNodeExpressionLoadFromString("fltPolyMaterial == 2 && fltPolyAltColor == -1");
