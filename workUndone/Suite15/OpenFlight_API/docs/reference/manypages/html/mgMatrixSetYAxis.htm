mgMatrixSetYAxis

FUNCTION NAME

mgMatrixSetYAxis - sets the local Y axis of a matrix.

SYNOPSIS       (C and Python)

void mgMatrixSetYAxis (mgmatrix*matrix,
const mgcoord3d*coord );

DESCRIPTION

mgMatrixSetYAxis sets the local Y axis of matrix. The Y axis is composed of the first three elements in the 2nd row of the matrix: matrix[4], matrix[5] and matrix[6].

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

matrix
matrix to set the local Y axis of
coord
local Y axis to set to

ACCESS LEVEL

Level 1
