mgGuiSetToolTip

FUNCTION NAME

mgGuiSetToolTip - assigns a tool tip string for a control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGuiSetToolTip (mgguicontrol,
const char*toolTip );

DESCRIPTION

mgGuiSetToolTip will assign the specified tool tip string toolTip to the given control control. This tool tip string will automatically be displayed when the user positions the mouse over the control.

To clear the tool tip string of a control, set toolTip to MG_NULL or the empty string.

PARAMETERS

control
the control whose tool tip will be set
toolTip
the tool tip string

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following example assigns the tool tip to a control using a string that is stored in a plug-in resource. The id of the string is tooltipId.
mggui control;
mgstringid tooltipId;
mgresource resource;  // loaded by mgLoadResource
char toolTip[100];

// load the string from the plug-in module resource
mgResourceGetString (resource, tooltipId, toolTip, 100);
mgGuiSetToolTip (control, toolTip);

// or you can simply pass string literals
mgGuiSetToolTip (control, "Show this when user hovers over control");

ACCESS LEVEL

Level 4
