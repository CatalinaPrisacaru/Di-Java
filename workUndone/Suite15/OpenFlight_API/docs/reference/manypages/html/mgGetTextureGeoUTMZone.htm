mgGetTextureGeoUTMZone

FUNCTION NAME

mgGetTextureGeoUTMZone - gets UTM zone from a georeference info object.

C SPECIFICATION

mgbool mgGetTextureGeoUTMZone (mgimagegeoinfogeoInfo,
int*zone );

PYTHON SPECIFICATION

mgbool, zone mgGetTextureGeoUTMZone (mgimagegeoinfogeoInfo )

DESCRIPTION

mgGetTextureGeoUTMZone gets the UTM zone from the specified georeference info object geoInfo.

Valid values for the UTM zone attribute are 1 through 60.

PARAMETERS

geoInfo
the georeference info object
zone
address of integer to receive UTM zone value

RETURNS

mgbool
MG_TRUE if zone retrieved successfully, MG_FALSE otherwise.
zone
If function is successful, zone contains the retrieved projection, otherwise zone is undefined.

ACCESS LEVEL

Level 1
