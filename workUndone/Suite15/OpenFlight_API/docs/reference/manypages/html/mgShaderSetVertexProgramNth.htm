mgShaderSetVertexProgramNth

FUNCTION NAME

mgShaderSetVertexProgramNth - sets the file name for an existing vertex program on a GLSL shader.

SYNOPSIS       (C and Python)

mgbool mgShaderSetVertexProgramNth (mgrec*shader,
intnth,
const char*programFileName );

DESCRIPTION

mgShaderSetVertexProgramNth sets the file name for the nth vertex program on the specified GLSL shader. The first vertex program is specified by nth equal to 0.

Note: If shader is not a GLSL shader or nth does not refer to an existing vertex progrom on the GLSL shader, this function will fail.

PARAMETERS

shader
the GLSL shader
nth
the index of the GLSL vertex program file to set
programFileName
the GLSL shader vertex program file

RETURNS

Returns MG_TRUE if shader is a GLSL shader and the nth vertex program was set successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
