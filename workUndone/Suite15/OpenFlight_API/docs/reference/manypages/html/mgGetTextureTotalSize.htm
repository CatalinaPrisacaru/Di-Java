mgGetTextureTotalSize

FUNCTION NAME

mgGetTextureTotalSize - gets the total size of the all the textures in a palette.

SYNOPSIS       (C and Python)

int mgGetTextureTotalSize (mgrec*db );

DESCRIPTION

mgGetTextureTotalSize returns the total size (in bytes) of all entries contained in the texture palette of database db.

PARAMETERS

db
the database

RETURNS

Returns the total size of all texture palette entries (in bytes) if successful, or returns -1 otherwise.

ACCESS LEVEL

Level 1

mgGetTextureCount, mgGetTextureSize
