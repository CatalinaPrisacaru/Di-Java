mgSetCompressVertexTableFlag

FUNCTION NAME

mgSetCompressVertexTableFlag - sets the preference that determines if the vertex table of the OpenFlight file is compressed when saved by mgWriteDb, mgSaveAsDb, or mgExportDb.

SYNOPSIS       (C and Python)

void mgSetCompressVertexTableFlag (mgboolflag );

DESCRIPTION

Normally, when an OpenFlight file is saved, the vertices contained in the file are compressed such that only "unique" vertices are written in the table. This results in smaller file sizes but can take much longer to write.

The default value for this preference is MG_TRUE.

If flag is MG_TRUE (the default), vertex tables will be compressed when writing OpenFlight files. If flag is MG_FALSE, vertex tables will not be compressed.

PARAMETERS

flag
whether or not to compress vertex table on write

ACCESS LEVEL

Level 2
