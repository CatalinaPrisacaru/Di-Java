mgSetCoord3f

FUNCTION NAME

mgSetCoord3f - sets all three attribute values in a fltCoord3f record.

SYNOPSIS       (C and Python)

mgbool mgSetCoord3f (mgrec*rec,
mgcodecode,
floatx,
floaty,
floatz );

DESCRIPTION

Given a record rec, and the name of one of its fltCoord3f attribute records, code, mgSetCoord3f sets the x, y, z attribute values in the fltCoord3f attribute record code.

PARAMETERS

rec
the record whose fltCoord3f record is to be set
code
the name of the fltCoord3f attribute record of rec
x
the x value to set in the fltCoord3f attribute record
y
the y value to set in the fltCoord3f attribute record
z
the z value to set in the fltCoord3f attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
