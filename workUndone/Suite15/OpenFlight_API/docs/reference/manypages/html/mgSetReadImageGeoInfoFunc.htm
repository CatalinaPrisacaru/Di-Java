mgSetReadImageGeoInfoFunc

FUNCTION NAME

mgSetReadImageGeoInfoFunc - sets a get geoinfo function. for an image importer plug-in tool.

SYNOPSIS       (C and Python)

mgstatus mgSetReadImageGeoInfoFunc (mgpluginplugin,
mgplugintoolpluginTool,
mgimagegetgeoinfofuncgetGeoInfoFunc,
void*userData );

DESCRIPTION

When you register an image importer plug-in tool, you can add a get geoinfo function using mgSetReadImageGeoInfoFunc. The function you register is called when the OpenFlight API needs to get georeferencing information about an image that your tool has opened.

PARAMETERS

plugin
the plug-in module that registered the image importer plug-in tool
pluginTool
the image importer plug-in tool to which the get geoinfo function is to be added
getGeoInfoFunc
the get geoinfo function
userData
user defined data that will be passed to getGeoInfoFunc when it is called

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
