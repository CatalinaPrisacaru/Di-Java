mgSetPolyColorRGB

FUNCTION NAME

mgSetPolyColorRGB - sets the primary RGB color values in a fltPolygon or fltMesh record.

SYNOPSIS       (C and Python)

mgbool mgSetPolyColorRGB (mgrec*rec,
shortred,
shortgreen,
shortblue );

DESCRIPTION

Given a fltPolygon or fltMesh record, rec, this function sets the red, green, and blue primary color values of rec to the given red, green, and blue values.

Note that if the database containing rec is color index mode, this function will convert the given color values to the closest index and intensity pair and set the fltPolyPrimeColor and fltPolyPrimeIntensity attributes of rec accordingly.

PARAMETERS

rec
the fltPolygon or fltMesh record on which to set the color values
red
the red value to set in the fltPolyPrimeColor attribute record
green
the green value to set in the fltPolyPrimeColor attribute record
blue
the blue value to set in the fltPolyPrimeColor attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

mgrec* poly;
mgrec* mesh;
short r, g, b;
mgbool status;

r = g = b = 50;
poly = mgNewRec (fltPolygon);
mesh = mgNewRec (fltMesh);
status = mgSetPolyColorRGB (poly, r, g, b);
status = mgSetPolyColorRGB (mesh, r, g, b);

ACCESS LEVEL

Level 2
