mgSetPolyCurrentMaterial

FUNCTION NAME

mgSetPolyCurrentMaterial - sets the material attribute in a fltPolygon or fltMesh record to the current modeling material.

SYNOPSIS       (C and Python)

mgbool mgSetPolyCurrentMaterial (mgrec*db,
mgrec*rec );

DESCRIPTION

Given a fltPolygon or fltMesh record, rec, this function sets its material attribute (fltPolyMaterial) according to the current modeling material selected by the user in Creator for database db.

PARAMETERS

db
the database whose current modeling material will be used
rec
the fltPolygon or fltMesh record on which to set the material attribute

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

mgrec* poly;
mgbool status;

poly = mgNewRec (fltPolygon);
// set material attribute to the current material in the palette
status = mgSetPolyCurrentMaterial (db, poly);

ACCESS LEVEL

Level 4
