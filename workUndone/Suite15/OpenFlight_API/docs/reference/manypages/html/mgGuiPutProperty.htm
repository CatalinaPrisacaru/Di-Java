mgGuiPutProperty

FUNCTION NAME

mgGuiPutProperty - associates user defined data with a gui item.

C SPECIFICATION       (Not Available For Python)

mgbool mgGuiPutProperty (mgguigui,
const mgpropertynamepropName,
mgpropertyvaluepropValue );

DESCRIPTION

mgGuiPutProperty allows you to associate data of any type, propValue, with a dialog or control. The user defined data is associated with a gui item, gui, and identified by a property name string, propName, allowing any number of data records to be associated with the gui item, as long as all property names are unique.

After using this function to associate user data to a gui item, use the function mgGuiGetProperty to retrieve the data. Then when you no longer need the data associated to the gui item, use mgGuiDeleteProperty to disassociate the data from the gui item. Note that mgGuiDeleteProperty does not free or otherwise affect the actual data that was associated. If this data was dynamically allocated, it is the callers responsibility to deallocate it.

PARAMETERS

gui
the gui item to put property on
propName
the name of the property to assign
propValue
the value of the property to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

The following example associates a string property to a control in a dialog, retrieves the string and prints it out. Finally, it deletes the property from the control.
#define PROPERTY_NAME   "My String Property"
#define USER_STRING     "This String is associated to my control"

char* string;

// allocate the string to be associated to the control
string = mgMalloc (strlen (USER_STRING) + 1);
strcpy (string, USER_STRING);

// associate the string as a property of the control
mgGuiPutProperty (gui, PROPERTY_NAME, string);

// Then later, retrieve the string property, print it out

char* string;

// Retrieve the string property from the control
string = (char*) mgGuiGetProperty (gui, PROPERTY_NAME);
printf ("%s: %s", PROPERTY_NAME, string);

// Output is:
// My String Property: This String is associated to my control

// Then finally, when you no longer need the string to be
// associated to the control

char* string;

// Retrieve the string property from the control (need to free it)
string = (char*) mgGuiGetProperty (gui, PROPERTY_NAME);
mgFree (string);
mgGuiDeleteProperty (gui, PROPERTY_NAME);
