mgMakeBox

FUNCTION NAME

mgMakeBox - makes a box given two (min and max) points.

SYNOPSIS       (C and Python)

mgboxd mgMakeBox (const mgcoord3d*min,
const mgcoord3d*max );

DESCRIPTION

mgMakeBox returns a double precision axis aligned box record representing the box whose lower left front point is min and whose upper right back point is max.

PARAMETERS

min
address of coordinate that defines the lower left point of the box
max
address of coordinate that defines the upper right back point of the box

RETURNS

Returns a double precision box record.

ACCESS LEVEL

Level 1
