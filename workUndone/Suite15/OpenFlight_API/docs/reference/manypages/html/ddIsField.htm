ddIsField

FUNCTION NAME

ddIsField - checks if a field code is a valid field of a record.

SYNOPSIS       (C and Python)

mgbool ddIsField (mgcodercode,
mgcodefcode );

DESCRIPTION

ddIsField checks if the specified field code fcode is a valid element contained in the specified record code rcode.

PARAMETERS

rcode
the record code to be searched
fcode
the field code to be matched

RETURNS

Returns MG_TRUE if fcode is found in the record, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
