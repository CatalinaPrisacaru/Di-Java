mgParamAddDouble

FUNCTION NAME

mgParamAddDouble - adds a double precision floating point parameter to a parameter block.

C SPECIFICATION       (Not Available For Python)

mgparam mgParamAddDouble (mgparamblockparamBlock,
const char*paramName,
doubledefaultValue );

DESCRIPTION

mgParamAddDouble creates a new double precision floating point parameter named paramName and adds it to the parameter block paramBlock. The parameter value is initialized with defaultValue.

You can use mgParamSetDoubleMinGT, mgParamSetDoubleMinGE, mgParamSetDoubleMaxLT and mgParamSetDoubleMaxLE to define the valid range for your parameter value.

After the parameter value is defined, use mgParamGetDouble and mgParamSetDouble to query and change its value, respectively.

Note: If the named parameter already exists in paramBlock, it is replaced.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
defaultValue
default value of the parameter

RETURNS

Returns the parameter if created successfully, MG_NULL otherwise.

ACCESS LEVEL

Level 4
