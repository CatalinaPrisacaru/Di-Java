mgkeysequence

STRUCT NAME

mgkeysequence - Keyboard key sequence used by shortcuts.

SYNOPSIS

typedef struct  {
   unsigned int keyboardFlags;
   mgkeytype key;
} mgkeysequence;

MEMBERS

keyboardFlags
unsigned int - the state of the Alt, Ctrl, and Shift keys for the key sequence - the value of this field will be a bitwise combination of MKB_ALTKEY, MKB_CTRLKEY and MKB_SHIFTKEY
key
mgkeytype - the key code for the key sequence

DESCRIPTION

This structure represents the state of the keyboard when a shortcut is entered by the user.
