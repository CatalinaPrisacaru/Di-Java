mgNewPaneLayout

FUNCTION NAME

mgNewPaneLayout - allocates a new mgpanelayout record.

C SPECIFICATION       (Not Available For Python)

mgpanelayout mgNewPaneLayout ( void )

DESCRIPTION

mgNewPaneLayout allocates a new mgpanelayout record that can be used to store and restore pane settings. Pane settings constitute location of splitters and the pane type (graphics view, structure view, etc). Each call to mgNewPaneLayout should be matched with a call to mgFreePaneLayout in order to prevent memory leaks.

RETURNS

If successful a new mgpanelayout record is returned, MG_NULL otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgFreePaneLayout, mgGetPaneLayout, mgSetPaneLayout
