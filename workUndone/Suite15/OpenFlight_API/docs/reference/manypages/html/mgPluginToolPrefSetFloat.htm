mgPluginToolPrefSetFloat

FUNCTION NAME

mgPluginToolPrefSetFloat - sets a float preference value in the plug-in tool preferences.

C SPECIFICATION       (Not Available For Python)

mgbool mgPluginToolPrefSetFloat (mgplugintoolplugintool,
const char*prefName,
floatprefValue );

DESCRIPTION

mgPluginToolPrefSetFloat stores a float value for a named preference key, prefName for pluginTool. If the key prefName does not exist for the pluginTool, it is created with a value of prefValue.

Preference keys created using this function are stored relative to the individual plug-in tools within the plug-in library, and are separate from the keys used for entire plug-in library.

PARAMETERS

plugintool
the plug-in tool handle
prefName
the preference key name
prefValue
value to store for prefName

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
