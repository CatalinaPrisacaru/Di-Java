mgMatrixFormReflect

FUNCTION NAME

mgMatrixFormReflect - creates a matrix that will reflect about a specified axis.

C SPECIFICATION

void mgMatrixFormReflect (mgmatrix*matrix,
const mgcoord3d*coord );

PYTHON SPECIFICATION

mgmatrix mgMatrixFormReflect (mgcoord3dcoord )

DESCRIPTION

mgMatrixFormReflect gets a matrix that will reflect about coord.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
coord
address of coordinates of axis to reflect about, should have a length above zero

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
