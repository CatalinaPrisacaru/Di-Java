mgDrawControl

FUNCTION NAME

mgDrawControl - queues a GL control for drawing.

C SPECIFICATION       (Not Available For Python)

mgstatus mgDrawControl (mgguicontrol );

DESCRIPTION

mgDrawControl queues the specified GL control control for drawing.

If the draw control event has been selected for the control callback of the specified GL control, the draw event is sent to the control at the time the queue is serviced. The draw usually occurs immediately after the function scope in which mgDrawControl is called exits but this is not guaranteed. If you want the control to draw immediately, use mgDrawControlNow.

PARAMETERS

control
the GL control to draw

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4

SEE ALSO

mgDrawControlNow, mgSetGuiCallback, MGCB_DRAW
