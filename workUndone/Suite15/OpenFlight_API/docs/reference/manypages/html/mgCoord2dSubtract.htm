mgCoord2dSubtract

FUNCTION NAME

mgCoord2dSubtract - calculates the difference of two 2D coordinates.

SYNOPSIS       (C and Python)

mgcoord2d mgCoord2dSubtract (const mgcoord2d*coord1,
const mgcoord2d*coord2 );

DESCRIPTION

mgCoord2dSubtract calculates and returns the difference of two 2D double precision floating point coordinates, coord1 - coord2.

The original coordinates coord1 and coord2 are not affected.

Note: For efficiency, coord1 and coord2 are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

coord1
address of coordinate to subtract coord2 from
coord2
address of coordinate to subtract coord1 by

RETURNS

Returns a double precision 2D coordinate record.

ACCESS LEVEL

Level 1
