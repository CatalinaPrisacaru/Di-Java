mgPtrArrayReplace

FUNCTION NAME

mgPtrArrayReplace - replace an item in a pointer array.

C SPECIFICATION       (Not Available For Python)

void mgPtrArrayReplace (mgptrarrayptrArray,
intpos,
void*item );

DESCRIPTION

mgPtrArrayReplace replaces the item at position pos in the pointer array ptrArray with the specified item.

The first item in a pointer array is at index 1. Therefore, to replace the first item of the array, specify pos as 1. To replace the item at the end of the array, specify pos as the length of the pointer array.

PARAMETERS

ptrArray
the pointer array
pos
the position of the item in the array to replace (in range 1..length)
item
the item that is to replace the item at pos in the pointer array

ACCESS LEVEL

Level 1
