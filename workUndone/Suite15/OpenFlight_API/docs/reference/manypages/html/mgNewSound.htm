mgNewSound

FUNCTION NAME

mgNewSound - allocates a new record for a sound palette entry.

C SPECIFICATION

mgrec* mgNewSound (mgrec*db,
char*name,
int*index );

PYTHON SPECIFICATION

mgrec, index mgNewSound (mgrec*db,
stringname )

DESCRIPTION

Given a database node db, mgNewSound allocates a sound record with the name given and puts it in the database’s sound palette. If successful, the sound record is returned. If unsuccessful, MG_NULL is returned. This sound attributes can then be set by using mgSetAttList with the returned record. The index of the entry in the sound palette is returned in index.

The code of the record returned will be fltSoundPalette .

PARAMETERS

db
the database
name
the name of the sound entry record
index
the returned index of the sound entry record

RETURNS

mgrec
Returns the new sound palette entry record if successful, MG_NULL otherwise.
index
If a palette record is returned, its index will be returned in index, otherwise index is undefined.

C EXAMPLE

mgrec* sound;
int index;
mgrec* db;
db = mgNewDb ("newfile.flt");
sound = mgNewSound (db, "NewSound", &index);

ACCESS LEVEL

Level 2
