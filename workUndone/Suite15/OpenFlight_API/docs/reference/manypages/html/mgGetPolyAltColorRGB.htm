mgGetPolyAltColorRGB

FUNCTION NAME

mgGetPolyAltColorRGB - gets the alternate RGB color values out of a fltPolygon or fltMesh record.

C SPECIFICATION

mgbool mgGetPolyAltColorRGB (mgrec*rec,
short*red,
short*green,
short*blue );

PYTHON SPECIFICATION

mgbool, red, green, blue mgGetPolyAltColorRGB (mgrec*rec )

DESCRIPTION

Given a fltPolygon or fltMesh record, rec, this function gets the red, green, and blue values of its fltPolyAltColor attribute.

Colors set in index mode are automatically converted to RGB values. The color values returned are in the range 0..255.

PARAMETERS

rec
the fltPolygon or fltMesh record from which to get the color values
red
the red component
green
the green component
blue
the blue component

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
red
If function is successful, red contains the retrieved value, otherwise red is undefined.
green
If function is successful, green contains the retrieved value, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved value, otherwise blue is undefined.

C EXAMPLE

mgrec* db;
mgrec* rec;
short r, g, b;
mgbool status;

db = mgOpenDb ("file.flt");

// create polygon or Mesh rec
rec = UserMakePolyOrMeshRec (db);

status = mgGetPolyAltColorRGB (rec, &r, &g, &b);
if (status == MG_TRUE) {
   // r, g and b contain the color components
}

ACCESS LEVEL

Level 1
