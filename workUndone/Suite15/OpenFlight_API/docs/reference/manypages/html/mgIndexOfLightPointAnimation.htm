mgIndexOfLightPointAnimation

FUNCTION NAME

mgIndexOfLightPointAnimation - gets the index of a named entry in the light point appearance palette.

SYNOPSIS       (C and Python)

int mgIndexOfLightPointAnimation (mgrec*db,
const char*name );

DESCRIPTION

mgIndexOfLightPointAnimation returns the index of the entry named name in the light point appearance palette for database db. If the entry is not found, -1 is returned.

PARAMETERS

db
the database
name
the name of the entry to look up

RETURNS

Returns the index of the palette entry if found, -1 otherwise.

ACCESS LEVEL

Level 1
