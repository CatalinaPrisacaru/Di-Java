mgGuiGetViewSize

FUNCTION NAME

mgGuiGetViewSize - retrieves the view dimensions of a control or dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGuiGetViewSize (mgguigui,
int*width,
int*height );

DESCRIPTION

mgGuiGetViewSize returns the view width and/or height of the specified control or dialog, gui, measured in pixels.

For both controls and dialogs, the dimensions returned define the "inner" or "view" dimensions of the gui item. The view dimensions do not include the margin, or in the case of dialogs, the title bar dimensions. For most controls, the view dimensions are identical to the outer dimensions of the control. For dialogs, the view dimensions are almost always smaller than the view dimensions. As stated above, this is due to margins and the title bar of the dialog.

If either of the dimension parameters are MG_NULL, that particular dimension will not be returned.

PARAMETERS

gui
the control or dialog whose view dimensions are to be returned
width
address of value to receive width dimension
height
address of value to receive height dimension

ACCESS LEVEL

Level 4

SEE ALSO

mgGuiGetSize, mgGuiSetSize, mgGuiGetPos, mgGuiSetPos

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
width
If function is successful, width contains the width, otherwise undefined.
height
If function is successful, height contains the height, otherwise undefined.
