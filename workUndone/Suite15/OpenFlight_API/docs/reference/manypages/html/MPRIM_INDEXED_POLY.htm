MPRIM_INDEXED_POLY

NAME

MPRIM_INDEXED_POLY - Mesh Primitive Type Indexed Poly

DESCRIPTION

This is the mesh primitive type defining an indexed polygon. An indexed polygon represents a single N-sided polygon defined by the vertices of the mesh primitive.

One polygon is defined per indexed polygon mesh primitive. If an indexed polygon mesh primitive has N vertices, 1 N-sided closed polygon or 1 (N-1)-sided unclosed polygon is defined.
