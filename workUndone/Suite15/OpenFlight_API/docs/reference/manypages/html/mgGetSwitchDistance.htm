mgGetSwitchDistance

FUNCTION NAME

mgGetSwitchDistance - gets the current switch distance for a database

C SPECIFICATION

mgbool mgGetSwitchDistance (mgrec*db,
double*distance );

PYTHON SPECIFICATION

mgbool, distance mgGetSwitchDistance (mgrec*db )

DESCRIPTION

mgGetSwitchDistance gets the current switch distance for the database db.

PARAMETERS

db
the database
distance
address of value to receive distance

RETURNS

mgbool
MG_TRUE if distance retrieved successfully, MG_FALSE otherwise.
distance
If function is successful, distance contains the retrieved switch distance, otherwise distance is undefined.

ACCESS LEVEL

Level 1
