mgGetAttrChild

FUNCTION NAME

mgGetAttrChild - gets the attribute child of a node.

SYNOPSIS       (C and Python)

mgrec* mgGetAttrChild (mgrec*rec );

DESCRIPTION

mgGetAttrChild returns the attribute child of the specified node rec.

PARAMETERS

rec
the node to get attribute child for

RETURNS

Returns the node if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
