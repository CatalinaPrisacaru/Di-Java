mgFreePtrStack

FUNCTION NAME

mgFreePtrStack - frees a pointer stack.

C SPECIFICATION       (Not Available For Python)

void mgFreePtrStack (mgptrstackptrStack );

DESCRIPTION

mgFreePtrStack frees the dynamic memory associated with the pointer stack ptrStack.

Note that this function does not free or otherwise affect the actual items that may be in the stack when called. If the stack contains any items that were dynamically allocated, it is the callers responsibility to deallocate them.

The pointer stack is assumed to have been created using the function mgNewPtrStack.

PARAMETERS

ptrStack
the pointer stack

ACCESS LEVEL

Level 1
