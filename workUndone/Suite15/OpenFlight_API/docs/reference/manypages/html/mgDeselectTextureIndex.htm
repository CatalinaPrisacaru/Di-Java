mgDeselectTextureIndex

FUNCTION NAME

mgDeselectTextureIndex - de-selects a texture in the palette.

SYNOPSIS       (C and Python)

mgstatus mgDeselectTextureIndex (mgrec*db,
intindex );

DESCRIPTION

mgDeselectTextureIndex de-selects the texture with the specified index in the texture palette of the database db.

PARAMETERS

db
the database
index
the index of the texture palette entry to de-select

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
