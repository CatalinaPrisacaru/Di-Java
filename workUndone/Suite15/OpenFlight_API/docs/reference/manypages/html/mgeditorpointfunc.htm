mgeditorpointfunc

CALLBACK NAME

mgeditorpointfunc - Editor 2D point function

SYNOPSIS

void (*mgeditorpointfunc) ( mgeditorcontext editorContext,
mgpointinputdata* pointInputData,
void* editorToolData );

DESCRIPTION

This is the signature for editor 2D point functions. If your editor tool requires 2D point input at any time while it is active, you will assign a 2D point function to your editor tool of this form.

2D point functions are called to notify your editor tool when a 2D screen coordinate has been entered by the user via the mouse.

PARAMETERS

editorContext
the editor context that identifies the editor instance receiving the 2D point input
pointInputData
address of record describing the 2D point input
editorToolData
user defined tool instance data set in toolData field of callData parameter (mgeditorcallbackrec) during editor tool start function mgtoolstartfunc
