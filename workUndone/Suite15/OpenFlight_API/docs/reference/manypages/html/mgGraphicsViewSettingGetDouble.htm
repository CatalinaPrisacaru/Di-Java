mgGraphicsViewSettingGetDouble

FUNCTION NAME

mgGraphicsViewSettingGetDouble - gets an double view setting value from the current graphics view.

C SPECIFICATION

mgbool mgGraphicsViewSettingGetDouble (mgrec*db,
const mgsettingnamesettingName,
double*settingValue,
doubledefaultValue );

PYTHON SPECIFICATION

mgbool, settingValue mgGraphicsViewSettingGetDouble (mgrec*db,
mgsettingnamesettingName,
doubledefaultValue )

DESCRIPTION

mgGraphicsViewSettingGetDouble retrieves an double value from a named graphics view setting, settingName, and stores it in settingValue. If settingName does not exist, the value specified by defaultValue is returned.

Currently only a small subset of the graphics view settings are accessible. For a complete list, see mgsettingname.

Note that the prefix of the setting name, either MGVD or MGVI, indicates whether the corresponding setting value is double or integer, respectively. If the value is double, use mgGraphicsViewSettingGetDouble and mgGraphicsViewSettingSetDouble to get and set the value, respectively. If the value is integer, use mgGraphicsViewSettingGetInteger and mgGraphicsViewSettingSetInteger.

PARAMETERS

db
the database
settingName
the graphics view setting name
settingValue
storage location for the setting value
defaultValue
default value returned if settingName not found

RETURNS

mgbool
MG_TRUE if there is a graphics view visible in Creator, settingName represents a valid graphics view setting and the value can be represented by a double value, MG_FALSE otherwise.
settingValue
If function is successful, settingValue contains the retrieved value, otherwise settingValue is undefined.

ACCESS LEVEL

Level 4

SEE ALSO

mgGraphicsViewSettingSetDouble, mgGraphicsViewSettingGetInteger, mgGraphicsViewSettingSetInteger
