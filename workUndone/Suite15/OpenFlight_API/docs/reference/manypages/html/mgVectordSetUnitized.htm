mgVectordSetUnitized

FUNCTION NAME

mgVectordSetUnitized - unitize a vector.

SYNOPSIS       (C and Python)

void mgVectordSetUnitized (mgvectord*vec );

DESCRIPTION

mgVectordSetUnitized unitizes the specified double precision vector record vec.

This function changes the components of the vector vec passed directly.

Note: For efficiency, vec is not verified before being dereferenced. You must not pass as MG_NULL. If the magnitude of vec is less than 0.000000000001, it is set to (0,0,1).

PARAMETERS

vec
address of vector to unitize

ACCESS LEVEL

Level 1
