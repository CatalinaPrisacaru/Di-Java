mgVectordEqual

FUNCTION NAME

mgVectordEqual - checks to see if two double precision vectors are exactly equal.

SYNOPSIS       (C and Python)

mgbool mgVectordEqual (const mgvectord*vector1,
const mgvectord*vector2 );

DESCRIPTION

mgVectorfEqual compares two double precision vectors, vector1 and vector2 to see if they are exactly equal.

PARAMETERS

vector1
first vector to check for equality
vector2
second vector to check for equality

RETURNS

Returns MG_TRUE if vector1 and vector2 are equal, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
