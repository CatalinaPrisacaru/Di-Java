mgSetName

FUNCTION NAME

mgSetName - assigns a name to a node record.

SYNOPSIS       (C and Python)

mgbool mgSetName (mgrec*rec,
const char*name );

DESCRIPTION

mgSetName sets the name of a node rec to name. The name must be unique in the database containing rec, otherwise the function will fail.

PARAMETERS

rec
the node record
name
the new name of the node record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if name is NULL or an empty string.

ACCESS LEVEL

Level 2
