mgtextactivatereason

ENUM NAME

mgtextactivatereason - Text activate reason type

SYNOPSIS

typedef enum mgtextactivatereason {
   MTEXT_LOSTFOCUS,
   MTEXT_TEXTCHANGED,
} mgtextactivatereason;

MEMBERS

MTEXT_LOSTFOCUS
Text control lost focus. This indicates that the event was sent because the text control lost focus. By default, this is the only reason the activate control event is sent to a text control.
MTEXT_TEXTCHANGED
Text control text changed. This indicates that the event was sent because the user typed in the text control. Note that the user may not have finished typing when this event is sent. By default, the activate control event is not sent for this reason. To enable this for a specific text control, use mgControlSetAttribute to set the MCA_TEXTACTIVATEONCHANGED attribute on the text control.

DESCRIPTION

This enumerated type is used when the activate control event is sent to text controls.
