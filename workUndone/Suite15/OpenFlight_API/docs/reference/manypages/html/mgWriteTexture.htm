mgWriteTexture

FUNCTION NAME

mgWriteTexture - writes a texture palette entry to disk.

SYNOPSIS       (C and Python)

mgbool mgWriteTexture (mgrec*db,
intindex,
const char*textureName );

DESCRIPTION

mgWriteTexture writes the specified texture image to disk in the SGI RGB format with the given textureName. This function also writes the attributes corresponding to the texture to disk using the given textureName appended with ".attr".

PARAMETERS

db
the database
index
the texture palette index
textureName
the name of the image file to create

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
