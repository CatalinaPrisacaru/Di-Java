mgGLTextureIndex

FUNCTION NAME

mgGLTextureIndex - specifies a texture image for GL rendering.

SYNOPSIS       (C and Python)

mgbool mgGLTextureIndex (mgglcontextglContext,
mgrec*db,
intindex );

DESCRIPTION

mgGLTextureIndex sets the current texture image for GL rendering to that specified by palette index, in database db.

This function loads the texels associated with the specified texture image in preparation for rendering into GL controls. It is the equivalent of calling the OpenGL routine glTexImage2D.

If you are rendering into a user defined GL control, you must explicitly enable texture rendering using the OpenGL routine glEnable(GL_TEXTURE_2D), before mgGLTextureIndex will have any affect. If you are rendering into a graphics view, texture rendering will be enabled according to the settings of the viewing parameters of that graphics view.

PARAMETERS

glContext
Gl rendering context into which texture image is to be loaded
db
database whose texture palette defines the texture to load
index
index of texture in database's palette to load

RETURNS

Returns MG_TRUE if texture image could be loaded successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
