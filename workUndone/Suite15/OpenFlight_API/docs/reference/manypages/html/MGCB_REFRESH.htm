MGCB_REFRESH

NAME

MGCB_REFRESH - Dialog/Control event Refresh

DESCRIPTION

This is the dialog/control event that is sent to both dialogs and controls to notify them that they need to be refreshed.

Dialog events are sent to dialogs via the dialog function.

Control events are sent to controls via the control callback function.
