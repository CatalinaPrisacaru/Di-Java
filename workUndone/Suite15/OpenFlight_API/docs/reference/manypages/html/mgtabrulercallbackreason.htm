mgtabrulercallbackreason

ENUM NAME

mgtabrulercallbackreason - Tab ruler callback reason type

SYNOPSIS

typedef enum mgtabrulercallbackreason {
   MTRUL_TAB_SELECTED,
   MTRUL_TAB_DESELECTED,
   MTRUL_TAB_DRAGGED,
   MTRUL_TAB_BUMPED,
   MTRUL_TAB_CHANGED,
   MTRUL_TAB_CREATED,
   MTRUL_TAB_DELETED,
   MTRUL_TAB_RELEASED,
} mgtabrulercallbackreason;

MEMBERS

MTRUL_TAB_SELECTED
The tab selected state. This indicates that the associated event was due to a tab getting selected.
MTRUL_TAB_DESELECTED
The tab deselected state. This indicates that the associated event was due to a tab getting deselected.
MTRUL_TAB_DRAGGED
The tab moved state. This indicates that the associated event was due to a tab getting moved. Specifically the tab was dragged.
MTRUL_TAB_BUMPED
The tab selected state. This indicates that the associated event was due to a tab getting moved. Specifically the tab was bumped by dragging another tab.
MTRUL_TAB_CHANGED
The tab selected state. This indicates that the associated event was due to a tab getting moved. Specifically the tab's value was changed by entering a new value in the tab's text box.
MTRUL_TAB_CREATED
The tab created state. This indicates that a new tab was created.
MTRUL_TAB_DELETED
The tab deleted state. This indicates that a tab was deleted.
MTRUL_TAB_RELEASED
The tab released state. This indicates the mouse up event occurred.

DESCRIPTION

This enumerated type is used when calling the tab ruler control callback function.
