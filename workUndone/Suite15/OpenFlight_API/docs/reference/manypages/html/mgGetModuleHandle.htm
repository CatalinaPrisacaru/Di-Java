mgGetModuleHandle

FUNCTION NAME

mgGetModuleHandle - returns a native plug-in module handle.

C SPECIFICATION       (Not Available For Python)

mgmodulehandle mgGetModuleHandle (mgpluginplugin );

DESCRIPTION

mgGetModuleHandle returns a platform specific handle to the specified plug-in module plugin.

PARAMETERS

plugin
the plug-in module being queried

RETURNS

On Windows, this function returns an object of type HINSTANCE.

ACCESS LEVEL

Level 4

mgLoadResource, mgGetModuleFilename
