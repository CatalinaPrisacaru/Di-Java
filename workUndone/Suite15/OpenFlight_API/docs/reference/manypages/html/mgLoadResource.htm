mgLoadResource

FUNCTION NAME

mgLoadResource - load the resource for a plug-in module.

C SPECIFICATION       (Not Available For Python)

mgresource mgLoadResource (mgmodulehandlemoduleHandle );

DESCRIPTION

mgLoadResource loads and initializes the resource associated with a plug-in module identified by moduleHandle.

After calling this function, the individual resource items (dialog templates, pixmaps, and string definitions) contained in the resource become available. Typically call this function one time when your plug-in module initializes. When you are through accessing items contained in the resource, you should use mgUnloadResource to let the API clean up.

PARAMETERS

moduleHandle
the module handle for your plug-in as returned by mgGetModuleHandle

RETURNS

Handle to resource if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
