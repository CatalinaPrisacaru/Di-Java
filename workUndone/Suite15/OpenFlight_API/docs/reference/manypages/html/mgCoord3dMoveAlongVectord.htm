mgCoord3dMoveAlongVectord

FUNCTION NAME

mgCoord3dMoveAlongVectord - calculates the position of a 3D coordinate translated along a double precision vector a given distance.

SYNOPSIS       (C and Python)

mgcoord3d mgCoord3dMoveAlongVectord (const mgcoord3d*coord,
const mgvectord*vec,
doubledistance );

DESCRIPTION

mgCoord3dMoveAlongVectord calculates and returns the position of the specified 3D coordinate, coord, translated the specified distance, distance, in the direction specified by the double precision floating point vector vec.

PARAMETERS

coord
address of 3D coordinate to move
vec
address of vector along which coord is to be moved
distance
distance along vector that coord is to be moved

RETURNS

Returns a double precision 3D coordinate record.

ACCESS LEVEL

Level 1
