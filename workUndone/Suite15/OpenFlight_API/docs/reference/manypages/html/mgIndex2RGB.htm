mgIndex2RGB

FUNCTION NAME

mgIndex2RGB - returns red, green, and blue values for a given color palette index and intensity.

C SPECIFICATION

mgbool mgIndex2RGB (mgrec*db,
unsigned intindex,
floatintensity,
short*red,
short*green,
short*blue );

PYTHON SPECIFICATION

mgbool, red, green, blue mgIndex2RGB (mgrec*db,
unsigned intindex,
floatintensity )

DESCRIPTION

Given a database node db, color palette index, index, and intensity, intensity, mgIndex2RGB returns the red, green, and blue values associated with them in the database’s color palette. The color values returned are in the range 0..255.

PARAMETERS

db
the database
index
the index into the color palette
intensity
the intensity component of the color (0.0-1.0)
red
address of value to receive red component of color
green
address of value to receive green component of color
blue
address of value to receive blue component of color

RETURNS

mgbool
MG_TRUE if the index could be converted successfully, MG_FALSE otherwise.
red
If function is successful, red contains the retrieved red component, otherwise red is undefined.
green
If function is successful, green contains the retrieved green component, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved blue component, otherwise blue is undefined.

ACCESS LEVEL

Level 1
