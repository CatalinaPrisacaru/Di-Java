mgEditorSetDeviceInputFunc

FUNCTION NAME

mgEditorSetDeviceInputFunc - sets the device input function for an editor tool.

C SPECIFICATION       (Not Available For Python)

void mgEditorSetDeviceInputFunc (mgplugintooleditorPluginTool,
mgeditordeviceinputfuncdevInputFunc );

DESCRIPTION

mgEditorSetDeviceInputFunc assigns a device input function to the specified editor tool editorPluginTool.

device input functions are called to notify your editor tool when an input device is activated by the user

PARAMETERS

editorPluginTool
the editor plug-in tool
devInputFunc
the device specific input function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
