MPRIM_QUAD_STRIP

NAME

MPRIM_QUAD_STRIP - Mesh Primitive Type Quadrilateral Strip

DESCRIPTION

This is the mesh primitive type defining a quadrilateral strip. A quadrilateral strip represents a series of 4-sided polygons (quadrilaterals) defined by the vertices of the mesh primitive.

One quadrilateral is defined for each pair of vertices after the first pair. For any n (starting at 0), the nth quadrilateral is defined by vertices 2n, 2n+1, 2n+3 and 2n+2.

If a quadrilateral strip mesh primitive has N vertices, (N/2)-1 quadrilaterals are defined.
