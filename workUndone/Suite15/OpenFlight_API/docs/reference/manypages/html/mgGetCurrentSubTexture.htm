mgGetCurrentSubTexture

FUNCTION NAME

mgGetCurrentSubTexture - returns index of current modeling subtexture.

C SPECIFICATION

mgbool mgGetCurrentSubTexture (mgrec*db,
int*index );

PYTHON SPECIFICATION

mgbool, index mgGetCurrentSubTexture (mgrec*db )

DESCRIPTION

mgGetCurrentSubTexture returns the index of the current modeling subtexture selected for database db. Subtexture indices are contiguous and range from 0..N-1 where N is the value returned by mgSubTextureCount.

PARAMETERS

db
the database
index
address of value to receive index of current modeling subtexture.

RETURNS

mgbool
MG_TRUE if current modeling subtexture for db could be determined, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

ACCESS LEVEL

Level 4
