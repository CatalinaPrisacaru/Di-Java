mgSetTextureAttributes

FUNCTION NAME

mgSetTextureAttributes - sets the attributes of a texture palette entry.

SYNOPSIS       (C and Python)

void mgSetTextureAttributes (mgrec*db,
intindex,
mgrec*imgRec );

DESCRIPTION

Given a database node, db, and a texture palette entry specified by index, mgSetTextureAttributes sets the attributes record of the texture palette entry to the given attributes record, imgRec of type fltImage .

PARAMETERS

db
the database
index
the index of the texture palette entry
imgRec
a pointer to the attributes record

ACCESS LEVEL

Level 2
