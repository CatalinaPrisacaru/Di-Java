mgGuiGetPos

FUNCTION NAME

mgGuiGetPos - retrieves the position of a control or dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGuiGetPos (mgguigui,
int*left,
int*top );

DESCRIPTION

mgGuiGetPos returns the left and/or top position of the specified control or dialog, gui, measured in pixels.

The position of a control is measured relative to the upper left corner of the parent dialog's "view" area. The upper left corner of a dialog's view area is at position (0, 0). Remember that the view area of a dialog does not include the margin or the title bar of the dialog.

The position of a dialog is measured relative to the upper left corner of the screen. The upper left corner of the screen is at position (0, 0).

If either of the position parameters are MG_NULL, that particular position will not be returned.

PARAMETERS

gui
the control or dialog whose position is to be returned
left
address of value to receive left position
top
address of value to receive top position

C EXAMPLE

The following example shows how to use this function to get the position of controls and dialogs.
mgstatus status;
int left, top;

// get JUST the left position of a dialog
status = mgGuiGetPos (dialog, &left, MG_NULL);

// get JUST the top position of a dialog
status = mgGuiGetPos (dialog, MG_NULL, &top);

// get BOTH the left and top positions of a control
status = mgGuiGetPos (control, &left, &top);

ACCESS LEVEL

Level 4

SEE ALSO

mgGuiSetPos, mgGuiGetSize, mgGuiSetSize, mgGuiGetViewSize

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
left
If function is successful, left contains the left position, otherwise undefined.
top
If function is successful, top contains the top position, otherwise undefined.
