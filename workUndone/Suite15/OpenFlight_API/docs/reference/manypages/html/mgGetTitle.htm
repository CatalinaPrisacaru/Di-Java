mgGetTitle

FUNCTION NAME

mgGetTitle - retrieves the title string for a dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGetTitle (mgguidialog,
char*string,
intmaxLen );

DESCRIPTION

mgGetTitle retrieves the title string of the specified dialog.

The title is truncated and null terminated if it is longer than maxLen characters.

PARAMETERS

dialog
the dialog to get title string for
string
character buffer to hold title retrieved
maxLen
the maximum number of characters to store in string

ACCESS LEVEL

Level 4

SEE ALSO

mgSetTitle, mgGetCaption, mgSetCaption

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
string
If function is successful, string contains the title, otherwise undefined.
