mgpInit

FUNCTION NAME

mgpInit - plug-in initialization function.

SYNOPSIS       (C and Python)

MGPIDECLARE(mgbool) mgpInit (mgpluginplugin,
int*argc,
char*argv [] );

DESCRIPTION

This is the plug-in initialization function required for all plug-in modules. You must supply this function so that your plug-in is properly recognized.

You must declare this function exactly as shown here, using the MGPIDECLARE macro, so that it is properly exported from your plug-in module library.

This function is called by the plug-in runtime environment at startup. In this function a plug-in module performs any initialization processing required. Typically, a plug-in module registers plug-in tools and/or data extensions for the runtime environment as well as load a resource containing GUI definitions required by the plug-in.

PARAMETERS

plugin
the plug-in module being initialized
argc
a pointer to the number of arguments contained in argv
argv []
array of arguments passed to the program when it was started

RETURNS

Return MG_TRUE if the plug-in was successfully initialized, MG_FALSE otherwise.

ACCESS LEVEL

Declared by Plug-in Module
