mgSetReadExtFlag

FUNCTION NAME

mgSetReadExtFlag - sets the preference that determines if mgOpenDb reads external reference database files contained in database files being opened.

SYNOPSIS       (C and Python)

void mgSetReadExtFlag (mgboolflag );

DESCRIPTION

mgSetReadExtFlag tells mgOpenDb how to act when it finds an external reference node in a database being opened. If flag is MG_TRUE, mgOpenDb will read external references. If flag is MG_FALSE (the default), mgOpenDb will not read external references.

The default value for this preference is MG_FALSE.

PARAMETERS

flag
whether or not to read external references

ACCESS LEVEL

Level 1
