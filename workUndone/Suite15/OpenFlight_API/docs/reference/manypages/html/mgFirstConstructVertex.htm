mgFirstConstructVertex

FUNCTION NAME

mgFirstConstructVertex - gets the first construction vertex node associated with an active editor tool instance.

SYNOPSIS       (C and Python)

mgrec* mgFirstConstructVertex (mgeditorcontexteditorContext );

DESCRIPTION

mgFirstConstructVertex gets the first construction vertex node associated with the editor tool instance identified by editor context, editorContext.

While an editor tool instance is active, all construction vertices created via calls to mgNewConstructVertex are maintained internally within the editor context record. In this way, you can use mgFirstConstructVertex and mgNextConstructVertex to traverse all construction vertices associated with the current editor tool instance.

PARAMETERS

editorContext
the editor context from which to get the first construction vertex node

RETURNS

Returns the first construction vertex node if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4

SEE ALSO
