mgpexitfunc

CALLBACK NAME

mgpexitfunc - Plug-in exit function

SYNOPSIS

void (*mgpexitfunc) ( mgplugin plugin );

DESCRIPTION

This is the signature for the plug-in exit function you must declare in your source code so that your plug-in is properly recognized.

PARAMETERS

plugin
the plug-in module being exited
