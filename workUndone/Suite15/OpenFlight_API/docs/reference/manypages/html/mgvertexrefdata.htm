mgvertexrefdata

UNION NAME

mgvertexrefdata - 3D Vertex Mouse Input Reference data

SYNOPSIS

typedef union  {
   mgrec* vertex;
   mgrec* edge[2];
   mgrec* curve;
   mgrec* face;
   mgplaned* trackPlane;
} mgvertexrefdata;

MEMBERS

vertex
mgrec* - the existing vertex referenced for MVRF_VERTEX
edge[2]
mgrec* - the existing edge referenced for MVRF_EDGE
curve
mgrec* - the existing curve referenced for MVRF_CURVE
face
mgrec* - the existing face referenced for MVRF_FACE
trackPlane
mgplaned* - the trackplane referenced for MVRF_TRACKPLANE

DESCRIPTION

When 3D vertex mouse input is reported to an editor tool via the corresponding 3D vertex input function, reference data may be included that specifies how the user entered the 3D vertex input. This record is used to specify this 3D vertex input reference data.
