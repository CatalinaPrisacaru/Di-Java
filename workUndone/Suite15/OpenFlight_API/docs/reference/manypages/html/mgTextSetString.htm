mgTextSetString

FUNCTION NAME

mgTextSetString - displays text in a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSetString (mgguicontrol,
const char*string );

DESCRIPTION

mgTextSetString sets the text displayed in the specified text control.

PARAMETERS

control
the text control
string
the text string to load into the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
