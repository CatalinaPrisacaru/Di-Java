mgTextSetDMS

FUNCTION NAME

mgTextSetDMS - displays DMS (degrees, minutes, seconds) formatted text in a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSetDMS (mgguicontrol,
doubledmsVal,
const char*fmtString );

DESCRIPTION

mgTextSetDMS displays the specified double value dmsVal (assumed to be in decimal degrees) in the specified text control formatted as degrees, minutes and seconds.

Depending on the fmtString you specify, you can control the format of the string displayed in the text control. You must specify either MDFMT_LAT or MDFMT_LON as described below.

If you specify MDFMT_LAT in fmtString, dmsVal is converted to a latitude measurement and is displayed in the form: DD~MM'SS"H, where DD is the number of degrees, MM is the number of minutes, SS is the number of seconds and H is either 'N' or 'S' depending on whether the value lies in the northern or southern hemisphere, respectively.

If you specify MDFMT_LON in fmtString, dmsVal is converted to a longitude measurement and is displayed in the form: DD~MM'SS"H, where DD is the number of degrees, MM is the number of minutes, SS is the number of seconds and H is either 'E' or 'W' depending on whether the value lies in the eastern or western hemisphere, respectively.

By convention, values in the northern and eastern hemispheres are positive (0.0 .. 180.0). Values in the southern and western hemispheres are negative (-180 .. 0.0). If you specify a value outside these ranges, the results are undefined.

PARAMETERS

control
the text control
dmsVal
double precision floating point value in decimal degrees to format and display
fmtString
latitude/longitude formatting string

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

Here are some examples.
mgstatus status;

// will display 37~30'00"N in text control
status = mgTextSetDMS (text, 37.5, MDFMT_LAT);

// will display 123~15'00"W in text control
status = mgTextSetDMS (text, -123.25, MDFMT_LON);
