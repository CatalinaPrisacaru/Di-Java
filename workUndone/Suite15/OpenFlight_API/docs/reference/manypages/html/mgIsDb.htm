mgIsDb

FUNCTION NAME

mgIsDb - checks if a file is an OpenFlight database file.

SYNOPSIS       (C and Python)

mgbool mgIsDb (const char*fileName );

DESCRIPTION

mgIsDb determines whether the file named by fileName is an OpenFlight database file.

PARAMETERS

fileName
the name of the file to be checked

RETURNS

Returns MG_TRUE if the specified file is an OpenFlight database file, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
