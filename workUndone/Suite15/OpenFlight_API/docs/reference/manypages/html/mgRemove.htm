mgRemove

FUNCTION NAME

mgRemove - temporarily deletes a node record from a database.

SYNOPSIS       (C and Python)

mgbool mgRemove (mgrec*rec );

DESCRIPTION

mgRemove temporarily deletes the specified node record rec. Node records that have been temporarily deleted in this way may be un-deleted later using the function mgUnRemove. This is useful for editor class plug-in tools to implement undo.

Records that have been temporarily deleted using this function is detached and appears to be completely deleted from the database. After rec has been removed in this way, the only OpenFlight API functions that accept rec are mgUnRemove which un-removes it and mgDelete which permanently deletes it.

This function is only applicable for node records in the Creator modeler environment. Calling this function in the stand-alone application environment does nothing.

PARAMETERS

rec
the record to remove

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgDelete, mgDetach, mgUnRemove
