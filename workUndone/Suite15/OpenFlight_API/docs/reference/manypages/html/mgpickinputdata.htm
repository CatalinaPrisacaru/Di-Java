mgpickinputdata

STRUCT NAME

mgpickinputdata - Pick Mouse Input data

SYNOPSIS

typedef struct  {
   unsigned int keyboardFlags;
   unsigned int buttonFlags;
   mgreclist pickList;
} mgpickinputdata;

MEMBERS

keyboardFlags
unsigned int - the state of the Alt, Ctrl, and Shift keys - the value of this field will be a bitwise combination of MKB_ALTKEY, MKB_CTRLKEY and MKB_SHIFTKEY
buttonFlags
unsigned int - the state of the Left, Middle and Right mouse buttons - the value of this field will be a bitwise combination of MMB_LEFTMOUSE, MMB_MIDDLEMOUSE and MMB_RIGHTMOUSE
pickList
mgreclist - the select list containing the nodes that were selected

DESCRIPTION

When pick mouse input is reported to an editor tool via the corresponding pick input function, a pointer to a record of this type is passed to the pick function that describes the pick input.
