mgRecDeleteProperty

FUNCTION NAME

mgRecDeleteProperty - deletes user defined data associated with a record.

C SPECIFICATION       (Not Available For Python)

void mgRecDeleteProperty (mgplugintoolpluginTool,
mgrec*rec,
const mgpropertynamepropName );

DESCRIPTION

Use mgRecDeleteProperty to delete the data named propName that was associated with the record rec using mgRecPutProperty.

Note that this function does not free or otherwise affect the actual data that was associated. If this data was dynamically allocated, it is the callers responsibility to deallocate it.

PARAMETERS

pluginTool
the plug-in tool handle
rec
the record
propName
the name of the property to delete

ACCESS LEVEL

Level 3

SEE ALSO

mgRecPutProperty, mgRecGetProperty
