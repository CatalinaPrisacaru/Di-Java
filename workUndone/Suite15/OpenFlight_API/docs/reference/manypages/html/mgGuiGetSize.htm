mgGuiGetSize

FUNCTION NAME

mgGuiGetSize - retrieves the dimensions of a control or dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGuiGetSize (mgguigui,
int*width,
int*height );

DESCRIPTION

mgGuiGetSize returns the width and/or height of the specified control or dialog, gui, measured in pixels.

For both controls and dialogs, the dimensions returned define the "outer" dimensions of the gui item. The outer dimensions include the margin, or in the case of dialogs, the title bar dimensions. For most controls, the outer dimensions are identical to the "inner" or "view" dimensions of the control. For dialogs, the outer dimensions are almost always larger than the view dimensions. As stated above, this is due to margins and the title bar of the dialog.

If either of the dimension parameters are MG_NULL, that particular dimension will not be returned.

PARAMETERS

gui
the control or dialog whose dimensions are to be returned
width
address of value to receive width dimension
height
address of value to receive height dimension

C EXAMPLE

The following example shows how to use this function to get the dimensions of controls and dialogs.
mgstatus status;
int width, height;

// get JUST the height of a dialog
status = mgGuiGetSize (dialog, MG_NULL, &height);

// get JUST the width of a dialog
status = mgGuiGetSize (dialog, &width, MG_NULL);

// get BOTH the width and height of a control
status = mgGuiGetSize (control, &width, &height);

ACCESS LEVEL

Level 4

SEE ALSO

mgGuiSetSize, mgGuiGetViewSize, mgGuiGetPos, mgGuiSetPos

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
width
If function is successful, width contains the width, otherwise undefined.
height
If function is successful, height contains the height, otherwise undefined.
