mgGetAbsoluteFilename

FUNCTION NAME

mgGetAbsoluteFilename - gets the absolute path of a file relative to a database.

C SPECIFICATION

mgbool mgGetAbsoluteFilename (mgrec*db,
const char*relFilename,
char*absFilename,
intmaxLen );

PYTHON SPECIFICATION

mgbool, absFilename mgGetAbsoluteFilename (mgrec*db,
const char*relFilename )

DESCRIPTION

Given a database db, and a relative filename relFilename, mgGetAbsoluteFilename forms and returns the absolute filename for the file. The file does not have to exist.

The following table shows the forms the relative filename relFilename can take and the absolute path results for db located in C:/folder1/folder2/db.flt:

Relative Filename Absolute Path Returned
../path1/file.ext C:/folder1/path1/file.ext
./file.ext C:/folder1/folder2/file.ext
./path1/file.ext C:/folder1/folder2/path1/file.ext
../../path1/path2/file.ext C:/path1/path2/file.ext

PARAMETERS

db
the database to which relFilename is relative
relFilename
the relative file name
absFilename
the address of string to receive absolute filename
maxLen
maximum number of characters to store in absFilename

RETURNS

mgbool
MG_TRUE if the absolute filename can be formed, MG_FALSE otherwise.
absFilename
If function is successful,this contains the absolute filename, otherwise it is undefined.

ACCESS LEVEL

Level 1
