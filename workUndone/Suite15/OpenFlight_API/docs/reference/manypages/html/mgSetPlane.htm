mgSetPlane

FUNCTION NAME

mgSetPlane - sets all four attribute values in a fltDPlane record.

SYNOPSIS       (C and Python)

mgbool mgSetPlane (mgrec*rec,
mgcodecode,
doublea,
doubleb,
doublec,
doubled );

DESCRIPTION

Given a record rec, and the name of one of its fltDPlane attribute records, code, mgSetPlane sets the a, b, c, d attribute values in the fltDPlane attribute record code.

PARAMETERS

rec
the record whose fltDPlane record is to be set
code
the fltDPlane attribute record code
a
the a value to set in the fltDPlane attribute record
b
the b value to set in the fltDPlane attribute record
c
the c value to set in the fltDPlane attribute record
d
the d value to set in the fltDPlane attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
