mgGetMatTextureLayer

FUNCTION NAME

mgGetMatTextureLayer - gets values out of a fltMatTextureLayer record.

C SPECIFICATION

mgbool mgGetMatTextureLayer (mgrec*rec,
mgcodematTextureCode,
int*textureIndex,
int*textureLayer );

PYTHON SPECIFICATION

mgbool, textureIndex, textureLayer mgGetMatTextureLayer (mgrec*rec,
mgcodematTextureCode )

DESCRIPTION

Given a record, rec, mgGetMatTextureLayer gets the values of fltMatTextureLayer attribute record matTextureCode from rec and stores them in textureIndex and textureLayer.

You do not have to provide addresses for both textureIndex and textureLayer values. For example, if you only want to receive the index value (but not layer), specify a valid address for textureIndex and leave textureLayer NULL.

PARAMETERS

rec
the record from which to get the fltMatTextureLayer attribute code
matTextureCode
the fltMatTextureLayer attribute code
textureIndex
address of value to receive texture index
textureLayer
address of value to receive texture layer

RETURNS

mgbool
MG_TRUE if the layer was retrieved successfully, MG_FALSE otherwise.
textureIndex
If function is successful, textureIndex contains the retrieved index, otherwise textureIndex is undefined.
textureLayer
If function is successful, textureLayer contains the retrieved layer, otherwise textureLayer is undefined.

C EXAMPLE

mgrec* db;
mgrec* matRec;
int materialIndex;
int textureIndex, textureLayer;
mgbool status;

db = mgOpenDb ("file.flt");
matRec = mgGetFirstMaterial (db, &materialIndex);
status = mgGetMatTextureLayer (matRec, fltAmbientExLayer1, &textureIndex, &textureLayer);

ACCESS LEVEL

Level 1
