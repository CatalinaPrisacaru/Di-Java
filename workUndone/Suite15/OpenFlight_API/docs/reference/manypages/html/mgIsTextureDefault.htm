mgIsTextureDefault

FUNCTION NAME

mgIsTextureDefault - determines if the default texture pattern has been substituted for a texture palette entry.

SYNOPSIS       (C and Python)

mgbool mgIsTextureDefault (mgrec*db,
intindex );

DESCRIPTION

When a texture palette entry is loaded into the texture palette, the file containing the image may or may not be found. In either case, the palette still contains the entry but when the image file is not found or cannot be read, the texels and/or attributes will be unknown. When this happens for a texture palette entry, a default pattern (a black X on a white background) is substituted in the palette for the missing texels.

mgIsTextureDefault determines whether the texels for a texture palette entry at the specified index contained in the texture palette of database db have been substituted by the default pattern texels.

PARAMETERS

db
the database
index
the index of the texture palette entry

RETURNS

Returns MG_TRUE if the texels for the entry have been substituted by the default pattern, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
