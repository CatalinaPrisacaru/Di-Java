mgMeshSetVtxCoord

FUNCTION NAME

mgMeshSetVtxCoord - sets the x,y,z coordinate for a vertex in a mesh.

SYNOPSIS       (C and Python)

mgbool mgMeshSetVtxCoord (mgrec*rec,
intvtxIndex,
doublex,
doubley,
doublez );

DESCRIPTION

This function sets the x, y, and z values of the vertex at index vtxIndex in the vertex pool of mesh node rec.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
x
the x coordinate
y
the y coordinate
z
the z coordinate

RETURNS

Returns MG_TRUE if the vertex data was set successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertex mask for the mesh does not specify that the vertices have coordinate positions.

ACCESS LEVEL

Level 2
