mgWriteMaterialFileVersion

FUNCTION NAME

mgWriteMaterialFileVersion - writes a database’s material palette to a file.

SYNOPSIS       (C and Python)

mgbool mgWriteMaterialFileVersion (mgrec*db,
const char*fileName );

DESCRIPTION

mgWriteMaterialFileVersion writes the material palette for database db to a file named fileName using the specified material palette format formatVersion. You can specify either MEFV_1630 or MEFV_1620 for formatVersion. If you specify MEFV_1630, extended material palette entries (OpenFlight version 16.3 and beyond) will be written. If you specify MEFV_1620, standard material palette entries (OpenFlight 16.2 and earlier) will be written.

The resulting file can be read into another database using mgReadMaterialFile or via the Load Palette command in the material palette in Creator.

Note that if you specify MEFV_1630 for formatVersion versions of Creator earlier than 3.4 will not recognize this material file format.

PARAMETERS

db
the database that contains the material palette to write
fileName
the material palette file name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2

SEE ALSO

mgWriteMaterialFile, mgReadMaterialFile
