mghierarchyviewdata

STRUCT NAME

mghierarchyviewdata - Hierarchy View data

SYNOPSIS

typedef struct  {
   mgglcontext glContext;
   unsigned int left;
   unsigned int bottom;
   unsigned int right;
   unsigned int top;
} mghierarchyviewdata;

MEMBERS

glContext
mgglcontext - Gl rendering context associated to this hierarchy view
left
unsigned int - left position of extension node rectangle
bottom
unsigned int - bottom position of extension node rectangle
right
unsigned int - right position of extension node rectangle
top
unsigned int - top position of extension node rectangle

DESCRIPTION

The hierarchy draw function for an extension node will be passed a pointer to an object of this type. This record contains a GL rendering context object and the extents of the rectangle in which the extension node is to be drawn. The draw function should not draw outside the bounds of this rectangle. The coordinates of this rectangle are measured relative to the lower left corner (0,0) of the enclosing hierarchy view.
