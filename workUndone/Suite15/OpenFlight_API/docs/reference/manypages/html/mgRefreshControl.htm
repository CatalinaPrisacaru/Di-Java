mgRefreshControl

FUNCTION NAME

mgRefreshControl - refreshes a control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRefreshControl (mgguicontrol );

DESCRIPTION

mgRefreshControl refreshes the specified control.

If the refresh control event has been selected for the callback function of the specified control, the refresh event is sent to the control.

PARAMETERS

control
the control to refresh

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4

SEE ALSO

mgSetGuiCallback, mgRefreshDialog
