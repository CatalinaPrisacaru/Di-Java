mgBoxGetYSize

FUNCTION NAME

mgBoxGetYSize - returns the Y dimension of a box.

SYNOPSIS       (C and Python)

double mgBoxGetYSize (const mgboxd*box );

DESCRIPTION

mgBoxGetYSize calculates and returns the Y dimension of a box.

PARAMETERS

box
address of box to get dimension for

RETURNS

Returns the Y dimension of box.

ACCESS LEVEL

Level 1
