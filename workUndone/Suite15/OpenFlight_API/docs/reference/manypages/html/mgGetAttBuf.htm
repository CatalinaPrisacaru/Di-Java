mgGetAttBuf

FUNCTION NAME

mgGetAttBuf - gets consecutive values from an attribute record and stores them in a data buffer.

C SPECIFICATION       (Not Available For Python)

mgbool mgGetAttBuf (mgrec*rec,
mgcodefcode,
void*buf );

DESCRIPTION

Given a record, rec, mgGetAttBuf gets the values of the attribute fcode from rec and stores those values in the data buffer buf. It is the caller's responsibility to ensure that sufficient storage is allocated.

This routine is used for getting all attribute values of a record, with the condition that the attributes are consecutive as defined by the data dictionary. Note: Simple records such as fltCoord3d and fltMatrix are guaranteed to be stored consecutively; more complex records are not. Exercise caution when using this routine.

PARAMETERS

rec
points to the record from which to get values
fcode
the attribute code
buf
points to the storage area for the attribute values

RETURNS

Returns MG_TRUE if data is stored in the buffer, MG_FALSE otherwise.

C EXAMPLE

The following example gets the attribute values of fltMatrix in polyRec.
mgmatrix matrix;

mgGetAttBuf (polyRec, fltMatrix, matrix);

ACCESS LEVEL

Level 1
