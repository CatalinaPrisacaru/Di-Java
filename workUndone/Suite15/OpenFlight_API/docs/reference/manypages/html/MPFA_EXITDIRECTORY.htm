MPFA_EXITDIRECTORY

NAME

MPFA_EXITDIRECTORY - File Browser parameter for receiving the name of the "exit" directory at the time the user closes the file browser

DESCRIPTION

The exit directory is the last directory selected by the user in the file browser while it is open.

Specify this as a parameter for mgPromptDialogFile followed by the address of the character string to receive the active directory name. If the user cancels the selection in the file browser, the directory returned is not defined.

C NOTES

The exit directory string returned is dynamically allocated by mgPromptDialogFile. When you are done accessing this string, you should dispose of it using mgFree.

PYTHON NOTES

This parameter is not supported in Python OpenFlight Script.
