mgProjectionConvertLLEtoXYZ

FUNCTION NAME

mgProjectionConvertLLEtoXYZ - converts projection (lat/lon/elevation) coordinates to database XYZ coordinates.

C SPECIFICATION

mgbool mgProjectionConvertLLEtoXYZ (mgprojectionprojection,
const mgprojcoord*lleCoord,
mgcoord3d*xyzCoord );

PYTHON SPECIFICATION

mgbool, xyzCoord mgProjectionConvertLLEtoXYZ (mgprojectionprojection,
const mgprojcoord*lleCoord )

DESCRIPTION

mgProjectionConvertLLEtoXYZ converts projection (lat/lon/elevation) coordinates to database XYZ coordinates using the specified projection. This function performs the same function as mgProject. If you need to convert multiple coordinates consecutively, it is more efficient to use mgProjectionConvertLLEtoXYZ.

PARAMETERS

projection
the projection object to use
lleCoord
projection (lat/lon/elevation) coordinates to convert
xyzCoord
the result database XYZ coordinates

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
xyzCoord
If function is successful, this is the database XYZ coordinate of lleCoord.

EXAMPLE

The following examples (in C and Python) show you how to convert from projection coordinates to database XYZ coordinates in database db.

C EXAMPLE

mgbool ok;
mgprojcoord lle;
mgcoord3d xyz;
mgprojection proj = mgNewProjection (db);

lle.lat = 37.0;
lle.lon = -122.0;
lle.height = 0.0;

ok = mgProjectionConvertLLEtoXYZ (proj, lle, &xyz);
printf ("XYZ Coordinates: %lf %lf %lf\n", xyz.x, xyz.y, xyz.z);
mgFreeProjection (proj);

PYTHON EXAMPLE

proj = mgNewProjection (db)
lle = mgprojcoord()
lle.lat = 37.0
lle.lon = -122.0
lle.height = 0.0
ok, xyz = mgProjectionConvertLLEtoXYZ (proj, lle)
print "XYZ Coordinates:", xyz.x, xyz.y, xyz.z

ACCESS LEVEL

Level 1
