mgTextGetSelectionRange

FUNCTION NAME

mgTextGetSelectionRange - retrieves the selection range from a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextGetSelectionRange (mgguicontrol,
int*start,
int*end );

DESCRIPTION

mgTextGetSelectionRange retrieves the start and end positions of the currently selected text displayed in the specified text control.

The first position in a text control is at index 0.

PARAMETERS

control
the text control
start
address of integer value to receive start position
end
address of integer value to receive end position

RETURNS

Use MSTAT_ISOK to check return value for success. If successful, the output parameters start and end will contain the start and end positions, respectively, otherwise they are undefined.

ACCESS LEVEL

Level 4
