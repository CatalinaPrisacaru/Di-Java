ddIsEdit

FUNCTION NAME

ddIsEdit - checks if a record or data code is editable in the modeler attribute page.

SYNOPSIS       (C and Python)

mgbool ddIsEdit (mgcodecode );

DESCRIPTION

ddIsEdit checks if the record or data identified by code is editable in the attribute page in the modeler. The value for the keyword NOEDIT controls this attribute.

By default, record and data codes are editable. Use the keyword NOEDIT to override the value for this attribute. This keyword is not valid for data of type MTYPE_FLAG if the ENUM attribute is set.

PARAMETERS

code
the record or data code

RETURNS

Returns MG_TRUE if record or data is editable, otherwise MG_FALSE.

ACCESS LEVEL

Level 1
