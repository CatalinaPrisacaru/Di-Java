mgShaderDeleteFragmentProgramNth

FUNCTION NAME

mgShaderDeleteFragmentProgramNth - deletes a fragment program from a GLSL shader.

SYNOPSIS       (C and Python)

mgbool mgShaderDeleteFragmentProgramNth (mgrec*shader,
intnth );

DESCRIPTION

mgShaderDeleteFragmentProgramNth deletes the nth fragment program from the specified GLSL shader. The first fragment program is specified by nth equal to 0.

Note: If shader is not a GLSL shader, this function will fail.

PARAMETERS

shader
the GLSL shader
nth
the index of the GLSL fragment program file to delete

RETURNS

Returns MG_TRUE if shader is a GLSL shader and the nth fragment program was deleted successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
