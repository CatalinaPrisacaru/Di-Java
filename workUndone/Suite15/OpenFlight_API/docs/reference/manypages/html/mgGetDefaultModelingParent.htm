mgGetDefaultModelingParent

FUNCTION NAME

mgGetDefaultModelingParent - returns default modeling parent.

SYNOPSIS       (C and Python)

mgrec* mgGetDefaultModelingParent (mgrec*db );

DESCRIPTION

mgGetDefaultModelingParent returns the default modeling parent node for the specified database db.

In Creator, the user specifies whether or not the default parent is allowed for modeling (this is set on the Preferences Window in Creator). If it is allowed, the default parent is always a node of type fltGroup.

PARAMETERS

db
the database to get modeling parent for

RETURNS

Returns the default modeling parent node for database db if this is allowed by the user, MG_NULL otherwise.

ACCESS LEVEL

Level 4
