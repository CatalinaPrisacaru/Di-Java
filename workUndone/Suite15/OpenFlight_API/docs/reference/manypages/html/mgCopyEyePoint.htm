mgCopyEyePoint

FUNCTION NAME

mgCopyEyePoint - copies from one eyepoint record to another.

SYNOPSIS       (C and Python)

mgbool mgCopyEyePoint (mgrec*to,
mgrec*from );

DESCRIPTION

mgCopyEyePoint copies all the eyepoint attributes from from to to.

PARAMETERS

to
the original fltEyePoint record
from
the new fltEyePoint record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
