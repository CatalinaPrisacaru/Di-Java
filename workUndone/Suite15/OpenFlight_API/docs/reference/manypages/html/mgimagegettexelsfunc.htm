mgimagegettexelsfunc

CALLBACK NAME

mgimagegettexelsfunc - Image Get Texels function for Image Importer Tools

SYNOPSIS

mgstatus (*mgimagegettexelsfunc) ( mgplugintool pluginTool,
void* userData,
int imageid,
int resolution,
unsigned char* texels );

DESCRIPTION

This is the signature for image get texels functions. When you register an image importer plug-in tool, provide a get texels function of this form that is called when the OpenFlight API needs to get the texels for all or part of an image that your tool has opened.

The format of the texels returned is described in the OpenFlight API Developer Guide (volume 2).

PARAMETERS

pluginTool
the plug-in tool
userData
user defined data specified when tool registered
imageid
image identifier indicating which image to get texels for - this is the value returned by the image open function when the image was first opened
resolution
the reduction factor of the texels to be returned (reserved for future enhancement), currently always 1
texels
the texels extracted from the image

RETURNS

Returns MIMG_NO_ERROR if successful, otherwise returns applicable texture error code. If successful, the parameter texels should be filled in with the texels extracted from the image.
