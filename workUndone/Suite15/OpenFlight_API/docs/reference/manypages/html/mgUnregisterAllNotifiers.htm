mgUnregisterAllNotifiers

FUNCTION NAME

mgUnregisterAllNotifiers - unregisters all notifiers for a plug-in tool.

C SPECIFICATION       (Not Available For Python)

void mgUnregisterAllNotifiers (mgplugintoolpluginTool );

DESCRIPTION

mgUnregisterAllNotifiers unregisters all notifiers currently registered for the specified plug-in tool pluginTool. Unregistering a notifier disables and destroys it.

Unregister a notifier when you are finished with it. Disable a notifier if you just want to stop the notifier from reporting events temporarily.

PARAMETERS

pluginTool
the plug-in tool for which all notifiers are being unregistered

ACCESS LEVEL

Level 4

SEE ALSO

mgRegisterNotifier, mgSetNotifierEnabled
