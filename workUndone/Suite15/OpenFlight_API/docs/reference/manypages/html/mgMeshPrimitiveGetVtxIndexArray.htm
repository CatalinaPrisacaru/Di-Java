mgMeshPrimitiveGetVtxIndexArray

FUNCTION NAME

mgMeshPrimitiveGetVtxIndexArray - gets the vertex index array for a mesh primitive.

C SPECIFICATION

int mgMeshPrimitiveGetVtxIndexArray (mgrec*rec,
intprimNo,
int*indexArray,
intmaxLen );

PYTHON SPECIFICATION

numIndices, array mgMeshPrimitiveGetVtxIndexArray (mgrec*rec,
intprimNo )

DESCRIPTION

This function gets the vertex index array for mesh primitive number primNo in mesh rec. The vertex index array of a mesh primitive is an array of integer values that define the vertex ordering of the vertices that make up the mesh primitive. The integer indices in the vertex index array refer to vertices stored in the mesh vertex pool.

Note: The first vertex in the mesh vertex pool is at index 0. The function mgMeshPrimitiveGetNumVtx can be used to determine the number of vertex indices contained in a primitive of a mesh. This result dictates the minimum size of the indexArray you pass in to mgMeshPrimitiveGetVtxIndexArray.

If successful, the indices of the vertices of the mesh primitive are returned in the output parameter indexArray, otherwise it is undefined. You must pass a buffer for indexArray that is large enough to receive at most maxLen integer values. The actual number of vertex indices returned is the smaller of maxLen and the number of vertex indices in the specified mesh primitive.

PARAMETERS

rec
the fltMesh node
primNo
the mesh primitive number starting from 0
indexArray
the array to receive the vertex indices
maxLen
the maximum number of indices to write into indexArray

RETURNS

numIndices
the number of vertex indices being returned in indexArray if successful, -1 otherwise. This function will fail if rec is not a valid mesh node or if primNo does not specify a valid mesh primitive number.
array
the array

ACCESS LEVEL

Level 1
