mgSetCoord3d

FUNCTION NAME

mgSetCoord3d - sets all three attribute values in a fltCoord3d record.

SYNOPSIS       (C and Python)

mgbool mgSetCoord3d (mgrec*rec,
mgcodecode,
doublex,
doubley,
doublez );

DESCRIPTION

Given a record rec, and the name of one of its fltCoord3d attribute records, code, mgSetCoord3d sets the x, y, z attribute values in the fltCoord3d attribute record code.

For setting vertex coordinates, it is much more efficient to call the convenience function mgSetVtxCoord.

PARAMETERS

rec
the record whose fltCoord3d record is to be set
code
the fltCoord3d attribute record code
x
the x value to set in the fltCoord3d attribute record
y
the y value to set in the fltCoord3d attribute record
z
the z value to set in the fltCoord3d attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2

SEE ALSO

mgSetCoord3f, mgSetCoord2i, mgSetVtxCoord
