mgDetach

FUNCTION NAME

mgDetach - unlinks a node record from the database hierarchy.

SYNOPSIS       (C and Python)

mgbool mgDetach (mgrec*rec );

DESCRIPTION

mgDetach unlinks a node record, rec, and its children from the database hierarchy. The node is left as an orphan, and is not deleted.

This function is for node and transformation records only.

PARAMETERS

rec
the record to detach

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function fails if rec is not attached in the hierarchy.

ACCESS LEVEL

Level 2
