mgUnRemove

FUNCTION NAME

mgUnRemove - unremoves a node record that was temporarily delete using mgRemove.

SYNOPSIS       (C and Python)

mgrec* mgUnRemove (mgrec*rec );

DESCRIPTION

mgRemove unremoves the specified node record rec that was previously removed by calling mgRemove.

Records that are unremoved in this way cannot be re-attached in the database. It is the caller's responsibility to attach the record back into the hierarchy at the appropriate place.

PARAMETERS

rec
the record to unremove

RETURNS

Returns rec if rec could be successfully unremoved, MG_NULL otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgRemove, mgDelete, mgDetach
