mgRegisterGetHeaderCode

FUNCTION NAME

mgRegisterGetHeaderCode - sets the Get Header Code (extension node helper) function for a database extension.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRegisterGetHeaderCode (mgpluginsitepluginSite,
mggetheadercodefuncgetHeaderCodeFunc );

DESCRIPTION

mgRegisterGetHeaderCode assigns the specified Get Header Code function getHeaderCodeFunc to the specified data extension site pluginSite.

PARAMETERS

pluginSite
the data extension site
getHeaderCodeFunc
the Get Header Code function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 3
