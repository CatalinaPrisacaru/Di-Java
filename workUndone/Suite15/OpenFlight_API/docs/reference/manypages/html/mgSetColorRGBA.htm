mgSetColorRGBA

FUNCTION NAME

mgSetColorRGBA - sets the attribute values in a fltColorRGBA record.

SYNOPSIS       (C and Python)

mgbool mgSetColorRGBA (mgrec*rec,
mgcodecolorRgbaCode,
floatred,
floatgreen,
floatblue,
floatalpha );

DESCRIPTION

Given a record, rec, and the name of one of its fltColorRGBA attribute records, colorRgbaCode, mgSetColorRGBA sets the red, green, blue, and alpha attribute values in the fltColorRGBA attribute record.

PARAMETERS

rec
the record whose fltColorRGBA record is to be set
colorRgbaCode
the name of the fltColorRGBA attribute code of rec
red
the red value to set in the fltColorRGBA attribute record
green
the green value to set in the fltColorRGBA attribute record
blue
the blue value to set in the fltColorRGBA attribute record
alpha
the alpha value to set in the fltColorRGBA attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
