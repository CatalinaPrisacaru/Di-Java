mgToggleButtonSetState

FUNCTION NAME

mgToggleButtonSetState - sets the state of a toggle button control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgToggleButtonSetState (mgguicontrol,
intstate );

DESCRIPTION

mgToggleButtonSetState sets the selection state of the specified toggle button control.

In two state toggle button controls, 1 indicates that the control is selected (or checked), and 0 indicates that it is not.

PARAMETERS

control
the toggle button control
state
the state of the control (1 is selected, 0 is not selected)

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
