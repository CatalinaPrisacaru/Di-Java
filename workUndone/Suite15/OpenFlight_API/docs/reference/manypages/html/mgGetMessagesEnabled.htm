mgGetMessagesEnabled

FUNCTION NAME

mgGetMessagesEnabled - determines if messages are enabled or disabled.

SYNOPSIS       (C and Python)

mgbool mgGetMessagesEnabled (mgmessageseverityenumseverity );

DESCRIPTION

mgGetMessagesEnabled determines if messages of level severity have been enabled or disabled via mgSetMessagesEnabled.

Note: Within the modeler environment, messages are always enabled and cannot be disabled. For this reason, this function always returns MG_TRUE when invoked from within the modeler environment.

PARAMETERS

severity
severity level of the messages to check whether enabled or disabled

RETURNS

Returns MG_TRUE if messages of level severity are enabled, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
