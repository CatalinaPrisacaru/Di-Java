mgSetUserData

FUNCTION NAME

mgSetUserData - sets the user data pointer of a record.

C SPECIFICATION       (Not Available For Python)

void mgSetUserData (mgrec*rec,
void*ptr );

DESCRIPTION

mgSetUserData sets the user data associated to the specified record rec to the value ptr.

PARAMETERS

rec
the record
ptr
the data pointer

Level 2
