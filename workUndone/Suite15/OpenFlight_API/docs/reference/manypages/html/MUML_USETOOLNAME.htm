MUML_USETOOLNAME

NAME

MUML_USETOOLNAME - Undo menu label Use Tool Name

DESCRIPTION

When an editor tool instance registers an undo-able action, MUML_USETOOLNAME can be specified for the menu label parameter. If this value is specified as the menu label, the name of the editor tool is displayed on the undo menu item in the Edit->Undo menu.
