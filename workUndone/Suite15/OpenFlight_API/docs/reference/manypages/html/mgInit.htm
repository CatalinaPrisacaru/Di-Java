mgInit

FUNCTION NAME

mgInit - initializes the OpenFlight API software development environment.

SYNOPSIS       (C and Python)

void mgInit (int*argc,
char*argv[] );

DESCRIPTION

mgInit initializes the OpenFlight API. In general this is the first function an application should call. This function performs the proper initialization of internal memory, the default palettes (color, material, light source, texture, etc), and loads the OpenFlight Data Dictionary. It is a mandatory call to activate the OpenFlight API programming environment.

There are some functions you can call in the OpenFlight API before this function. For example, your application can call mgSetMessagesEnabled to control what messages are displayed by mgInit when it is called. Also, if your stand-alone application specifies a custom folder from which plug-ins are to be loaded, you must call mgSetPluginFolder before mgInit.

Note: This function is for use in stand-alone applications only, and should not be called from inside a plug-in. Doing so may yield undefined results.

PARAMETERS

argc
pointer to an int containing the number of command line arguments
argv[]
the array of command line argument strings

ACCESS LEVEL

Level 1
