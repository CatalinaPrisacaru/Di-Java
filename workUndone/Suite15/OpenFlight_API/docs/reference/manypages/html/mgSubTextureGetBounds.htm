mgSubTextureGetBounds

FUNCTION NAME

mgSubTextureGetBounds - gets the bounds of a subtexture from a texture attribute record.

C SPECIFICATION

mgbool mgSubTextureGetBounds (mgrec*imgRec,
int*left,
int*bottom,
int*right,
int*top,
intindex );

PYTHON SPECIFICATION

mgbool, left, bottom, right, top mgSubTextureGetBounds (mgrec*imgRec,
intindex )

DESCRIPTION

mgSubTextureGetBounds gets the bounds (left, bottom, right and top coordinates) of the subtexture at the specified index within the specified texture attribute record imgRec (of type fltImage ). The bounds are returned in the output parameters, left, bottom, right and top.

Subtexture indices are contiguous and range from 0..N-1 where N is the value returned by mgSubTextureCount.

PARAMETERS

imgRec
the texture attribute record
left
address of value to receive lower left x coordinate of subtexture
bottom
address of value to receive lower left y coordinate of subtexture
right
address of value to receive upper right x coordinate of subtexture
top
address of value to receive upper right y coordinate of subtexture
index
the index of the subtexture

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
left
If function is successful, left contains the retrieved left value, otherwise left is undefined.
bottom
If function is successful, bottom contains the retrieved bottom value, otherwise bottom is undefined.
right
If function is successful, right contains the retrieved right value, otherwise right is undefined.
top
If function is successful, top contains the retrieved top value, otherwise top is undefined.

ACCESS LEVEL

Level 1
