mgwalkfunc

CALLBACK NAME

mgwalkfunc - Walk/Find callback function

SYNOPSIS

mgbool (*mgwalkfunc) ( mgrec* db,
mgrec* parent,
mgrec* rec,
void* userData );

DESCRIPTION

This is the signature for walk/find callback functions.

When you call mgWalk, mgWalkEx or mgWalkRecList, you specify a pre-visit and/or a post-visit function of this form. These functions are called before and after a node is visited, respectively.

When you call mgFind or mgFindEx, you specify a match function of this form.

When you call mgIterateRecList, you specify an iterate function of this form.

PARAMETERS

db
the database containing the node being visited
parent
the parent of the node being visited (MG_NULL if rec is db)
rec
the node currently being visited
userData
user defined data specified in call to mgWalk, mgWalkEx, mgFind, mgFindEx, mgWalkRecList, mgIterateRecList

RETURNS

When used for mgWalk, mgWalkEx or mgWalkRecList, return MG_TRUE to continue traversal, MG_FALSE to terminate.
When used for mgFind or mgFindEx, return MG_TRUE to add the node to the record list returned, MG_FALSE otherwise. When used for mgIterateRecList, return MG_TRUE to continue iteration of the record list, MG_FALSE to terminate.
