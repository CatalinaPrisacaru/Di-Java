mgResourceGetDialog

FUNCTION NAME

mgResourceGetDialog - creates a dialog instance derived from a dialog template contained in a resource.

C SPECIFICATION       (Not Available For Python)

mggui mgResourceGetDialog (mgguigui,
mgresourceresource,
mgdialogiddialogId,
unsigned intcallbackMask,
mgdialogfuncdialogFunc,
void*userData );

DESCRIPTION

mgResourceGetDialog builds a dialog instance using a dialog template extracted from the specified resource. The dialog template is identified in the resource by dialogId.

Specify a dialog function dialogFunc for the created dialog and select the dialog events you are interested in for your dialogFunc using the parameter callbackMask.

The dialog is created as a child of the dialog specified by gui. Note that gui can be a control, a dialog or MG_NULL. If gui is a control, it is assumed to be contained in the dialog that is to be the parent. If gui is a dialog, it is the dialog to be the parent. If gui is MG_NULL, the dialog is created as a child of the MultiGen modeling system main window.

The dialog created is not displayed until you explicitly do so by calling mgShowDialog.

PARAMETERS

gui
the dialog parent or a control in the dialog that is to be the parent
resource
the resource containing the dialog template
dialogId
the dialog template identifier
callbackMask
bitwise combination of dialog events selected for dialog function dialogfunc - choose any bitwise combination of MGCB_INIT, MGCB_SHOW, MGCB_REFRESH, MGCB_HIDE and MGCB_DESTROY
dialogFunc
dialog function to assign to dialog
userData
user defined data to pass to dialogfunc

RETURNS

If the dialog template is found in the resource and the dialog instance is successfully created, the dialog instance is returned, MG_NULL otherwise.

ACCESS LEVEL

Level 4
