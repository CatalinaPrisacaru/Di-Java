mgSetGuiCallback

FUNCTION NAME

mgSetGuiCallback - sets callback function for a control.

C SPECIFICATION       (Not Available For Python)

void mgSetGuiCallback (mgguicontrol,
unsigned intcallbackMask,
mgguifunccallback,
void*userData );

DESCRIPTION

mgSetGuiCallback assigns a callback function to the specified control.

You select which control events the callback function is to be called for using the parameter callbackMask. The value for this parameter may be any bitwise combination of MGCB_ACTIVATE, MGCB_REFRESH and MGCB_DRAW.

The MGCB_ACTIVATE and MGCB_REFRESH events are sent to all sent to all control types. The MGCB_DRAW event is only sent to GL control types.

PARAMETERS

control
the control whose callback function is to be set
callbackMask
mask indicating which control events are selected for callback function
callback
the control callback function
userData
user data to be passed to callback function when it is called

ACCESS LEVEL

Level 4
