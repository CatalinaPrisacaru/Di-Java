mgReadTexture

FUNCTION NAME

mgReadTexture - loads a texture and creates a new entry in a database's texture palette at a specified location and index.

SYNOPSIS       (C and Python)

mgbool mgReadTexture (mgrec*db,
const char*textureName,
intindex,
intx,
inty );

DESCRIPTION

mgReadTexture loads a texture (image and attributes) into the palette of the specified database db. The palette entry is loaded at the specified index and lower left hand location x, y.

The attributes are read from the file specified by the texture file name appended with ".attr". If there is no attributes file, default attributes are assigned.

PARAMETERS

db
the database
textureName
the name of the texture file on disk
index
the index to assign to the texture palette entry created
x
the x position of the texture in the palette (in pixels)
y
the y position of the texture in the palette (in pixels)

RETURNS

Returns MG_TRUE if the image is read successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
