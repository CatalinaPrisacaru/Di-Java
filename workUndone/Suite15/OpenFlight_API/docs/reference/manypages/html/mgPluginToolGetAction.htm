mgPluginToolGetAction

FUNCTION NAME

mgPluginToolGetAction - gets a plug-in tool action associated with a plug-in tool.

C SPECIFICATION       (Not Available For Python)

mgtoolaction mgPluginToolGetAction (mgplugintoolpluginTool,
const char*actionName );

DESCRIPTION

mgPluginToolGetAction gets the plug-in tool action identified by actionName associated with plug-in tool pluginTool.

PARAMETERS

pluginTool
the plug-in tool for which the action applies
actionName
the name of the tool action to lookup

RETURNS

Returns the plug-in action if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
