mgRegisterPostEdit

FUNCTION NAME

mgRegisterPostEdit - sets the Post-Edit action function for a data extension field.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRegisterPostEdit (mgpluginsitepluginSite,
mgcodecode,
mgposteditfuncpostEditFunc );

DESCRIPTION

mgRegisterPostEdit assigns the Post-Edit action function, postEditFunc to the data field, code for the data extension site pluginSite.

PARAMETERS

pluginSite
the data extension site
code
the field code
postEditFunc
the Post-Edit action function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 3

SEE ALSO

mgRegisterSite, mgRegisterPreEdit, mgposteditfunc
