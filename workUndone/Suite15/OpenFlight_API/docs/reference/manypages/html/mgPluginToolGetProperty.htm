mgPluginToolGetProperty

FUNCTION NAME

mgPluginToolGetProperty - retrieves user defined data associated with a plug-in tool.

C SPECIFICATION       (Not Available For Python)

mgpropertyvalue mgPluginToolGetProperty (mgplugintoolpluginTool,
const mgpropertynamepropName );

DESCRIPTION

Use mgPluginToolGetProperty to retrieve the data named propName that was associated with the plug-in tool pluginTool using mgPluginToolPutProperty.

PARAMETERS

pluginTool
the plug-in tool
propName
the name of the property to get

RETURNS

Returns the user-defined data if found, MG_NULL otherwise.

ACCESS LEVEL

Level 3

mgPluginToolPutProperty, mgPluginToolDeleteProperty, mgPluginToolGetUserData, mgPluginPutProperty, mgPluginGetProperty, mgPluginDeleteProperty
