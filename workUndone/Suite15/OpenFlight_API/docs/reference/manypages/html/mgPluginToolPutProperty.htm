mgPluginToolPutProperty

FUNCTION NAME

mgPluginToolPutProperty - associates user defined data with a plug-in tool.

C SPECIFICATION       (Not Available For Python)

mgbool mgPluginToolPutProperty (mgplugintoolpluginTool,
const mgpropertynamepropName,
mgpropertyvaluepropValue );

DESCRIPTION

mgPluginToolPutProperty allows you to associate data of any type, propValue, with a plug-in tool. The user-defined data is associated with the plug-in tool, pluginTool, and identified by a property name string, propName, allowing any number of data records to be associated with the plug-in tool, as long as all property names are unique.

After using this function to associate user data to a plug-in tool, use the function mgPluginToolGetProperty to retrieve the data. Then when you no longer need the data associated to the plug-in tool, use mgPluginToolDeleteProperty to disassociate the data from the plug-in tool. Note that mgPluginToolDeleteProperty does not free or otherwise affect the actual data that was associated. If this data was dynamically allocated, it is the callers responsibility to deallocate it.

PARAMETERS

pluginTool
the plug-in tool to which the data is to be associated
propName
the name of the property to assign
propValue
the value of the property to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

The following example associates a string property to a plug-in tool, retrieves the string and prints it out. Finally, it deletes the property from the plug-in tool.
#define PROPERTY_NAME   "My String Property"
#define USER_STRING     "This String is associated to my plug-in tool"

char* string;

// allocate the string to be associated to the plug-in tool
string = mgMalloc (strlen (USER_STRING) + 1);
strcpy (string, USER_STRING);

// associate the string as a property of the plug-in
mgPluginToolPutProperty (pluginTool, PROPERTY_NAME, string);

// Then later, retrieve the string property, print it out

char* string;

// Retrieve the string property from the plug-in
string = (char*) mgPluginToolGetProperty (pluginTool, PROPERTY_NAME);
printf ("%s: %s", PROPERTY_NAME, string);

// Output is:
// My String Property: This String is associated to my plug-in tool

// Then finally, when you no longer need the string to be
// associated to the plug-in tool

char* string;

// Retrieve the string property from the plug-in tool (need to free it)
string = (char*) mgPluginToolGetProperty (pluginTool, PROPERTY_NAME);
mgFree (string);
mgPluginToolDeleteProperty (pluginTool, PROPERTY_NAME);
