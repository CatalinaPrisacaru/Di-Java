mgrendererfunc

CALLBACK NAME

mgrendererfunc - Renderer function

SYNOPSIS

void (*mgrendererfunc) ( mgrenderercontext rendererContext,
mgrenderercallbackreason callbackReason,
void* rendererToolData,
void* callData );

DESCRIPTION

This is the signature for renderer functions.

PARAMETERS

rendererContext
the renderer context that identifies the renderer instance
callbackReason
the renderer event that triggered the callback
rendererToolData
user defined tool instance data set in toolData field of callData parameter (mgrenderercallbackrec) during renderer tool start function mgtoolstartfunc
callData
callback specific data

RETURNS

Currently, the value returned by mgrendererfunc is ignored and reserved for future enhancement. For now always return MSTAT_OK.
