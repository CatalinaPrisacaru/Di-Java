mgTextGetInteger

FUNCTION NAME

mgTextGetInteger - retrieves integer value from text a control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextGetInteger (mgguicontrol,
int*intVal );

DESCRIPTION

mgTextGetInteger attempts to interpret the text value displayed in the specified text control as an integer value.

If the text displayed does represent an integer value, the value will be returned in the parameter intVal.

PARAMETERS

control
the text control
intVal
address of integer value to receive numeric value from text control

RETURNS

Use MSTAT_ISOK to check return value for success. If function is successful, intVal contains the corresponding numeric value, otherwise intVal is undefined.

ACCESS LEVEL

Level 4
