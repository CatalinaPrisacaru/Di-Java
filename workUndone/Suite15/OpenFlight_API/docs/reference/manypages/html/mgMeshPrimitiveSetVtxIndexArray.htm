mgMeshPrimitiveSetVtxIndexArray

FUNCTION NAME

mgMeshPrimitiveSetVtxIndexArray - sets the vertex index array for a mesh primitive.

SYNOPSIS       (C and Python)

int mgMeshPrimitiveSetVtxIndexArray (mgrec*rec,
intprimNo,
int*indexArray,
intnumVtx );

DESCRIPTION

This function creates the vertex index array for mesh primitive number primNo in mesh rec and fills in the values of the array created using the indices in indexArray. The vertex index array created will be large enough to hold numVtx indices.

The vertex index array of a mesh primitive is an array of integer values that define the vertex ordering of the vertices that make up the mesh primitive. The integer indices in the vertex index array refer to vertices stored in the mesh vertex pool.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
primNo
the mesh primitive number starting from 0
indexArray
the vertex indices for this primitive
numVtx
the number of vertex indices to write from indexArray

RETURNS

Returns the number of vertex indices that were set if successful, -1 otherwise. This function will fail if rec is not a valid mesh node or if primNo does not specify a valid mesh primitive number.

ACCESS LEVEL

Level 2
