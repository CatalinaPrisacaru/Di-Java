mgGetCurrentLightPointAppearance

FUNCTION NAME

mgGetCurrentLightPointAppearance - returns index of current modeling light point appearance.

C SPECIFICATION

mgbool mgGetCurrentLightPointAppearance (mgrec*db,
int*index );

PYTHON SPECIFICATION

mgbool, index mgGetCurrentLightPointAppearance (mgrec*db )

DESCRIPTION

mgGetCurrentLightPointAppearance returns the index of the current modeling light point appearance selected for database db.

PARAMETERS

db
the database
index
address of value to receive index of current modeling light point appearance

RETURNS

mgbool
MG_TRUE if current modeling light point appearance for db could be determined, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

ACCESS LEVEL

Level 4
