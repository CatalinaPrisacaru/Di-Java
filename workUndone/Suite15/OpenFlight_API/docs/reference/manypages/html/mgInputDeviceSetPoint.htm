mgInputDeviceSetPoint

FUNCTION NAME

mgInputDeviceSetPoint - sets a point on an input device.

SYNOPSIS       (C and Python)

mgstatus mgInputDeviceSetPoint (mginputdeviceinputDevice,
intx,
inty );

DESCRIPTION

mgInputDeviceSetPoint sets a 2D point with coordinates x, y on the input device inputDevice obtained by mgInputDeviceGetHandle. The event is sent by calling mgInputDeviceSendEvent.

PARAMETERS

inputDevice
the input device obtained from mgInputDeviceGetHandle
x
x coordinate of the point
y
y coordinate of the point

RETURNS

Use MSTAT_ISOK to check return value for success

ACCESS LEVEL

Level 4

SEE ALSO

mgInputDeviceSendEvent, mgInputDeviceGetHandle, mgInputDeviceSetVertex
