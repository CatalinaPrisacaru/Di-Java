mgMeshGetVtxOffset

FUNCTION NAME

mgMeshGetVtxOffset - gets the offset for vertex data in a mesh.

SYNOPSIS       (C and Python)

int mgMeshGetVtxOffset (mgrec*rec,
unsigned intbit );

DESCRIPTION

This function retrieves the offset for the vertex data specified by mesh vertex mask bit bit in the vertex pool of mesh rec.

The vertex data offset for a particular kind of data within the mesh vertex pool indicates the number of bytes from the start of the vertex pool to the first byte of the data in the first vertex.

Use a single Mesh Vertex Mask Bit for bit to specify which data you want the offset for:
MMESH_VTXCOORD,
MMESH_VTXCOLOR,
MMESH_VTXCOLORRGB,
MMESH_VTXNORMAL,
MMESH_VTXUV0,
MMESH_VTXUV1,
MMESH_VTXUV2,
MMESH_VTXUV3,
MMESH_VTXUV4,
MMESH_VTXUV5,
MMESH_VTXUV6, or
MMESH_VTXUV7

You can use the values returned by this function together with the vertex stride returned by mgMeshGetVtxStride to extract data from the mesh vertex pool. Note: This is not the recommended technique for extracting data from the mesh vertex pool. Instead, it is much simpler to use the functions mgMeshGetVtxCoord, mgMeshGetVtxColor, mgMeshGetVtxColorRGBA, mgMeshGetVtxColorRGB, mgMeshGetVtxColorAlpha, mgMeshGetVtxNormal and mgMeshGetVtxUV.

PARAMETERS

rec
the fltMesh node
bit
the mesh vertex mask bit specifying the data to get offset for

RETURNS

Returns the offset for the specified data in the vertex pool of mesh rec if successful, -1 otherwise. This function will fail if rec is not a valid mesh node or if the vertices in the mesh do not have the data specified by bit.

ACCESS LEVEL

Level 1
