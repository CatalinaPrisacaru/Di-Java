mgGetTextString

FUNCTION NAME

mgGetTextString - gets the string in a fltText record.

SYNOPSIS       (C and Python)

char* mgGetTextString (mgrec*rec );

DESCRIPTION

Given a fltText record, rec, mgGetTextString returns the string stored in the record.

Use mgSetTextString to change the string of a fltText record.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

rec
the record

RETURNS

Returns the fltText string if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
