mgpalettenotifyfunc

CALLBACK NAME

mgpalettenotifyfunc - Palette notifier callback function

SYNOPSIS

void (*mgpalettenotifyfunc) ( mgnotifier notifier,
mgnotifierevent event,
mgrec* db,
int index,
void* userData );

PARAMETERS

notifier
the notifier
event
the notification event that triggered the callback
db
the database to which or in which the event occurred
index
the index of the palette entry that is affected
userData
user defined data specified when notifier registered
