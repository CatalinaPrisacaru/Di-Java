mgSetTextureGeoNumCtlPts

FUNCTION NAME

mgSetTextureGeoNumCtlPts - sets the number of control points attribute (used to geo reference the image) for a georeference info object.

C SPECIFICATION       (Not Available For Python)

mgbool mgSetTextureGeoNumCtlPts (mgimagegeoinfogeoInfo,
intnumCoords );

DESCRIPTION

mgSetTextureGeoImageOrigin sets the number of control points attribute for the specified georeference info object geoInfo to the specified value numCoords.

When the image get geo info function, mgimagegetgeoinfofunc is called for an image importer plug-in tool, the get geo info function uses this function to set the number of control points attribute for the georeference info object.

Valid values for the number of control points attribute are any positive integer.

The default value is 0.

PARAMETERS

geoInfo
the georeference info object
numCoords
the number of control points

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
