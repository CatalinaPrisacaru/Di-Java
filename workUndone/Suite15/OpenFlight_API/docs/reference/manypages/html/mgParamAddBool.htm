mgParamAddBool

FUNCTION NAME

mgParamAddBool - adds a boolean parameter to a parameter block.

C SPECIFICATION       (Not Available For Python)

mgparam mgParamAddBool (mgparamblockparamBlock,
const char*paramName,
mgbooldefaultValue );

DESCRIPTION

mgParamAddBool creates a new boolean parameter named paramName and adds it to the parameter block paramBlock. The parameter value is initialized with defaultValue.

After the parameter value is defined, use mgParamGetBool and mgParamSetBool to query and change its value, respectively.

Note: If the named parameter already exists in paramBlock, it is replaced.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
defaultValue
default value of the parameter

RETURNS

Returns the parameter if created successfully, MG_NULL otherwise.

ACCESS LEVEL

Level 4
