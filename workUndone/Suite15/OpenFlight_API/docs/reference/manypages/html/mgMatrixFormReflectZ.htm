mgMatrixFormReflectZ

FUNCTION NAME

mgMatrixFormReflectZ - creates a matrix that will reflect about the Z axis.

C SPECIFICATION

void mgMatrixFormReflectZ (mgmatrix*matrix );

PYTHON SPECIFICATION

mgmatrix mgMatrixFormReflectZ ( )

DESCRIPTION

mgMatrixFormReflectZ gets a matrix that will reflect about the Z axis or yaw 180 degrees.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
