MGAPIFUNC

MACRO NAME

MGAPIFUNC - exports a symbol from an OpenFlight API library module.

SYNOPSIS

MGAPIFUNC ( type );

PARAMETERS

type
the return type (for function symbols) or object type (for variables) exported by the OpenFlight API.

DESCRIPTION

This is a macro used ONLY by the OpenFlight API to export symbols (functions or variables) from OpenFlight API library modules. All exported functions in the OpenFlight API are declared using this macro in corresponding include files.

Third party developers should NOT use this macro to declare symbols in their plug-in modules or stand-alone applications. Instead, the macro MGPIDECLARE macro should be used to export symbols from third party developed plug-in modules.
