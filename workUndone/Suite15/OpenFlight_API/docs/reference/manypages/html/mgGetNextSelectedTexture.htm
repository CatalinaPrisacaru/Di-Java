mgGetNextSelectedTexture

FUNCTION NAME

mgGetNextSelectedTexture - gets successive selected entries in a texture palette.

C SPECIFICATION

mgbool mgGetNextSelectedTexture (mgrec*db,
int*index,
char*textureName );

PYTHON SPECIFICATION

mgbool, index, name mgGetNextSelectedTexture (mgrec*db )

DESCRIPTION

mgGetNextSelectedTexture gets the index and name of the next selected texture contained in the texture palette of database db.

After calling mgGetFirstSelectedTexture, successive calls to mgGetNextSelectedTexture returns successive texture palette entries.

C NOTES

If you pass MG_NULL for textureName, the name of the texture is not returned. If you do pass a character string for textureName, the buffer should be at least 1024 characters.

PARAMETERS

db
the database
index
address of value to receive index
textureName
address of string to receive texture name

RETURNS

mgbool
MG_TRUE if the texture palette entry is found, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.
name
If function is successful, name contains the retrieved name, otherwise name is undefined.

EXAMPLE

See mgGetFirstSelectedTexture for an example of how to use this function.

ACCESS LEVEL

Level 4
