mgPluginToolGetUserData

FUNCTION NAME

mgPluginToolGetUserData - gets the user data associated with a plug-in tool.

C SPECIFICATION       (Not Available For Python)

void* mgPluginToolGetUserData (mgplugintoolpluginTool );

DESCRIPTION

mgPluginToolGetUserData gets the user data that was associated to the plug-in tool pluginTool when it was registered via mgRegisterImporter, mgRegisterExporter, mgRegisterImageImporter, mgRegisterViewer, mgRegisterEditor or mgRegisterInputDevice.

PARAMETERS

pluginTool
the plug-in tool

RETURNS

Returns the user data associated to pluginTool if found, MG_NULL otherwise.

ACCESS LEVEL

Level 3

SEE ALSO

mgRegisterImporter, mgRegisterExporter, mgRegisterImageImporter, mgRegisterViewer, mgRegisterEditor, mgRegisterInputDevice, mgPluginToolGetProperty
