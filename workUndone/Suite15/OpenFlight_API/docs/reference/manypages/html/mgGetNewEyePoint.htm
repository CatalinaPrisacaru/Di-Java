mgGetNewEyePoint

FUNCTION NAME

mgGetNewEyePoint - returns a new eyepoint record.

SYNOPSIS       (C and Python)

mgrec* mgGetNewEyePoint (mgrec*db );

DESCRIPTION

mgGetNewEyePoint returns a new eyepoint record that must solely be used as a parameter to mgGetCurrentEyePoint, mgSetCurrentEyePoint and mgSetEyePoint.

Use mgFreeNewEyePoint when you are done with the record.

PARAMETERS

db
the database

ACCESS LEVEL

Level 4

SEE ALSO

mgCopyEyePoint, mgFreeNewEyePoint
