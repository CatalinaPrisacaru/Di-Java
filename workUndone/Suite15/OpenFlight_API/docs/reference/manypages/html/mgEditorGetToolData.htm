mgEditorGetToolData

FUNCTION NAME

mgEditorGetToolData - gets the user defined tool instance data associated with an editor context.

C SPECIFICATION       (Not Available For Python)

void* mgEditorGetToolData (mgeditorcontexteditorContext );

DESCRIPTION

mgEditorGetToolData returns the user defined tool instance data that is associated with the specified editor context, editorContext.

When an editor tool instance is active, a unique association exists between the tool instance and the corresponding editor tool instance data. This function is used to retrieve the tool data associated with a particular editor tool instance.

PARAMETERS

editorContext
the editor context to get associated tool instance data for

RETURNS

Returns editor tool instance data if defined for this tool instance, MG_NULL otherwise.

ACCESS LEVEL

Level 4
