mgWriteImageAttributes

FUNCTION NAME

mgWriteImageAttributes - writes image attributes to disk.

SYNOPSIS       (C and Python)

mgbool mgWriteImageAttributes (const char*textureName,
mgrec*rec );

DESCRIPTION

Given a fltImage record, rec, and a textureName, mgWriteImageAttribute writes an image’s attributes file to disk using the given file name, textureName appended with ".attr".

PARAMETERS

textureName
the image attributes file name, without the extension ".attr"
rec
a pointer to the fltImage (image attributes) record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
