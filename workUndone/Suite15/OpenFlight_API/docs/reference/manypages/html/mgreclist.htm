mgreclist

TYPE NAME

mgreclist - Abstract type used to represent a general record list

DESCRIPTION

This node list is used for select lists and construct lists.

The node list maintains a list of node records and a traversal pointer to keep track of which node in the list is next to be traversed by mgGetNextRecInList.
