mgWriteDb

FUNCTION NAME

mgWriteDb - writes an OpenFlight database to disk.

SYNOPSIS       (C and Python)

mgbool mgWriteDb (mgrec*db );

DESCRIPTION

Given the database, db, mgWriteDb writes this database to disk using the file name stored in the database header. The file name must first be opened and initialized with either the mgOpenDb or mgNewDb functions, which associate the database with a file name.

Note: If you call this function in Creator, it will fail if db is open on the Creator desktop.

PARAMETERS

db
the database to write

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

mgrec *db1, *db2;
db1 = mgOpenDb ("file1.flt");
db2 = mgNewDb ("newfile.flt");

// make some changes to the databases...

mgWriteDb (db1);
mgWriteDb (db2);

mgCloseDb (db1);
mgCloseDb (db2);

ACCESS LEVEL

Level 2
