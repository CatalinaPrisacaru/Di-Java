mgExtensionFieldSetName

FUNCTION NAME

mgExtensionFieldSetName - sets the name of an extension field.

SYNOPSIS       (C and Python)

mgbool mgExtensionFieldSetName (const char*fieldGuid,
const char*name );

DESCRIPTION

mgExtensionFieldSetName sets the name of an extension field identified by fieldGuid.

PARAMETERS

fieldGuid
the GUID of the site
name
the name to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
