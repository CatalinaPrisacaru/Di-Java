mgGetRecListLevel

FUNCTION NAME

mgGetRecListLevel - returns the level of the record list.

SYNOPSIS       (C and Python)

mgcode mgGetRecListLevel (mgreclistrecList );

DESCRIPTION

mgGetRecListLevel returns the node level of the first item in the specified record list recList.

Nodes contained in a record list are not guaranteed to all be of the same level. This level returned by this function may not be indicative of all nodes contained in the record list.

PARAMETERS

recList
the record list to get level

RETURNS

Returns the node level of the first item in the list, and returns 0 if list is empty.

ACCESS LEVEL

Level 1
