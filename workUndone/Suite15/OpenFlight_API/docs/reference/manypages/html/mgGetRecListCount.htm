mgGetRecListCount

FUNCTION NAME

mgGetRecListCount - counts the number of items in a record list.

SYNOPSIS       (C and Python)

int mgGetRecListCount (mgreclistrecList );

DESCRIPTION

mgGetRecListCount returns the number of items contained in the specified record list recList.

PARAMETERS

recList
the record list to count items for

RETURNS

Returns the number of items in the record list, and returns 0 if list is empty.

ACCESS LEVEL

Level 1
