mgguicallbackreason

TYPE NAME

mgguicallbackreason - Callback reason type used to specify dialog and control events

DESCRIPTION

When a control callback or dialog function is called, the callbackReason parameter passed in of this type will specify which event triggered the callback.

Dialog and control callback functions are called to report these events.

The events that are sent to dialog functions are:
MGCB_INIT
MGCB_SHOW
MGCB_SIZE
MGCB_REFRESH
MGCB_HIDE
MGCB_DESTROY

The events that are sent to control callback functions are:
MGCB_ACTIVATE
MGCB_REFRESH
MGCB_DRAW
