mgIsReference

FUNCTION NAME

mgIsReference - determines if a node is a reference node.

SYNOPSIS       (C and Python)

mgbool mgIsReference (mgrec*rec );

DESCRIPTION

mgIsReference determines if a node rec is a reference node. A reference node is one that is referenced by other nodes.

PARAMETERS

rec
the node in question

RETURNS

Returns MG_TRUE if the node is a reference, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
