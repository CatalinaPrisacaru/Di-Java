mgTabRulerSetGuiCallback

FUNCTION NAME

mgTabRulerSetGuiCallback - sets callback function for a tab ruler control.

C SPECIFICATION       (Not Available For Python)

void mgTabRulerSetGuiCallback (mgguicontrol,
unsigned intcallbackMask,
mgguifunccallback,
void*userData );

DESCRIPTION

mgTabRulerSetGuiCallback assigns a callback function to the specified tab ruler control.

You select which control events the callback function is to be called for using the parameter callbackMask. The value for this parameter may be any bitwise combination of MTRUL_TAB_SELECTED, MTRUL_TAB_DESELECTED, MTRUL_TAB_DRAGGED MTRUL_TAB_BUMPED, MTRUL_TAB_CHANGED, MTRUL_TAB_CREATED, MTRUL_TAB_DELETED and MTRUL_TAB_RELEASED

PARAMETERS

control
the control whose callback function is to be set
callbackMask
mask indicating which control events are selected for callback function
callback
the control callback function
userData
user data to be passed to callback function when it is called

ACCESS LEVEL

Level 4
