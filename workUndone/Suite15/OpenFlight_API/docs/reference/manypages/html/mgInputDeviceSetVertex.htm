mgInputDeviceSetVertex

FUNCTION NAME

mgInputDeviceSetVertex - sets a vertex on an input device.

SYNOPSIS       (C and Python)

mgstatus mgInputDeviceSetVertex (mginputdeviceinputDevice,
doublex,
doubley,
doublez );

DESCRIPTION

mgInputDeviceSetVertex sets a vertex with coordinates x, y, z on the input device inputDevice obtained by mgInputDeviceGetHandle. The event is sent by calling mgInputDeviceSendEvent.

PARAMETERS

inputDevice
the input device obtained from mgInputDeviceGetHandle
x
x coordinate of the vertex
y
y coordinate of the vertex
z
z coordinate of the vertex

RETURNS

Use MSTAT_ISOK to check return value for success

ACCESS LEVEL

Level 4

SEE ALSO

mgInputDeviceSendEvent, mgInputDeviceGetHandle, mgInputDeviceSetPoint
