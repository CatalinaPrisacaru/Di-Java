mgReadImageInfo

FUNCTION NAME

mgReadImageInfo - reads the header information from an image file.

C SPECIFICATION

mgstatus mgReadImageInfo (const char*textureName,
mgimageinfo*textureInfo );

PYTHON SPECIFICATION

mgstatus, textureInfo mgReadImageInfo (stringtextureName )

DESCRIPTION

mgReadImageInfo reads the header of an image on disk and returns an image info object containing the information read. The parameter textureName is the name of the image file.

C NOTES

When you are through with the returned image info object, you must free it by calling mgFreeImageInfo.

PYTHON NOTES

It is not necessary to free the returned image info object in Python.

PARAMETERS

textureName
the name of the image file
textureInfo
address of object to receive image info object allocated and filled

RETURNS

mgstatus
MIMG_NO_ERROR if successful, otherwise applicable texture error code.
textureInfo
If function is successful, textureInfo contains the image info object, otherwise textureInfo is undefined.

EXAMPLE

The following examples (in C and Python) illustrate the use of mgReadImageInfo to read and examine an image info object for a texture.

C EXAMPLE

int imgError;
mgimageinfo info = MG_NULL;
imgError = mgReadImageInfo ("c:/MyTextures/ah6.rgb", &info);
if (imgError == MIMG_NO_ERROR)
{
   mgbool ok;
   int width, height, type;
   ok = mgGetTextureWidth (info, &width);
   if (ok == MG_TRUE)
      printf ("Width: %d\n", width);
   ok = mgGetTextureHeight (info, &height);
   if (ok == MG_TRUE)
      printf ("Height: %d\n", height);
   ok = mgGetTextureType (info, &type);
   if (ok == MG_TRUE)
      printf ("Type: %d\n", type);
}

PYTHON EXAMPLE

imgError, info = mgReadImageInfo ("c:/MyTextures/ah6.rgb")
if (imgError == MIMG_NO_ERROR):
   ok, width = mgGetTextureWidth (info)
   if (ok == MG_TRUE):
      print "Width:", width
   ok, height = mgGetTextureHeight (info)
   if (ok == MG_TRUE):
      print "Height:", height
   ok, type = mgGetTextureType (info)
   if (ok == MG_TRUE):
      print "Type:", type

Level 1
