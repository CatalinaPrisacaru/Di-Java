ddIsFieldFollowInLine

FUNCTION NAME

ddIsFieldFollowInLine - checks if a specific code is a legal field of a record; search includes the record itself and all subsequent in-line records.

SYNOPSIS       (C and Python)

mgbool ddIsFieldFollowInLine (mgcodercode,
mgcodefcode );

DESCRIPTION

ddIsFieldFollowInLine searches all fields of record rcode for a match with fcode. If fcode is not found in rcode, ddIsFieldFollowInLine continues the search for fcode, following the attribute list of every in-line record of rcode until either a match is found, or until all subsequent in-line records are searched.

PARAMETERS

rcode
the record being searched
fcode
the field code to match

RETURNS

Returns MG_TRUE if fcode is a legal field of rcode, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
