mgglmousedatarec

STRUCT NAME

mgglmousedatarec - generic callback structure for callData parameter of mgglmousefunc

SYNOPSIS

typedef struct  {
   mgglmousedatatype callDataType;
} mgglmousedatarec;

MEMBERS

callDataType
mgglmousedatatype - call data type that indicates the type of call data this object really is

DESCRIPTION

When a GL mouse function is called, the callData parameter passed in will be a pointer to an object type that is dependent on the context of the mouse function. The first field of each data structure is the same as this structure. In this way, the callData parameter can always safely be cast to a pointer to an object of this type. Doing so, you can then examine the callDataType field to determine which callback record structure has actually been passed.
