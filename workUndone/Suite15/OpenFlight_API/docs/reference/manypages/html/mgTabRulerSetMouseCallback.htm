mgTabRulerSetMouseCallback

FUNCTION NAME

mgTabRulerSetMouseCallback - sets mouse callback function for a tab ruler control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTabRulerSetMouseCallback (mgguicontrol,
unsigned intactionMask,
mgglmousefunccallback,
void*userData );

DESCRIPTION

mgTabRulerSetGuiCallback assigns a mouse callback function to the specified tab ruler control.

PARAMETERS

control
the control whose callback function is to be set
actionMask
mask indicating which mouse actions are selected for mouse function
callback
the control callback function
userData
user data to be passed to callback function when it is called

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
