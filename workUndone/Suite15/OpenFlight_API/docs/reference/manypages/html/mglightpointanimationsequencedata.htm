mglightpointanimationsequencedata

STRUCT NAME

mglightpointanimationsequencedata - Light Point Animation Sequence data

SYNOPSIS

typedef struct  {
   unsigned int state;
   float duration;
   unsigned int colorIndex;
   float colorIntensity;
} mglightpointanimationsequencedata;

MEMBERS

state
unsigned int - state of this item in the sequence - On = 0, Off = 1, Color Change = 2
duration
float - time (in seconds) for this item in the sequence
colorIndex
unsigned int - index of color for this item in the sequence if state is On or Color Change, ignored when state is Off
colorIntensity
float - intensity of color for this item in the sequence if state is On or Color Change, ignored when state is Off

DESCRIPTION

Use an array of mglightpointanimationsequencedata to add and get light point animation sequence records.
