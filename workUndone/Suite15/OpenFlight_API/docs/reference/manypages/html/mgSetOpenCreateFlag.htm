mgSetOpenCreateFlag

FUNCTION NAME

mgSetOpenCreateFlag - sets the preference that determines if a new database is created when mgOpenDb is instructed to open a file that does not exist.

SYNOPSIS       (C and Python)

void mgSetOpenCreateFlag (mgboolflag );

DESCRIPTION

mgSetOpenCreateFlag tells mgOpenDb how to act when it is asked to open a database that does not exist. If flag is MG_TRUE, mgNewDb is called to create a new file. If flag is MG_FALSE (the default), mgOpenDb fails and returns.

The default value for this preference is MG_FALSE.

PARAMETERS

flag
whether or not to create a new database

C EXAMPLE

mgrec* db;
mgSetOpenCreateFlag (MG_TRUE);
db = mgOpenDb ("file1.flt");

ACCESS LEVEL

Level 1
