mgEditorSetPickFunc

FUNCTION NAME

mgEditorSetPickFunc - sets the pick function for an editor tool.

C SPECIFICATION       (Not Available For Python)

void mgEditorSetPickFunc (mgplugintooleditorPluginTool,
mgeditorpickfuncpickFunc );

DESCRIPTION

mgEditorSetPickFunc assigns a pick function to the specified editor tool editorPluginTool.

Pick functions are called to notify your editor tool when geometry has been picked by the user.

PARAMETERS

editorPluginTool
the editor plug-in tool
pickFunc
the pick input function

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following is a typical pick function.
mgEditorSetPickFunc (pluginTool, PickFunc);

static void PickFunc (mgeditorcontext editorContext,
  mgpickinputdata* pickInputData,
  void* toolData)
{
   toolrec* toolRec = (toolrec*) toolData;

   unsigned keyboardFlags  = pickInputData->keyboardFlags;
   unsigned buttonFlags    = pickInputData->buttonFlags;

   int leftMouseDown       = buttonFlags & MMB_LEFTMOUSE;
   int middleMouseDown     = buttonFlags & MMB_MIDDLEMOUSE;
   int rightMouseDown      = buttonFlags & MMB_RIGHTMOUSE;

   int ctrlKeyDown         = keyboardFlags & MKB_CTRLKEY;
   int altKeyDown          = keyboardFlags & MKB_ALTKEY;
   int shiftKeyDown        = keyboardFlags & MKB_SHIFTKEY;
}
