mgNewTextureMapping

FUNCTION NAME

mgNewTextureMapping - allocates a new record for an entry into a database’s texture mapping palette.

C SPECIFICATION

mgrec* mgNewTextureMapping (mgrec*db,
inttype,
const char*name,
int*index );

PYTHON SPECIFICATION

mgrec, index mgNewTextureMapping (mgrec*db,
inttype,
stringname )

DESCRIPTION

Given a database node db, a texture mapping type type, and a texture mapping entry name name, mgNewTextureMapping allocates a record containing a new texture mapping palette entry of the specified type, adds it to the database’s texture mapping palette, and returns the record. The index of the new texture mapping entry is returned in index.

Texture mapping entries can be any of these types:

1 = 3 point put
2 = 4 point put
4 = Spherical project
5 = Radial project
6 = Environment

The texture mapping properties can be set by using mgSetAttList with the returned texture mapping record.

Note: After setting or modifying any field of a texture mapping palette entry, you must call mgUpdateTextureMapping to make sure the internal data in the mapping entry is updated properly.

The code of the record returned will be fltTextureMapping .

PARAMETERS

db
the database
type
the type of the new texture mapping
name
the name of the new texture mapping (may be MG_NULL )
index
pointer to receive the index assigned to the new texture mapping palette entry record

RETURNS

mgrec
Returns the new texture mapping palette entry record if successful, MG_NULL otherwise.
index
If a palette record is returned, its index will be returned in index, otherwise index is undefined.

ACCESS LEVEL

Level 2
