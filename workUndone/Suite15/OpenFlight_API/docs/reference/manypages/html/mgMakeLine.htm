mgMakeLine

FUNCTION NAME

mgMakeLine - makes a line between two points.

SYNOPSIS       (C and Python)

mglined mgMakeLine (const mgcoord3d*coord1,
const mgcoord3d*coord2 );

DESCRIPTION

mgMakeLine returns a double precision line record representing the line passing through the two specified points, coord1 and coord2.

PARAMETERS

coord1
address of first coordinate that defines the line
coord2
address of second coordinate that defines the line

RETURNS

Returns a double precision line record.

ACCESS LEVEL

Level 1
