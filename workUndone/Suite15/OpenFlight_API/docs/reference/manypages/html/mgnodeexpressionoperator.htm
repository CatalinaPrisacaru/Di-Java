mgnodeexpressionoperator

ENUM NAME

mgnodeexpressionoperator - Node expression type

SYNOPSIS

typedef enum mgnodeexpressionoperator {
   MEOP_EQ,
   MEOP_NE,
   MEOP_LT,
   MEOP_LE,
   MEOP_GT,
   MEOP_GE,
   MEOP_LOGAND,
   MEOP_LOGOR,
   MEOP_ADD,
   MEOP_SUB,
   MEOP_MUL,
   MEOP_DIV,
   MEOP_FUNC,
   MEOP_LIT,
   MEOP_GETATT,
   MEOP_GETVAR,
} mgnodeexpressionoperator;

MEMBERS

MEOP_EQ
The equal to operator.
MEOP_NE
The not equal to operator.
MEOP_LT
The less than operator.
MEOP_LE
The less than or equal to operator.
MEOP_GT
The greater than operator.
MEOP_GE
The greater than or equal to operator.
MEOP_LOGAND
The logical and operator.
MEOP_LOGOR
The logical or operator.
MEOP_ADD
The addition operator.
MEOP_SUB
The subtraction operator.
MEOP_MUL
The multiplication operator.
MEOP_DIV
The division operator.
MEOP_FUNC
The indication to use a function as the operator.
MEOP_LIT
The indication of a literal (no operation performed).
MEOP_GETATT
The indication to use the mgGetAttList function.
MEOP_GETVAR
The indication of a variable.

DESCRIPTION

This enumerated type is used to indicate the operator to be embodied by the the expression returned from the mgNewNodeExpressionOperator call
