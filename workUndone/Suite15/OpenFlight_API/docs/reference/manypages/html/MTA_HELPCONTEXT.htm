MTA_HELPCONTEXT

NAME

MTA_HELPCONTEXT - Tool Attribute Help Context

DESCRIPTION

This string attribute specifies the help topic for context sensitive help for your plug-in tool. The help context identifies the location in your help file where the plug-in tool context sensitive help is found.
