mgSetAutoUpdatePriorityFlag

FUNCTION NAME

mgSetAutoUpdatePriorityFlag - sets the preference that determines if mgSetAttList called on a node to set its relative priority attribute automatically repositions the node among its siblings according to the priority value set.

SYNOPSIS       (C and Python)

void mgSetAutoUpdatePriorityFlag (mgboolflag );

DESCRIPTION

mgSetAutoUpdatePriorityFlag tells mgSetAttList how to act when called on a node to set one of the relative priority attributes fltPolyPrio , fltObjPrio , or fltGrpPrio . If flag is MG_TRUE (the default), mgSetAttList will reposition the node such that all siblings to its left have priority values less than or equal to the new value set. If flag is MG_FALSE (the default), mgSetAttList will not change the position of the node.

The default value for this preference is MG_TRUE.

PARAMETERS

flag
whether or not to reposition the node when fltPolyPrio , fltObjPrio or fltGrpPrio is set by mgSetAttList.

ACCESS LEVEL

Level 1
