mgMeshPrimitiveSetType

FUNCTION NAME

mgMeshPrimitiveSetType - sets the type of a mesh primitive.

SYNOPSIS       (C and Python)

int mgMeshPrimitiveSetType (mgrec*rec,
intprimNo,
inttype );

DESCRIPTION

This function sets the type for mesh primitive number primNo in mesh rec. A mesh node is composed of one or more mesh primitives. Each mesh primitive is one of four types:

MPRIM_TRI_STRIP - Triangle Strip
MPRIM_TRI_FAN - Triangle Fan
MPRIM_QUAD_STRIP - Quadrilateral Strip
MPRIM_INDEXED_POLY - Indexed Polygon

The first mesh primitive in a mesh is number 0. The fltMeshNumPrimitives attribute of a mesh node specifies how many primitives are contained in the mesh.

PARAMETERS

rec
the fltMesh node
primNo
the mesh primitive number starting from 0
type
the primitive type

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if primNo does not specify a valid mesh primitive number, or type does not specify a valid mesh primitive type.

ACCESS LEVEL

Level 2
