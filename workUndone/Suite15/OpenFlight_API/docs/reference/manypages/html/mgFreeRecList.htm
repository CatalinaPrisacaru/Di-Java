mgFreeRecList

FUNCTION NAME

mgFreeRecList - frees a record list.

C SPECIFICATION       (Not Available For Python)

void mgFreeRecList (mgreclistrecList );

DESCRIPTION

mgFreeRecList frees the contents of a record list allocated by a call to mgGetSelectList, mgGetConstructList, of mgGetTriangleList. When you are done accessing a record list, dispose of it using this function.

Only the internal contents of the record list is disposed. This function does not affect the actual nodes that were contained in the list.

PYTHON NOTES

It is not necessary to dispose of record lists in Python.

PARAMETERS

recList
the record list to free

ACCESS LEVEL

Level 1
