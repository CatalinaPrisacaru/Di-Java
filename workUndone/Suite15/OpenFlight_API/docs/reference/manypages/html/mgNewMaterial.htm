mgNewMaterial

FUNCTION NAME

mgNewMaterial - allocates a new record for an entry into a database’s material palette.

C SPECIFICATION

mgrec* mgNewMaterial (mgrec*db,
const char*name,
int*index );

PYTHON SPECIFICATION

mgrec, index mgNewMaterial (mgrec*db,
stringname )

DESCRIPTION

Given a database node db and a material entry name name, mgNewMaterial allocates a record containing a new material palette entry, adds it to the database’s material palette, and returns the record. The index of the new material entry is returned in index. The material properties can be set by using mgSetAttList or mgSetNormColor with the returned material record.

The code of the record returned will be fltFMaterial .

PARAMETERS

db
the database
name
the name of the new material (may be MG_NULL )
index
pointer to receive the index assigned to the new material palette entry record

RETURNS

mgrec
Returns the new material palette entry record if successful, MG_NULL otherwise.
index
If a palette record is returned, its index will be returned in index, otherwise index is undefined.

ACCESS LEVEL

Level 2
