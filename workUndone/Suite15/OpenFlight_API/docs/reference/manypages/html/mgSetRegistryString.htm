mgSetRegistryString

FUNCTION NAME

mgSetRegistryString - sets the value of a Creator registry key.

SYNOPSIS       (C and Python)

mgbool mgSetRegistryString (const char*keyName,
const char*keyString );

DESCRIPTION

mgSetRegistryString sets the value of the specified Creator registry key keyName to the string specified by keyString.

This function is only applicable on the Windows operation system platform in the Creator modeler environment. This function will always return MG_FALSE when called by a stand-alone application or from any code running on IRIX.

The key is written to the Windows system registry corresponding to that currently in use by the running version of Creator.

PARAMETERS

keyName
the name of the key to set in the Creator registry
keyString
the value for the key

RETURNS

Returns MG_TRUE if the the key was set successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
