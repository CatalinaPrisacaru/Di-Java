mgnodeexpressionfunc

CALLBACK NAME

mgnodeexpressionfunc - Expression tree callback function

SYNOPSIS

mgnodeexpression (*mgnodeexpressionfunc) ( mgrec* db,
mgrec* parent,
mgrec* rec,
void* userData );

DESCRIPTION

This is the signature for expression tree callback functions, embodied by the expression returned from mgNewNodeExpressionCallback

PARAMETERS

db
the database containing the node being visited
parent
the parent of the node being visited (MG_NULL if rec is db)
rec
the node currently being visited
userData
user defined data specified in call to mgFind, mgFindEx
