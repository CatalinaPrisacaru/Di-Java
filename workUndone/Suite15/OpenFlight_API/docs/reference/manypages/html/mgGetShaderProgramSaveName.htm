mgGetShaderProgramSaveName

FUNCTION NAME

mgGetShaderProgramSaveName - gets the save name of a shader program file.

SYNOPSIS       (C and Python)

char* mgGetShaderProgramSaveName (mgrec*db,
const char*programFilename );

DESCRIPTION

mgGetShaderProgramSaveName returns the save name of a shader program file contained in the shader palette of database db.

The save name of a shader program is the actual name that is saved in the OpenFlight database file for this shader palette entry. The save name may or may not include the full path specification of the shader program file. If the shader program files for a database are saved using relative names, the save name of a shader program will be a relative name and will begin with the character sequence "./". If the shader program files are saved using absolute names, the save name will include the full path specification of the shader program. If the shader program files are saved with no path, the save name will simply be the name of the file.

You can determine how shader program names are saved using the function mgShaderGetSavePathType. Similarly, you can change how shader program names are saved using mgShaderSetSavePathType.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

db
the database
programFilename
the shader program file

RETURNS

Returns the save name of the shader program if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1

SEE ALSO

mgShaderGetSavePathType, mgShaderSetSavePathType
