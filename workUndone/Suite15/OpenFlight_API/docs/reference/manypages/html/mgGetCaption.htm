mgGetCaption

FUNCTION NAME

mgGetCaption - retrieves caption string for a control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGetCaption (mgguicontrol,
char*string,
intmaxLen );

DESCRIPTION

mgGetCaption retrieves the caption string of the specified control.

The caption is truncated and null terminated if it is longer than maxLen characters.

PARAMETERS

control
the control to get caption string for
string
character buffer to hold caption retrieved
maxLen
the maximum number of characters to store in string

ACCESS LEVEL

Level 4

SEE ALSO

mgSetCaption, mgSetTitle, mgGetTitle

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
string
If function is successful, string contains the retrieved caption, otherwise string is undefined.
