mgCoord3dMultiply

FUNCTION NAME

mgCoord3dMultiply - calculates the product of 3D coordinates and a scalar.

SYNOPSIS       (C and Python)

mgcoord3d mgCoord3dMultiply (const mgcoord3d*coord,
doublescalar );

DESCRIPTION

mgCoord3dMultiply calculates and returns the product of 3D double precision floating point coordinates and a double precision scalar, coord * scalar.

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

coord
address of coordinates to multiply by scalar
scalar
scalar to multiply coord by

RETURNS

Returns a double precision 3D coordinate record.

ACCESS LEVEL

Level 1
