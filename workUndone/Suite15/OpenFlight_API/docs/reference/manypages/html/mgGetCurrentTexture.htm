mgGetCurrentTexture

FUNCTION NAME

mgGetCurrentTexture - returns index of current modeling texture.

C SPECIFICATION

mgbool mgGetCurrentTexture (mgrec*db,
int*index );

PYTHON SPECIFICATION

mgbool, index mgGetCurrentTexture (mgrec*db )

DESCRIPTION

mgGetCurrentTexture returns the index of the current modeling texture selected for database db.

PARAMETERS

db
the database
index
address of value to receive index of current modeling texture.

RETURNS

mgbool
MG_TRUE if current modeling texture for db could be determined, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

ACCESS LEVEL

Level 4
