mgGetTextureMapping

FUNCTION NAME

mgGetTextureMapping - gets an entry record from a database’s texture mapping palette.

SYNOPSIS       (C and Python)

mgrec* mgGetTextureMapping (mgrec*db,
intindex );

DESCRIPTION

mgGetTextureMapping gets the texture mapping entry record defined by index from the texture mapping palette of database node db.

The code of the record returned will be fltTextureMapping .

PARAMETERS

db
the database
index
the texture mapping palette index

RETURNS

Returns the texture mapping entry record if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1

SEE ALSO

mgIndexOfTextureMapping, mgNewTextureMapping
