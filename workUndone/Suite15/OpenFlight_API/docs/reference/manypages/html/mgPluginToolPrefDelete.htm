mgPluginToolPrefDelete

FUNCTION NAME

mgPluginToolPrefDelete - deletes a preference value from the plug-in tool preferences.

C SPECIFICATION       (Not Available For Python)

mgbool mgPluginToolPrefDelete (mgplugintoolplugintool,
const char*prefName );

DESCRIPTION

mgPluginToolPrefDelete deletes a named preference key, prefName from plugintool.

PARAMETERS

plugintool
the plug-in tool handle
prefName
the preference key name to delete

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
