mgSetModelingParent

FUNCTION NAME

mgSetModelingParent - sets the current modeling parent.

SYNOPSIS       (C and Python)

mgbool mgSetModelingParent (mgrec*db,
mgrec*parent );

DESCRIPTION

mgSetModelingParent sets the current modeling parent node for the specified database db to parent.

PARAMETERS

db
the database to set modeling parent for
parent
the node to set as parent

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if parent is already the current modeling parent, if parent is contained within an externally referenced database in db or if parent is otherwise not eligible to be a parent node.

ACCESS LEVEL

Level 4
