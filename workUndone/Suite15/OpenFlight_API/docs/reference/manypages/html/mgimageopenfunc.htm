mgimageopenfunc

CALLBACK NAME

mgimageopenfunc - Image Open function for Image Importer Tools

SYNOPSIS

mgstatus (*mgimageopenfunc) ( mgplugintool pluginTool,
void* userData,
const char* filename,
const char* filemode,
int* imageid );

DESCRIPTION

This is the signature for image open functions. When you register an image importer plug-in tool, provide an open function of this form that is called when the OpenFlight API needs your tool to open an image file.

PARAMETERS

pluginTool
the plug-in tool
userData
user defined data specified when tool registered
filename
name of image file to open
filemode
mode to open file 'r for read, 'w' for write, currently only 'r' supported for image importers
imageid
address of image identifier filled in by image importer to identify this image for subsequent interactions between the OpenFlight API and the tool

RETURNS

Return MIMG_NO_ERROR if successful, otherwise applicable texture error code. If successful, the parameter imageId should be filled in with a unique integer value that will be used to identify the opened image in subsequent interactions between the OpenFlight API and your image importer tool.
