MTA_MENULABEL

NAME

MTA_MENULABEL - Tool Attribute Menu Label

DESCRIPTION

This string attribute specifies the menu label that is placed in the corresponding menu MTA_MENULOCATION for a tool. When the user selects this menu item, the plug-in tool is launched.

You must specify both MTA_MENULOCATION and MTA_MENULABEL if you want your tool to be launched from a menu.

To put a horizontal separator before or after your new menu item prepend or append, respectively, your menu label with the "-" character. See the examples that follow.

C EXAMPLE

Specify the following values for this attribute to put a horizontal separator before/after your new menu item:
-Launch My Tool : Horizontal separator added before your menu item.

Launch My Tool- : Horizontal separator added after your menu item.
