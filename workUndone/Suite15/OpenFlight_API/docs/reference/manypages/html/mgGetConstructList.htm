mgGetConstructList

FUNCTION NAME

mgGetConstructList - gets the construction vertices and edges in a database.

SYNOPSIS       (C and Python)

mgselectlist mgGetConstructList (mgrec*db );

DESCRIPTION

mgGetConstructList creates a new node list containing all the construction vertex and edge nodes in the specified database db. The traversal pointer of the list returned is set to the first item in the list.

This function returns a list object that reflects the state of the database constructions when it was called. Typically call this function, examine the contents of the list returned and then dispose of the list when you are done. If you retain the list, its contents are not guaranteed to be valid if the modeler subsequently adds or deletes a construction vertex or edge.

When you are done accessing the list returned by this function, you should dispose of it using mgFreeRecList.

PYTHON NOTES

It is not necessary to dispose of the record list returned in Python.

PARAMETERS

db
the database to get the construction list for

RETURNS

Returns a record list object containing all the construction nodes in database, MG_NULL if no construction nodes found.

C EXAMPLE

The following example gets the construction list for a database db and traverses each node it contains.
mgrec* rec;
mgmatrix constructMatrix;
mgreclist constructList = mgGetConstructList (db);

rec = mgGetNextRecInList (constructList, &constructMatrix);
while (rec)
{
   // do something with rec
   if (mgGetNext (rec)) {
      // rec is an "edge"
   }
   else {
      // rec is a "vertex"
   }
   // get next construct rec in list
   rec = mgGetNextRecInList (constructList, &constructMatrix);
}
mgFreeRecList (constructList);

ACCESS LEVEL

Level 4
