ddGetSiteNo

FUNCTION NAME

ddGetSiteNo - gets the runtime site number of a record or data code.

SYNOPSIS       (C and Python)

int ddGetSiteNo (mgcodecode );

DESCRIPTION

ddGetSiteNo returns the runtime site number assigned to the data extension site that defined record or data code code.

Runtime site numbers are assigned by the API and are unique only within one invocation of the modeler or standalone application.

PARAMETERS

code
the data or record code to check

RETURNS

Returns the runtime site number. Site number 0 is reserved for OpenFlight.

ACCESS LEVEL

Level 1
