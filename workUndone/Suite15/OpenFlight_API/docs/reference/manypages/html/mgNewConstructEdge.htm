mgNewConstructEdge

FUNCTION NAME

mgNewConstructEdge - creates a construction edge node.

SYNOPSIS       (C and Python)

mgrec* mgNewConstructEdge (mgeditorcontexteditorContext,
const mgcoord3d*coord1,
const mgcoord3d*coord2 );

DESCRIPTION

mgNewConstructEdge creates a new construction edge node with the specified endpoint coordinates, coord1 and coord2. The new construction node is created in the database associated with the specified editor context, editorContext.

If either coord1 or coord2 is MG_NULL, the corresponding coordinate of the edge is assigned (0.0, 0.0, 0.0).

A construction edge is actually composed of two construction vertex nodes. This function returns the first vertex node of the edge created. Calling mgGetNext for the first vertex of the edge is return the second vertex of the edge.

The attributes of either vertex of a construction edge can be accessed like any other fltVertex node using mgGetAttList and mgSetAttList. The edge returned, however, should not be explicitly attached to other geometry nodes in the database. Doing so will yield undefined results.

PARAMETERS

editorContext
the editor context used to identify the database in which the construction edge is to be created
coord1
address of record containing initial coordinates of first vertex of construction edge
coord2
address of record containing initial coordinates of second vertex of construction edge

RETURNS

Returns first vertex node of construction edge created if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
