mgParamAppendDouble2

FUNCTION NAME

mgParamAppendDouble2 - appends a value to a double2 array type parameter in a parameter block.

SYNOPSIS       (C and Python)

mgbool mgParamAppendDouble2 (mgparamblockparamBlock,
const char*paramName,
doubleparamValue1,
doubleparamValue2 );

DESCRIPTION

mgParamAppendDouble2 appends the value (comprised of paramValue1 and paramValue2) to the double2 array type parameter named paramName in the parameter block paramBlock. If the parameter does not exist, no value is appended.

When setting up a double2 array type parameter, use mgParamSetDouble2 to set the first item in the array. When you call mgParamSetDouble2, the parameter array is truncated to a single item. To add the second and subsequent items to the array, use mgParamAppendDouble2 as needed.

Note that if the parameter does not exist, it is NOT created in the parameter block. Also, if the parameter is not an array type parameter, this function will do nothing.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
paramValue1
value 1 to append for paramName
paramValue2
value 2 to append for paramName

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if paramName is not a parameter in paramBlock, if paramValue1 or paramValue2 is not a valid value for the paramName parameter as defined by its constraints (if any), or if the maximum dimension of paramName would be exceeded by appending this item.

ACCESS LEVEL

Level 4
