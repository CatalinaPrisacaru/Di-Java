mgRegisterGraphicDraw

FUNCTION NAME

mgRegisterGraphicDraw - registers a graphic draw function for an extension node type.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRegisterGraphicDraw (mgpluginsitepluginSite,
mgcoderCode,
mggraphicdrawfuncdrawFunc );

DESCRIPTION

mgRegisterGraphicDraw registers a graphic draw function for the specified extension node type rCode. The function you register, drawFunc will be called when nodes of type rCode are drawn in the graphics view of Creator. In this way extension developers can modify how a node they develop appears in the graphic view.

Note: You can only assign graphic draw functions to node types that your extension has defined.

PARAMETERS

pluginSite
the data extension site
rCode
the record code
drawFunc
the graphic draw function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
