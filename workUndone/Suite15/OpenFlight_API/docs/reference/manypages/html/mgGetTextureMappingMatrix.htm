mgGetTextureMappingMatrix

FUNCTION NAME

mgGetTextureMappingMatrix - gets the matrix of a 3-point put or 4-point put texture mapping.

C SPECIFICATION

mgbool mgGetTextureMappingMatrix (mgrec*db,
intindex,
mgmatrix*textureMappingMatrix );

PYTHON SPECIFICATION

mgbool, textureMappingMatrix mgGetTextureMappingMatrix (mgrec*db,
intindex )

DESCRIPTION

mgGetTextureMappingMatrix gets the mapping defined by index, and puts the mapping’s matrix into matrix. The result is suitable for use with OpenGL’s glTexGen function linear mappings.

Note: The matrix is only valid for 3-point put and 4-point put mappings.

PARAMETERS

db
the database
index
the index of the mapping
textureMappingMatrix
the matrix of the 3-point put or 4-point put texture mapping.

RETURNS

mgbool
MG_TRUE if a valid matrix is found, MG_FALSE otherwise.
textureMappingMatrix
If function is successful, textureMappingMatrix contains the retrieved matrix, otherwise textureMappingMatrix is undefined.

C EXAMPLE

int index;
mgrec* db;
mgmatrix matrix;
if ( (mgGetTextureMappingType (db, index) == 1) ||
     (mgGetTextureMappingType (db, index) == 2) )
{
   if (mgGetTextureMappingMatrix (db, index, &matrix))
   {
      // Do Stuff
   }
}
