mgRegisterCreate

FUNCTION NAME

mgRegisterCreate - sets the create action function for an extension node type.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRegisterCreate (mgpluginsitepluginSite,
mgcoderCode,
mgcreatefunccreateFunc );

DESCRIPTION

mgRegisterCreate registers a create function for the specified extension node type rCode. The function you register, createFunc will be called after a user creates a node of type rCode using either the Extensions Menu or the Create Tool. In this way extension developers can be notified after their extension nodes are created and can perform actions on the new node.

Note: You can only assign create functions to node types that your extension has defined.

PARAMETERS

pluginSite
the data extension site
rCode
the record code
createFunc
the create action function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
