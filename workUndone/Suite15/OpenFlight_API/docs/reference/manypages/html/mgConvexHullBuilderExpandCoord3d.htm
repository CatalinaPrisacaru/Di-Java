mgConvexHullBuilderExpandCoord3d

FUNCTION NAME

mgConvexHullBuilderExpandCoord3d - adds a 3D point to a convex hull builder.

SYNOPSIS       (C and Python)

mgbool mgConvexHullBuilderExpandCoord3d (mgconvexhullbuilderhullBuilder,
const mgcoord3d*c );

DESCRIPTION

mgConvexHullBuilderExpandCoord3d adds a 3D point to the set of points used by a convex hull builder object hullBuilder to build a convex hull. The convex hull created will be built around this set of points.

PARAMETERS

hullBuilder
the convex hull builder
c
address of coordinate to add

RETURNS

Returns MG_TRUE if the point is added sucessfully, MG_FALSE otherwise.

EXAMPLE

See mgNewConvexHullBuilder for an example of how to use a convex hull builder.

ACCESS LEVEL

Level 1
