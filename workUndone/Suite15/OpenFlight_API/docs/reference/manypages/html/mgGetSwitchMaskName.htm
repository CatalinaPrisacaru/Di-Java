mgGetSwitchMaskName

FUNCTION NAME

mgGetSwitchMaskName - gets the name of a switch mask.

SYNOPSIS       (C and Python)

char* mgGetSwitchMaskName (mgrec*rec,
intmaskNo );

DESCRIPTION

Given a fltSwitch node, rec, mgGetSwitchMaskName returns the name assigned to the mask maskNo.

The value of maskNo must be between 0 and the number of masks of rec minus 1 (the first mask is maskNo 0).

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

rec
the fltSwitch node
maskNo
switch mask number starting from 0

RETURNS

Returns the name of the switch mask if it exists and is named, MG_NULL otherwise. This function will fail if maskNo does not specify a valid mask of rec.

ACCESS LEVEL

Level 1
