mgRegisterInputDevice

FUNCTION NAME

mgRegisterInputDevice - registers an input device tool.

SYNOPSIS       (C and Python)

mgplugintool mgRegisterInputDevice (mgpluginplugin,
const char*toolName,
intinputFlags,
mginputdevicestartfuncstartFunc,
mginputdevicestopfuncstopFunc,
mginputdevicesetinputtypefuncsetInputTypeFunc,
void*userData,
... );

DESCRIPTION

mgRegisterInputDevice registers an input device tool for the specified plug-in module plugin with the specified tool attributes.

The required tool attributes include the name of the tool toolName, start and stop functions for collecting events from the input device, a set input type function for notifying the device which input type is desired, and user defined callback data userData (may be MG_NULL) that is passed to the callback functions when they are called.

PARAMETERS

plugin
the plug-in module registering the tool
toolName
the name of the tool
inputFlags
input device capability flags, bitwise combination of MMSI_VERTEXINPUT, MMSI_POINTINPUT, and MMSI_DEVICEINPUT
startFunc
start function called when input desired from your device
stopFunc
stop function called when input is no longer desired from your device
setInputTypeFunc
set input type function called to notify your device which kind of device input is desired
userData
user defined data that will be passed to startFunc, stopFunc, and setInputTypeFunc
...
additional input device tool specific attributes in variable argument style

RETURNS

Returns the plug-in tool registered if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
