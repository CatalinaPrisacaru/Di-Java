mgReference

FUNCTION NAME

mgReference - changes an existing node record into an instance node record.

SYNOPSIS       (C and Python)

mgbool mgReference (mgrec*rec,
mgrec*refRec );

DESCRIPTION

Given an existing node record rec, mgReference causes rec to become an instance of the reference node record refRec.

Alternatively, you can use mgAttach to attach a child as the first child of a parent or mgAppend to attach a child as the last child of a parent.

The node refRec must not be a child node of any node in the database.

PARAMETERS

rec
a node record, which will be an instance of the reference node
refRec
a reference node record.

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2

SEE ALSO

mgDeReference, mgAttach, mgAppend, mgInsert
