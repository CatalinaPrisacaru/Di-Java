mgVectorfDot

FUNCTION NAME

mgVectorfDot - calculates the dot product of two vectors.

SYNOPSIS       (C and Python)

float mgVectorfDot (const mgvectorf*vector1,
const mgvectorf*vector2 );

DESCRIPTION

mgVectorfDot calculates and returns the dot product of the two vectors vector1 DOT vector2.

The original vectors vector1 and vector2 are not affected.

Note: For efficiency, vector1 and vector2 are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

vector1
address of first vector
vector2
address of second vector

RETURNS

Returns a single precision floating point value (vector1 DOT vector2).

ACCESS LEVEL

Level 1
