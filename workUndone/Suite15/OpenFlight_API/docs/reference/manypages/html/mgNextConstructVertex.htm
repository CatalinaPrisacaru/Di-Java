mgNextConstructVertex

FUNCTION NAME

mgNextConstructVertex - gets the next construction vertex node associated with an active editor tool instance.

SYNOPSIS       (C and Python)

mgrec* mgNextConstructVertex (mgeditorcontexteditorContext,
mgrec*constructVtx );

DESCRIPTION

mgNextConstructVertex gets the construction vertex node that follows the specified construction node, constructVtx, associated with the editor tool instance identified by editor context, editorContext.

After calling mgFirstConstructVertex to get the first construction vertex node associated with an editor tool instance, you can call mgNextConstructVertex to get successive construction vertex nodes.

PARAMETERS

editorContext
the editor context from which to get the next construction vertex node
constructVtx
the construction vertex node to get next construction vertex for

RETURNS

Returns the next construction vertex node if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4
