mgtoolterminationreason

ENUM NAME

mgtoolterminationreason - Editor Tool Termination Reason type

SYNOPSIS

typedef enum mgtoolterminationreason {
   MTRM_DONE,
   MTRM_CANCEL,
   MTRM_SELF,
   MTRM_SYSTEM,
} mgtoolterminationreason;

MEMBERS

MTRM_DONE
Tool terminated because user pressed Done button (MBT_DONE).
MTRM_CANCEL
Tool terminated because user pressed Cancel button (MBT_CANCEL) or tool terminated itself by calling mgEditorCancelTool.
MTRM_SELF
Tool terminated itself by calling mgEditorTerminateTool.
MTRM_SYSTEM
Tool terminated by modeling system (window closed, tool chain, etc)
