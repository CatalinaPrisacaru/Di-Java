mgSetCoord2i

FUNCTION NAME

mgSetCoord2i - sets all three attribute values in a fltCoord2i record.

SYNOPSIS       (C and Python)

mgbool mgSetCoord2i (mgrec*rec,
mgcodecode,
intx,
inty );

DESCRIPTION

Given a record, rec, and the name of one of its fltCoord2i attribute records, mgSetCoord2i sets the x and y attribute values in the fltCoord2i attribute record.

PARAMETERS

rec
the record whose fltCoord2i record is to be set
code
the name of the fltCoord2i attribute code of rec
x
the x value to set in the fltCoord2i attribute record
y
the y value to set in the fltCoord2i attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
