mgNewTexture

FUNCTION NAME

mgNewTexture - creates a new entry in a database's texture palette.

SYNOPSIS       (C and Python)

mgbool mgNewTexture (mgrec*db,
intindex,
unsigned char*texels,
const char*textureName,
mgrec*imgRec,
intx,
inty );

DESCRIPTION

Given a database node db, a texture palette index index, an array of texels texels, a texture name textureName, an attribute record imgRec (of type fltImage ), and a palette position x, y, mgNewTexture creates a new texture in the database’s texture palette with the specified name, attributes and position.

PARAMETERS

db
the database
index
the index of the new texture palette entry
texels
the array of texels that make up the texture image
textureName
the name of the texture
imgRec
the attribute record describing the new texture
x
the X position in the texture palette
y
the Y position in the texture palette

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. If successful, the corresponding texture record is returned in the output parameter, imgRec, otherwise it is undefined.

ACCESS LEVEL

Level 2
