mgGetXformType

FUNCTION NAME

mgGetXformType - determines the type of a transformation node.

SYNOPSIS       (C and Python)

mgxfllcode mgGetXformType (mgrec*rec );

DESCRIPTION

mgGetXformType determines the type of the specified transformation node, rec. The returned transformation type will be one of the following:
MXLL_TRANSLATE
MXLL_SCALE
MXLL_ROTEDGE
MXLL_ROTPT
MXLL_PUT
MXLL_TOPOINT
MXLL_GENERAL

PARAMETERS

rec
the transformation node

RETURNS

Returns the type of the transformation node. Note that if rec is not a transformation node, the return value of this function is not defined.

EXAMPLE

The following examples (in C and Python) examine the first transformation (and its type) associated to node rec.

C EXAMPLE

if (mgHasXform (rec)) {
      mgrec* xForm = mgGetXform (rec);
      mgxfllcode xType = mgGetXformType (xForm);
      switch (xType)
      {
      case MXLL_TRANSLATE:
       printf ("MXLL_TRANSLATE\\n");
       break;
      case MXLL_SCALE:
       printf ("MXLL_SCALE\\n");
       break;
      case MXLL_ROTEDGE:
       printf ("MXLL_ROTEDGE\\n");
       break;
      case MXLL_ROTPT:
       printf ("MXLL_ROTPT\\n");
       break;
      case MXLL_PUT:
       printf ("MXLL_PUT\\n");
       break;
      case MXLL_TOPOINT:
       printf ("MXLL_TOPOINT\\n");
       break;
      case MXLL_GENERAL:
       printf ("MXLL_GENERAL\\n");
       break;
      }
}

PYTHON EXAMPLE

if (mgHasXform (rec)):
   xForm = mgGetXform (rec)
   xType = mgGetXformType (xForm)
      if xType == MXLL_TRANSLATE:
       print "MXLL_TRANSLATE"
      elif xType == MXLL_SCALE:
       print "MXLL_SCALE"
      elif xType == MXLL_ROTEDGE:
       print "MXLL_ROTEDGE"
      elif xType == MXLL_ROTPT:
       print "MXLL_ROTPT"
      elif xType == MXLL_PUT:
       print "MXLL_PUT"
      elif xType == MXLL_TOPOINT:
       print "MXLL_TOPOINT"
      elif xType == MXLL_GENERAL:
       print "MXLL_GENERAL"
