mgVectorfToVectord

FUNCTION NAME

mgVectorfToVectord - converts a vector from single to double precision floating point.

C SPECIFICATION       (Not Available For Python)

mgvectord mgVectorfToVectord (const mgvectorf*vec );

DESCRIPTION

mgVectorfToVectord converts the specified singe precision floating point vector vec to a double precision floating point vector.

The original vector vec is not affected.

PARAMETERS

vec
address of single precision vector to convert

RETURNS

Returns double precision vector record.

ACCESS LEVEL

Level 1
