mgGetVtxBackColorRGB

FUNCTION NAME

mgGetVtxBackColorRGB - gets the RGB back color values out of a fltVertex record.

C SPECIFICATION

mgbool mgGetVtxBackColorRGB (mgrec*vtx,
short*red,
short*green,
short*blue );

PYTHON SPECIFICATION

mgbool, red, green, blue mgGetVtxBackColorRGB (mgrec*vtx )

DESCRIPTION

Given a fltVertex record, vtx, mgGetVtxBackColorRGB gets the red, green, and blue values of its back bolor (if applicable).

Colors set in index mode are automatically converted to RGB values. The color values returned are in the range 0..255.

You do not have to provide addresses for all red, green, and blue values. For example, if you only want to receive the red and blue components (but not green), specify valid addresses for red and blue and leave green NULL.

PARAMETERS

vtx
the fltVertex record from which to get the color values
red
the red component
green
the green component
blue
the blue component

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
red
If function is successful, red contains the retrieved value, otherwise red is undefined.
green
If function is successful, green contains the retrieved value, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved value, otherwise blue is undefined.

ACCESS LEVEL

Level 1
