mgimageclosefunc

CALLBACK NAME

mgimageclosefunc - Image Close function for Image Importer Tools

SYNOPSIS

mgstatus (*mgimageclosefunc) ( mgplugintool pluginTool,
void* userData,
int imageid );

DESCRIPTION

This is the signature for image close functions. When you register an image importer plug-in tool, provide a close function of this form that is called when the OpenFlight API needs your tool to close an image file.

PARAMETERS

pluginTool
the plug-in tool
userData
user defined data specified when tool registered
imageid
image identifier indicating which image to close - this is the value returned by the image open function when the image was first opened

RETURNS

Returns MIMG_NO_ERROR if successful, otherwise returns applicable texture error code.
