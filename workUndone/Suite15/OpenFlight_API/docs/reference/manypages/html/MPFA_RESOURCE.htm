MPFA_RESOURCE

NAME

MPFA_RESOURCE - File Browser parameter for specifying your resource when using a custom dialog template for your file browser

DESCRIPTION

Specify this as a parameter for mgPromptDialogFile followed by the mgresource of your plug-in containing your dialog template.

Typically you specify this parameter as well as MPFA_DIALOGID, MPFA_DIALOGFUNC, MPFA_CALLBACKMASK and MPFA_USERDATA together.

PYTHON NOTES

This parameter is not supported in Python OpenFlight Script.
