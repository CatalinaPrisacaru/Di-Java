mgGetRegistryString

FUNCTION NAME

mgGetRegistryString - returns the value of a Creator registry key.

C SPECIFICATION

mgbool mgGetRegistryString (const char*keyName,
char*keyString,
intmaxLen );

PYTHON SPECIFICATION

mgbool, string mgGetRegistryString ( )

DESCRIPTION

mgGetRegistryString returns the value of the specified Creator registry key keyName in the output string keyString. The output string is truncated and null terminated if it is longer than maxLen characters.

This function is only applicable on the Windows operation system platform in the Creator modeler environment. This function will always return MG_FALSE when called by a stand-alone application or from any code running on IRIX.

The key is looked up in the Windows system registry corresponding to that currently in use by the running version of Creator.

PARAMETERS

keyName
the name of the key to look up in the Creator registry
keyString
character buffer to hold the key value
maxLen
maximum number of characters to store in keyString

RETURNS

mgbool
MG_TRUE if registry string retrieved successfully, MG_FALSE otherwise.
string
If function is successful, string contains the registry string, otherwise undefined.

ACCESS LEVEL

Level 4
