mgImageToXYZMatrix

FUNCTION NAME

mgImageToXYZMatrix - projects georeferenced image coordinates to georeferenced database coordinates.

C SPECIFICATION

void mgImageToXYZMatrix (mgrec*imgRec,
mgimagegeoinfogeoinfo,
mgrec*db,
mgmatrix*matrix );

PYTHON SPECIFICATION

mgmatrix mgImageToXYZMatrix (mgrec*imgRec,
mgimagegeoinfogeoinfo,
mgrec*db )

DESCRIPTION

mgImageToXYZMatrix projects georeferenced image coordinates to georeferenced database coordinates

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

imgRec
image attribute record
geoinfo
image georeference information
db
database containing the projection to use
matrix
address of matrix to receive result

RETURNS

mgmatrix
the matrix that converts image coords to database coords

ACCESS LEVEL

Level 1
