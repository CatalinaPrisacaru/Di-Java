mgVectordTransform

FUNCTION NAME

mgVectordTransform - transforms a double precision vector using a specified matrix.

SYNOPSIS       (C and Python)

mgvectord mgVectordTransform (const mgmatrixmatrix,
const mgvectord*vec );

DESCRIPTION

mgVectordTransform applies the specified matrix to the specified double precision floating point vector vec and returns the resulting vector.

The original vector vec is not affected.

Note: For efficiency, matrix and vec are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

matrix
matrix to apply to vec
vec
vector to transform

RETURNS

Returns the transformed double precision vector record.

ACCESS LEVEL

Level 1
