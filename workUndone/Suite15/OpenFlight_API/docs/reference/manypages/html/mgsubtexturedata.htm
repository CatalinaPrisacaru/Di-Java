mgsubtexturedata

STRUCT NAME

mgsubtexturedata - Subtexture Definition data

SYNOPSIS

typedef struct  {
   char name[MIMG_MAXNAMELEN];
   int l;
   int b;
   int r;
   int t;
} mgsubtexturedata;

MEMBERS

name[MIMG_MAXNAMELEN]
char - name of subtexture in image
l
int - lower left x coordinate of subtexture in image
b
int - lower left y coordinate of subtexture in image
r
int - upper right x coordinate of subtexture in image
t
int - upper right y coordinate of subtexture in image

DESCRIPTION

Use an array of mgsubtexturedata to add and get subtexture records.
