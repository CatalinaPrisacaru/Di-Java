mgMakeUnitVectord

FUNCTION NAME

mgMakeUnitVectord - makes a unitized vector from two points.

SYNOPSIS       (C and Python)

mgvectord mgMakeUnitVectord (const mgcoord3d*coord1,
const mgcoord3d*coord2 );

DESCRIPTION

Like mgMakeVectord, mgMakeUnitVectord returns a double precision vector record representing the vector from coord1 to coord2. The vector returned by this function, however, is unitized.

The original coordinates coord1 and coord2 are not affected.

Note: For efficiency, coord1 and coord2 are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

coord1
address of first coordinate that defines the vector
coord2
address of second coordinate that defines the vector

RETURNS

Returns a unitized double precision vector record. If either coord1 or coord2 are MG_NULL, the vector returned by this function is undefined. If the distance from coord1 to coord2 is less than 0.000000000001, this function returns (0,0,1).

ACCESS LEVEL

Level 1
