mgGLColor3

FUNCTION NAME

mgGLColor3 - specifies a color for GL rendering.

SYNOPSIS       (C and Python)

mgbool mgGLColor3 (mgglcontextglContext,
shortred,
shortgreen,
shortblue );

DESCRIPTION

mgGLColor3 sets the current color for GL rendering to that specified by the red, green and blue color components.

This function loads the specified red, green and blue color components in preparation for rendering into GL controls. It is the equivalent of calling the OpenGL routine glColor3*().

PARAMETERS

glContext
Gl rendering context into which color is to be loaded
red
red component of color to load
green
green component of color to load
blue
blue component of color to load

RETURNS

Returns MG_TRUE if color could be loaded successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
