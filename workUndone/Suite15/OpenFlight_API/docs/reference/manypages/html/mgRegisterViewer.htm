mgRegisterViewer

FUNCTION NAME

mgRegisterViewer - registers a viewer tool.

C SPECIFICATION       (Not Available For Python)

mgplugintool mgRegisterViewer (mgpluginplugin,
const char*toolName,
mgtoolstartfuncstartFunc,
void*userData,
... );

DESCRIPTION

mgRegisterViewer registers a viewer tool for the specified plug-in module plugin with the specified tool attributes.

The required tool attributes include the name of the tool toolName, a start function for the tool, startFunc and user defined callback data, userData (may be MG_NULL) that is passed to the start function when it is called.

Additional required tool attributes include launch location attributes. Your tool can be launched from either a tool palette or menu. If your tool is to be launched from a tool palette, you must specify MTA_PALETTELOCATION and MTA_PALETTEICON. If your tool is to be launched from a menu, you must specify MTA_MENULOCATION and MTA_MENULABEL. Additional optional tool attributes for tools launched from a menu include MTA_MENUPOSITION and MTA_MENUSUBMENU.

Additional optional tool attributes for all viewer tools include MTA_VERSION, MTA_HELPCONTEXT and MTA_TOOLTIP. If you specify that your tool be launched from a tool palette and do not specify MTA_TOOLTIP, the default tool tip text displayed is the name of your tool as specified in the parameter toolName.

All additional (required and optional) tool attributes are passed using variable argument style and must be terminated with MG_NULL.

PARAMETERS

plugin
the plug-in module registering the tool
toolName
the name of the tool
startFunc
the tool start function that is called when the tool is launched
userData
user defined data that will be passed to startFunc when it is called
...
additional viewer tool specific attributes in variable argument style

RETURNS

Pointer to plug-in tool registered if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
