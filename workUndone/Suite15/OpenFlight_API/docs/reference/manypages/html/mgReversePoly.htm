mgReversePoly

FUNCTION NAME

mgReversePoly - Reverses the order of the vertices in a polygon.

SYNOPSIS       (C and Python)

mgbool mgReversePoly (mgrec*poly );

DESCRIPTION

mgReversePoly reverses the order of the vertices in the specified polygon poly. This has the effect of reversing which side of the polygon is considered the "front".

PARAMETERS

poly
polygon whose vertices are to be reversed

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. If successful, the vertices of poly will be in the reverse order after the function completes, otherwise the vertex order will be unchanged.

ACCESS LEVEL

Level 1
