mgSetNameUnique

FUNCTION NAME

mgSetNameUnique - assigns a unique name to a node record.

SYNOPSIS       (C and Python)

mgbool mgSetNameUnique (mgrec*rec,
const char*name );

DESCRIPTION

mgSetNameUnique sets the name of a node rec based on name. If name is unique in the database containing rec, name will be assigned. If name is not unique in the database containing rec, a name derived from name will be assigned. The derived name assigned will be the first unique name of the form name_N where N is 1,2,3, etc. If name is already of the form name_N, the derived name assigned will be the first unique name of the form name_M where M is N+1, N+2, N+3, etc.

PARAMETERS

rec
the node record
name
the name on which the new node name is based

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

char* name;
mgbool status;

status = mgSetNameUnique (node, "building");
if (status == MG_TRUE) {
   char* name = mgGetName (node);
   // name may be building, building_1, building_2, etc
   mgFree (name);
}

Level 2
