mgParamGetBool

FUNCTION NAME

mgParamGetBool - gets a boolean parameter value from a parameter block.

C SPECIFICATION

mgbool mgParamGetBool (mgparamblockparamBlock,
const char*paramName,
mgbool*paramValue,
mgbooldefaultValue );

PYTHON SPECIFICATION

mgbool, paramValue mgParamGetBool (mgparamblockparamBlock,
const char*paramName )

DESCRIPTION

mgParamGetBool retrieves a boolean parameter value named paramName from the parameter block paramBlock. If the parameter exists, its value is returned in paramValue. If the parameter does not exist, paramValue is set to defaultValue.

If the parameter is not an array type parameter, this is the standard way to retrieve its boolean value. If the parameter is an array type parameter, this function gets the first item in the array and is equivalent to calling mgParamGetBoolNth passing 1 as nth.

Note that if the parameter does not exist, it is NOT created in the parameter block.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
paramValue
storage location for the parameter value returned
defaultValue
default value returned if parameter does not exist

RETURNS

mgbool
MG_TRUE if the parameter exists, MG_FALSE otherwise.
paramValue
If successful, paramValue will be the parameter value, otherwise paramValue will be defaultValue.

ACCESS LEVEL

Level 4
