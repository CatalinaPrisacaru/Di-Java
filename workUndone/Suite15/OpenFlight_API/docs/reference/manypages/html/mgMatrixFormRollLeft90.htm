mgMatrixFormRollLeft90

FUNCTION NAME

mgMatrixFormRollLeft90 - creates a matrix that will roll left 90 degrees.

C SPECIFICATION

void mgMatrixFormRollLeft90 (mgmatrix*matrix );

PYTHON SPECIFICATION

mgmatrix mgMatrixFormRollLeft90 ( )

DESCRIPTION

mgMatrixFormRollLeft90 gets a matrix that will roll left 90 degrees.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
