mgNewLightSource

FUNCTION NAME

mgNewLightSource - creates a new entry in the light source palette.

C SPECIFICATION

mgrec* mgNewLightSource (mgrec*db,
char*name,
int*index );

PYTHON SPECIFICATION

mgrec, index mgNewLightSource (mgrec*db,
stringname )

DESCRIPTION

mgNewLightSource creates a new light source palette entry for database db. The new entry is assigned the specified name and returned if created successfully. The index assigned to the new entry is also returned in index. The attributes of the new entry can be set using mgSetAttList.

The code of the record returned will be fltLightSourcePalette .

PARAMETERS

db
the database
name
name to assign to the new light source
index
address of value to receive index assigned to new entry in the palette

RETURNS

mgrec
Returns the new light source palette entry record if successful, MG_NULL otherwise.
index
If a palette record is returned, its index will be returned in index, otherwise index is undefined.

ACCESS LEVEL

Level 2
