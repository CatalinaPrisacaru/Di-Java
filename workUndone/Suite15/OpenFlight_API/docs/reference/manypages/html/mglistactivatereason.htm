mglistactivatereason

ENUM NAME

mglistactivatereason - List activate reason type

SYNOPSIS

typedef enum mglistactivatereason {
   MLIST_SELECT,
   MLIST_DESELECT,
   MLIST_DOUBLECLICK,
} mglistactivatereason;

MEMBERS

MLIST_SELECT
List item selected. This indicates that the user selected an item in the list.
MLIST_DESELECT
List item deselected. This indicates that the user deselected an item in the list.
MLIST_DOUBLECLICK
List item double clicked. This indicates that the user double clicked an item in the list.

DESCRIPTION

This enumerated type is used when the activate control event is sent to list controls.
