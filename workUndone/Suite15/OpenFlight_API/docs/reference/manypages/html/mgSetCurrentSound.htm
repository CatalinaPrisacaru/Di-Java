mgSetCurrentSound

FUNCTION NAME

mgSetCurrentSound - sets the current modeling sound index.

SYNOPSIS       (C and Python)

mgbool mgSetCurrentSound (mgrec*db,
intindex );

DESCRIPTION

mgSetCurrentSound sets the current modeling sound palette index for database db.

PARAMETERS

db
the database
index
index to set current modeling sound

RETURNS

Returns MG_TRUE if current modeling sound for db could be set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
