mgcoord3d

STRUCT NAME

mgcoord3d - Double precision 3 dimensional coordinate record

SYNOPSIS

typedef struct mgcoord3d {
   double x;
   double y;
   double z;
} mgcoord3d;

MEMBERS

x
double - x component of coordinate
y
double - y component of coordinate
z
double - z component of coordinate
