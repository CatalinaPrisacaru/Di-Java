mgGetPlane

FUNCTION NAME

mgGetPlane - gets values out of a fltDPlane record.

C SPECIFICATION

mgbool mgGetPlane (mgrec*rec,
mgcodeplaneCode,
double*a,
double*b,
double*c,
double*d );

PYTHON SPECIFICATION

mgbool, a, b, c, d mgGetPlane (mgrec*rec,
mgcodeplaneCode )

DESCRIPTION

Given a record rec, mgGetPlane gets the values of fltDPlane attribute record planeCode from rec and stores them in a, b, c and d.

You do not have to provide addresses for all a, b, c, and d values. For example, if you only want to receive the a, b and c coefficients (but not d), specify valid addresses for a, b, and c and leave d NULL.

PARAMETERS

rec
the record from which to get the fltDPlane attribute record
planeCode
the fltDPlane attribute record code
a
address of value to receive a coefficient
b
address of value to receive b coefficient
c
address of value to receive c coefficient
d
address of value to receive d coefficient

RETURNS

mgbool
MG_TRUE if the plane was retrieved successfully, MG_FALSE otherwise.
a
If function is successful, a contains the retrieved a coefficient, otherwise a is undefined.
b
If function is successful, b contains the retrieved b coefficient, otherwise b is undefined.
c
If function is successful, c contains the retrieved c coefficient, otherwise c is undefined.
d
If function is successful, d contains the retrieved d coefficient, otherwise d is undefined.

C EXAMPLE

mgrec* db;
mgrec* clipRec;
mgplaned plane;

db = mgOpenDb ("file.flt");

// create clip rec
clipRec = UserMakeClipRec (db);

mgGetPlane (clipRec, fltClipPlane0, &plane.a, &plane.b,
     &plane.c, &plane.d);

ACCESS LEVEL

Level 1
