mgExtensionSiteSetName

FUNCTION NAME

mgExtensionSiteSetName - sets the name of an extension site.

SYNOPSIS       (C and Python)

mgbool mgExtensionSiteSetName (const char*siteGuid,
const char*name );

DESCRIPTION

mgExtensionSiteSetName sets the name of an extension site identified by siteGuid. This name can be any character string of any length.

PARAMETERS

siteGuid
the GUID of the site
name
the name to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
