mgPtrStackTop

FUNCTION NAME

mgPtrStackTop - returns the item on top of a pointer stack.

C SPECIFICATION       (Not Available For Python)

void* mgPtrStackTop (mgptrstackptrStack );

DESCRIPTION

mgPtrStackTop gets the item on top of the pointer stack ptrStack and returns that item.

PARAMETERS

ptrStack
the pointer stack

RETURNS

Returns the item that is currently at the top of the stack, MG_NULL if stack is empty.

ACCESS LEVEL

Level 1
