mgMatrixRotateX

FUNCTION NAME

mgMatrixRotateX - multiply an X-axis rotation matrix into the given matrix.

C SPECIFICATION

mgbool mgMatrixRotateX (const mgmatrixmatrix,
doubletheta,
mgmatrix*resultMatrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixRotateX (mgmatrixmatrix,
doubletheta )

DESCRIPTION

mgMatrixRotateX calculates a rotation matrix for rotation about the X axis and multiplies that matrix into the given matrix matrix.

Call the rotation matrix representing the rotation of theta degrees about the X coordinate axis R. Call the value (on input) of matrix M. The resulting matrix will be the product R X M. The parameter theta specifies the angle (in degrees) to rotate counter-clockwise (right hand rule). This function operates safely when the input and output matrices are the same matrix.

C NOTES

This function writes the resulting matrix into the output parameter resultMatrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
matrix to multiply
theta
angle to rotate measured counter-clockwise in degrees about X axis (right hand rule)
resultMatrix
address of matrix to receive result

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix result.

ACCESS LEVEL

Level 1
