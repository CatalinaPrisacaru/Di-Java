mgSelectOneEdge

FUNCTION NAME

mgSelectOneEdge - select a vertex node record as an edge.

SYNOPSIS       (C and Python)

mgstatus mgSelectOneEdge (mgrec*rec );

DESCRIPTION

mgSelectOneEdge selects the specified vertex node record rec as an edge.

When a vertex node is selected, it may be selected as a vertex or as an edge. Use this function to select a vertex node as an edge. Use mgSelectOne to select a vertex node as a vertex.

If any nodes are already selected, they remain selected and the specified rec is added to the select list. To make a node the only node selected, call mgDeselectAll before mgSelectOneEdge.

If selecting the specified node would result in a recursive select list (that is either an ancestor or descendant of rec is already selected), the selection is not allowed.

PARAMETERS

rec
the vertex node to select as an edge

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
