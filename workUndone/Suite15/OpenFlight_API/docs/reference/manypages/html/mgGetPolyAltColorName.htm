mgGetPolyAltColorName

FUNCTION NAME

mgGetPolyAltColorName - gets the alternate color name of a fltPolygon record.

SYNOPSIS       (C and Python)

char* mgGetPolyAltColorName (mgrec*poly );

DESCRIPTION

Given a fltPolygon record, poly, mgGetPolyAltColorName returns the alternate color name for the polygon.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

poly
a fltPolygon record

RETURNS

Returns the alternate color name for the polygon if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
