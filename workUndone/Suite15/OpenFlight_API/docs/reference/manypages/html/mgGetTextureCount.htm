mgGetTextureCount

FUNCTION NAME

mgGetTextureCount - gets the number of entries contained in a texture palette.

SYNOPSIS       (C and Python)

int mgGetTextureCount (mgrec*db );

DESCRIPTION

mgGetTextureCount returns the number of entries contained in the texture palette of database db.

PARAMETERS

db
the database

RETURNS

Returns the number of texture palette entries if successful, or returns -1 otherwise.

ACCESS LEVEL

Level 1
