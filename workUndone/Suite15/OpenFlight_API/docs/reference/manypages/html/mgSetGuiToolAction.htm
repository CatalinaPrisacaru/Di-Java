mgSetGuiToolAction

FUNCTION NAME

mgSetGuiToolAction - binds a plug-in tool action to a control.

C SPECIFICATION       (Not Available For Python)

void mgSetGuiToolAction (mgguicontrol,
mgtoolactiontoolAction );

DESCRIPTION

mgSetGuiToolAction binds the plug-in tool action toolAction to the specified control. If your plug-in tool dialog allows keyboard shortcuts to activate controls it contains, you do so via plug-in tool actions. Use this function to "bind" a plug-in tool action to a specific control in your dialog. Use mgToolActionSetShortCut to assign a shortcut key sequence to a plug-in tool action.

See the example code in mgPluginToolNewAction.

PARAMETERS

control
the control to bind toolAction to
toolAction
the tool action to bind to control
