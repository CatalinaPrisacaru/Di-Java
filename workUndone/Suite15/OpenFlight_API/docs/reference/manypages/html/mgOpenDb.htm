mgOpenDb

FUNCTION NAME

mgOpenDb - opens an existing OpenFlight database for reading or writing.

SYNOPSIS       (C and Python)

mgrec* mgOpenDb (const char*fileName );

DESCRIPTION

mgOpenDb opens the file named by fileName. It returns the root database node upon successful completion.

The behavior of this function is affected by the functions: mgSetOpenCreateFlag, mgSetReadExtFlag and mgSetReadTexturesFlag.

Note: If your plug-in or script opens a file in the Creator modeling environment with this function or mgNewDb, the file is not opened in an active window on the Creator desktop. Your plug-in or script can still access the database but must explicitly save and close the file when done.

Note: If invoked from within the modeler environment, this function will fail if the named file is already open in an active window.

PARAMETERS

fileName
the name of the file to be opened

RETURNS

Returns a pointer to the opened database node if successful, MG_NULL otherwise.

C EXAMPLE

The following example opens an existing database.
mgrec* db;
db = mgOpenDb ("existingfile.flt");

ACCESS LEVEL

Level 1
