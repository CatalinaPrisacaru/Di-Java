mgGetModelingParentMatrix

FUNCTION NAME

mgGetModelingParentMatrix - returns current modeling parent and a matrix representing the cumulative transformation matrix applied to this parent node.

C SPECIFICATION

mgrec* mgGetModelingParentMatrix (mgrec*db,
mgmatrix*matrix );

PYTHON SPECIFICATION

mgrec, mgmatrix mgGetModelingParentMatrix (mgrec*db )

DESCRIPTION

Similar to mgGetModelingParent, mgGetModelingParentMatrix returns the current modeling parent node for the specified database db. Additionally, mgGetModelingParentMatrix returns a matrix that represents the cumulative transformation matrix applied to the parent node returned.

The function mgMatrixInvert can be used to obtain the inverse of this modeling parent matrix. This inverse matrix describes the transformation that transforms a coordinate in world space back to local space under the current modeling parent.

PARAMETERS

db
the database to get parent matrix for
matrix
address of matrix to receive parent matrix

RETURNS

mgrec
The current modeling parent node for database if one is set, MG_NULL otherwise.
mgmatrix
If a parent node is returned, the composite matrix representing the cumulative transformations applied to this parent node is returned in the output parameter matrix.

ACCESS LEVEL

Level 4
