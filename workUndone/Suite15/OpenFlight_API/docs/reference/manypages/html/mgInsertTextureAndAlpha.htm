mgInsertTextureAndAlpha

FUNCTION NAME

mgInsertTextureAndAlpha - loads a texture and alpha mask as a single texture palette entry.

SYNOPSIS       (C and Python)

int mgInsertTextureAndAlpha (mgrec*db,
const char*textureName,
const char*alphaName,
const char*mergeName );

DESCRIPTION

mgInsertTextureAndAlpha loads a texture and alpha mask from files textureName and alphaName, respectively, into the texture palette for database db. The created palette entry is automatically positioned in the palette such that no other textures are covered. The index of the new texture is also automatically assigned.

PARAMETERS

db
the database
textureName
the name of the texture file on disk
alphaName
the name of the alpha mask file on disk
mergeName
the name to associate with the merged texture in the palette

RETURNS

Returns the index assigned to the new palette entry if successful, or returns -1 otherwise.

ACCESS LEVEL

Level 1
