mgPluginToolDeleteProperty

FUNCTION NAME

mgPluginToolDeleteProperty - deletes user defined data associated with a plug-in tool.

C SPECIFICATION       (Not Available For Python)

void mgPluginToolDeleteProperty (mgplugintoolpluginTool,
const mgpropertynamepropName );

DESCRIPTION

Use mgPluginToolDeleteProperty to delete the data named propName that was associated with the plug-in tool pluginTool using mgPluginToolPutProperty.

Note that this function does not free or otherwise affect the actual data that was associated. If this data was dynamically allocated, it is the callers responsibility to deallocate it.

PARAMETERS

pluginTool
the plug-in tool
propName
the name of the property to delete

ACCESS LEVEL

Level 3
