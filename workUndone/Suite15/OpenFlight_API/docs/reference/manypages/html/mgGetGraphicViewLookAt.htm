mgGetGraphicViewLookAt

FUNCTION NAME

mgGetGraphicViewLookAt - returns the eyepoint of a graphic view in "look at" form.

SYNOPSIS       (C and Python)

mgbool mgGetGraphicViewLookAt (mggraphicviewgraphicView,
double*eyex,
double*eyey,
double*eyez,
double*centerx,
double*centery,
double*centerz,
double*upx,
double*upy,
double*upz );

DESCRIPTION

mgGetGraphicViewLookAt returns the eyepoint of the graphic view graphicView using "look at" style parameters. The "look at" parameters include 3 sets of values. The first set, eyex, eyey, and eyez is the position of the eye point (the "look from"). The second set, centerx, centery, centerz is the position of the reference point (the "look at"). The third set, upx, upy, and upz is the direction of the up vector.

PARAMETERS

graphicView
the graphic view
eyex
the x component of the eyepoint position
eyey
the y component of the eyepoint position
eyez
the z component of the eyepoint position
centerx
the x component of the reference point
centery
the y component of the reference point
centerz
the z component of the reference point
upx
the i component of the up vector
upy
the j component of the up vector
upz
the k component of the up vector

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
