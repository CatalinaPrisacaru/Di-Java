mgNewImageInfo

FUNCTION NAME

mgNewImageInfo - reads the header information from an image file.

SYNOPSIS       (C and Python)

mgimageinfo mgNewImageInfo (const char*textureName );

DESCRIPTION

mgNewImageInfo reads the header of an image on disk and then returns an image object with the information read. The parameter textureName is the name of the image file.

This function is similar to mgReadImageInfo except that it returns the image info object directly instead of returning it as an output parameter as mgReadImageInfo does. Use mgReadImageInfo if you need the texture error code on failure.

C NOTES

When you are through with the returned image info object, you must free it by calling mgFreeImageInfo.

PYTHON NOTES

It is not necessary to free the returned image info object in Python.

PARAMETERS

textureName
the name of the image file

RETURNS

Returns image info object if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
