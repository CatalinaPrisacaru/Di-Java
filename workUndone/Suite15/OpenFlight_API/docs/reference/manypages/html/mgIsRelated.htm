mgIsRelated

FUNCTION NAME

mgIsRelated - determines if a node is related to another node in the hierarchy.

SYNOPSIS       (C and Python)

mgbool mgIsRelated (mgrec*parent,
mgrec*child );

DESCRIPTION

mgIsRelated determines if the node child is a descendant of node parent.

PARAMETERS

parent
the parent node record
child
the node or transformation record

RETURNS

Returns MG_TRUE if the child is a descendant of parent, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
