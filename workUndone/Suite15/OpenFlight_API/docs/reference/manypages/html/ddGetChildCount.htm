ddGetChildCount

FUNCTION NAME

ddGetChildCount - gets the number of valid child codes for a record.

SYNOPSIS       (C and Python)

int ddGetChildCount (mgcodercode );

DESCRIPTION

ddGetChildCount returns the number of valid child codes defined in the data dictionary for a record identified by rcode.

PARAMETERS

rcode
the record code

RETURNS

Returns the child count if valid, 0 if not valid or no children.

ACCESS LEVEL

Level 1
