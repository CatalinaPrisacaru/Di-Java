mgRegisterPaletteNotifier

FUNCTION NAME

mgRegisterPaletteNotifier - registers a palette notifier.

C SPECIFICATION       (Not Available For Python)

mgnotifier mgRegisterPaletteNotifier (mgplugintoolpluginTool,
mgnotifiereventevent,
mgrec*db,
mgpalettenotifyfuncnotifyFunc,
void*userData );

DESCRIPTION

mgRegisterPaletteNotifier registers a model-time palette notifier on behalf of the specified editor or viewer plug-in tool, pluginTool. When the specified model time palette event occurs for the specified database db, the palette notification callback function, notifyFunc is called. The affected database and palette entry index are passed to the notification callback function with the specified user defined data, userData.

Typically, you register a model-time palette notifier for a particular database node db in the scene. If you do this, your notification callback function will be called only when the specified event occurs within the palette of that specific db node. If you want your notification callback function to be called whenever the event occurs within the palette of any database node, set db to MG_NULL.

Note: Model-time palette notifier events only apply to viewer and editor tools. Database importers and exporters, image importers and input device plug-in cannot receive model-time palette notification in this way.

The following palette notifiers are valid for this function:
MNOTIFY_COLORPALETTECHANGED
MNOTIFY_TEXTUREPALETTECHANGED
MNOTIFY_MATERIALPALETTECHANGED
MNOTIFY_TEXTUREMAPPINGPALETTECHANGED
MNOTIFY_LIGHTPOINTAPPEARANCEPALETTECHANGED
MNOTIFY_LIGHTPOINTANIMATIONPALETTECHANGED
MNOTIFY_SHADERPALETTECHANGED
MNOTIFY_SOUNDPALETTECHANGED
MNOTIFY_LIGHTSOURCEPALETTECHANGED

PARAMETERS

pluginTool
the plug-in tool that is registering the notifier
event
notification event you are interested in
db
the database for which you are registering the event
notifyFunc
the palette notification callback function
userData
user data to be passed to notification callback function when it is called

RETURNS

Returns a notifier object if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
