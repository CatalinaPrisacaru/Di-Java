mggraphicdrawmode

ENUM NAME

mggraphicdrawmode - Graphic View Draw Mode type

SYNOPSIS

typedef enum mggraphicdrawmode {
   MGDM_WIREFRAME,
   MGDM_SOLID,
   MGDM_WIRESOLID,
   MGDM_SELECT,
} mggraphicdrawmode;

MEMBERS

MGDM_WIREFRAME
Graphic View Draw Mode is Wireframe
MGDM_SOLID
Graphic View Draw Mode is Solid
MGDM_WIRESOLID
Graphic View Draw Mode is Wireframe over Solid
MGDM_SELECT
Graphic View Draw Mode is Select
