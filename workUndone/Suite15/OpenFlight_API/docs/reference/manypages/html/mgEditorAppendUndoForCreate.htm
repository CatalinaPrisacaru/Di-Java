mgEditorAppendUndoForCreate

FUNCTION NAME

mgEditorAppendUndoForCreate - appends undo/redo callback actions for an editor tool instance that creates new geometry in the scene.

SYNOPSIS       (C and Python)

mgstatus mgEditorAppendUndoForCreate (mgeditorcontexteditorContext,
mgrec*node );

DESCRIPTION

mgEditorAppendUndoForCreate performs the same function as mgEditorAddUndoForCreate but instead of creating a new entry in the Edit->Undo menu, mgEditorAppendUndoForCreate appends the undo callback actions to an existing undo entry for the editor tool instance.

See mgEditorAddUndoForCreate for a complete description of this function.

Calling mgEditorAppendUndoForCreate without first calling mgEditorAddUndo, mgEditorAddUndoForCreate, mgEditorAddUndoForDelete, or mgEditorAddUndoForMove within the editor tool instance is equivalent to calling mgEditorAddUndoForCreate and specifying MUML_USETOOLNAME for the menu label.

PARAMETERS

editorContext
the editor context to register undo callback for
node
the node created for which undo/redo is desired

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
