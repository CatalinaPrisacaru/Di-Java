MKB_ALTKEY

NAME

MKB_ALTKEY - Keyboard flag Alt Key Down

DESCRIPTION

When the appropriate mouse input function is called to report mouse input activity to an editor tool or to a GL control, this keyboard flag mask will be set if the Alt Key is down when the input was reported.
