mgNextConstructEdge

FUNCTION NAME

mgNextConstructEdge - gets the next construction edge node associated with an active editor tool instance.

SYNOPSIS       (C and Python)

mgrec* mgNextConstructEdge (mgeditorcontexteditorContext,
mgrec*constructEdge );

DESCRIPTION

mgNextConstructEdge gets the construction edge node that follows the specified construction node, constructEdge, associated with the editor tool instance identified by editor context, editorContext.

After calling mgFirstConstructEdge to get the first construction edge node associated with an editor tool instance, you can call mgNextConstructEdge to get successive construction edge nodes.

PARAMETERS

editorContext
the editor context from which to get the next construction edge node
constructEdge
the construction edge node to get next construction edge for

RETURNS

Returns the next construction edge node if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4
