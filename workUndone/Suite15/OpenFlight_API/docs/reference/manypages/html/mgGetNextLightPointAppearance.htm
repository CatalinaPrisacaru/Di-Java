mgGetNextLightPointAppearance

FUNCTION NAME

mgGetNextLightPointAppearance - gets the next entry in a light point appearance palette.

C SPECIFICATION

mgrec* mgGetNextLightPointAppearance (mgrec*lightPointAppearance,
int*index );

PYTHON SPECIFICATION

mgrec, index mgGetNextLightPointAppearance (mgrec*lightPointAppearance )

DESCRIPTION

mgGetNextLightPointAppearance gets the entry that follows lightPointAppearance in the light point appearance palette for database db.

The code of the record returned will be fltLpAppearancePalette .

PARAMETERS

lightPointAppearance
the palette entry to get next for
index
address of value to receive index of entry returned

RETURNS

mgrec
The next light point appearance entry record, or MG_NULL if no next light point appearance in palette.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

EXAMPLE

See mgGetFirstLightPointAppearance for an example of how to use this function.

ACCESS LEVEL

Level 1
