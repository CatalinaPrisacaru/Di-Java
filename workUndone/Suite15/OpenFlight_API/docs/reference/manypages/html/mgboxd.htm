mgboxd

STRUCT NAME

mgboxd - Double precision box record

SYNOPSIS

typedef struct mgboxd {
   mgcoord3d min;
   mgcoord3d max;
} mgboxd;

MEMBERS

min
mgcoord3d - lower left front point of box
max
mgcoord3d - upper right back point of box

DESCRIPTION

This record contains 2 3-D coordinates that define an axis aligned box.
