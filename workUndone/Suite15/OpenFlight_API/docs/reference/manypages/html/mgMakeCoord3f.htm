mgMakeCoord3f

FUNCTION NAME

mgMakeCoord3f - makes a 3 dimensional single precision floating point coordinate from individual values.

SYNOPSIS       (C and Python)

mgcoord3f mgMakeCoord3f (floatx,
floaty,
floatz );

DESCRIPTION

mgMakeCoord3f returns a 3 dimensional single precision floating point record representing the coordinate whose X, Y and Z values are x, y and z, respectively.

PARAMETERS

x
X value for the coordinate
y
Y value for the coordinate
z
Z value for the coordinate

RETURNS

Returns a 3 dimensional single precision floating point coordinate record.

ACCESS LEVEL

Level 1
