mgMatrixCompose

FUNCTION NAME

mgMatrixCompose - composes a matrix by specifying its 3 local axes and a translation.

C SPECIFICATION

void mgMatrixCompose (mgmatrix*matrix,
const mgcoord3d*xAxis,
const mgcoord3d*yAxis,
const mgcoord3d*zAxis,
const mgcoord3d*translation );

PYTHON SPECIFICATION

mgmatrix mgMatrixCompose (mgcoord3dxAxis,
mgcoord3dyAxis,
mgcoord3dzAxis,
mgcoord3dtranslation )

DESCRIPTION

mgMatrixCompose composes a matrix with xAxis as the local X axis, yAxis as the local Y axis, zAxis as the local Z axis and translation as the translation.

The original coordinates xAxis, yAxis, zAxis and translation are not affected.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
xAxis
local X axis to set to
yAxis
local Y axis to set to
zAxis
local Z axis to set to
translation
translation to set to

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
