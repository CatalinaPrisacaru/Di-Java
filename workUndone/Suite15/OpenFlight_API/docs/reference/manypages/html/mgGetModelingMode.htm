mgGetModelingMode

FUNCTION NAME

mgGetModelingMode - returns current modeling mode.

SYNOPSIS       (C and Python)

mgcode mgGetModelingMode (mgrec*db );

DESCRIPTION

mgGetModelingMode returns the current modeling mode for the specified database db.

Note that the edge modeling mode is a special case of vertex mode. Since there is no "edge" type in OpenFlight, you must interpret the return value of this function carefully to differentiate between vertex and edge mode. In both vertex and edge modes, this function will return fltVertex. To know if the modeling mode is really edge mode, call mgIsModelingModeEdge. It will return MG_TRUE if the modeling mode is really edge, MG_FALSE if the modeling mode is truly vertex.

PARAMETERS

db
the database to get modeling mode for

RETURNS

Returns current modeling mode for database if valid, 0 (zero) otherwise.

EXAMPLE

The following examples (in C and Python) show how to use this function for different modeling modes:

C EXAMPLE

mgcode mode;

mode = mgGetModelingMode (db);
if (mode == fltGroup) {
   printf ("modeling mode is group\n");
}
else if (mode == fltVertex) {
   // modeling mode is vertex OR edge
   mgbool isEdge = mgIsModelingModeEdge (db);
   if (isEdge == MG_TRUE) {
      printf ("modeling mode is edge\n")
   }
   else {
      printf ("modeling mode is really vertex\n")
   }
}

PYTHON EXAMPLE

mode = mgGetModelingMode (db)
if (mode == fltGroup):
   print "modeling mode is group"
elif (mode == fltVertex:
   # modeling mode is vertex OR edge
   isEdge = mgIsModelingModeEdge (db)
   if (isEdge == MG_TRUE):
      print "modeling mode is edge"
   else:
      print "modeling mode is really vertex"

Level 4
