mgSetPolyCurrentAltColor

FUNCTION NAME

mgSetPolyCurrentAltColor - sets the alternate color attribute in a fltPolygon or fltMesh record to the current alternate modeling color.

SYNOPSIS       (C and Python)

mgbool mgSetPolyCurrentAltColor (mgrec*db,
mgrec*rec );

DESCRIPTION

Given a fltPolygon or fltMesh record, rec, this function sets its alternate color attribute according to the current alternate modeling color selected by the user in Creator for database db.

If db is color index mode, this function will set the fltPolyAltColor and fltPolyAltIntensity attributes of rec. If db is RGB color mode, this function will set the alternate red, green and blue color values of rec.

PARAMETERS

db
the database whose current alternate modeling color will be used
rec
the fltPolygon or fltMesh record on which to set the color attribute

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

C EXAMPLE

mgrec* poly;
mgbool status;

poly = mgNewRec (fltPolygon);
// set alternate color attribute to the current alternate color in the palette
status = mgSetPolyCurrentAltColor (db, poly);

ACCESS LEVEL

Level 4
