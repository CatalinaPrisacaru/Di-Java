mgTextSetDouble

FUNCTION NAME

mgTextSetDouble - displays double precision floating point formatted text in a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSetDouble (mgguicontrol,
doubledoubleVal,
const char*fmtString );

DESCRIPTION

mgTextSetDouble displays the specified double value doubleVal using the specified printf style formatting string fmtString in the specified text control.

If you specify MG_NULL in fmtString, the default double precision floating point formatting string "%lf" is used.

PARAMETERS

control
the text control
doubleVal
double precision floating point value to display
fmtString
printf style formatting string

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
