mgEditorSetTrackPlane

FUNCTION NAME

mgEditorSetTrackPlane - selects a tracking plane to control 3D vertex input for an editor tool instance.

C SPECIFICATION       (Not Available For Python)

mgstatus mgEditorSetTrackPlane (mgeditorcontexteditorContext,
mgplaned*trackPlane );

DESCRIPTION

mgEditorSetTrackPlane selects a tracking plane trackPlane, that controls 3D vertex input for the specified editor context, editorContext.

When a 3D vertex input callback function is active for an editor tool instance, you can use this function to specify that all coordinate input be projected onto the specified plane. In this way, you can override the tracking plane(s) set up by the modeler in Creator.

The tracking plane you select does not affect vertex and edge references or coordinates input via the Track Coordinate Window. Also, Snap To Grid, if set in Creator, is ignored for your selected tracking plane.

If trackPlane is specified as MG_NULL, the tracking plane set by the modeler in Creator will be used again for 3D vertex input.

PARAMETERS

editorContext
the editor context to select the tracking plane for
trackPlane
the tracking plane to control 3D vertex input

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
