mgGetNextTexture

FUNCTION NAME

mgGetNextTexture - gets successive entries in a texture palette.

C SPECIFICATION

mgbool mgGetNextTexture (mgrec*db,
int*index,
char*textureName );

PYTHON SPECIFICATION

mgbool, index, textureName mgGetNextTexture (mgrec*db )

DESCRIPTION

mgGetNextTexture gets the index and name of the next texture contained in the texture palette of database db.

After calling mgGetFirstTexture, successive calls to mgGetNextTexture return successive texture palette entries.

If the texture was located when it was loaded, the textureName includes the full path specification of the file where it was found on disk. If the texture was not located at load time, the textureName returned by this function will be the same name returned by mgGetTextureSaveName.

If you pass MG_NULL for index, the index of the texture is not returned.

If you pass MG_NULL for textureName, the name of the texture is not returned. If you do pass a character string for textureName, the buffer should be at least 1024 characters.

PARAMETERS

db
the database
index
address of value to receive index
textureName
address of string to receive texture name

RETURNS

mgbool
MG_TRUE if the next texture palette entry is found, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.
textureName
If function is successful, textureName contains the retrieved name, otherwise textureName is undefined.

EXAMPLE

See mgGetFirstTexture for an example of how to use this function.

ACCESS LEVEL

Level 1
