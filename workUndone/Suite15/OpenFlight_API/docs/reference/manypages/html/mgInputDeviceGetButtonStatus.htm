mgInputDeviceGetButtonStatus

FUNCTION NAME

mgInputDeviceGetButtonStatus - gets the state of a button on an input device.

SYNOPSIS       (C and Python)

mgmousestate mgInputDeviceGetButtonStatus (mginputdeviceinputDevice,
intbuttonNo );

DESCRIPTION

mgInputDeviceGetButtonStatus gets the state of the button buttonNo from the input device inputDevice.

Note: The mouse pointer device does not set button states.

PARAMETERS

inputDevice
the input device
buttonNo
the button number

RETURNS

Returns the state of the button buttonNo on the input device.

ACCESS LEVEL

Level 4
