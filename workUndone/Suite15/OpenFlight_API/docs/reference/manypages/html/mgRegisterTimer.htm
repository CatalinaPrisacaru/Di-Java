mgRegisterTimer

FUNCTION NAME

mgRegisterTimer - submit a timer request.

C SPECIFICATION       (Not Available For Python)

mgtimer mgRegisterTimer (mgguigui,
unsigned inttimeout,
mgtimerfunctimerFunc,
void*userData );

DESCRIPTION

mgRegisterTimer submits a timer request for a specified time-out value timeout. When the time-out expires, the specified timer function timerFunc will be called.

If the timer request was accepted, a timer object will be created and returned by this function. If you need to cancel the timer before it expires, you can call mgUnregisterTimer passing to it the timer object.

Note: The timer object is destroyed if either your timer function returns MG_FALSE or you call mgUnregisterTimer to remove the timer request before it expires.

PARAMETERS

gui
dialog or control to associate timer to
timeout
time-out value in milliseconds
timerFunc
timer function to call when time-out expires
userData
user data to be passed to timer function when it is called

RETURNS

Returns the timer object if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4

mgUnregisterTimer, mgtimerfunc
