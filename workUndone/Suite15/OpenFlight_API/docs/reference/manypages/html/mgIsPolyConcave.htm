mgIsPolyConcave

FUNCTION NAME

mgIsPolyConcave - determines if a polygon is concave.

SYNOPSIS       (C and Python)

mgbool mgIsPolyConcave (mgrec*rec );

DESCRIPTION

Given the fltPolygon record poly, mgIsPolyConcave determines if this polygon is concave. If any line on the plane of the polygon crosses more than two edges of the polygon, then the polygon is concave.

Note: By convention, a polygon with no vertices is NOT considered concave by this function.

PARAMETERS

rec
the polygon to check

RETURNS

Returns MG_TRUE if the polygon is concave, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
