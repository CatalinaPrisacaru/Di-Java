mgDeleteConstructVertex

FUNCTION NAME

mgDeleteConstructVertex - deletes a construction vertex node.

SYNOPSIS       (C and Python)

mgstatus mgDeleteConstructVertex (mgrec*constructVtx );

DESCRIPTION

mgDeleteConstructVertex deletes the specified construction vertex node constructVtx.

This function is typically called in the termination function of your editor tool.

PARAMETERS

constructVtx
the construction vertex to delete

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
