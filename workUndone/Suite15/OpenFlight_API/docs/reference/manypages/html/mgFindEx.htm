mgFindEx

FUNCTION NAME

mgFindEx - walks a node hierarchy and collects a list of nodes that match a user specified criteria.

SYNOPSIS       (C and Python)

mgreclist mgFindEx (mgrec*node,
mgmatrixstartMatrix,
mgwalkfuncmatchFunc,
void*userData,
unsigned intflags );

DESCRIPTION

mgFindEx performs the same function as mgFind, but in addition also collects a matrix for each node in the record list returned. This matrix represents the cumulative transformations between the root node and the node in the record list. When using mgFindEx in this way, you specify a starting matrix, startMatrix, to use when accumulating a matrix stack during the traversal.

Note: If flags does not specify MWALK_MATRIXSTACK, startMatrix is ignored and no matrices are collected for the nodes in the record list.

For more information and example code, see mgFind.

C NOTES

When you are done accessing the record list returned by this function, you should dispose of it using mgFreeRecList.

PYTHON NOTES

It is not necessary to dispose of the record list returned in Python.

PARAMETERS

node
the starting node of the traversal
startMatrix
the starting matrix to use when accumulating a matrix stack during the traversal - ignored if MWALK_MATRIXSTACK is not specified in flags
matchFunc
the match function invoked for each node visited
userData
pointer to user defined data that will be passed to the matchFunc callback function
flags
traversal modifiers that allow you to select certain types of nodes for visiting. This parameter is the bitwise combination of the following masks:
MWALK_NEXT
MWALK_VERTEX
MWALK_MASTER
MWALK_MASTERALL
MWALK_ON
MWALK_NOREADONLY
MWALK_NONESTED
MWALK_MATRIXSTACK

RETURNS

Returns a record list containing all the nodes visited for which the matchFunc function returned MG_TRUE.

ACCESS LEVEL

Level 1
