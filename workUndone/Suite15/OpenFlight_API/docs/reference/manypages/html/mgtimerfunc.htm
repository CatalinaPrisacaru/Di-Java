mgtimerfunc

CALLBACK NAME

mgtimerfunc - Timer function

SYNOPSIS

mgbool (*mgtimerfunc) ( mgtimer timerId,
void* userData );

DESCRIPTION

This is the signature for timer functions. When you create a timer object using mgRegisterTimer, you will provide a timer function of this form that will be called when your timer expires.

PARAMETERS

timerId
the timer object that identifies the timer that has expired
userData
user defined data specified when timer was submitted

RETURNS

Return MG_TRUE if you want to resubmit the timer request, MG_FALSE otherwise. In this way you can set up a recurring timer object by returning MG_TRUE.

mgRegisterTimer, mgUnregisterTimer
