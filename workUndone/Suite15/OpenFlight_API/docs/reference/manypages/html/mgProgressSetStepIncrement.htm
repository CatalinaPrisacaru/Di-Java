mgProgressSetStepIncrement

FUNCTION NAME

mgProgressSetStepIncrement - sets the step increment for a progress control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgProgressSetStepIncrement (mgguicontrol,
doubleincrement );

DESCRIPTION

mgProgressSetStepIncrement sets the step increment value for the specified progress control to the specified value, increment.

The step increment value is the value that a progress control is incremented when mgProgressStepValue is called.

The step increment value specified must be positive. The value will be ignored if it is not. The default step increment value for a progress control is 10.

PARAMETERS

control
the progress control
increment
the step increment value for the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
