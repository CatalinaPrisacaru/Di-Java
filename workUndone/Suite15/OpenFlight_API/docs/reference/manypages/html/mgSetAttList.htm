mgSetAttList

FUNCTION NAME

mgSetAttList - sets the values of record attributes using varargs argument style.

SYNOPSIS       (C and Python)

int mgSetAttList (mgrec*rec,
< attrCode >,
< attrVal >,
... );

DESCRIPTION

mgSetAttList sets one or more values of rec's attributes named by the attribute codes attrCode. Values are specified by the corresponding attrVal. Note that rec must be a proper record and cannot be a value record.

C NOTES

Variable-length argument lists cannot be type-checked by the compiler, so using varargs functions can be a source of bugs. Be sure to terminate the argument list with MG_NULL and be sure that the type of each argument is as expected.

PYTHON NOTES

It is not necessary to terminate the argument list with MG_NULL in Python.

PARAMETERS

rec
the record in which to set values
< attrCode >
the attribute record code to set
< attrVal >
the attribute value to which attrCode is set
...
the argument list is in pairs of attrCode and attrVal ending with MG_NULL terminator

RETURNS

Returns the number of attribute code/address pairs set successfully.

C EXAMPLE

mgrec* poly;
short texture = 1;
short material = 1;

mgSetAttList (poly,
        fltPolyMaterial, material,
        fltPolyTexture, texture,
        MG_NULL);

PYTHON EXAMPLE

texture = 1
material = 1
mgSetAttList (poly,
        fltPolyMaterial, material,
        fltPolyTexture, texture)

ACCESS LEVEL

Level 2
