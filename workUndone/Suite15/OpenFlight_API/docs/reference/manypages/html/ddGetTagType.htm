ddGetTagType

FUNCTION NAME

ddGetTagType - gets the data type for a record or data code.

SYNOPSIS       (C and Python)

mgtagtype ddGetTagType (mgcodecode );

DESCRIPTION

ddGetTagType gets the data type for the specified record or data code, code.

PARAMETERS

code
the record or data code

RETURNS

Returns the data type of the code. See mgtagtype for a description of the possible data types returned.

ACCESS LEVEL

Level 1
