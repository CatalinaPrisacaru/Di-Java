mgParamAddDouble2

FUNCTION NAME

mgParamAddDouble2 - adds a parameter to a parameter block composed of two double precision floating point numbers.

C SPECIFICATION       (Not Available For Python)

mgparam mgParamAddDouble2 (mgparamblockparamBlock,
const char*paramName,
doubledefaultValue1,
doubledefaultValue2 );

DESCRIPTION

mgParamAddDouble2 creates a new parameter composed of two double precision floating point numbers named paramName and adds it to the parameter block paramBlock. The parameter value is initialized with defaultValue1 and defaultValue2.

After the parameter value is defined, use mgParamGetDouble2 and mgParamSetDouble2 to query and change its value, respectively.

Note: If the named parameter already exists in paramBlock, it is replaced.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
defaultValue1
default value 1 of the parameter
defaultValue2
default value 2 of the parameter

RETURNS

Returns the parameter if created successfully, MG_NULL otherwise.

ACCESS LEVEL

Level 4
