mgrenderercallbackreason

TYPE NAME

mgrenderercallbackreason - Callback reason type used to renderer events

DESCRIPTION

When a renderer callback function function is called, the callbackReason parameter passed in of this type will specify which event triggered the callback.

Renderer callback functions are called to report these events.

The events that are sent to renderer callback functions are:
MRCB_INIT
MRCB_SHOW
MRCB_SIZE
MRCB_DRAW
MRCB_HIDE
MRCB_DESTROY
