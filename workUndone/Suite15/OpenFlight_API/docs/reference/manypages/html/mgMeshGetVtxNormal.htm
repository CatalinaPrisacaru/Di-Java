mgMeshGetVtxNormal

FUNCTION NAME

mgMeshGetVtxNormal - retrieves the normal vector for a vertex in a mesh.

C SPECIFICATION

mgbool mgMeshGetVtxNormal (mgrec*rec,
intvtxIndex,
float*i,
float*j,
float*k );

PYTHON SPECIFICATION

mgbool, i, j, k mgMeshGetVtxNormal (mgrec*rec,
intvtxIndex )

DESCRIPTION

This function retrieves the i, j, and k values of the normal vector of the vertex at index vtxIndex in the vertex pool of mesh node rec.

You do not have to provide addresses for all i, j, and k values. For example, if you only want to receive the i and j components (but not k), specify valid addresses for i and j and leave k NULL.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
i
address of value to receive i component of normal vector
j
address of value to receive j component of normal vector
k
address of value to receive k component of normal vector

RETURNS

mgbool
MG_TRUE if the vertex data was retrieved successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertices in the mesh do not color values.
i
If function is successful, i contains the retrieved i coordinate, otherwise i is undefined.
j
If function is successful, j contains the retrieved j coordinate, otherwise j is undefined.
k
If function is successful, k contains the retrieved k coordinate, otherwise k is undefined.

ACCESS LEVEL

Level 1
