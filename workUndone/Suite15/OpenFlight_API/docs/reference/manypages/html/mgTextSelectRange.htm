mgTextSelectRange

FUNCTION NAME

mgTextSelectRange - selects a range of text in a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSelectRange (mgguicontrol,
intfirstPos,
intlastPos );

DESCRIPTION

mgTextSelectRange selects the specified range of characters displayed in the specified text control.

The first position is at index 0.

If firstPos is 0 and lastPos is -1, all the text in the text control is selected. If firstPos is -1, any current selection is removed. The caret is placed at the end of the selection indicated by the greater of the two values firstPos and lastPos.

PARAMETERS

control
the text control
firstPos
position of first character to select
lastPos
position of last character to select

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
