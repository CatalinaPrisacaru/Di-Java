mgExtensionFieldSetFloat

FUNCTION NAME

mgExtensionFieldSetFloat - sets the single precision floating point value of an extension field on a node.

SYNOPSIS       (C and Python)

mgbool mgExtensionFieldSetFloat (mgrec*rec,
const char*fieldGuid,
floatfieldValue );

DESCRIPTION

mgExtensionFieldSetFloat sets the value of an extension field identified by fieldGuid, attached to node rec to fieldValue.

Note: If the extension field is already attached to rec and is not of type MTYPE_FLOAT, fieldValue is coerced to the type of the existing extension field. For example, if the current extension field is of type MTYPE_FLAG, a value of 0 will be set on rec if fieldValue is 0.0, a value of 1 will be set otherwise. If the current extension field is of type MTYPE_INT or MTYPE_DOUBLE, fieldValue will simply be converted to the corresponding numeric type. Note that if the type is MTYPE_INT, fieldValue will be truncated. If the current extension field is of type MTYPE_TEXT or MTYPE_XML fieldValue will converted to the corresponding ASCII text.

PARAMETERS

rec
the node to set extension value on
fieldGuid
the GUID of the extension field
fieldValue
the value to set

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
