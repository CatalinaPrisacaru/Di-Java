mgEditorSetSnapFace

FUNCTION NAME

mgEditorSetSnapFace - enables or disables "face snapping" to control 3D vertex input for an editor tool instance.

C SPECIFICATION       (Not Available For Python)

mgstatus mgEditorSetSnapFace (mgeditorcontexteditorContext,
mgboolflag );

DESCRIPTION

mgEditorSetSnapFace controls whether 3D vertex input is automatically snapped to the face below the mouse for the specified editor context, editorContext.

When a 3D vertex input callback function is active for an editor tool instance, you can use this function to specify that all coordinate input be "snapped to" (projected onto) the face below (pointed to by) the mouse. If you set flag to MG_TRUE, face snapping will be enabled. If you set flag to MG_FALSE, face snapping will be disabled.

When face snapping is enabled and the modeler uses the mouse to enter a 3D vertex in the graphics view, the 3D vertex coordinate returned will be projected onto the plane in which the face below the mouse lies. When more than one face is below the mouse, the face selected is that which is "topmost" in Z ordering. If no faces are below the mouse, the 3D vertex returned will be determined by the current trackplane.

PARAMETERS

editorContext
the editor context in which to enable or disable face snapping
flag
MG_TRUE to enable face snapping, MG_FALSE to disable

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
