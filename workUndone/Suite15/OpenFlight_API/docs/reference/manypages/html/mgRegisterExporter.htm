mgRegisterExporter

FUNCTION NAME

mgRegisterExporter - registers a database exporter tool.

C SPECIFICATION       (Not Available For Python)

mgplugintool mgRegisterExporter (mgpluginplugin,
const char*toolName,
mgtoolstartfuncstartFunc,
void*userData,
... );

DESCRIPTION

mgRegisterExporter registers a database exporter tool for the specified plug-in module plugin with the specified tool attributes.

The required tool attributes include the name of the tool toolName, a start function for the tool, startFunc and user defined callback data, userData (may be MG_NULL) that is passed to the start function when it is called.

Additional required database exporter tool attributes include MTA_FILTER and MTA_FILETYPE. Additional optional tool attributes include MTA_VERSION and MTA_HELPCONTEXT. These additional tool attributes are passed using variable argument style and must be terminated with MG_NULL.

PARAMETERS

plugin
the plug-in module registering the tool
toolName
the name of the tool
startFunc
the tool start function that is called when the tool is launched
userData
user defined data that will be passed to startFunc when it is called
...
additional database exporter tool specific attributes in variable argument style

RETURNS

Pointer to plug-in tool registered if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
