mgRendererSetRendererFunc - registers a renderer function for a renderer.

DESCRIPTION

mgRendererSetRendererFunc registers the renderer function for a renderer.

rendererPluginTool
the renderer plug-in tool to which this renderer function is to be applied
rendererFunc
the renderer function
rendererContext
the renderer context to get associated view handle
db
the database
eyex
the x component of the eyepoint position
eyey
the y component of the eyepoint position
eyez
the z component of the eyepoint position
centerx
the x component of the reference point
centery
the y component of the reference point
centerz
the z component of the reference point
upx
the i component of the up vector
upy
the j component of the up vector
upz
the k component of the up vector
