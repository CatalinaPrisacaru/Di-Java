mgSendMessage

FUNCTION NAME

mgSendMessage - reports a system message.

SYNOPSIS       (C and Python)

mgbool mgSendMessage (mgmessageseverityenumseverity,
const char*message,
... );

DESCRIPTION

mgSendMessage reports a system message containing the text message tagged with the specified severity level severity.

If invoked from within the stand-alone program environment, the reported message is directed to stdout. If invoked from the modeler environment, the reported message is directed to the Status Bar/Status Log Window.

The message string can contain printf style formatting, in which case the appropriate number of additional parameters must be passed using variable argument style.

PARAMETERS

severity
severity level of the message
message
message string, which may contain formatting, and should be followed with the appropriate parameters, if any
...
optional additional parameters to support formatted output

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if messages are not enabled.

ACCESS LEVEL

Level 1
