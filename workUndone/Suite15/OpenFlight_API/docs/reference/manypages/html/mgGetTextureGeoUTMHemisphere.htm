mgGetTextureGeoUTMHemisphere

FUNCTION NAME

mgGetTextureGeoUTMHemisphere - gets hemisphere from a georeference info object.

C SPECIFICATION

mgbool mgGetTextureGeoUTMHemisphere (mgimagegeoinfogeoInfo,
int*hemisphere );

PYTHON SPECIFICATION

mgbool, hemisphere mgGetTextureGeoUTMHemisphere (mgimagegeoinfogeoInfo )

DESCRIPTION

mgGetTextureGeoUTMHemisphere gets the hemisphere from the specified image info object geoInfo.

Valid values for the hemisphere attribute are MIMG_GEOHEMISPHERE_NORTH, and MIMG_GEOHEMISPHERE_SOUTH.

PARAMETERS

geoInfo
the georeference info object
hemisphere
address of integer to receive UTM hemisphere value

RETURNS

mgbool
MG_TRUE if hemisphere retrieved successfully, MG_FALSE otherwise.
hemisphere
If function is successful, hemisphere contains the retrieved projection, otherwise hemisphere is undefined.

ACCESS LEVEL

Level 1
