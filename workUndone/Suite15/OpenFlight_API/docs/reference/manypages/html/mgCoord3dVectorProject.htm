mgCoord3dVectorProject

FUNCTION NAME

mgCoord3dVectorProject - calculates the vector projection of 3D coordinates onto a specified axis.

SYNOPSIS       (C and Python)

mgcoord3d mgCoord3dVectorProject (const mgcoord3d*coord1,
const mgcoord3d*coord2 );

DESCRIPTION

mgCoord3dVectorProject calculates and returns the vector projection of 3D double precision floating point coordinates onto a specified axis.

The original coordinates coord1 and coord2 are not affected.

Note: For efficiency, coord1 and coord2 are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

coord1
address of coordinates to project
coord2
address of coordinates of axis to project onto, should have a length above zero

RETURNS

Returns a double precision 3D coordinate record.

ACCESS LEVEL

Level 1
