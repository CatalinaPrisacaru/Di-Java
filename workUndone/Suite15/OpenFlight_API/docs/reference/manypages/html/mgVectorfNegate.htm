mgVectorfNegate

FUNCTION NAME

mgVectorfNegate - calculates vector multiplied by -1.0.

SYNOPSIS       (C and Python)

mgvectorf mgVectorfNegate (const mgvectorf*vector );

DESCRIPTION

mgVectorfNegate calculates and returns the product of double precision floating point vector and -1.0.

The original vector vector is not affected.

Note: For efficiency, vector is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

vector
address of vector

RETURNS

Returns a single precision vector record.

ACCESS LEVEL

Level 1
