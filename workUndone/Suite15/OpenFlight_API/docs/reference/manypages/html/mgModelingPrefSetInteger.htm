mgModelingPrefSetInteger

FUNCTION NAME

mgModelingPrefSetInteger - sets an integer preference value in the modeling preferences.

SYNOPSIS       (C and Python)

mgbool mgModelingPrefSetInteger (const mgmodelingprefnameprefName,
intprefValue );

DESCRIPTION

mgModelingPrefSetInteger sets the modeling preference named prefName to the value prefValue.

Currently only a small subset of the modeling preference values are accessible. For a complete list, see mgmodelingprefname.

Note that the prefix of the key, either MPREFD or MPREFI, indicates whether the corresponding preference value is double or integer, respectively.

PARAMETERS

prefName
the preference key name
prefValue
the value to set

RETURNS

Returns MG_TRUE if the key prefName represents a valid modeling preference and the value was set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
