mgGetTextureGeoImageOrigin

FUNCTION NAME

mgGetTextureGeoImageOrigin - gets image origin from a georeference info object.

C SPECIFICATION

mgbool mgGetTextureGeoImageOrigin (mgimagegeoinfogeoInfo,
int*imageOrigin );

PYTHON SPECIFICATION

mgbool, imageOrigin mgGetTextureGeoImageOrigin (mgimagegeoinfogeoInfo )

DESCRIPTION

mgGetTextureGeoImageOrigin gets the width attribute from the specified georeference info object geoInfo. The image origin specifies the origin and the positive y direction used by image geographic control points.

Valid values for the image origin attribute are MIMG_GEOIMAGEORIGIN_LL (the image origin is in the lower left corner and y increases toward the top), and MIMG_GEOIMAGEORIGIN_UL (the image origin is in the upper left corner of the image and increases toward the bottom).

PARAMETERS

geoInfo
the georeference info object
imageOrigin
address of integer to receive image origin value

RETURNS

mgbool
MG_TRUE if image origin retrieved successfully, MG_FALSE otherwise.
imageOrigin
If function is successful, imageOrigin contains the retrieved image origin, otherwise imageOrigin is undefined.

ACCESS LEVEL

Level 1
