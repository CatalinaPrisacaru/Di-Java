mgCopyTexture

FUNCTION NAME

mgCopyTexture - makes a copy of a texture palette entry.

SYNOPSIS       (C and Python)

int mgCopyTexture (mgrec*dstDb,
mgrec*srcDb,
const char*newTextureName,
intsrcIndex );

DESCRIPTION

mgCopyTexture makes a new texture palette entry from a source database srcDb, and index srcIndex, and loads it in the destination database’s dstDb texture palette using the first available index and the given texture name newTextureName.

The texture can be copied from one database to another, or it can be copied to a new index in the same database by specifying the same database for dstDb and srcDb.

PARAMETERS

dstDb
the destination database node
srcDb
the source database node
newTextureName
the name of the new entry (the copy)
srcIndex
the index of the texture palette entry to be copied

RETURNS

Returns the index of the new entry.

ACCESS LEVEL

Level 2
