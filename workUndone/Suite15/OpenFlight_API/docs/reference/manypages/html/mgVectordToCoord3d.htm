mgVectordToCoord3d

FUNCTION NAME

mgVectordToCoord3d - converts a double precision floating point vector to a 3D double precision floating point coordinate.

SYNOPSIS       (C and Python)

mgcoord3d mgVectordToCoord3d (const mgvectord*vec );

DESCRIPTION

mgVectordToCoord3d converts the specified double precision floating point vector vec to a 3D double precision floating point coordinate.

The original vector vec is not affected.

PARAMETERS

vec
address of double precision vector to convert

RETURNS

Returns a 3D double precision floating point coordinate.

ACCESS LEVEL

Level 1
