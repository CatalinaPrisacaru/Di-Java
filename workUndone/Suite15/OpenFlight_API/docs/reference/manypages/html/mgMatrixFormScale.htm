mgMatrixFormScale

FUNCTION NAME

mgMatrixFormScale - forms a scale matrix.

C SPECIFICATION

mgbool mgMatrixFormScale (mgmatrix*matrix,
doublex,
doubley,
doublez );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixFormScale (doublex,
doubley,
doublez )

DESCRIPTION

mgMatrixFormScale gets a matrix for scaling about a local origin (0.0, 0.0, 0.0). The parameters x, y, and z are the scale factors to apply along the coordinate axes, X, Y, and Z, respectively.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
x
scale factor in the X direction
y
scale factor in the Y direction
z
scale factor in the Z direction

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix.

ACCESS LEVEL

Level 1
