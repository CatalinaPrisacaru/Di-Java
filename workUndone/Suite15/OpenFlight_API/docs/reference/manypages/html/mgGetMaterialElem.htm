mgGetMaterialElem

FUNCTION NAME

mgGetMaterialElem - gets the material components of a record.

C SPECIFICATION

float*ambientR,
float*ambientG,
float*ambientB,
float*diffuseR,

float*diffuseB,

float*specularG,
float*specularB,

float*emissiveG,

PYTHON SPECIFICATION

mgbool, ambientR, ambientG, ambientB, diffuseR, diffuseG, diffuseB, specularR, specularG, specularB, emissiveR, emissiveG, emissiveB, shininess, alpha mgGetMaterialElem (mgrec*rec )

DESCRIPTION

Given a fltIMaterial or fltPolygon record, this function extracts the individual material components of that record. If rec is a fltIMaterial record, this function extracts the components directly from rec. If rec is a fltPolygon record, this function extracts the components from the material assigned to the polygon.

PARAMETERS

rec
the material record pointer
ambientR
the pointer for the ambient red value
ambientG
the pointer for the ambient green value
ambientB
the pointer for the ambient blue value
diffuseR
the pointer for the diffuse red value
diffuseG
the pointer for the diffuse green value
diffuseB
the pointer for the diffuse blue value
specularR
the pointer for the specular red value
specularG
the pointer for the specular green value
specularB
the pointer for the specular blue value
emissiveR
the pointer for the emissive red value
emissiveG
the pointer for the emissive green value
emissiveB
the pointer for the emissive blue value
shininess
the pointer for the shininess value
alpha
the pointer for the alpha (transparency) value

RETURNS

mgbool
MG_TRUE if the material components are found. MG_FALSE if the record doesn’t have material properties.

emissiveR

emissiveB
