mgGetNextRecInList

FUNCTION NAME

mgGetNextRecInList - returns the next item in the record list.

C SPECIFICATION

mgrec* mgGetNextRecInList (mgreclistrecList,
mgmatrix*matrix );

PYTHON SPECIFICATION

mgrec, mgmatrix mgGetNextRecInList (mgreclistrecList )

DESCRIPTION

mgGetNextRecInList returns the item in the specified record list recList that corresponds to the current position of the traversal pointer.

Before this function returns, it advances the position of the traversal pointer of the record list so that the next call to mgGetNextRecInList returns the proper list node.

If this function is successful and the output parameter matrix is not MG_NULL, it is loaded with the composite matrix representing the cumulative transformations applied to this item. If you do not need this matrix, you may specify matrix as MG_NULL.

PARAMETERS

recList
the record list to traverse
matrix
address of a matrix to receive the composite of all transformations above the returned record list item

RETURNS

mgrec
The next item in the record list, MG_NULL if no more items are in the list.
mgmatrix
If an item is returned, the output parameter matrix (if specified) is loaded with the composite matrix representing the cumulative transformations applied to this item.

ACCESS LEVEL

Level 1
