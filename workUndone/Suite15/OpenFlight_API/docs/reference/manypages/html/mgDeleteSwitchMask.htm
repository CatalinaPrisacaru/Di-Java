mgDeleteSwitchMask

FUNCTION NAME

mgDeleteSwitchMask - deletes a mask from a fltSwitch node.

SYNOPSIS       (C and Python)

int mgDeleteSwitchMask (mgrec*rec,
intmaskNo );

DESCRIPTION

mgDeleteSwitchMask deletes the mask specified by maskNo from a fltSwitch node. Upon successful completion, the indices of all masks after the one deleted are decremented by one.

The value of maskNo must be between 0 and the number of masks of rec minus 1 (the first mask is maskNo 0).

Note: Since a switch must have at least one mask, mgDeleteSwitchMask will not delete the last mask.

PARAMETERS

rec
the fltSwitch node
maskNo
mask index to be deleted, 0 is the first mask

RETURNS

Returns the new current mask index if successful, -1 otherwise. This function will fail if maskNo does not specify a valid mask of rec.

ACCESS LEVEL

Level 2
