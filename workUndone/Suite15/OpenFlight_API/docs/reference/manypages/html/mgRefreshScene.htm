mgRefreshScene

FUNCTION NAME

mgRefreshScene - forces a database to be redrawn.

SYNOPSIS       (C and Python)

mgstatus mgRefreshScene (mgeditorcontexteditorContext );

DESCRIPTION

mgRefreshScene forces the database associated with the specified editor context, editorContext to be redrawn in Creator.

While an editor tool instance is active, it may perform actions within a database node that change the appearance of the database contents in the graphics or hierarchy view or both. For example, the tool may create new geometry or modify a visual attribute of an existing geometry node. When such an action is performed, mgRefreshScene can be called to make sure the graphic and hierarchy views of the corresponsing database are redrawn.

PARAMETERS

editorContext
the editor context used to identify the database that is to be redrawn

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
