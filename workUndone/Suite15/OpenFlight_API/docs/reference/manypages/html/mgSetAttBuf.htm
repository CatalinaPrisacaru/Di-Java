mgSetAttBuf

FUNCTION NAME

mgSetAttBuf - sets the values of consecutive attributes using a data buffer.

SYNOPSIS       (C and Python)

mgbool mgSetAttBuf (mgrec*rec,
mgcodefcode,
void*buf );

DESCRIPTION

Given a record, rec, mgSetAttBuf sets the values of the attribute fcode from from the data buffer buf.

This routine is used for setting all the attribute values of a record, with the condition that the attributes are consecutive as defined by the data dictionary. Note: Simple records such as fltCoord3d and fltMatrix are guaranteed to be stored consecutively; more complex records are not. Exercise caution when using this routine.

PARAMETERS

rec
the record in which to set values
fcode
the attribute code to be set from buf
buf
the data buffer from which to assign attributes

RETURNS

Returns MG_TRUE if the attribute values are assigned, MG_FALSE otherwise

C EXAMPLE

The following example sets the attribute record fltCoord3d in vtx.
mgrec* vtx;
double icoord1 = {1.0, 0.0, 0.0};

mgSetAttBuf (vtx, fltCoord3d, icoord1);

ACCESS LEVEL

Level 2
