mgListIsItemAtPosSelected

FUNCTION NAME

mgListIsItemAtPosSelected - determines if an item at a specified position in a list control is selected.

C SPECIFICATION       (Not Available For Python)

mgbool mgListIsItemAtPosSelected (mgguicontrol,
intitemPos );

DESCRIPTION

mgListIsItemAtPosSelected checks if the item at a specified position itemPos is selected in the specified list control.

Position 1 indicates the first item in the control.

PARAMETERS

control
the list control
itemPos
the position of the item to check

RETURNS

Returns MG_TRUE if item at specified position in list is selected. MG_FALSE otherwise.

ACCESS LEVEL

Level 4
