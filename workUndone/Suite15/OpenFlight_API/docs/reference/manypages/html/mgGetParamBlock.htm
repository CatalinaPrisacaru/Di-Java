mgGetParamBlock

FUNCTION NAME

mgGetParamBlock - gets a parameter block for a tool.

SYNOPSIS       (C and Python)

mgparamblock mgGetParamBlock (const char*commandName );

DESCRIPTION

mgGetParamBlock creates a new parameter block initialized with the values for the Creator command identified by commandName. The parameter block returned is typically used to pass to mgExecute to execute a Creator command.

See the example code in mgExecute.

C NOTES

The parameter block is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFreeParamBlock.

PYTHON NOTES

It is not necessary to free the returned parameter block in Python.

PARAMETERS

commandName
the Creator command for the parameter block returned

RETURNS

Pointer to parameter block if created successfully, MG_NULL otherwise.

ACCESS LEVEL

Level 4
