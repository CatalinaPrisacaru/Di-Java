mgReadLightPointFile

FUNCTION NAME

mgReadLightPointFile - loads a database's light point appearance and animation palettes from a file.

SYNOPSIS       (C and Python)

mgbool mgReadLightPointFile (mgrec*db,
const char*fileName );

DESCRIPTION

mgReadLightPointFile loads the light point appearance and animation palettes for database db from a file named fileName. The palette entries for db are replaced by those read from fileName.

The file fileName was created by mgWriteLightPointFile, or via the Save Palette command in the light point palette in Creator.

Existing light point palette records become invalid.

Note that even though the light point appearance and animation palettes are really separate, this function, as well as mgWriteLightPointFile groups the entries in both into one single file.

PARAMETERS

db
the database
fileName
the light point file name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
