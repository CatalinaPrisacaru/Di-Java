mgtabruleroption

ENUM NAME

mgtabruleroption - tab ruler control Options

SYNOPSIS

typedef enum mgtabruleroption {
   MTRUL_ORIENTATION,
   MTRUL_TABBAROFFSET,
   MTRUL_LABELOFFSET,
   MTRUL_SCALEBAROFFSET,
   MTRUL_MARGINWIDTH,
   MTRUL_TICHEIGHT,
   MTRUL_AUTOTICS,
   MTRUL_LABELEDTICSEVERY,
   MTRUL_TICSEVERY,
   MTRUL_SCALEMIN,
   MTRUL_SCALEMAX,
   MTRUL_ELASTICMIN,
   MTRUL_ELASTICMAX,
} mgtabruleroption;

MEMBERS

MTRUL_ORIENTATION
Tab Orientation, either MTRUL_HORIZONTAL or MTRUL_VERTICAL. The Default is horizontal.
MTRUL_TABBAROFFSET
Tab Bar Offset in pixels The Default is 0.
MTRUL_LABELOFFSET
Tab Label Offset in pixels The Default is 0.
MTRUL_SCALEBAROFFSET
Scale Bar Offset in pixels The Default is 0.
MTRUL_MARGINWIDTH
Margin Width in pixels The Default is 0.
MTRUL_TICHEIGHT
Tic Height specifies the height in pixels of the largest tic-marks on the tab ruler control. The tab ruler automatically decrements for finer level tics. The default is 10.
MTRUL_AUTOTICS
Auto Calculate Tics uses an algorithm to automatically generate tic marks for the tab ruler control. This parameter cannot be used in conjunction with MTRUL_TICKSEVERY
MTRUL_LABELEDTICSEVERY
Labeled Tics Every specifies the labeled tic frequency for the tab ruler control. Setting this value turns off MTRUL_AUTOTICS
MTRUL_TICSEVERY
Tics Every specifies the unlabeled tic frequency for the tab ruler control. Setting this value turns off MTRUL_AUTOTICS
MTRUL_SCALEMIN
Scale Min specifies the minimum value of the scale extents of the tab ruler control. The default is 0.
MTRUL_SCALEMAX
Scale Max specifies the maximum value of the scale extents of the tab ruler control. The default is 100.
MTRUL_ELASTICMIN
Elastic Minimum specifies whether the scale extents of the tab ruler control should grow when a tab is dragged outside the current scale minimum. Default is to assign the same value as MTRUL_SCALEMIN.
MTRUL_ELASTICMAX
Elastic Maximum specifies whether the scale extents of the tab ruler control should grow when a tab is dragged outside the current scale maximum. Default is to assign the same values as MTRUL_SCALEMAX.

DESCRIPTION

This type is used to set the options for a new tab ruler control created using the mgNewTabRuler function.
