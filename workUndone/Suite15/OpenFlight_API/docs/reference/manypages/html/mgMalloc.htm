mgMalloc

FUNCTION NAME

mgMalloc - allocates a dynamic memory buffer.

C SPECIFICATION       (Not Available For Python)

void* mgMalloc (mgsize_tnumBytes );

DESCRIPTION

Given the number of bytes to allocate, numBytes, mgMalloc returns a pointer to a memory buffer of that size. All of the bytes of the buffer are preset to zero. The data type of the returned buffer is a void pointer. The calling function should use a cast statement to convert the buffer to the desired data type. When the buffer is no longer needed, use the function mgFree to deallocate it.

PARAMETERS

numBytes
the number of bytes to allocate

RETURNS

Returns the pointer to the allocated and zeroed buffer.

ACCESS LEVEL

Level 1
