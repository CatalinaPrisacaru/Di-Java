ddGetPrefix

FUNCTION NAME

ddGetPrefix - gets the ID prefix of a record.

SYNOPSIS       (C and Python)

const char* ddGetPrefix (mgcodercode );

DESCRIPTION

ddGetPrefix returns the ID prefix associated with a record identified by rcode. This is the prefix that was specified when the data extension was registered via mgRegisterSite.

Note: Do not free or modify the returned string.

PARAMETERS

rcode
the record code

RETURNS

Returns the prefix string if defined, MG_NULL otherwise.

ACCESS LEVEL

Level 1
