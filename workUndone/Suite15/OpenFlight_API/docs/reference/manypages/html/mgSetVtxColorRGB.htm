mgSetVtxColorRGB

FUNCTION NAME

mgSetVtxColorRGB - sets the RGB color values in a fltVertex record.

SYNOPSIS       (C and Python)

mgbool mgSetVtxColorRGB (mgrec*vtx,
shortred,
shortgreen,
shortblue );

DESCRIPTION

Given a fltVertex record, vtx, mgSetVtxColorRGB sets the red, green, and blue values of its fltVColor attribute to the given red, green, and blue.

Note that if the database containing vtx is color index mode, this function will convert the given color values to the closest index and intensity pair and set the fltVColor and fltVIntensity attributes of vtx accordingly.

PARAMETERS

vtx
the fltVertex record on which to set the color values
red
the red value to set
green
the green value to set
blue
the blue value to set

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
