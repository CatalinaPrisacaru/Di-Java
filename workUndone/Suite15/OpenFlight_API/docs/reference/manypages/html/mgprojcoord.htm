mgprojcoord

STRUCT NAME

mgprojcoord - A projection coordinate expressed in latitude, longitude and height

SYNOPSIS

typedef struct mgprojcoord {
   double lat;
   double lon;
   double height;
} mgprojcoord;

MEMBERS

lat
double - the latitude in degrees, positive is north
lon
double - the longitude in degrees, positive is east
height
double - the elevation above mean sea level in meters
