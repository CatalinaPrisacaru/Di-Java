ddGetKey

FUNCTION NAME

ddGetKey - gets the value defined for keyword KEY for a record or data code.

SYNOPSIS       (C and Python)

int ddGetKey (mgcodecode );

DESCRIPTION

ddGetKey returns the value defined for keyword KEY for the specified record or data code, code.

PARAMETERS

code
the record or data code

RETURNS

Returns the keyword value if defined, 0 otherwise.

ACCESS LEVEL

Level 1
