mgScaleSetSpinIncrement

FUNCTION NAME

mgScaleSetSpinIncrement - sets the spin increment for a scale control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgScaleSetSpinIncrement (mgguicontrol,
doubleincrement );

DESCRIPTION

mgScaleSetSpinIncrement sets the spin increment value for the specified scale control to the specified value, increment.

The spin increment value is the value that a scale control is incremented or decremented when the user clicks the up/down arrow buttons of the spin buddy control associated to the scale control. The default value for this spin increment is 1.0.

If increment is positive, the up arrow of the spin control will increase the value of the scale control and the down arrow will decrease it. If increment is negative, the up arrow will decrease the scale's value and the down arrow will increase it.

PARAMETERS

control
the scale control
increment
the increment value for the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
