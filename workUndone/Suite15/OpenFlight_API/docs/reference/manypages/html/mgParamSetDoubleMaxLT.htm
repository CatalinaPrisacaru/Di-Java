mgParamSetDoubleMaxLT

FUNCTION NAME

mgParamSetDoubleMaxLT - defines the maximum value (exclusive) for a double precision floating point parameter in a parameter block.

C SPECIFICATION       (Not Available For Python)

mgbool mgParamSetDoubleMaxLT (mgparamparam,
doublemaxValue );

DESCRIPTION

mgParamSetDoubleMaxLT defines the maximum value maxValue for the double precision floating point parameter param such that the parameter must be less than maxValue.

See the example code in mgParamSetDoubleMinGT to see how to set up valid ranges for double precision floating point parameters.

PARAMETERS

param
the parameter to set max value for
maxValue
the maximum value for paramName

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
