mgProject

FUNCTION NAME

mgProject - projects projection coordinates to database coordinates.

SYNOPSIS       (C and Python)

mgcoord3d mgProject (const mgprojcoord*coord,
mgrec*db );

DESCRIPTION

mgProject projects projection coordinates to database coordinates. This function performs the same function as mgProjectionConvertLLEtoXYZ. If you need to convert multiple coordinates consecutively, it is more efficient to use mgProjectionConvertLLEtoXYZ.

PARAMETERS

coord
projection coordinates to project
db
database containing the projection to use

RETURNS

Returns the database coordinates represented by coord.

ACCESS LEVEL

Level 1
