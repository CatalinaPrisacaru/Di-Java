mgReadTexturePalette

FUNCTION NAME

mgReadTexturePalette - loads a database's texture palette from a file.

SYNOPSIS       (C and Python)

int mgReadTexturePalette (mgrec*db,
const char*fileName );

DESCRIPTION

mgReadTexturePalette loads the texture palette for database db from a file named fileName. The palette entries for db are replaced by those read from fileName.

The file fileName was created by mgWriteTexturePalette, or via the Save Palette command in the texture palette in Creator.

PARAMETERS

db
the database
fileName
the name of the texture palette file to read

RETURNS

Returns the number of textures loaded if successful, -1 otherwise.

ACCESS LEVEL

Level 1
