mgShaderGetFragmentProgramNth

FUNCTION NAME

mgShaderGetFragmentProgramNth - gets the file name for a fragment program on a shader.

SYNOPSIS       (C and Python)

char* mgShaderGetFragmentProgramNth (mgrec*shader,
intnth );

DESCRIPTION

mgShaderGetFragmentProgramNth gets the program file name for the nth fragment program on the specified shader. The first fragment program is specified by nth equal to 0.

If shader is a GLSL shader and nth does not refer to an existing fragment progrom on the GLSL shader, this function will fail. If shader is a Cg shader and nth is 0, this function will return "the" fragment program on the Cg shader. If shader is a Cg shader and nth is not 0 or the Cg shader does not have a fragment program, this function will fail.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

shader
the shader
nth
the index of the fragment program file to get

RETURNS

Returns the name of the nth fragment program if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
