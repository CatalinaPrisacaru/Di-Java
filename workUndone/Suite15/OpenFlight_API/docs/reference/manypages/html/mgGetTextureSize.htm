mgGetTextureSize

FUNCTION NAME

mgGetTextureSize - gets the size of a texture palette entry.

SYNOPSIS       (C and Python)

int mgGetTextureSize (mgrec*db,
intindex );

DESCRIPTION

mgGetTextureSize returns the size (in bytes) of the texture palette entry at the specified index contained in the texture palette of database db.

The position of a texture palette entry is the lower left corner where it is displayed in the texture palette. This position is measured in pixels.

PARAMETERS

db
the database
index
the index of the texture palette entry

RETURNS

Returns the size of the texture palette entry (in bytes) if found, -1 otherwise.

ACCESS LEVEL

Level 1
