mgMeshGetVtxColorAlpha

FUNCTION NAME

mgMeshGetVtxColorAlpha - retrieves the Alpha color value for a vertex in a mesh.

C SPECIFICATION

mgbool mgMeshGetVtxColorAlpha (mgrec*rec,
intvtxIndex,
short*alpha );

PYTHON SPECIFICATION

mgbool, alpha mgMeshGetVtxColorAlpha (mgrec*rec,
intvtxIndex )

DESCRIPTION

This function retrieves the alpha alpha color value of the vertex at index vtxIndex in the vertex pool of mesh node rec.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
alpha
address of value to receive alpha component

RETURNS

mgbool
MG_TRUE if the vertex data was retrieved successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertices in the mesh do not color values.
alpha
If function is successful, alpha contains the retrieved color alpha, otherwise alpha is undefined.

ACCESS LEVEL

Level 1
