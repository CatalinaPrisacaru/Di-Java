mgNewMatrixStack

FUNCTION NAME

mgNewMatrixStack - creates and returns a matrix stack object.

SYNOPSIS       (C and Python)

mgmatrixstack mgNewMatrixStack ( void )

DESCRIPTION

mgNewMatrixStack creates and returns a matrix stack object which can be used to perform matrix stack operations.

C NOTES

When you are done with the matrix stack, dispose of it using mgFreeMatrixStack.

PYTHON NOTES

It is not necessary to free the returned matrix stack in Python.

RETURNS

Returns a matrix stack object (initialized with the identity matrix) if creation is successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
