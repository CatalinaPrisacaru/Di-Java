mgeditorbuttonfunc

CALLBACK NAME

mgeditorbuttonfunc - Editor dialog button function

SYNOPSIS

void (*mgeditorbuttonfunc) ( mgeditorcontext editorContext,
int whichButton,
void* editorToolData );

DESCRIPTION

This is the signature for editor dialog button functions. If your editor tool displays a dialog, you can assign a button function to your editor tool of this form.

Editor dialog button functions are called to notify your editor tool when the user has pressed one of the pre-defined buttons on your dialog.

PARAMETERS

editorContext
the editor context that identifies the editor instance receiving the button event
whichButton
the button that was pressed
MBT_DONE - Done button
MBT_CANCEL - Cancel button
MBT_NEXT - Next button
MBT_UNDO - Undo button
MBT_REDO - Redo button
MBT_HELP - Help button (reserved for future enhancement)
editorToolData
user defined tool instance data set in toolData field of callData parameter (mgeditorcallbackrec) during editor tool start function mgtoolstartfunc
