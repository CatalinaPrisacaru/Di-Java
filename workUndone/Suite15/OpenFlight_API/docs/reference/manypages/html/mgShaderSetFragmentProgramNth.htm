mgShaderSetFragmentProgramNth

FUNCTION NAME

mgShaderSetFragmentProgramNth - sets the file name for an existing fragment program on a GLSL shader.

SYNOPSIS       (C and Python)

mgbool mgShaderSetFragmentProgramNth (mgrec*shader,
intnth,
const char*programFileName );

DESCRIPTION

mgShaderSetFragmentProgramNth sets the file name for the nth fragment program on the specified GLSL shader. The first fragment program is specified by nth equal to 0.

Note: If shader is not a GLSL shader or nth does not refer to an existing fragment progrom on the GLSL shader, this function will fail.

PARAMETERS

shader
the GLSL shader
nth
the index of the GLSL fragment program file to set
programFileName
the GLSL shader fragment program file

RETURNS

Returns MG_TRUE if shader is a GLSL shader and the nth fragment program was set successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
