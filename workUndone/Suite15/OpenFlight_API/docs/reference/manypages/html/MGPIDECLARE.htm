MGPIDECLARE

MACRO NAME

MGPIDECLARE - exports a symbol from a plug-in module.

SYNOPSIS

MGPIDECLARE ( type );

PARAMETERS

type
the return type (for function symbols) or object type (for variables) you are exporting.

DESCRIPTION

When you declare symbols (functions or variables) in the source code for your plug-in module that need to be exported from your plug-in module library, you should use this macro to do so. Using this macro ensures that your code is portable between the different platforms supported by the OpenFlight API.

C EXAMPLE

The following shows how you would use this macro to export your plug-in initialization function, mgpInit in a portable fashion.
MGPIDECLARE(mgbool) mgpInit (mgplugin plugin, int* argc, char* argv [])
{
   // your plug-in initialization processing goes here
}
