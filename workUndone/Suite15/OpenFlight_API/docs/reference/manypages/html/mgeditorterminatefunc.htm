mgeditorterminatefunc

CALLBACK NAME

mgeditorterminatefunc - Editor termination function

SYNOPSIS

void (*mgeditorterminatefunc) ( mgeditorcontext editorContext,
mgtoolterminationreason reason,
void* editorToolData );

DESCRIPTION

This is the signature for editor termination callback functions. If your editor tool displays a dialog, you can assign a termination function to your editor tool of this form.

Termination functions are called to notify your editor tool when it has been terminated. Refer to mgtoolterminationreason for a list of reasons an editor tool can be terminated.

PARAMETERS

editorContext
the editor context that identifies the editor instance being terminated
reason
the reason editor is being terminated
MTRM_DONE
MTRM_CANCEL
MTRM_SELF
MTRM_SYSTEM
editorToolData
user defined tool instance data set in toolData field of callData parameter (mgeditorcallbackrec) during editor tool start function mgtoolstartfunc
