mgGLSetMouseFunc

FUNCTION NAME

mgGLSetMouseFunc - sets mouse function for GL control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGLSetMouseFunc (mgguicontrol,
unsigned intactionMask,
mgglmousefuncmouseFunc,
void*userData );

DESCRIPTION

mgGLSetMouseFunc assigns a GL mouse function to the specified GL control.

You select which mouse events the callback function is to be called for using the parameter actionMask. The value for this parameter may be any bitwise combination of MGMA_BUTTON, MGMA_DOUBLECLICK, MGMA_MOTION and MGMA_WHEEL.

GL mouse functions are called to notify your GL control that the user has entered some form of mouse input into the GL control.

PARAMETERS

control
the GL control
actionMask
mask indicating which mouse actions are selected for mouse function
mouseFunc
the GL mouse function
userData
user data to be passed to mouse function when it is called

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
