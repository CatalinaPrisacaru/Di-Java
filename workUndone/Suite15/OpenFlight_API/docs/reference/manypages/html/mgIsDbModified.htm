mgIsDbModified

FUNCTION NAME

mgIsDbModified - checks if a database has been modified since the last "save" operation.

SYNOPSIS       (C and Python)

mgbool mgIsDbModified (mgrec*db );

DESCRIPTION

mgIsDbModified determines whether the database db has been modified since it was last written out to disk.

Note: This function is only applicable in the Creator modeler environment. If called from the stand application environment, the value returned is not reliable.

PARAMETERS

db
the database to check

RETURNS

Returns MG_TRUE if the specified database has been modified since it was last saved, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
