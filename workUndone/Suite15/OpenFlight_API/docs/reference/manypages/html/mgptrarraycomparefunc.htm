mgptrarraycomparefunc

CALLBACK NAME

mgptrarraycomparefunc - Pointer Array compare function

SYNOPSIS

int (*mgptrarraycomparefunc) ( const void* item1Addr,
const void* item2Addr );

DESCRIPTION

This is the signature for pointer array compare functions. When you call mgPtrArraySort, you provide a function of this type to perform the comparisons on the items in your array.

This function compares two array elements and returns a value specifying their relationship.

PARAMETERS

item1Addr
the address of the first element to compare
item2Addr
the address of the second element to compare

RETURNS

Return
< 0 - if item pointed to by item1Addrr is less than item pointed to by item2Addr
0 - if item pointed to by item1Addrr is equal to item pointed to by item2Addr
> 0 - if item pointed to by item1Addrr is greater than item pointed to by item2Addr
