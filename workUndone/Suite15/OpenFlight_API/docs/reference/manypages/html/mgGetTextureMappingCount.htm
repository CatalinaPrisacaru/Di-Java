mgGetTextureMappingCount

FUNCTION NAME

mgGetTextureMappingCount - gets the number of entries in a database’s texture mapping palette.

SYNOPSIS       (C and Python)

int mgGetTextureMappingCount (mgrec*db );

DESCRIPTION

Given a database node, db, mgGetTextureMappingCount gets the number of entries in the database’s texture mapping palette.

PARAMETERS

db
the database

RETURNS

Returns the number of entries in the database’s texture mapping palette.

ACCESS LEVEL

Level 1
