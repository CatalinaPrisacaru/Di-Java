mgFreeNewEyePoint

FUNCTION NAME

mgFreeNewEyePoint - frees an eyepoint record.

C SPECIFICATION       (Not Available For Python)

mgbool mgFreeNewEyePoint (mgrec*db,
mgrec*eyeRec );

DESCRIPTION

mgFreeEyePoint frees the memory associated with an eyepoint record previously obtained by mgGetNewEyePoint.

Do not use mgFreeNewEyePoint on an eyepoint record obtained from mgGetEyePoint.

PARAMETERS

db
the database
eyeRec
the eyepoint record to be freed

ACCESS LEVEL

Level 2
