mgListAddItem

FUNCTION NAME

mgListAddItem - adds an item to a list control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgListAddItem (mgguicontrol,
const char*itemString,
intitemPos,
mgboolselect );

DESCRIPTION

mgListAddItem adds the specified itemString to the specified list control at the specified position itemPos.

If the list control has the LBS_SORT style, this function does not cause the list to be sorted.

PARAMETERS

control
the list control
itemString
the string to add
itemPos
the position of the new item in the list - a value of 1 makes the new item the first item in the list, 2 makes it the second item and so on - a value of 0 (zero) makes the new item the last item in the list
select
enum mgbool MG_TRUE to make new item selected, MG_FALSE to make new item unselected

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
