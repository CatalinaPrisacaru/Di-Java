mgToolActionGetShortCut

FUNCTION NAME

mgToolActionGetShortCut - gets a plug-in tool action shortcut key sequence.

C SPECIFICATION       (Not Available For Python)

mgstatus mgToolActionGetShortCut (mgtoolactiontoolAction,
mgkeysequence*keySequence );

DESCRIPTION

mgToolActionGetShortCut returns the key sequence representing the shortcut assigned to the plug-in tool action toolAction in the output parameter keySequence.

PARAMETERS

toolAction
the tool action
keySequence
the key sequence that is currently set as the shortcut for this action

RETURNS

Use MSTAT_ISOK to check return value for success. If successful, the output parameter keySequence will contain the shortcut key sequence of the plug-in tool action, otherwise it is undefined.

ACCESS LEVEL

Level 4
