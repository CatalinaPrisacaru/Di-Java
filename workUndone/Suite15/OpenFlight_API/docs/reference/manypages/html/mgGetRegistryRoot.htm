mgGetRegistryRoot

FUNCTION NAME

mgGetRegistryRoot - returns the root name of the Creator registry.

C SPECIFICATION

mgbool mgGetRegistryRoot (char*rootString,
intmaxLen );

PYTHON SPECIFICATION

mgbool, string mgGetRegistryRoot ( )

DESCRIPTION

mgGetRegistryRoot returns the root path of the registry currently in use by Creator. The output string is truncated and null terminated if it is longer than maxLen characters.

This function is only applicable on the Windows operation system platform in the Creator modeler environment. This function will always return MG_FALSE when called by a stand-alone application or from any code running on IRIX.

PARAMETERS

rootString
character buffer to hold the root path
maxLen
maximum number of characters to store in rootString

RETURNS

mgbool
MG_TRUE if registry root retrieved successfully, MG_FALSE otherwise.
string
If function is successful, string contains the root path, otherwise undefined.

ACCESS LEVEL

Level 4
