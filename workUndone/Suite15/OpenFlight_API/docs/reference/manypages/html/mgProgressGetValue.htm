mgProgressGetValue

FUNCTION NAME

mgProgressGetValue - retrieves the current value of a progress control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgProgressGetValue (mgguicontrol,
double*value );

DESCRIPTION

mgProgressGetValue returns the current value of the specified progress control.

PARAMETERS

control
the progress control
value
address of double value to receive current value from control

RETURNS

Use MSTAT_ISOK to check return value for success. If function is successful, value contains the current value of the control, otherwise value is undefined.

ACCESS LEVEL

Level 4
