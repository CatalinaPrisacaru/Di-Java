mgdialogfunc

CALLBACK NAME

mgdialogfunc - Dialog function

SYNOPSIS

mgstatus (*mgdialogfunc) ( mggui dialog,
mgdialogid dialogId,
mgguicallbackreason callbackReason,
void* userData,
void* callData );

DESCRIPTION

This is the signature for dialog functions. When you create a dialog instance, you will provide a dialog function of this form that will be called to notify your plug-in tool of significant dialog events.

PARAMETERS

dialog
the dialog
dialogId
the identifier of the dialog
callbackReason
the dialog event that triggered the callback
userData
user defined data specified when dialog was created
callData
callback specific data - you can always safely cast this pointer to aa pointer to an object of type mgguicallbackrec to determine the type of call data this object really is

RETURNS

Currently, the value returned by mgdialogfunc is ignored and reserved for future enhancement. For now always return MSTAT_OK.
