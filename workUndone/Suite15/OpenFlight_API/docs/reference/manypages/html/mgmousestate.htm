mgmousestate

ENUM NAME

mgmousestate - Mouse State type

SYNOPSIS

typedef enum mgmousestate {
   MMSS_START,
   MMSS_CONTINUE,
   MMSS_STOP,
   MMSS_NONE,
} mgmousestate;

MEMBERS

MMSS_START
The mouse-start state. This indicates that the associated event was the first (down) event in a sequence of mouse down/drag/release events. The event sequence will have at most, one MMSS_START event which will always be the first event sent.
MMSS_CONTINUE
The drag-mouse state. This indicates that the associated event was a drag event in a sequence of mouse down/drag/release events. The event sequence will have 0-n MMSS_CONTINUE events, the first of which will always follow the MMSS_START event, and the last of which will always precede the MMSS_STOP event.
MMSS_STOP
The mouse-stop state. This indicates that the associated event was the last (release) event in a sequence of mouse down/drag/release events. The event sequence will have at most, one MMSS_STOP event which will always be the last event sent.
MMSS_NONE
The mouse-none state. This indicates that the mouse was not involved with the associated event.

DESCRIPTION

This enumerated type is used in the following contexts:

When reporting mouse input events to editor tools. See mgRegisterEditor, mgEditorSetVertexFunc, mgEditorSetPointFunc, mgEditorSetPickFunc for more information.

When reporting the MGCB_ACTIVATE control event to scale controls. See mgscaleactivatecallbackrec for more information.

When reporting the MGCB_ACTIVATE control event to text controls that have an associated spin buddy control. See mgtextactivatecallbackrec for more information.
