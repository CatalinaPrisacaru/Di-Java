mgSetTextureHeight

FUNCTION NAME

mgSetTextureHeight - sets height attribute for an image object.

C SPECIFICATION       (Not Available For Python)

mgbool mgSetTextureHeight (mgimageinfotextureInfo,
intheight );

DESCRIPTION

mgSetTextureHeight sets the height attribute for the specified image object textureInfo to the specified value height.

When the image get info function, mgimagegetinfofunc is called for an image importer plug-in tool, the get info function will use this function to set the height attribute for the image object.

There is no default value for this attribute, so this attribute must be set.

PARAMETERS

textureInfo
the image object
height
the height attribute value

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
