mgVectordUnitize

FUNCTION NAME

mgVectordUnitize - calculates vector coordinates scaled uniformly to have a length of 1.0.

SYNOPSIS       (C and Python)

mgvectord mgVectordUnitize (const mgvectord*vec );

DESCRIPTION

mgVectordUnitize calculates and returns double precision vector coordinates scaled uniformly to have a length of 1.0.

The original vector vec is not affected.

Note: For efficiency, vec is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

vec
vector to unitize

RETURNS

Returns a double precision vector record. If the magnitude of vec is less than 0.000000000001, this function returns (0,0,1).

ACCESS LEVEL

Level 1
