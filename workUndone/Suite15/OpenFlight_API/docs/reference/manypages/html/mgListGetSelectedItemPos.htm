mgListGetSelectedItemPos

FUNCTION NAME

mgListGetSelectedItemPos - retrieves position of selected item in a listcontrol.

C SPECIFICATION       (Not Available For Python)

mgstatus mgListGetSelectedItemPos (mgguicontrol,
int*itemPos );

DESCRIPTION

mgListGetSelectedItemPos retrieves the position of the selected item in the specified list control.

For multi-select list controls with more than one item currently selected, the position retrieved is the position of the first item selected.

Position 1 indicates the first item in the control.

PARAMETERS

control
the list control
itemPos
address of value to receive selected position

ACCESS LEVEL

Level 4

SEE ALSO

mgListGetSelectedItemString, mgListSelectItemAtPos

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
itemPos
If function is successful, itemPos contains the position of the selected item. Otherwise, itemPos is undefined.
