mgcontrolattribute

ENUM NAME

mgcontrolattribute - Control attribute name type

SYNOPSIS

typedef enum mgcontrolattribute {
   MCA_GLBORDERSTYLE,
   MCA_BOLDFONT,
   MCA_ITALICFONT,
   MCA_GLMOUSEWRAP,
   MCA_SPINAUTOINCREMENT,
   MCA_TEXTACTIVATEONCHANGED,
} mgcontrolattribute;

MEMBERS

MCA_GLBORDERSTYLE
GL Control Border Style. This control attribute specifies what kind (if any) of border to draw around the GL control.
The possible values for this attribute include:
MGLBS_NONE (default)
MGLBS_SUNKEN
MGLBS_RAISED
MGLBS_SOLID.
MCA_BOLDFONT
Control Bold Font Style. This control attribute specifies whether or not a control's font is bold.
The possible values for this attribute include:
1 - bold
0 - not bold (default)
MCA_ITALICFONT
Control Italic Font Style. This control attribute specifies whether or not a control's font is italic.
The possible values for this attribute include:
1 - italic
0 - not italic (default)
MCA_GLMOUSEWRAP
GL Control Mouse Wrap. This control attribute specifies whether or not the MGMA_BUTTON mouse action reported to a GL mouse function allows the mouse position to go outside the bounds of the physical screen display. Specifically, when the user tries to drag the mouse past the "edge" of the screen (top, bottom, left or right), this attribute controls the mouse behavior as follows:
If Mouse Wrap is not enabled, the mouse position is clamped to the bounds of the screen and the mouse is not allowed to move any further in that direction.
If Mouse Wrap is enabled, the mouse position is automatically "wrapped" to the edge of the screen that is opposite the edge past which the user is trying to drag.
For example, if the user drags the mouse past the right edge of the screen, the mouse is automatically moved to the left edge of the screen. This allows for more continuous mouse motion in the GL control.
Note: When Mouse Wrap is enabled, the coordinates contained in the mgglmousebuttondatarec structure reported to the GL mouse function will be adjusted automatically to account for this "wrapping" behavior. Again, consider the example of the user dragging the mouse past the right edge of the screen. If the X coordinate reported just before the mouse "wraps" is N, The X coordinate reported just after the mouse "wraps" will be N+1.
The possible values for this attribute include:
1 - Mouse Wrap enabled
0 - Mouse Wrap not enabled (default)
MCA_SPINAUTOINCREMENT
Spin Control Auto Increment. This control attribute specifies whether or not the increment value of a spin buddy control attached to a text or scale control is automatically calculated by Creator based on the current value of the text or scale control.
When setting this attribute you must call mgControlSetAttribute after mgSetGuiCallback and after you have assigned the spin buddy to the control.
The possible values for this attribute include:
1 - Auto Increment enabled
0 - Auto Increment not enabled (default)
MCA_TEXTACTIVATEONCHANGED
Text Activate on Changed. Normally a text control receives the MGCB_ACTIVATE event only when the control loses focus. This control attribute lets you specify that a text control is to receive this event as the user types. This may be useful in certain situations.
When setting this attribute you must call mgControlSetAttribute after mgSetGuiCallback.
The possible values for this attribute include:
1 - Text control receives MGCB_ACTIVATE event as user types and when control loses focus.
0 - Text control receives MGCB_ACTIVATE event only when control loses focus. (default)

DESCRIPTION

This type is used to enumerate the control attributes that can be set.
