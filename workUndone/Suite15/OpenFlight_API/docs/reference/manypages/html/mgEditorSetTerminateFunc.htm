mgEditorSetTerminateFunc

FUNCTION NAME

mgEditorSetTerminateFunc - sets the termination function for an editor tool.

C SPECIFICATION       (Not Available For Python)

void mgEditorSetTerminateFunc (mgplugintooleditorPluginTool,
mgeditorterminatefuncterminateFunc );

DESCRIPTION

mgEditorSetTerminateFunc assigns a termination function to the specified editor tool editorPluginTool.

Termination functions are called to notify your editor tool when it has been terminated. Refer to mgtoolterminationreason for a list of reasons an editor tool can be terminated.

PARAMETERS

editorPluginTool
the editor plug-in tool
terminateFunc
the editor termination function

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following is a typical dialog button function.
mgEditorSetButtonFunc (pluginTool, ButtonFunc);

static void TerminateFunc (mgeditorcontext editorContext,
  mgtoolterminationreason terminateReason,
  void* toolData)
{
   toolrec* toolRec = (toolrec*) toolData;

   switch (terminateReason)
   {
   case MTRM_DONE:
      break;
   case MTRM_SYSTEM:
      break;
   case MTRM_CANCEL:
      break;
   case MTRM_SELF:
      break;
   }
}
