mgSendWarning

FUNCTION NAME

mgSendWarning - reports an error message.

C SPECIFICATION       (Not Available For Python)

mgbool mgSendWarning (mgplugintoolpluginTool,
const char*message,
... );

DESCRIPTION

mgSendWarning reports a warning message containing the text message. This corresponds to message severity level MMSG_WARNING. The warning message is identified with the name of the plug-in tool that sent the message.

If invoked from within the stand-alone program environment, the reported message is directed to stdout. If invoked from the modeler environment, the reported message is directed to the Status Bar/Status Log Window.

The message string can contain printf style formatting, in which case the appropriate number of additional parameters must be passed.

PARAMETERS

pluginTool
the plug-in tool sending the message
message
message string, which may contain formatting, and should be followed with the appropriate parameters, if any
...
optional additional parameters to support formatted output

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
