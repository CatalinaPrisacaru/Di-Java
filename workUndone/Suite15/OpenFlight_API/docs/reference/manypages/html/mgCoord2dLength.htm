mgCoord2dLength

FUNCTION NAME

mgCoord2dLength - calculates the length of 2D coordinates.

SYNOPSIS       (C and Python)

double mgCoord2dLength (const mgcoord2d*coord );

DESCRIPTION

mgCoord2dLength calculates and returns the length of 2D double precision floating point coordinates.

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

coord
address of coordinates

RETURNS

Returns a double precision floating point value.

ACCESS LEVEL

Level 1
