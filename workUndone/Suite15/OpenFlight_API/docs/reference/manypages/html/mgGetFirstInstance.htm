mgGetFirstInstance

FUNCTION NAME

mgGetFirstInstance - gets the first instance node of a reference node.

SYNOPSIS       (C and Python)

mgrec* mgGetFirstInstance (mgrec*ref );

DESCRIPTION

Instancing is a technique in which one node is a child of several parent nodes. The child is said to be a reference node and each parent node is an instance node of the child. A parent node can have either a regular child or a reference child, but not both.

mgGetFirstInstance returns the first instance node of the specified reference node ref.

PARAMETERS

ref
the reference node to get first instance for

RETURNS

Returns the first instance node of the reference node, if found. If ref is not a reference node, MG_NULL is returned.

C EXAMPLE

mgrec* ref;
mgrec* inst;
inst = mgGetFirstInstance (ref);
while (inst) {
   // Do something with inst
   inst = mgGetNextInstance (inst);
}

ACCESS LEVEL

Level 1
