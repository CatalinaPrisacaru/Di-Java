mgGetCoord2i

FUNCTION NAME

mgGetCoord2i - gets values out of a fltCoord2i record.

C SPECIFICATION

mgbool mgGetCoord2i (mgrec*rec,
mgcodecode,
int*x,
int*y );

PYTHON SPECIFICATION

mgbool, x, y mgGetCoord2i (mgrec*rec,
mgcodecode )

DESCRIPTION

Given a record, rec, mgGetCoord2i gets the values of fltCoord2i attribute record code from rec and stores them in x and y.

PARAMETERS

rec
points to the record from which to get values
code
the code of the point record to get
x
address of value to receive x coordinate
y
address of value to receive y coordinate

RETURNS

mgbool
MG_TRUE if the coord was retrieved successfully, MG_FALSE otherwise.
x
If function is successful, x contains the retrieved x component, otherwise x is undefined.
y
If function is successful, y contains the retrieved y component, otherwise y is undefined.

C EXAMPLE

mgrec* db;
mgrec* imgRec;
int x, y;

db = mgOpenDb ("file.flt");

// Get image rec

mgGetCoord2i (imgRec, fltImgUp, &x, &y);

ACCESS LEVEL

Level 1
