MTA_PALETTELOCATION

NAME

MTA_PALETTELOCATION - Tool Attribute Palette Location

DESCRIPTION

Specify this tool attribute if you want your viewer or editor tool to be launched from a tool palette.

This string attribute specifies the tool palette where the corresponding palette icon MTA_PALETTEICON is placed for a tool.

You must specify both MTA_PALETTELOCATION and MTA_PALETTEICON if you want your tool to be launched from a tool palette.

Possible values for this tool attribute are:
MPAL_FACETOOLS - Face Tools palette
MPAL_GEOMETRYTOOLS - Geometry Tools palette
MPAL_MANEUVERTOOLS - Maneuver Tools palette
MPAL_DUPLICATE - Duplicate palette
MPAL_DEFORMTOOLS - Deform Tools palette
MPAL_BOOLEANTOOLS - Boolean Tools palette
MPAL_MODIFYGEOMETRY - Modify Geometry palette
MPAL_MODIFYFACE - Modify Face palette
MPAL_MODIFYVERTEX - Modify Vertex palette
MPAL_PROPERTIES - Properties palette
MPAL_MAPTEXTURE - Map Texture palette
MPAL_MODIFYTEXTURE - Modify Texture palette
MPAL_CONSTRUCTIONTOOLS - Construction Tools palette
MPAL_CREATETOOLS - Create Tools palette
MPAL_HIERARCHYTOOLS - Hierarchy Tools palette
