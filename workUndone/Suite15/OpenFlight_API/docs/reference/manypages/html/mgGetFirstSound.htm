mgGetFirstSound

FUNCTION NAME

mgGetFirstSound - gets the first sound entry from a database’s sound palette.

C SPECIFICATION

mgrec* mgGetFirstSound (mgrec*db,
int*index );

PYTHON SPECIFICATION

mgrec, index mgGetFirstSound (mgrec*db )

DESCRIPTION

Given a database node db, mgGetFirstSound gets the database’s first sound entry record. The index of the sound record in the palette is returned in index, if there is one.

The code of the record returned will be fltSoundPalette .

PARAMETERS

db
the database
index
address of value to receive palette index of entry returned, -1 if none

RETURNS

mgrec
The first sound palette entry in the database’s sound palette, MG_NULL if no sounds in palette.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

C EXAMPLE

mgrec* sound;
mgrec* next;
mgrec* db;
int index;
db = mgNewDb ("newfile.flt");
sound = mgGetFirstSound (db, index);
next = mgGetNextSound (sound, index);

ACCESS LEVEL

Level 1
