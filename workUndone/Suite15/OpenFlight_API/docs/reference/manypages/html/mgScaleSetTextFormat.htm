mgScaleSetTextFormat

FUNCTION NAME

mgScaleSetTextFormat - sets the text format string for a scale control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgScaleSetTextFormat (mgguicontrol,
const char*fmtString );

DESCRIPTION

mgScaleSetTextFormat sets the format string for the text buddy of a scale control. The format string, fmtString, is assumed to be a printf style format string and is used to automatically format the numeric value that is displayed in the text buddy control associated to the scale control.

Since the value represented by a scale control is of type double, you may, for example, use this format string to effectively display integer values in the scale control by using a format string similar to "%.0lf". See printf for more details on allowable format strings.

The default format string for a text buddy control is "%lf".

PARAMETERS

control
the scale control
fmtString
the format string for the text buddy

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
