mgSetSwitchBit

FUNCTION NAME

mgSetSwitchBit - sets the on/off status of a specific bit within a switch mask.

SYNOPSIS       (C and Python)

mgbool mgSetSwitchBit (mgrec*rec,
intmaskNo,
intbitNo,
mgboolon );

DESCRIPTION

Given a fltSwitch node, rec, mgSetSwitchBit sets the value of the bit specified by bitNo within the mask maskNo on or off based on the value of on.

The value of bitNo must be between 0 and the number of children of rec minus 1 (the first child is bitNo 0). Similarly, the value of maskNo must be between 0 and the number of masks of rec minus 1 (the first mask is maskNo 0).

PARAMETERS

rec
the fltSwitch node
maskNo
mask index starting from 0
bitNo
bit number starting from 0
on
value to set bits to

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. This function will fail if maskNo does not specify a valid mask of rec.

ACCESS LEVEL

Level 2
