mgExtensionFieldDelete

FUNCTION NAME

mgExtensionFieldDelete - deletes an extension field from a node.

SYNOPSIS       (C and Python)

mgbool mgExtensionFieldDelete (mgrec*rec,
const char*fieldGuid );

DESCRIPTION

mgExtensionFieldDelete deletes the extension field identified by fieldGuid from node rec.

PARAMETERS

rec
the node to from which to delete the extension field
fieldGuid
the GUID of the extension field

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
