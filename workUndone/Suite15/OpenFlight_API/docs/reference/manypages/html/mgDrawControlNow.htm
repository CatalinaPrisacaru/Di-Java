mgDrawControlNow

FUNCTION NAME

mgDrawControlNow - draws a GL control immediately.

C SPECIFICATION       (Not Available For Python)

mgstatus mgDrawControlNow (mgguicontrol );

DESCRIPTION

mgDrawControlNow draws the specified GL control control immediately.

If the draw control event has been selected for the control callback of the specified GL control, the draw event is sent to the control.

PARAMETERS

control
the GL control to draw

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4

SEE ALSO

mgDrawControl, mgSetGuiCallback, MGCB_DRAW
