mgEvaluateAsBool

FUNCTION NAME

mgEvaluateAsBool - provides a walkfunc that wraps an expression tree.

SYNOPSIS       (C and Python)

mgbool mgEvaluateAsBool (mgrec*db,
mgrec*parent,
mgrec*rec,
void*userData );

DESCRIPTION

mgEvaluateAsBool provides a walkfunc that wraps an expression tree. This walkfunc is to be used in mgFind and mgFindEx calls.

PARAMETERS

db
the database containing the node being visited
parent
the parent of the node being visited (MG_NULL if rec is db)
rec
the node currently being visited
userData
user defined data specified in call to mgFind, mgFindEx

C EXAMPLE

The following example collects all polygons of material index 2 in a database.
mgreclist list;

mgnodeexpression poly = mgNewNodeExpressionOperator(MEOP_EQ);
mgnodeexpression divide = mgNewNodeExpressionOperator(MEOP_DIV);
mgNodeExpressionSetLeft(poly, mgNewNodeExpressionGetAtt(fltPolyMaterial));
mgNodeExpressionSetLeft(divide, mgNewNodeExpressionInteger(4));
mgNodeExpressionSetRight(divide, mgNewNodeExpressionInteger(2));
mgNodeExpressionSetRight(poly, divide);
list = mgFind(toolRec->db, mgEvaluateAsBool, poly, MWALK_ON);

// do something with list, free it when done

mgFreeRecList (list);

Level 1
