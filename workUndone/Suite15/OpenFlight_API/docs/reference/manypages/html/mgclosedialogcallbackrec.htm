mgclosedialogcallbackrec

STRUCT NAME

mgclosedialogcallbackrec - callback structure for editor close dialog functions

SYNOPSIS

typedef struct  {
   int buttonEvent;
} mgclosedialogcallbackrec;

MEMBERS

buttonEvent
int - if your editor close dialog function returns MG_TRUE, this is the button event you want sent to the dialog button function for your editor tool

DESCRIPTION

If you have assigned a close dialog function for you editor tool and the user clicks the Close Box, the close dialog function is called. The close dialog function is passed a pointer to a record of this type in the callData parameter.

This record contains a field that your tool fills in and returns if a dialog is required by your tool. If your editor tool needs to display a dialog to continue its work, it must set the dialogRequired field to MG_TRUE. Additionally, you can fill in the toolData field with tool defined data that is passed to subsequent editor interactions while your tool is active on the desktop.

If you do not need to display a dialog, set the dialogRequired field to MG_FALSE.
