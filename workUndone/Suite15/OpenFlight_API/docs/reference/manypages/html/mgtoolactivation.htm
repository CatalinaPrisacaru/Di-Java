mgtoolactivation

TYPE NAME

mgtoolactivation - Abstract type used to represent a tool activation object.

DESCRIPTION

When the user launches a plug-in tool you have registered, an object of this type is created and passed to your tool's start function. You can query this object to obtain useful information about the context in which your tool is running.
