mgGetAttRec

FUNCTION NAME

mgGetAttRec - gets pointer to one of a record's attribute records.

SYNOPSIS       (C and Python)

mgrec* mgGetAttRec (mgrec*rec,
mgcodercode,
mgrec*recOut );

DESCRIPTION

mgGetAttRec returns a pointer to the attribute record identified by rcode from rec. Use this function to retrieve an attribute record as opposed to an attribute value. Use mgGetAttList to retrieve an attribute value.

If you specify MG_NULL for recOut, a new record will be allocated and returned. If you specify a valid mgrec* for recOut, the record retrieved will be copied into recOut. In this way, you can reuse records.

Note: Any recOut that you pass in must have been returned from a previous call to mgGetAttRec or must be MG_NULL. This ensures that the contents of other records do not inadvertently get overwritten.

PARAMETERS

rec
points to the record from which to get the attribute record
rcode
the attribute record code
recOut
points to the attribute record

RETURNS

Returns the attribute record pointer if found, MG_NULL otherwise.

C EXAMPLE

The following example gets the attribute record pointer of fltBoundingBox in groupRec through attRec.
mgrec* groupRec;
mgrec* attRec;
int boundType;

attRec = mgGetAttRec (groupRec, fltBoundingBox, MG_NULL);
mgGetAttList (attRec, fltBoundingType, &boundType, MG_NULL);

ACCESS LEVEL

Level 1
