mglistactivatecallbackrec

STRUCT NAME

mglistactivatecallbackrec - callback structure for the activate control event sent to list controls

SYNOPSIS

typedef struct  {
   mgguicalldatatype callDataType;
   mglistactivatereason activateReason;
} mglistactivatecallbackrec;

MEMBERS

callDataType
mgguicalldatatype - call data type that indicates the type of call data this object really is - will always be MGCD_LISTACTIVATE
activateReason
mglistactivatereason - reason the activate control event was sent to the list control - possible values are:
MLIST_SELECT - The user selected an item in the list.
MLIST_DESELECT - The user deselected an item in the list.
MLIST_DOUBLECLICK - The user double clicked an item in the list.

DESCRIPTION

When the MGCB_ACTIVATE control event is sent to a list control via its mgguifunc, the calldata parameter passed in will point to an object of this type.
