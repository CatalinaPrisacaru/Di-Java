mgVectordFromLine

FUNCTION NAME

mgVectordFromLine - makes a vector from a line record.

SYNOPSIS       (C and Python)

mgvectord mgVectordFromLine (const mglined*line );

DESCRIPTION

mgVectordFromLine returns a double precision vector record representing the vector from the first coordinate to the second coordinate of the specified line record, line.

The original line record line is not affected. The resulting vector is not unitized.

Note: For efficiency, line is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

line
line used to derive vector

RETURNS

Returns double precision vector record.

ACCESS LEVEL

Level 1
