mgIsFileOnDesktop

FUNCTION NAME

mgIsFileOnDesktop - checks if an OpenFlight file is open on the Creator desktop.

SYNOPSIS       (C and Python)

mgbool mgIsFileOnDesktop (const char*fileName );

DESCRIPTION

mgIsFileOnDesktop determines whether the OpenFlight file fileName is open on the Creator desktop.

Note: This function is only applicable in the Creator modeler environment and will always return MG_FALSE when called by a stand-alone application.

PARAMETERS

fileName
the name of the file to check

RETURNS

Returns MG_TRUE if the specified file is open on the Creator desktop, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
