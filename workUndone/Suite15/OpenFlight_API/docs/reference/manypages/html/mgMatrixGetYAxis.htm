mgMatrixGetYAxis

FUNCTION NAME

mgMatrixGetYAxis - returns the local Y axis of a matrix.

SYNOPSIS       (C and Python)

mgcoord3d mgMatrixGetYAxis (const mgmatrixmatrix );

DESCRIPTION

mgMatrixGetYAxis returns the local Y axis of matrix. The Y axis is composed of the first three elements in the 2nd row of the matrix: matrix[4], matrix[5] and matrix[6].

The original matrix matrix is not affected.

PARAMETERS

matrix
matrix to get the local Y axis of

RETURNS

Returns the local Y axis of matrix.

ACCESS LEVEL

Level 1
