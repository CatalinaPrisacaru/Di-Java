mgMeshGetVtxUV

FUNCTION NAME

mgMeshGetVtxUV - retrieves the u,v texture coordinate for a vertex in a mesh.

C SPECIFICATION

mgbool mgMeshGetVtxUV (mgrec*rec,
intvtxIndex,
intlayer,
float*u,
float*v );

PYTHON SPECIFICATION

mgbool, u, v mgMeshGetVtxUV (mgrec*rec,
intvtxIndex,
intlayer )

DESCRIPTION

This function retrieves the u and v texture coordinate values for layer layer of the vertex at index vtxIndex in the vertex pool of mesh node rec.

You do not have to provide addresses for both u and v values. For example, if you only want to receive u (but not v), specify a valid address for u and leave v NULL.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
layer
the layer number 0..7
u
address of value to receive u texture coordinate
v
address of value to receive v texture coordinate

RETURNS

mgbool
MG_TRUE if the vertex data was retrieved successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertices in the mesh do not color values.
u
If function is successful, u contains the retrieved u coordinate, otherwise u is undefined.
v
If function is successful, v contains the retrieved v coordinate, otherwise v is undefined.

ACCESS LEVEL

Level 1
