mgConvexHullGetTriangleNth

FUNCTION NAME

mgConvexHullGetTriangleNth - gets the nth triangle of a convex hull.

C SPECIFICATION

mgbool mgConvexHullGetTriangleNth (mgconvexhullhull,
intnth,
mgconvexhulltriangle*triangle );

PYTHON SPECIFICATION

mgbool, triangle mgConvexHullGetTriangleNth (mgconvexhullhull )

DESCRIPTION

mgConvexHullGetTriangleNth returns the nth triangle of the specified convex hull. The first triangle is specified by nth equal to 1.

PARAMETERS

hull
the convex hull
nth
which triangle to get
triangle
the triangle retrieved

RETURNS

mgbool
MG_TRUE if the triangle was retrieved successfully, MG_FALSE otherwise.
triangle
If function is successful, this contains the retrieved triangle, otherwise this is undefined.

EXAMPLE

See mgGetConvexHull for an example of how to build and examine a convex hull.

ACCESS LEVEL

Level 1
