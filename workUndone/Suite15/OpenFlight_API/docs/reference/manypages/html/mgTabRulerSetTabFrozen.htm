mgTabRulerSetTabFrozen

FUNCTION NAME

mgTabRulerSetTabFrozen - freezes or unfreezes a tab on a tab ruler control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTabRulerSetTabFrozen (mgguicontrol,
inttabIndex,
mgbooltabFrozen );

DESCRIPTION

mgTabRulerSetTabFrozen sets the tab tabIndex on the tab ruler control that is associated with the GL control control to frozen. A frozen tab cannot be moved.

The first tab is at index 1.

PARAMETERS

control
tab ruler control
tabIndex
index of tab to set frozen
tabFrozen
new forzen state of tab

RETURNS

Use MSTAT_ISOK to check return value for success. If the function succeeds, the tab tabIndex was set to tabFrozen.

ACCESS LEVEL

Level 4
