mgGetTextureSaveName

FUNCTION NAME

mgGetTextureSaveName - gets the save name of a texture palette entry from its index.

SYNOPSIS       (C and Python)

char* mgGetTextureSaveName (mgrec*db,
intindex );

DESCRIPTION

mgGetTextureSaveName returns the save name of a texture palette entry at the specified index contained in the texture palette of database db.

The save name of a texture is the actual name that is saved in the OpenFlight database file for this texture palette entry. The save name may or may not include the full path specification of the texture file. If the texture files for a database are saved using relative texture names, the save name of a texture will be a relative name and will begin with the character sequence "./". If the texture files are saved using absolute texture names, the save name will include the full path specification of the texture. If the texture files are saved with no path, the save name will simply be the name of the file.

You can determine how texture names are saved using the function mgTextureGetSavePathType. Similarly, you can change how texture names are saved using mgTextureSetSavePathType.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

db
the database
index
the index of the texture palette entry

RETURNS

Returns the save name of the texture palette entry if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
