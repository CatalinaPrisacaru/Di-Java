mgNewRecDb

FUNCTION NAME

mgNewRecDb - allocates a new node record for a specific database.

SYNOPSIS       (C and Python)

mgrec* mgNewRecDb (mgcodercode,
mgrec*db );

DESCRIPTION

Given a node record code code and a database node db, mgNewRecDb allocates and initializes a node record of the type defined by code. A new node is returned if the action is successful. The new node is an orphan, meaning that it is not attached to any other node in the hierarchy. Use mgAttach, mgAppend or mgInsert to attach the node into the database hierarchy.

Valid codes for this function include all node type record codes except fltHeader . Since fltHeader is the root of a database, use mgNewDb or mgOpenDb to create a fltHeader node type. You can also use this function to create transformation node records using the codes fltXmTranslate , fltXmScale , etc.

The new node created is explicitly bound to the db you specify. After you create a new node using this function you can only attach it to other nodes that are contained in db.

PARAMETERS

rcode
the code for the new record
db
the database in which you are going to attach the record

RETURNS

Returns the pointer to the newly created node record if successful, MG_NULL otherwise.

EXAMPLE

The following examples (in C and Python) create and attach a new polygon to an object in the database.

C EXAMPLE

static void MakeNewPoly (mgrec* object)
{
   mgrec* db = mgRec2Db (object);
   mgrec* poly = mgNewRecDb (fltPolygon, db);
   mgAttach (object, poly);  // poly is first child of object
}

PYTHON EXAMPLE

def MakeNewPoly (object):
   db = mgRec2Db (object)
   poly = mgNewRecDb (fltPolygon, db)
   mgAttach (object, poly)  # poly is first child of object

ACCESS LEVEL

Level 2
