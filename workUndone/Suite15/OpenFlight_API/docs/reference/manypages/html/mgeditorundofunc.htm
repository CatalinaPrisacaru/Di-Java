mgeditorundofunc

CALLBACK NAME

mgeditorundofunc - Editor undo callback function

SYNOPSIS

void (*mgeditorundofunc) ( mgrec* db,
void* undoData );

DESCRIPTION

This is the signature for editor undo callback functions. If your editor tool performs an undo-able action, register an undo callback function for your editor tool instance of this form.

Editor undo functions are called when the user selects the corresponding Undo "ToolName" menu item from the Edit->Undo menu. This function must "undo" any actions performed by the corresponding editor tool instance.

PARAMETERS

db
the database for which the undo/redo callback is being called
undoData
user defined undo data specified when undo/redo callback was registered via mgEditorAddUndo, mgEditorAppendUndo, mgEditorAddRedo, mgEditorAddUndoForRedo, mgEditorAddUndoForCreate, mgEditorAppendUndoForCreate, mgEditorAddUndoForDelete, mgEditorAppendUndoForDelete, mgEditorAddUndoForMove or mgEditorAppendUndoForMove
