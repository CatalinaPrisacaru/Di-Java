mgUnregisterAllTools

FUNCTION NAME

mgUnregisterAllTools - unregisters all tools for a plug-in.

C SPECIFICATION       (Not Available For Python)

void mgUnregisterAllTools (mgpluginplugin );

DESCRIPTION

mgUnregisterAllTools unregisters all previously registered tools for a specified plug-in module plugin.

After calling this function, all tools registered for the plug-in become inaccessible. You should only call this function from your plug-in termination function mgpExit.

PARAMETERS

plugin
the plug-in module to unregister all tools for

ACCESS LEVEL

Level 3
