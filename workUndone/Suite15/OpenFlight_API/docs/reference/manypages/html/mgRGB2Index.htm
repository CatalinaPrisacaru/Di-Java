mgRGB2Index

FUNCTION NAME

mgRGB2Index - finds the color palette entry and intensity which most closely matches RGB values.

C SPECIFICATION

mgbool mgRGB2Index (mgrec*db,
shortred,
shortgreen,
shortblue,
unsigned int*index,
float*intensity );

PYTHON SPECIFICATION

mgbool, index, intensity mgRGB2Index (mgrec*db,
intred,
intgreen,
intblue )

DESCRIPTION

Given a database node db, and red, green, and blue values red, green, and blue, mgRGB2Index returns the index, index, and intensity, intensity, of the closest match in the database’s color palette. The best match is determined by the least sum of squares method.

The color components supplied must range from 0 to 255. The intensity value returned will range from 0.0-1.0.

PARAMETERS

db
the database
red
the red value, which ranges from 0 to 255
green
the green value, which ranges from 0 to 255
blue
the blue value, which ranges from 0 to 255
index
address of value to receive palette index
intensity
address of value to receive intensity value

RETURNS

mgbool
MG_TRUE if the index could be converted successfully, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.
intensity
If function is successful, intensity contains the retrieved intensity, otherwise intensity is undefined.

ACCESS LEVEL

Level 1
