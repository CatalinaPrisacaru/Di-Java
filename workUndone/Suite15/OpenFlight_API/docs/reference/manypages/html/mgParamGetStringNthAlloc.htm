mgParamGetStringNthAlloc

FUNCTION NAME

mgParamGetStringNthAlloc - allocates and gets the nth item of a string parameter value in a parameter block.

C SPECIFICATION       (Not Available For Python)

mgbool mgParamGetStringNthAlloc (mgparamblockparamBlock,
const char*paramName,
unsigned intnth,
char**paramValue,
const char*defaultValue );

DESCRIPTION

mgParamGetStringNthAlloc allocates and retrieves the nth item of a string parameter value named paramName in the parameter block paramBlock. If the parameter exists and contains an nth item, a string is allocated, initialized to the nth parameter value and returned in paramValue. If the parameter does not exist or does not contain an nth item, a string is allocated, initialized to defaultValue and returned in paramValue.

The string returned for paramValue is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

The first item of an array parameter is nth equal to 1.

This function is similar to mgParamGetStringNth. Use mgParamGetStringNth if the value of the string parameter you want is a known or fixed length and you can guarantee the buffer you pass in paramValue is big enough. If you do not know the length of the string parameter value or want to make sure the entire string is retrieved use mgParamGetStringNthAlloc.

Note that if the parameter does not exist, it is NOT created in the parameter block.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name
nth
which value to return from the array parameter
paramValue
the address of value to receive parameter value returned
defaultValue
default value returned if parameter does not exist

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
