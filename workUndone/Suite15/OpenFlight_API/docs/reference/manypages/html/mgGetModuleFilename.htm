mgGetModuleFilename

FUNCTION NAME

mgGetModuleFilename - returns the path and file name associated with a plug-in.

C SPECIFICATION       (Not Available For Python)

char* mgGetModuleFilename (mgpluginplugin );

DESCRIPTION

mgGetModuleFilename returns the full path and file name specification for the plug-in module plugin.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

plugin
the plug-in module being queried

RETURNS

Returns the path and file name if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 4
