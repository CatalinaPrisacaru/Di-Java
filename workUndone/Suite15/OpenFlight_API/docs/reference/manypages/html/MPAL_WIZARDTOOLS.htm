MPAL_WIZARDTOOLS

NAME

MPAL_WIZARDTOOLS - Palette Location Wizard

DESCRIPTION

This is a valid value for Tool Attribute Palette Location MTA_PALETTELOCATION

If you specify this value for MTA_PALETTELOCATION when you register an editor or viewer tool, the palette icon for your tool is placed in the Wizards Tool palette.
