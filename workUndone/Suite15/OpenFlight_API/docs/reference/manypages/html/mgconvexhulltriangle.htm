mgconvexhulltriangle

STRUCT NAME

mgconvexhulltriangle - Convex Hull triangle structure used by mgconvexhull

SYNOPSIS

typedef struct  {
   mgcoord3d p1;
   mgcoord3d p2;
   mgcoord3d p3;
} mgconvexhulltriangle;

MEMBERS

p1
mgcoord3d - the coordinate position of the first vertex of the convex hull triangle
p2
mgcoord3d - the coordinate position of the second vertex of the convex hull triangle
p3
mgcoord3d - the coordinate position of the third vertex of the convex hull triangle

DESCRIPTION

This structure represents a single triangle of a convex hull. The vertices of the triangle are ordered counter-clockwise and the direction of the normal is based on the right hand rule.
