mgGuiSetPos

FUNCTION NAME

mgGuiSetPos - sets the position of a control or dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgGuiSetPos (mgguigui,
intleft,
inttop );

DESCRIPTION

mgGuiSetPos sets the left and/or top position of the specified control or dialog, gui, measured in pixels.

The position of a control is measured relative to the upper left corner of the parent dialog's "view" area. The upper left corner of a dialog's view area is at position (0, 0). Remember that the view area of a dialog does not include the margin or the title bar of the dialog.

The position of a dialog is measured relative to the upper left corner of the screen. The upper left corner of the screen is at position (0, 0).

If either of the input parameters are MGSP_DONOTCHANGE, that particular position will not be changed. In this way, you can set one position without having to respecify the value for the other. For example, if you specify a value for top, but MGSP_DONOTCHANGE for left, the top position of the gui item will be changed, but the left position will not be affected.

PARAMETERS

gui
the control or dialog whose position is to be set
left
new left position
top
new top position

C EXAMPLE

The following example shows how to use this function to set the position of controls and dialogs.
mgstatus status;

// set the top position of a dialog, leave the left position as is
status = mgGuiSetPos (dialog, MGSP_DONOTCHANGE, 200);

// set the left position of a dialog, leave the top position as is
status = mgGuiSetPos (dialog, 100, MGSP_DONOTCHANGE);

// set BOTH the left and top positions of a control
status = mgGuiSetPos (control, 20, 10);
