mgGetLastError

FUNCTION NAME

mgGetLastError - retrieves the text of the last reported message.

C SPECIFICATION

mgbool mgGetLastError (char*message,
intmaxLen );

PYTHON SPECIFICATION

mgbool, string mgGetLastError ( )

DESCRIPTION

The text of every message generated by a call to mgSendMessage, mgSendStatus, mgSendWarning, and mgSendError is saved until the next call, and can be retrieved by mgGetLastError.

C NOTES

A copy of the message is stored in message, up to the length defined by maxLen.

PARAMETERS

message
character buffer to hold the message retrieved
maxLen
maximum number of characters to store in message

RETURNS

mgbool
MG_TRUE if last error message retrieved successfully, MG_FALSE otherwise.
string
If function is successful, string contains the message, otherwise undefined.

ACCESS LEVEL

Level 1

SEE ALSO

mgSendMessage, mgSendStatus, mgSendWarning, mgSendError
