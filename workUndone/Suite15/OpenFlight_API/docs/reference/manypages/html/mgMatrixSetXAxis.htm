mgMatrixSetXAxis

FUNCTION NAME

mgMatrixSetXAxis - sets the local X axis of a matrix.

SYNOPSIS       (C and Python)

void mgMatrixSetXAxis (mgmatrix*matrix,
const mgcoord3d*coord );

DESCRIPTION

mgMatrixSetXAxis sets the local X axis of matrix. The X axis is composed of the first three elements in the 1st row of the matrix: matrix[0], matrix[1] and matrix[2].

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

matrix
matrix to set the local X axis of
coord
local X axis to set to

ACCESS LEVEL

Level 1
