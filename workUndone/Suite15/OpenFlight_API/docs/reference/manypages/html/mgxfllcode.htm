mgxfllcode

ENUM NAME

mgxfllcode - The set of possible transformation matrix types for fltXmHeader records

SYNOPSIS

typedef enum mgxfllcode {
   MXLL_TRANSLATE,
   MXLL_SCALE,
   MXLL_ROTEDGE,
   MXLL_ROTPT,
   MXLL_PUT,
   MXLL_TOPOINT,
   MXLL_GENERAL,
} mgxfllcode;

MEMBERS

MXLL_TRANSLATE
a translation matrix
MXLL_SCALE
a scale matrix
MXLL_ROTEDGE
a rotation about an edge
MXLL_ROTPT
a rotation about a point
MXLL_PUT
a simultaneous translation, rotation and scale matrix
MXLL_TOPOINT
a simultaneous scale and rotation matrix
MXLL_GENERAL
a matrix of one or more undefined transformations
