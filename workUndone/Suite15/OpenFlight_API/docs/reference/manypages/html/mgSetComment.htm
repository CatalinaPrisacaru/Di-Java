mgSetComment

FUNCTION NAME

mgSetComment - assigns comment text to a node record.

SYNOPSIS       (C and Python)

mgbool mgSetComment (mgrec*rec,
const char*comment );

DESCRIPTION

mgSetComment allocates space and copies comment into the comment text of the node record rec. Pre-existing comments are deleted before the new comment space is allocated. Note that vertex node records do not have comment text.

PARAMETERS

rec
the node record whose comment text is to be set
comment
the comment text for the node record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise,

ACCESS LEVEL

Level 2

SEE ALSO

mgGetComment, mgDeleteComment
