mgGetCurrentGridPoints

FUNCTION NAME

mgGetCurrentGridPoints - gets the current modeling grid points.

C SPECIFICATION

mgbool mgGetCurrentGridPoints (mgrec*db,
mgcoord3d*origin,
mgcoord3d*xAxis,
mgcoord3d*third );

PYTHON SPECIFICATION

mgbool, origin, xAxis, third mgGetCurrentGridPoints (mgrec*db )

DESCRIPTION

mgGetCurrentGridPoints returns the three points that define the location and orientation of the grid in the current Creator graphics view. These points are returned in the output parameter, origin, xAxis, and third where origin is the origin of the grid, xAxis is a point on the x axis of the grid and third is a point that along with origin and xAxis define the plane in which the grid is located.

PARAMETERS

db
the database
origin
address of 3D coordinate to receive grid origin
xAxis
address of 3D coordinate to receive x axis point
third
address of 3D coordinate to receive third point on the grid plane

RETURNS

mgbool
MG_TRUE if RGB components of current primary modeling color for db could be determined, MG_FALSE otherwise.
origin
If function is successful, origin contains the retrieved origin coordinate, otherwise origin is undefined.
xAxis
If function is successful, xAxis contains the retrieved xAxis coordinate, otherwise xAxis is undefined.
third
If function is successful, third contains the retrieved third coordinate, otherwise third is undefined.

ACCESS LEVEL

Level 4
