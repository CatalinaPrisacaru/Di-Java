mgGetBoundsForSelect

FUNCTION NAME

mgGetBoundsForSelect - retrieves the bounding box for the selection.

C SPECIFICATION

mgbool mgGetBoundsForSelect (mgrec*db,
mgboxd*box );

PYTHON SPECIFICATION

mgbool, box mgGetBoundsForSelect (mgrec*db )

DESCRIPTION

mgGetBoundsForSelect retrieves the axis aligned bounding box for the selected node records in database db. The bounding box is in world coordinates.

PARAMETERS

db
the database
box
address of box to receive bounds

RETURNS

mgbool
MG_TRUE if the box was retrieved successfully, MG_FALSE otherwise.
box
If function is successful, box contains the retrieved coordinates, otherwise box is undefined.

ACCESS LEVEL

Level 1
