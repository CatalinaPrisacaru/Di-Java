mgMatrixStackRotateX

FUNCTION NAME

mgMatrixStackRotateX - multiplies a rotation matrix into a matrix stack.

SYNOPSIS       (C and Python)

mgbool mgMatrixStackRotateX (mgmatrixstackmatrixStack,
doubletheta );

DESCRIPTION

mgMatrixStackRotateX calculates a rotation matrix representing the rotation of theta degrees about the X coordinate axis and multiplies that matrix by the current top-of-stack matrix in the specified matrix stack matrixStack. The resulting matrix replaces the top-of-stack matrix in matrixStack.

The parameter theta specifies the angle (in degrees) to rotate counter-clockwise (right hand rule).

If the top-of-stack matrix is T before this function, and the rotation matrix representing the rotation of theta degrees about the X coordinate axis is R, the new top-of-stack matrix after this function will be the product R X T.

PARAMETERS

matrixStack
matrix stack to apply operation
theta
angle to rotate measured counter-clockwise in degrees about vector (right hand rule)

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
