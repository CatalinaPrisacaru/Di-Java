mgSetMessagesEnabled

FUNCTION NAME

mgSetMessagesEnabled - enables or disables system messages.

SYNOPSIS       (C and Python)

void mgSetMessagesEnabled (mgmessageseverityenumseverity,
mgboolenabled );

DESCRIPTION

According to the switch enabled, mgSetMessagesEnabled either enables (MG_TRUE) or disables (MG_FALSE) the display of messages of level severity within the stand-alone program environment. By default, messages for all severities are enabled.

Note: Within the modeler environment, messages are always enabled and cannot be disabled. For this reason, this function is ignored when invoked from within the modeler environment.

You can invoke this function before you call mgInit in the stand alone program environment to turn on or off messages displayed by the OpenFlight API initialization function.

PARAMETERS

severity
severity level of the messages to enable or disable
enabled
MG_TRUE to enable messages, MG_FALSE to disable them

EXAMPLE

The following examples (in C and Python) show how to disable all messages before calling mgInit and then re-enabling them after.

C EXAMPLE

// disable all messages
mgSetMessagesEnabled (MMSG_STATUS, MG_FALSE);
mgSetMessagesEnabled (MMSG_WARNING, MG_FALSE);
mgSetMessagesEnabled (MMSG_ERROR, MG_FALSE);

mgInit (argc, argv);

// re-enable all messages
mgSetMessagesEnabled (MMSG_STATUS, MG_TRUE);
mgSetMessagesEnabled (MMSG_WARNING, MG_TRUE);
mgSetMessagesEnabled (MMSG_ERROR, MG_TRUE);

// continue with stand-alone application processing

PYTHON EXAMPLE

# disable all messages
mgSetMessagesEnabled (MMSG_STATUS, MG_FALSE)
mgSetMessagesEnabled (MMSG_WARNING, MG_FALSE)
mgSetMessagesEnabled (MMSG_ERROR, MG_FALSE)

mgInit (argc, argv)

# re-enable all messages
mgSetMessagesEnabled (MMSG_STATUS, MG_TRUE)
mgSetMessagesEnabled (MMSG_WARNING, MG_TRUE)
mgSetMessagesEnabled (MMSG_ERROR, MG_TRUE)

# continue with stand-alone script processing
