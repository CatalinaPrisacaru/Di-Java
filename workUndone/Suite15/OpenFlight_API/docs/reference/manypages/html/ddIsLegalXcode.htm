ddIsLegalXcode

FUNCTION NAME

ddIsLegalXcode - checks the tag-along parent and child relationship between two codes.

SYNOPSIS       (C and Python)

mgbool ddIsLegalXcode (mgcodeparentCode,
mgcodecode );

DESCRIPTION

ddIsLegalXcode checks if code is a legal tag-along extension for parent code parentCode.

PARAMETERS

parentCode
the parent code
code
the extension code

RETURNS

Returns MG_TRUE if code is a legal tag-along extension code for parent code parentCode, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
