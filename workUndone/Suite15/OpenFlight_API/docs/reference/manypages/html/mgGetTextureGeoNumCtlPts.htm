mgGetTextureGeoNumCtlPts

FUNCTION NAME

mgGetTextureGeoNumCtlPts - gets number of control points from a georeference info object.

SYNOPSIS       (C and Python)

mgbool mgGetTextureGeoNumCtlPts (mgimagegeoinfogeoInfo,
int*numCoords );

DESCRIPTION

mgGetTextureGeoNumCtlPts gets the number of control points from the specified georeference info object geoInfo.

PARAMETERS

geoInfo
the georeference info object
numCoords
address of integer to receive number of control points

ACCESS LEVEL

Level 1

SEE ALSO

mgReadImageGeoInfo, mgGetTextureGeoType, mgGetTextureGeoProjection, mgGetTextureGeoEarthModel, mgGetTextureGeoUTMZone, mgGetTextureGeoUTMHemisphere, mgGetTextureGeoImageOrigin, mgGetTextureGeoCtlPt, mgFreeImageGeoInfo

RETURNS

mgbool
MG_TRUE if numCoords retrieved successfully, MG_FALSE otherwise.
numCoords
If function is successful, numCoords contains the retrieved number of coords, otherwise numCoords is undefined.
