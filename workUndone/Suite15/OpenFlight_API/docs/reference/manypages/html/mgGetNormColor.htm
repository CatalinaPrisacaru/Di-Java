mgGetNormColor

FUNCTION NAME

mgGetNormColor - gets values out of a fltNormColor record.

C SPECIFICATION

mgbool mgGetNormColor (mgrec*rec,
mgcodenormColorCode,
float*red,
float*green,
float*blue );

PYTHON SPECIFICATION

mgbool, red, green, blue mgGetNormColor (mgrec*rec,
mgcodenormColorCode )

DESCRIPTION

Given a record, rec, mgGetNormColor gets the values of fltNormColor attribute record normColorCode from rec and stores them in red, green, and blue.

You do not have to provide addresses for all red, green, and blue values. For example, if you only want to receive the red and blue components (but not green), specify valid addresses for red and blue and leave green NULL.

PARAMETERS

rec
the record from which to get the fltNormColor attribute code
normColorCode
the fltNormColor attribute code
red
address of value to receive red component value
green
address of value to receive green component value
blue
address of value to receive blue component value

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
red
If function is successful, red contains the retrieved value, otherwise red is undefined.
green
If function is successful, green contains the retrieved value, otherwise green is undefined.
blue
If function is successful, blue contains the retrieved value, otherwise blue is undefined.

C EXAMPLE

mgrec* db;
mgrec* matRec;
float r, g, b;
int index;
mgbool status;

db = mgOpenDb ("file.flt");
matRec = mgGetFirstMaterial (db, &index);
status = mgGetNormColor (matRec, fltAmbient, &r, &g, &b);

ACCESS LEVEL

Level 1
