mgpaletteid

ENUM NAME

mgpaletteid - Palette Id type

SYNOPSIS

typedef enum mgpaletteid {
   MPID_COLORPALETTE,
   MPID_ALTCOLORPALETTE,
   MPID_IRCOLORPALETTE,
   MPID_MATERIALPALETTE,
   MPID_LIGHTSOURCEPALETTE,
   MPID_TEXTUREPALETTE,
   MPID_TEXTUREMAPPINGPALETTE,
   MPID_SOUNDPALETTE,
   MPID_LINESTYLEPALETTE,
   MPID_LIGHTPOINTPALETTE,
   MPID_SHADERPALETTE,
} mgpaletteid;

MEMBERS

MPID_COLORPALETTE
Primary Color Palette Id. This palette id specifies the Primary Color Palette.
MPID_ALTCOLORPALETTE
Alternate Color Palette Id. This palette id specifies the Alternate Color Palette.
MPID_IRCOLORPALETTE
Infrared Color Palette Id. This palette id specifies the Infrared Color Palette.
MPID_MATERIALPALETTE
Material Palette Id. This palette id specifies the Material Palette.
MPID_LIGHTSOURCEPALETTE
Light Source Palette Id. This palette id specifies the Light Source Palette.
MPID_TEXTUREPALETTE
Texture Palette Id. This palette id specifies the Texture Palette.
MPID_TEXTUREMAPPINGPALETTE
Texture Mapping Palette Id. This palette id specifies the Texture Mapping Palette.
MPID_SOUNDPALETTE
Sound Palette Id. This palette id specifies the Sound Palette.
MPID_LINESTYLEPALETTE
Line Style Palette Id. This palette id specifies the Line Style Palette.
MPID_LIGHTPOINTPALETTE
Light Point Palette Id. This palette id specifies the Light Point Palette.
MPID_SHADERPALETTE
Shader Palette Id. This palette id specifies the Shader Palette.

DESCRIPTION

This type is used to enumerate the database palettes that can be displayed using the mgShowPalette function.
