mgGetDbOnDesktop

FUNCTION NAME

mgGetDbOnDesktop - gets the database associated with an OpenFlight file open on the Creator desktop.

SYNOPSIS       (C and Python)

mgrec* mgGetDbOnDesktop (const char*fileName );

DESCRIPTION

mgGetDbOnDesktop gets the database record associated with the OpenFlight file fileName if that file is open on the Creator desktop.

Note: This function is only applicable in the Creator modeler environment and will always return MG_NULL when called by a stand-alone application.

PARAMETERS

fileName
the name of the file to check

RETURNS

Returns database open on desktop if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4
