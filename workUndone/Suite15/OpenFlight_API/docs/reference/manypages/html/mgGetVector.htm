mgGetVector

FUNCTION NAME

mgGetVector - gets values out of a fltVector record.

C SPECIFICATION

mgbool mgGetVector (mgrec*rec,
mgcodevectorCode,
float*i,
float*j,
float*k );

PYTHON SPECIFICATION

mgbool, i, j, k mgGetVector (mgrec*rec,
mgcodevectorCode )

DESCRIPTION

Given a record rec, mgGetVector gets the values of fltVector attribute record vectorCode from rec and stores them in i, j, and k.

For retrieving vertex normals, it is much more efficient to call the convenience function mgGetVtxNormal.

You do not have to provide addresses for all i, j, and k values. For example, if you only want to receive the i and j components (but not k), specify valid addresses for i and j and leave k NULL.

PARAMETERS

rec
the record from which to get the fltVector attribute record
vectorCode
the fltVector attribute record code
i
address of value to receive i component
j
address of value to receive j component
k
address of value to receive k component

RETURNS

mgbool
MG_TRUE if the vector was retrieved successfully, MG_FALSE otherwise.
i
If function is successful, i contains the retrieved i coordinate, otherwise i is undefined.
j
If function is successful, j contains the retrieved j coordinate, otherwise j is undefined.
k
If function is successful, k contains the retrieved k coordinate, otherwise k is undefined.

C EXAMPLE

mgrec* eyeRec;
float i, j, k;

db = mgOpenDb ("file.flt");

// create eyepoint rec
eyeRec = UserMakeEyeRec (db);

mgGetVector (eyeRec, fltEyeEyeDir, &i, &j, &k);

ACCESS LEVEL

Level 1
