mgMatrixIdentity

FUNCTION NAME

mgMatrixIdentity - forms the identity matrix.

C SPECIFICATION

mgbool mgMatrixIdentity (mgmatrix*matrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixIdentity ( )

DESCRIPTION

mgMatrixIdentity creates and returns the identity matrix.

C NOTES

This function writes the identity matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the matrix if successful.

PARAMETERS

matrix
address of matrix to receive result

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix.

ACCESS LEVEL

Level 1
