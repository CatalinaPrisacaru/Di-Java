mgGetBounds

FUNCTION NAME

mgGetBounds - retrieves the bounding box for a node.

C SPECIFICATION

mgbool mgGetBounds (mgrec*rec,
mgboxd*box );

PYTHON SPECIFICATION

mgbool, box mgGetBounds (mgrec*rec )

DESCRIPTION

mgGetBounds retrieves the axis aligned bounding box for a node record rec. The bounding box is in local coordinates.

PARAMETERS

rec
a node record
box
address of box to receive bounds

RETURNS

mgbool
MG_TRUE if the box was retrieved successfully, MG_FALSE otherwise.
box
If function is successful, box contains the retrieved coordinates, otherwise box is undefined.

EXAMPLE

The following examples (in C and Python) show you might use mgGetBounds:

C EXAMPLE

mgbool ok;
mgboxd bounds;

ok = mgGetBounds (rec, &bounds);
if (ok == MG_TRUE) {
   printf ("Min: ( %f, %f, %f ), Max: ( %f, %f, %f )",
     bounds.min.x, bounds.min.y, bounds.min.z,
     bounds.max.x, bounds.max.y, bounds.max.z);
}

PYTHON EXAMPLE

ok, bounds = mgGetBounds (rec)
if (ok == MG_TRUE):
   print "Min: (", bounds.min.x, bounds.min.y, bounds.min.z, "), " \\
  "Max: (", bounds.max.x, bounds.max.y, bounds.max.z, ")"

ACCESS LEVEL

Level 1
