mggeocoorddata

TYPE NAME

mggeocoorddata - Geospecific Coordinate data

MEMBERS

u
double - x coordinate in image
v
double - y coordinate in image
lat
double - latitude position that maps to x coordinate
lon
double - longitude position that maps to y coordinate

DESCRIPTION

Use an array of mggeocoorddata to add and get geospecific coordinates.

SEE ALSO

mgGeoCoordGet, mgGeoCoordAdd
