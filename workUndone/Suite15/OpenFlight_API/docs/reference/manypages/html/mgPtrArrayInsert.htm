mgPtrArrayInsert

FUNCTION NAME

mgPtrArrayInsert - insert an item into a pointer array.

C SPECIFICATION       (Not Available For Python)

void mgPtrArrayInsert (mgptrarrayptrArray,
intpos,
void*item );

DESCRIPTION

mgPtrArrayInsert inserts the specified item into the pointer array ptrArray at the position pos.

This function inserts a new item into the pointer array such that the new item, after it is inserted, is at position pos. This causes the item that was previously at this position (and all items after it) to get shifted up one index.

The first item in a pointer array is at index 1. Therefore, to insert an item at the beginning of the array, specify pos as 1. To insert an item at the end of the array, specify pos as the length of the pointer array plus one (or use mgPtrArrayAppend).

PARAMETERS

ptrArray
the pointer array
pos
the position in array to insert the item (in range 1..length+1)
item
the item to be inserted into the pointer array

ACCESS LEVEL

Level 1
