mgSetSaveNonIndexedLightPoints

FUNCTION NAME

mgSetSaveNonIndexedLightPoints - sets the preference that determines how light point records are saved in an OpenFlight file.

SYNOPSIS       (C and Python)

void mgSetSaveNonIndexedLightPoints (mgboolflag );

DESCRIPTION

mgSetSaveNonIndexedLightPoints tells mgWriteDb and mgSaveAsDb how to write light point records in the OpenFlight file. If flag is MG_TRUE, light points will be saved in the "non-indexed" format used in OpenFlight 15.7 and earlier. This format is represented in the OpenFlight format as raw record code 111. If flag is MG_FALSE (the default), light points will be saved in the "indexed" format introduced in OpenFlight 15.8. This format is represented in the OpenFlight format as raw record code 130.

The default value for this preference is MG_FALSE.

PARAMETERS

flag
whether or not to save light points in the non-indexed format

ACCESS LEVEL

Level 2
