mgimagegetinfofunc

CALLBACK NAME

mgimagegetinfofunc - Image Get Info function for Image Importer Tools

SYNOPSIS

mgstatus (*mgimagegetinfofunc) ( mgplugintool pluginTool,
void* userData,
int imageid,
mgimageinfo imageInfo );

DESCRIPTION

This is the signature for image get info functions. When you register an image importer plug-in tool, provide a get info function of this form that is called when the OpenFlight API needs to get information about an image that your tool has opened.

PARAMETERS

pluginTool
the plug-in tool
userData
user defined data specified when tool is registered
imageid
image identifier indicating which image to get info for - this is the value returned by the image open function when the image was first opened
imageInfo
image info object that your tool loads values into for info attributes width, height, type, etc corresponding to the specified image

RETURNS

Returns MIMG_NO_ERROR if successful, otherwise returns applicable texture error code. If successful, the parameter imageInfo should be filled in with all pertinent image information using the mgSetTextureXXX functions listed below.
