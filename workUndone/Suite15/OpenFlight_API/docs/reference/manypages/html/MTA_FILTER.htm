MTA_FILTER

NAME

MTA_FILTER - Tool Attribute Filter String

DESCRIPTION

This string attribute specifies the pattern string that is used in the appropriate File Selection Dialog to match files for database and image importer and database exporter tools.

This attribute is typically of the form "*.xyz" where xyz is the suffix used to identify applicable files for the importer or exporter tool. To include multiple filters, separate each using a semi-colon as in "*.xyz;*.abc;*.ijk";
