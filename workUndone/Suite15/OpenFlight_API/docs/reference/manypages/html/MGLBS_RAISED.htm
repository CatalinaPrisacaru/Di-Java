MGLBS_RAISED

NAME

MGLBS_RAISED - Control Attribute GL Border Style Raised

DESCRIPTION

This is a valid value for Control Attribute GL Border Style MCA_GLBORDERSTYLE.

If you specify this value for MCA_GLBORDERSTYLE when you call mgControlSetAttribute, a raised frame will be drawn around the specified GL control.
