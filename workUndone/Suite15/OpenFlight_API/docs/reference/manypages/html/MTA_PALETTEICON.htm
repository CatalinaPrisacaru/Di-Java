MTA_PALETTEICON

NAME

MTA_PALETTEICON - Tool Attribute Palette Icon

DESCRIPTION

This mgpixmap attribute specifies the tool icon that is placed in the corresponding tool palette MTA_PALETTELOCATION for a tool. When the user clicks on this icon, the plug-in tool is launched.

You must specify both MTA_PALETTELOCATION and MTA_PALETTEICON if you want your tool to be launched from a tool palette.
