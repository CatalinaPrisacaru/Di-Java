mgDeselectAll

FUNCTION NAME

mgDeselectAll - deselects all node records.

SYNOPSIS       (C and Python)

mgstatus mgDeselectAll (mgrec*db );

DESCRIPTION

mgDeselectAll deselects all node records for the specified database db.

PARAMETERS

db
the database to de-select all for

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
