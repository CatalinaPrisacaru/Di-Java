mgMatrixFormQuadToQuad

FUNCTION NAME

mgMatrixFormQuadToQuad - forms a "quad to quad" transformation matrix.

C SPECIFICATION

mgbool mgMatrixFormQuadToQuad (mgmatrix*matrix,
const mgcoord3d*from1,
const mgcoord3d*from2,
const mgcoord3d*from3,
const mgcoord3d*from4,
const mgcoord3d*to1,
const mgcoord3d*to2,
const mgcoord3d*to3,
const mgcoord3d*to4 );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixFormQuadToQuad (mgcoord3dfrom1,
mgcoord3dfrom2,
mgcoord3dfrom3,
mgcoord3dfrom4,
mgcoord3dto1,
mgcoord3dto2,
mgcoord3dto3,
mgcoord3dto4 )

DESCRIPTION

mgMatrixFormQuadToQuad calculates a transformation matrix that transforms one quadrilateral to another. The "from" quadrilateral is specified by four points, from1, from2, from3 and from4. Similarly, the "to" quadrilateral is specified by to1, to2, to3 and to4.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
from1
first point of the "from" quadrilateral
from2
second point of the "from" quadrilateral
from3
third point of the "from" quadrilateral
from4
third point of the "from" quadrilateral
to1
first point of the "to" quadrilateral
to2
second point of the "to" quadrilateral
to3
third point of the "to" quadrilateral
to4
third point of the "to" quadrilateral

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
If function is successful, the matrix.

ACCESS LEVEL

Level 1
