mgProgressSetValue

FUNCTION NAME

mgProgressSetValue - sets the value of a progress control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgProgressSetValue (mgguicontrol,
doublevalue );

DESCRIPTION

mgProgressSetValue sets the value of a progress control, to the specified value.

If the value you specify is greater than the maximum value allowed for the progress control, the progress value will be set to the maximum progress value. Similarly, if value is less than the minimum progress value, the progress value will be set to the minimum progress value.

PARAMETERS

control
the progress control
value
the value for the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
