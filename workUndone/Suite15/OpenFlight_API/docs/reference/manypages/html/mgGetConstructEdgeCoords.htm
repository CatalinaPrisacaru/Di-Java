mgGetConstructEdgeCoords

FUNCTION NAME

mgGetConstructEdgeCoords - gets the endpoint coordinates of a construction edge node.

C SPECIFICATION

mgstatus mgGetConstructEdgeCoords (mgrec*constructEdge,
mgcoord3d*coord1,
mgcoord3d*coord2 );

PYTHON SPECIFICATION

mgstatus, coord1, coord2 mgGetConstructEdgeCoords (mgrec*constructEdge )

DESCRIPTION

mgGetConstructEdgeCoords gets the coordinates of the endpoints of a construction edge node constructEdge.

If either of the coordinate parameters (coord1 or coord2) are MG_NULL, that particular coordinate will not be returned. In this way you can query one endpoint of the construction edge without the other.

PARAMETERS

constructEdge
the construction edge to get coordinates for
coord1
address of record to receive coordinates of first vertex of construction edge
coord2
address of record to receive coordinates of second vertex of construction edge

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success.
coord1
If function is successful, coord1 contains coordinates of first vertex of construction edge, otherwise coord1 is undefined.
coord2
If function is successful, coord2 contains coordinates of second vertex of construction edge, otherwise coord2 is undefined.

ACCESS LEVEL

Level 4
