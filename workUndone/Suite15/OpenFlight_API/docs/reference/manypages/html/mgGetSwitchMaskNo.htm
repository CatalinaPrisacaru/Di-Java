mgGetSwitchMaskNo

FUNCTION NAME

mgGetSwitchMaskNo - gets the current mask index of a fltSwitch node.

C SPECIFICATION

mgbool mgGetSwitchMaskNo (mgrec*rec,
int*maskNo );

PYTHON SPECIFICATION

mgbool, maskNo mgGetSwitchMaskNo (mgrec*rec )

DESCRIPTION

A fltSwitch node contains at least one mask. mgGetSwitchMaskNo returns the mask index currently being used to define the on/off display behavior of the switch node’s children. The first switch mask is number 0. The mask index is set to -1 if unsuccessful.

PARAMETERS

rec
the fltSwitch node
maskNo
address of value to receive mask number

RETURNS

mgbool
MG_TRUE if switch mask index retrieved successfully, MG_FALSE otherwise.
maskNo
If function is successful, maskNo contains the retrieved index, otherwise maskNo is undefined.

ACCESS LEVEL

Level 1
