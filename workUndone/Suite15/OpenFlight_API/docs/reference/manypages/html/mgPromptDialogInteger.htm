mgPromptDialogInteger

FUNCTION NAME

mgPromptDialogInteger - prompts the user to enter integer value.

C SPECIFICATION

mgstatus mgPromptDialogInteger (mgguiparent,
const char*messageText,
int*value );

PYTHON SPECIFICATION

mgstatus, number mgPromptDialogInteger (mgguiparent,
stringmessageText,
intvalue )

DESCRIPTION

mgPromptDialogInteger displays a modal dialog in which the user is asked to enter an integer value.

The dialog displays a text message and contains a text control and two push button controls - OK and Cancel. The text control is used to control the numeric value contained in the parameter value. The OK and Cancel push button controls will be used to dismiss the dialog.

C NOTES

The dialog is created as a child of the specified dialog parent. If you specify parent as MG_NULL, the dialog is created as a child of the Creator main window. If the user enters a valid numeric value and chooses the OK button to dismiss the dialog, the value is returned in the parameter value.

PYTHON NOTES

Currently the parent parameter is reserved for future enhancement and should be specified as None. For now the dialog is created as a child of the main Creator main window. If the user enters a valid numeric value and chooses the OK button to dismiss the dialog, the value is returned.

PARAMETERS

parent
the dialog parent to attach the prompt dialog
messageText
a text string that is displayed in the dialog as a message to the user
value
C - on input, the address of the initial integer value to display in the text control, on output will receive the integer value entered
Python - the initial integer value to display in the text control

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success. Success indicates that the value entered was a valid integer value and the user dismissed the dialog using the OK push button.
number
If function is successful, contains the number entered by the user.

EXAMPLE

The following examples (in C and Python) prompt the user to enter an integer number (default value 2).

C EXAMPLE

int number = 2;
mgstatus status = mgPromptDialogInteger (MG_NULL, "Enter Number", &number);
if (MSTAT_ISOK (status)) {
   printf ("Number: %d", number);
}

PYTHON EXAMPLE

status,number = mgPromptDialogInteger (None, "Enter Number", 2)
if (MSTAT_ISOK (status)):
   print "Number:", number

Level 4
