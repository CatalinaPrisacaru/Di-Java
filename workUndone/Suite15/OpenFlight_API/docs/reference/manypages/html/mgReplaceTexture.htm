mgReplaceTexture

FUNCTION NAME

mgReplaceTexture - replaces a texture palette entry with a texture from a given file.

SYNOPSIS       (C and Python)

mgbool mgReplaceTexture (mgrec*db,
intindex,
const char*textureName );

DESCRIPTION

mgReplaceTexture replaces the texture palette entry specified by index with an image and attributes from files specified by textureName (the attributes file is textureName appended with ".attr"). If there is no attributes file, default attributes are created and used.

PARAMETERS

db
the database
index
the index of the texture palette entry
textureName
the name of the texture file to replace index

RETURNS

Returns MG_TRUE if index is valid for this db and the new texture assigned to the palette has a different name than the original texture at index, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
