mgGetVtxCoord

FUNCTION NAME

mgGetVtxCoord - retrieves the x,y,z coordinate from a vertex record.

C SPECIFICATION

mgbool mgGetVtxCoord (mgrec*rec,
double*x,
double*y,
double*z );

PYTHON SPECIFICATION

mgbool, x, y, z mgGetVtxCoord (mgrec*rec )

DESCRIPTION

mgGetVtxCoord retrieves the x, y, and z values of the vertex coordinate from a vertex record rec.

PARAMETERS

rec
a vertex record
x
address of value to receive x coordinate
y
address of value to receive y coordinate
z
address of value to receive z coordinate

RETURNS

mgbool
MG_TRUE if the coord was retrieved successfully, MG_FALSE otherwise.
x
If function is successful, x contains the retrieved x coordinate, otherwise x is undefined.
y
If function is successful, y contains the retrieved y coordinate, otherwise y is undefined.
z
If function is successful, z contains the retrieved z coordinate, otherwise z is undefined.

ACCESS LEVEL

Level 1
