mgSetModelingMode

FUNCTION NAME

mgSetModelingMode - sets current modeling mode.

SYNOPSIS       (C and Python)

mgbool mgSetModelingMode (mgrec*db,
mgcodemode );

DESCRIPTION

mgSetModelingMode sets the current modeling mode for the specified database db to the specified level, mode.

Use this function to set modeling mode to any node type except edge. Note that the edge modeling mode is a special case of vertex mode. Since there is no "edge" type in OpenFlight, you must use mgSetModelingModeEdge to set modeling mode to edge.

PARAMETERS

db
the database to set modeling mode for
mode
the modeling mode to set

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

EXAMPLE

The following examples (in C and Python) set the modeling mode to different levels:

C EXAMPLE

mgbool ok;

// set modeling mode to group
ok = mgSetModelingMode (db, fltGroup);

// set modeling mode to vertex
ok = mgSetModelingMode (db, fltVertex);

// set modeling mode to edge
ok = mgSetModelingModeEdge (db);

PYTHON EXAMPLE

# set modeling mode to group
ok = mgSetModelingMode (db, fltGroup)

# set modeling mode to vertex
ok = mgSetModelingMode (db, fltVertex)

# set modeling mode to edge
ok = mgSetModelingModeEdge (db)

ACCESS LEVEL

Level 4
