mgIsFirstInstance

FUNCTION NAME

mgIsFirstInstance - determines if a node is the first instance of its referenced node.

SYNOPSIS       (C and Python)

mgbool mgIsFirstInstance (mgrec*rec );

DESCRIPTION

mgIsFirstInstance determines if a node rec is the first instance node of its reference node. This routine is useful when used with mgGetFirstInstance and mgGetNextInstance to visit all instances of a node, or when used by itself to visit only the first instance.

PARAMETERS

rec
an instance node

RETURNS

Returns MG_TRUE if rec is the first instance of its referenced node, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
