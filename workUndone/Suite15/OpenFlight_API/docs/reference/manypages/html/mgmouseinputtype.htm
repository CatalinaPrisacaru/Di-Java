mgmouseinputtype

ENUM NAME

mgmouseinputtype - Mouse Input Selector type

SYNOPSIS

typedef enum mgmouseinputtype {
   MMSI_NOINPUT,
   MMSI_VERTEXINPUT,
   MMSI_POINTINPUT,
   MMSI_PICKINPUT,
   MMSI_DEVICEINPUT,
} mgmouseinputtype;

MEMBERS

MMSI_NOINPUT
No mouse input selected
MMSI_VERTEXINPUT
Select 3D vertex mouse input
MMSI_POINTINPUT
Select 2D point mouse input
MMSI_PICKINPUT
Select geometry pick mouse input
MMSI_DEVICEINPUT
Select device specific input
