mgSetFocusVertex

FUNCTION NAME

mgSetFocusVertex - set the current focus vertex.

C SPECIFICATION       (Not Available For Python)

mgstatus mgSetFocusVertex (mgeditorcontexteditorContext,
mgrec*vertexRec );

DESCRIPTION

mgSetFocusVertex sets the focus vertex to the specified vertex node, vertexRec for the specified editor context, editorContext.

PARAMETERS

editorContext
the editor context used to identify the database in which the focus vertex is to be set
vertexRec
the vertex node to become the new focus vertex

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
