mgMatrixMultiply

FUNCTION NAME

mgMatrixMultiply - multiplies two matrices.

C SPECIFICATION

mgbool mgMatrixMultiply (const mgmatrixmatrixA,
const mgmatrixmatrixB,
mgmatrix*resultMatrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixMultiply (mgmatrixmatrixA,
mgmatrixmatrixB )

DESCRIPTION

mgMatrixMultiply multiplies two matrices matrixA and matrixB (matrixA X matrixB). This function operates safely when either of the input matrices and the output matrix are the same matrix.

C NOTES

This function writes the resulting matrix into the output parameter resultMatrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrixA
matrix to multiply
matrixB
matrix to multiply
resultMatrix
address of matrix to receive result

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix result.

ACCESS LEVEL

Level 1
