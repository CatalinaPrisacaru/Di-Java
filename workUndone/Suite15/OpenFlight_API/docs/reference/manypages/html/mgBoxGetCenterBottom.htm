mgBoxGetCenterBottom

FUNCTION NAME

mgBoxGetCenterBottom - returns the center bottom of a box.

SYNOPSIS       (C and Python)

mgcoord3d mgBoxGetCenterBottom (const mgboxd*box );

DESCRIPTION

mgBoxGetCenterBottom calculates and returns the coordinates of the center bottom of a box.

PARAMETERS

box
address of box to get center bottom for

RETURNS

Returns the coordinates of the center bottom of box.

ACCESS LEVEL

Level 1
