mgShaderGetVertexProgramNth

FUNCTION NAME

mgShaderGetVertexProgramNth - gets the file name for a vertex program on a shader.

SYNOPSIS       (C and Python)

char* mgShaderGetVertexProgramNth (mgrec*shader,
intnth );

DESCRIPTION

mgShaderGetVertexProgramNth gets the program file name for the nth vertex program on the specified shader. The first vertex program is specified by nth equal to 0.

If shader is a GLSL shader and nth does not refer to an existing vertex progrom on the GLSL shader, this function will fail. If shader is a Cg shader and nth is 0, this function will return "the" vertex program on the Cg shader. If shader is a Cg shader and nth is not 0 or the Cg shader does not have a vertex program this function will fail.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

shader
the shader
nth
the index of the vertex program file to get

RETURNS

Returns the name of the nth vertex program if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
