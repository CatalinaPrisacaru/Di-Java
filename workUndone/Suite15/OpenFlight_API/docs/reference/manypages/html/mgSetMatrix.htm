mgSetMatrix

FUNCTION NAME

mgSetMatrix - sets the attribute values in a fltMatrix record.

SYNOPSIS       (C and Python)

mgbool mgSetMatrix (mgrec*rec,
mgcodematrixCode,
mgmatrixmatrix );

DESCRIPTION

Given a record rec which contains a fltMatrix record identified by the record code matrixCode, mgSetMatrix sets the corresponding matrix of rec using the values contained in matrix.

PARAMETERS

rec
a node or record containing a fltMatrix record
matrixCode
the record code of the fltMatrix to set
matrix
the matrix whose values will be copied

RETURNS

Returns MG_TRUE if successful, MG_FALSE if rec has no such matrix or otherwise unsuccessful.

ACCESS LEVEL

Level 2
