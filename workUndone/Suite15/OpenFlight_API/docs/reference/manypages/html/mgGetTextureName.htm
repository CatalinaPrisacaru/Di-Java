mgGetTextureName

FUNCTION NAME

mgGetTextureName - gets the full name of a texture palette entry from its index.

SYNOPSIS       (C and Python)

char* mgGetTextureName (mgrec*db,
intindex );

DESCRIPTION

mgGetTextureName returns the full name of a texture palette entry at the specified index contained in the texture palette of database db.

If the texture was located when it was loaded, the full name includes the full path specification of the file where it was found on disk. If the texture was not located at load time, the name returned by this function will be the same name returned by mgGetTextureSaveName.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

db
the database
index
the index of the texture palette entry

RETURNS

Returns the name of the texture palette entry if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
