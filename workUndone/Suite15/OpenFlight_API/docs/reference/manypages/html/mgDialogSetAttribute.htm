mgDialogSetAttribute

FUNCTION NAME

mgDialogSetAttribute - sets the attribute values of a dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgDialogSetAttribute (mgguidialog,
mgdialogattributedialogAttr,
intattrValue );

DESCRIPTION

mgDialogSetAttribute sets the specified dialog attribute dialogAttr for the dialog to the specified value attrValue.

Dialog attributes define how dialogs behave in different situations. The dialog attributes and a brief description of each are listed below. These are the possible values for dialogAttr. For more information on the meaning of each attribute, see the description of that particular attribute.

MDA_RESIZEWIDTH - specifies whether or not the dialog's width can be resized.

MDA_RESIZEHEIGHT - specifies whether or not the dialog's height can be resized.

MDA_MINWIDTH - specifies the minimum width the dialog can be sized.

MDA_MAXWIDTH - specifies the maximum width the dialog can be sized.

MDA_MINHEIGHT - specifies the minimum height the dialog can be sized.

MDA_MAXHEIGHT - specifies the maximum height the dialog can be sized.

PARAMETERS

dialog
the dialog whose attribute is to be set
dialogAttr
the dialog attribute to set
attrValue
the value to which the attribute is set

RETURNS

Use MSTAT_ISOK to check return value for success.

C EXAMPLE

The following example sets the minimum size the user can resize a dialog to be its current dimentions. This would be done, most likely, when the MGCB_INIT dialog event is handled by the dialog function.
mggui dialog;
int dlgW, dlgH;

mgGuiGetSize (dialog, &dlgW, &dlgH);
mgDialogSetAttribute (dialog, MDA_MINWIDTH, dlgW);
mgDialogSetAttribute (dialog, MDA_MINHEIGHT, dlgH);

ACCESS LEVEL

Level 4
