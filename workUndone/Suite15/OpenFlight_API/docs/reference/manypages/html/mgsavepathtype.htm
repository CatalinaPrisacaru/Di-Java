mgsavepathtype

ENUM NAME

mgsavepathtype - Save path type for external files associated with a database file (textures and shaders)

SYNOPSIS

typedef enum mgsavepathtype {
   MSPT_ABSOLUTE,
   MSPT_RELATIVE,
   MSPT_NOPATH,
   MSPT_UNKNOWN,
} mgsavepathtype;

MEMBERS

MSPT_ABSOLUTE
Save absolute file paths
MSPT_RELATIVE
Save relative file paths
MSPT_NOPATH
Save no path information with file names
MSPT_UNKNOWN
Unknown save path type
