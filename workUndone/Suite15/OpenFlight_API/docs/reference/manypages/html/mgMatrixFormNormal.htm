mgMatrixFormNormal

FUNCTION NAME

mgMatrixFormNormal - creates a matrix to transform normals by based on the matrix by which the coordinates are transformed.

C SPECIFICATION

void mgMatrixFormNormal (mgmatrix*matrix,
const mgmatrixgeomMatrix );

PYTHON SPECIFICATION

mgmatrix mgMatrixFormNormal (mgmatrixgeomMatrix )

DESCRIPTION

mgMatrixFormNormal gets a matrix that can be used to transform normals if geomMatrix is the matrix by which coordinates are transformed. Normals transformed by the resulting matrix will remain orthogonal to a surface transformed by geomMatrix even when geomMatrix is a non rigid transform such as a skew or scale (including non uniform scaling). Note that unit normals transformed by matrix may not remain unit lengthed on certain transformation such as scales and thus in most cases, the resulting normals will most likely need to be reunitized. Also note that for rigid transformations such as rotations, translations and reflections about an axis, this function is unnecessary and simply transforming unit normals by the same matrix without translation will maintain the normals to be orthogonal to the surface and of unit length.

The original matrix geomMatrix is not affected.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result
geomMatrix
transformation of geometry

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
