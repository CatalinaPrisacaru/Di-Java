mgTabRulerSetOptions

FUNCTION NAME

mgTabRulerSetOptions - changes options for a tab ruler control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTabRulerSetOptions (mgguicontrol,
... );

DESCRIPTION

mgTabRulerSetOptions changes the options for an existing tab ruler control. control, is the GL control that has an associated tab ruler control.

The variable arguments are a set of key/value pairs, where key is a 32-bit value. The key/value list MUST be terminated with a tag of 'MG_NULL'.

PARAMETERS

control
GL control to associate with a tab ruler control
...
optional input parameters in variable argument style using mgtabruleroption as the keys

RETURNS

Use MSTAT_ISOK to check return value for success. If the function succeeds, the options for the tab ruler control were successfully changed.

ACCESS LEVEL

Level 4
