mgDelete

FUNCTION NAME

mgDelete - deletes a node, transformation, tag-along extension or texture attribute record.

SYNOPSIS       (C and Python)

mgbool mgDelete (mgrec*rec );

DESCRIPTION

Given a node, transformation, tag-along extension or texture attribute record rec, mgDelete deletes it. If the record is a node in the database and has children, its descendants are recursively deleted as well. This deletion is permanent and there is no "undo".

When deleting node or transformation records, the corresponding record is deleted and removed from the database. For example if you delete a fltGroup node the group and children are deleted and removed from the database scene graph. Similarly if you delete a fltXmTranslate (or other transformation) record attached to a node in the database, the transformation is deleted and removed from that node. This effectively removes the transformation from the node.

When deleting texture attribute records, however, the corresponding record is deallocated only. The corresponding texture is not removed from any texture palette it might be in (use mgDeleteTexture to delete a texture from its palette). You should only use mgDelete to delete texture attribute records returned by mgReadImageAttributes, not mgGetTextureAttributes.

To delete a tag-along extension record, use mgGetExtRec to get the pointer to the tag-along extension record, then pass that pointer to mgDelete.

PARAMETERS

rec
the record to delete

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
