MTA_MENUPOSITION

NAME

MTA_MENUPOSITION - Tool Attribute Menu Position

DESCRIPTION

When you want your viewer or editor tool to be launched from a menu, you can select another item in the menu before or after which your new menu item is to be located.

This string attribute specifies an existing item in the menu you specify by MTA_MENULOCATION. To place your new menu item after the item specified by this attribute, precede the menu item with the ">" character. To place your new menu item before the specified item, use the "<" character. If you do not specify ">" or "<", your new menu item will be placed after the specified item. See the examples that follow.

C EXAMPLE

Specify the following values for this attribute to add your new menu item before/after the "Open" menu item:
< Open    : Add new menu item before the "Open" menu item.

> Open... : Add new menu item after the "Open" menu item.
Note: The "..." are not required and if included are ignored.

Open      : Add new menu item after the "Open" menu item.
Note: In the absence of ">" and "<", ">" is the default.
