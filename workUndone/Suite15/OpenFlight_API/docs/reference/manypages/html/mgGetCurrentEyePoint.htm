mgGetCurrentEyePoint

FUNCTION NAME

mgGetCurrentEyePoint - returns the attributes of the current eyepoint.

SYNOPSIS       (C and Python)

mgbool mgGetCurrentEyePoint (mgrec*db,
mgrec*eyeRec );

DESCRIPTION

mgGetCurrentEyePoint copies the current eyepoint into the eyepoint record eyeRec. eyeRec must have previously been obtained by a call to mgGetNewEyePoint

PARAMETERS

db
the database
eyeRec
the address of the record to receive the eyepoint record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise. If successful, the attributes of the current eyepoint are copied to the output parameter fltEyePoint record eyeRec, otherwise no attributes are copied.

ACCESS LEVEL

Level 4

SEE ALSO

mgSetCurrentEyePoint, mgGetNewEyePoint
