mgFreeParamBlock

FUNCTION NAME

mgFreeParamBlock - deallocates a parameter block.

C SPECIFICATION       (Not Available For Python)

void mgFreeParamBlock (mgparamblockparamBlock );

DESCRIPTION

mgFreeParamBlock deallocates a parameter block created by mgNewParamBlock or mgGetParamBlock.

Note: If you passed the parameter block to mgPluginToolSetDefaultParamBlock, you transfer ownership of the parameter block to the OpenFlight API and you should NOT deallocate the block with this function. The OpenFlight API will automatically dispose of the memory when it is appropriate to do so.

PARAMETERS

paramBlock
the parameter block to deallocate

ACCESS LEVEL

Level 4

SEE ALSO

mgparamblock, mgNewParamBlock, mgGetParamBlock, mgPluginToolSetDefaultParamBlock
