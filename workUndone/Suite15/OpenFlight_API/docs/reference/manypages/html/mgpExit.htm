mgpExit

FUNCTION NAME

mgpExit - plug-in termination function.

SYNOPSIS       (C and Python)

MGPIDECLARE(void) mgpExit (mgpluginplugin );

DESCRIPTION

This is the plug-in termination function required for all plug-in modules. You must supply this function so that your plug-in is properly recognized.

You must declare this function exactly as shown here, using the MGPIDECLARE macro, so that it is properly exported from your plug-in module library.

This function will be called by the plug-in runtime environment at termination. In this function, a plug-in module will perform any termination processing required. Typically a plug-in module will unregister any registered plug-in tools and/or data extensions and unload any loaded resources.

PARAMETERS

plugin
the plug-in module being exited

ACCESS LEVEL

Declared by Plug-in Module
