mgMatrixStackMultiply

FUNCTION NAME

mgMatrixStackMultiply - multiplies a matrix into a matrix stack.

SYNOPSIS       (C and Python)

mgbool mgMatrixStackMultiply (mgmatrixstackmatrixStack,
const mgmatrixmatrix );

DESCRIPTION

mgMatrixStackMultiply multiplies the current top-of-stack matrix in the specified matrix stack, matrixStack by a given matrix.

If the top-of-stack matrix is T before this function, the new top-of-stack matrix after this function will be the product matrix X T.

PARAMETERS

matrixStack
matrix stack to apply operation
matrix
matrix to multiply into the top-of-stack

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
