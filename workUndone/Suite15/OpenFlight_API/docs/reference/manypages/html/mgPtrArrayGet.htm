mgPtrArrayGet

FUNCTION NAME

mgPtrArrayGet - get an item from a pointer array.

C SPECIFICATION       (Not Available For Python)

void* mgPtrArrayGet (mgptrarrayptrArray,
intpos );

DESCRIPTION

mgPtrArrayGet returns the item in pointer array ptrArray at position pos.

The first item in a pointer array is at index 1. Therefore, to get the first item of the array, specify pos as 1. To get the item at the end of the array, specify pos as the length of the pointer array.

PARAMETERS

ptrArray
the pointer array
pos
the position of the item in the array to get (in range 1..length)

RETURNS

The item at the specified position if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1
