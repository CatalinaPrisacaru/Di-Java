mgDuplicateToDb

FUNCTION NAME

mgDuplicateToDb - duplicates a node or transformation record in one database for attaching into another database.

SYNOPSIS       (C and Python)

mgrec* mgDuplicateToDb (mgrec*rec,
mgrec*toDb );

DESCRIPTION

Given a node or transformation record rec in one database, mgDuplicateToDb duplicates this record, and in the case of a node record, all of its descendants, returning the new (duplicate) record that can be attached to a node contained in the database toDb. Unlike mgDuplicate, this function allows you to copy nodes from one database to another.

The record returned can be attached ONLY to a node contained in the database toDb. If you specify toDb as MG_NULL or the database that contains rec, it is equivalent to calling mgDuplicate.

Note: This function does not resolve palette indices between the source and destination databases. So if you copy between two databases that have "different" palette entries, the duplicate node (and nodes below) returned will reference palette entries in the destination database.

PARAMETERS

rec
the node to duplicate
toDb
the database where you are going to attach the duplicate

RETURNS

Returns the new node if successful, otherwise MG_NULL.

ACCESS LEVEL

Level 2
