mgInsertTexture

FUNCTION NAME

mgInsertTexture - loads a texture and creates a new entry in a database's texture palette.

SYNOPSIS       (C and Python)

int mgInsertTexture (mgrec*db,
const char*textureName );

DESCRIPTION

mgInsertTexture loads a texture from file textureName into the texture palette for database db. The created palette entry is automatically positioned in the palette such that no other textures are covered. The index of the new texture is also automatically assigned.

PARAMETERS

db
the database
textureName
the name of the texture file to load

RETURNS

Returns the index assigned to the new palette entry if successful, -1 otherwise.

ACCESS LEVEL

Level 1
