mgTabRulerGetSelectedTab

FUNCTION NAME

mgTabRulerGetSelectedTab - returns the index of the selected tab in a tab ruler control.

C SPECIFICATION       (Not Available For Python)

int mgTabRulerGetSelectedTab (mgguicontrol );

DESCRIPTION

mgTabRulerGetSelectedTab returns the index of the selected tab in the tab ruler control that is associated with the GL control control

PARAMETERS

control
tab ruler control

RETURNS

The index of the selected tab is returned, or -1 if no tab is selected.

ACCESS LEVEL

Level 4
