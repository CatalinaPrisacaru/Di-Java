mgSetDbModified

FUNCTION NAME

mgSetDbModified - sets the modified state of a database.

SYNOPSIS       (C and Python)

mgbool mgSetDbModified (mgrec*db,
mgboolmodified );

DESCRIPTION

mgSetDbModified sets or clears the modified state of the database db according to modified. If modified is MG_TRUE, db will be marked as modified (or "dirty"). If modified is MF_FALSE, db will be marked as not modified.

Note: This function is only applicable in the Creator modeler environment. If called from the stand application environment, the results are not reliable.

PARAMETERS

db
the database whose modified state is to be set
modified
the modified state to set on the database

RETURNS

Returns MG_TRUE if the modified state of the database was successfully set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
