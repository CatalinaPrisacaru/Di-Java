mgTextSetEditable

FUNCTION NAME

mgTextSetEditable - enables or disables editing of a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSetEditable (mgguicontrol,
mgbooleditable );

DESCRIPTION

Depending on the value of the parameter editable, mgTextSetEditable either enables (MG_TRUE) or disables (MG_FALSE) the editing of a text control.

Text controls that are editable can be modified by the user. Text controls that are not editable can be used to display "read-only" values.

PARAMETERS

control
the text control
editable
enum mgbool MG_TRUE to allow editing, MG_FALSE to disallow editing

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
