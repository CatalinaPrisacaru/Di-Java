mgFreeConvexHull

FUNCTION NAME

mgFreeConvexHull - frees a convex hull object.

C SPECIFICATION       (Not Available For Python)

void mgFreeConvexHull (mgconvexhullhull );

DESCRIPTION

mgFreeConvexHull frees the dynamic memory associated with the convex hull object hull.

The convex hull is assumed to have been created using either of the functions mgGetConvexHull or mgConvexHullBuilderClose.

PARAMETERS

hull
the convex hull to free

ACCESS LEVEL

Level 1

SEE ALSO

mgGetConvexHull, mgConvexHullBuilderClose
