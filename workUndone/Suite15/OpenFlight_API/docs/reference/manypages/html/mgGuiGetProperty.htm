mgGuiGetProperty

FUNCTION NAME

mgGuiGetProperty - retrieves user defined data associated with a gui item.

C SPECIFICATION       (Not Available For Python)

mgpropertyvalue mgGuiGetProperty (mgguigui,
const mgpropertynamepropName );

DESCRIPTION

Use mgGuiGetProperty to retrieve the data named propName that was associated with the control or dialog gui using mgGuiPutProperty.

PARAMETERS

gui
the gui item to get property for
propName
the name of the property to get

RETURNS

Returns the user defined data if found, MG_NULL otherwise.

ACCESS LEVEL

Level 4

SEE ALSO

mgGuiPutProperty, mgGuiDeleteProperty
