mgMatrixStackLoadIdentity

FUNCTION NAME

mgMatrixStackLoadIdentity - load an identity matrix onto a matrix stack.

SYNOPSIS       (C and Python)

mgbool mgMatrixStackLoadIdentity (mgmatrixstackmatrixStack );

DESCRIPTION

mgMatrixStackLoadIdentity loads an identity matrix onto the specified matrix stack matrixStack. An identity matrix replaces the current top-of-stack matrix in matrixStack.

PARAMETERS

matrixStack
matrix stack to load the identity matrix onto the top-of-stack

ACCESS LEVEL

Level 1
