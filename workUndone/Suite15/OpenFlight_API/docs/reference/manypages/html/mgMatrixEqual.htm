mgMatrixEqual

FUNCTION NAME

mgMatrixEqual - checks to see if two matrices are exactly equal.

SYNOPSIS       (C and Python)

mgbool mgMatrixEqual (const mgmatrixmatrixA,
const mgmatrixmatrixB );

DESCRIPTION

mgMatrixEqual compares two matrices, matrixA and matrixB to see if they are exactly equal.

PARAMETERS

matrixA
first matrix to check for equality
matrixB
second matrix to check for equality

RETURNS

Returns MG_TRUE if matrixA and matrixB are equal, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
