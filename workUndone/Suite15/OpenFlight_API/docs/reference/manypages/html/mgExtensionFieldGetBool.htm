mgExtensionFieldGetBool

FUNCTION NAME

mgExtensionFieldGetBool - gets the boolean value of an extension field on a node.

C SPECIFICATION

mgbool mgExtensionFieldGetBool (mgrec*rec,
const char*fieldGuid,
mgbool*fieldValue );

PYTHON SPECIFICATION

mgbool, fieldValue mgExtensionFieldGetBool (mgrec*rec,
const char*fieldGuid )

DESCRIPTION

mgExtensionFieldGetBool gets the boolean value of an extension field identified by fieldGuid, attached to node rec.

If successful, the value of the extension field is returned in the output parameter fieldValue, otherwise it is undefined.

Note: If the data type of the extension field is not MTYPE_FLAG, this function will attempt to coerce the extension field value to boolean and return the corresponding value in fieldValue. For example, a field of type MTYPE_FLAG will set fieldValue to 0.0 or 1.0. A field of type MTYPE_INT or MTYPE_DOUBLE will simply convert its value to float and set fieldValue to the result. A field of type MTYPE_TEXT or MTYPE_XML will try to "scan" the string for a float value and set fieldValue to the result.

Note: If the data type of the extension field is not MTYPE_FLAG, this function will attempt to coerce the extension field value to boolean and return the corresponding value in fieldValue. For example, a field of type MTYPE_INT, MTYPE_FLOAT or MTYPE_DOUBLE will will set fieldValue to MG_FALSE if the numeric value is 0, MG_TRUE otherwise. A field of type MTYPE_TEXT or MTYPE_XML will try to "scan" the string for "0" and set fieldValue to MG_FALSE if found, MG_TRUE otherwise.

PARAMETERS

rec
the node to get extension value from
fieldGuid
the GUID of the extension field
fieldValue
the address of value to receive field value

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise. This function will fail if fieldGuid does not specify a valid extension field currently attached to rec or fieldValue is MG_NULL.
fieldValue
If function is successful, fieldValue contains the retrieved value, otherwise it will not be defined.

EXAMPLE

The following examples (in C and Python) illustrate the use of mgExtensionFieldGetBool.

C EXAMPLE

mgbool fieldValue = MG_FALSE;
mgbool status;

status = mgExtensionFieldGetBool (rec, "732202BF-3EFA-4451-B8A6-B817DDB7BBAE", &fieldValue)

if (MSTAT_ISOK (status)) {
   printf ("Field Value: %s", fieldValue==MG_TRUE ? "MG_TRUE":"MG_FALSE");
}

PYTHON EXAMPLE

status, fieldValue = mgExtensionFieldGetBool (rec, "732202BF-3EFA-4451-B8A6-B817DDB7BBAE")
if (MSTAT_ISOK (status)):
   print "Field Value:","MG_TRUE" if fieldValue else "MG_FALSE"
