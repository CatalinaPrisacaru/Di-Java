mgGetCoord3d

FUNCTION NAME

mgGetCoord3d - gets values out of a fltCoord3d record.

C SPECIFICATION

mgbool mgGetCoord3d (mgrec*rec,
mgcodecode,
double*x,
double*y,
double*z );

PYTHON SPECIFICATION

mgbool, x, y, z mgGetCoord3d (mgrec*rec,
mgcodecode )

DESCRIPTION

Given a record rec, mgGetCoord3d gets the values of fltCoord3d attribute record code from rec and stores them in x, y, and z.

For retrieving vertex coordinates, it is much more efficient to call the convenience function mgGetVtxCoord.

You do not have to provide addresses for all x, y, and z values. For example, if you only want to receive the x and y components (but not z), specify valid addresses for x and y and leave z NULL.

PARAMETERS

rec
the record from which to get the fltCoord3d attribute record
code
the fltCoord3d attribute record code
x
address of value to receive x component
y
address of value to receive y component
z
address of value to receive z component

RETURNS

mgbool
MG_TRUE if the coord was retrieved successfully, MG_FALSE otherwise.
x
If function is successful, x contains the retrieved x component, otherwise x is undefined.
y
If function is successful, y contains the retrieved y component, otherwise y is undefined.
z
If function is successful, z contains the retrieved z component, otherwise z is undefined.

C EXAMPLE

mgrec* db;
mgrec* lodRec;
double x, y, z;

db = mgOpenDb ("file.flt");

// create lod rec
lodRec = UserMakeLODRec (db);

mgGetCoord3d (lodRec, fltLodCenterPoint, &x, &y, &z);

ACCESS LEVEL

Level 1
