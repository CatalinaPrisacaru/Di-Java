mgGetMatrix

FUNCTION NAME

mgGetMatrix - gets values out of a fltMatrix record.

C SPECIFICATION

mgbool mgGetMatrix (mgrec*rec,
mgcodematrixCode,
mgmatrix*matrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgGetMatrix (mgrec*rec,
mgcodematrixCode )

DESCRIPTION

Given a record rec which contains a fltMatrix record identified by the record code matrixCode, mgGetMatrix copies the corresponding matrix of rec into matrix.

C NOTES

Note: Memory is not allocated for matrix by this function.

PARAMETERS

rec
a node or record containing a fltMatrix record
matrixCode
the record code of the fltMatrix to retrieve
matrix
the address of the matrix to receive values

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE if rec has no such matrix or if otherwise unsuccessful.
mgmatrix
the corresponding matrix of rec.

ACCESS LEVEL

Level 1
