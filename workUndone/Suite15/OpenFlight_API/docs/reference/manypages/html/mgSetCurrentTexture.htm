mgSetCurrentTexture

FUNCTION NAME

mgSetCurrentTexture - sets the current modeling texture index.

SYNOPSIS       (C and Python)

mgbool mgSetCurrentTexture (mgrec*db,
intindex );

DESCRIPTION

mgSetCurrentTexture sets the current modeling texture palette index for database db.

PARAMETERS

db
the database
index
index to set current modeling texture

RETURNS

Returns MG_TRUE if current modeling texture for db could be set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
