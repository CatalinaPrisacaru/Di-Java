mgListGetSelectedItemString

FUNCTION NAME

mgListGetSelectedItemString - retrieves text of selected item in a list control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgListGetSelectedItemString (mgguicontrol,
char*itemString,
intmaxLen );

DESCRIPTION

mgListGetSelectedItemString retrieves the text string of the selected item in the specified list control.

For multi-select list controls with more than one item currently selected, the string retrieved is the text of the first item selected.

The string is truncated and null terminated if it is longer than maxLen characters.

PARAMETERS

control
the list control
itemString
character buffer to hold text of selected item retrieved
maxLen
the maximum number of characters to store in itemString

RETURNS

Use MSTAT_ISOK to check return value for success. If function is successful, itemString is loaded with the text of the selected item. Otherwise, itemString is undefined.

ACCESS LEVEL

Level 4
