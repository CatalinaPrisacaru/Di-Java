mgGetCurrentAltColor

FUNCTION NAME

mgGetCurrentAltColor - returns color index and intensity for the current alternate modeling color.

C SPECIFICATION

mgbool mgGetCurrentAltColor (mgrec*db,
unsigned int*index,
float*intensity );

PYTHON SPECIFICATION

mgbool, index, intensity mgGetCurrentAltColor (mgrec*db )

DESCRIPTION

mgGetCurrentAltColor returns the color palette index and intensity corresponding to the current alternate modeling color selected for database db.

PARAMETERS

db
the database
index
address of value to receive index of current alternate modeling color
intensity
address of value to receive intensity of current alternate modeling color

RETURNS

mgbool
MG_TRUE if current alternate modeling color for db could be determined, MG_FALSE otherwise.
index
If function is successful, index contains the retrieved color index, otherwise index is undefined.
intensity
If function is successful, intensity contains the retrieved color intensity, otherwise intensity is undefined.

ACCESS LEVEL

Level 4
