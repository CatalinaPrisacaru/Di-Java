mgSetEyePoint

FUNCTION NAME

mgSetEyePoint - stores an eyepoint record into the database.

SYNOPSIS       (C and Python)

void mgSetEyePoint (mgrec*db,
intindex,
mgrec*eyeRec );

DESCRIPTION

mgSetEyePoint stores the eyepoint record eyeRec in the position defined by index in the database db, index must be between 1 and 9. Eyepoint 0 is reserved for use by Presagis.

The eyeRec you pass here must have been created using mgGetNewEyePoint.

Note: The attribute fltEyeActive must be set for the eyepoint to be considered valid.

PARAMETERS

db
the database record
index
index into the eyepoint palette
eyeRec
a fltEyePoint record

ACCESS LEVEL

Level 2
