mgvertexreftype

ENUM NAME

mgvertexreftype - 3D Vertex Mouse Input Reference type

SYNOPSIS

typedef enum mgvertexreftype {
   MVRF_VERTEX,
   MVRF_EDGE,
   MVRF_CURVE,
   MVRF_TRACKPLANE,
   MVRF_UNKNOWN,
   MVRF_FACE,
} mgvertexreftype;

MEMBERS

MVRF_VERTEX
Vertex input references existing vertex node
MVRF_EDGE
Vertex input references existing edge
MVRF_CURVE
Vertex input references existing curve node
MVRF_TRACKPLANE
Vertex input references trackplane
MVRF_UNKNOWN
Unknown vertex input reference
MVRF_FACE
Vertex input references existing face node
