mgFreeProjection

FUNCTION NAME

mgFreeProjection - frees a projection object.

SYNOPSIS       (C and Python)

void mgFreeProjection (mgprojectionprojection );

DESCRIPTION

mgFreeProjection disposes of the dynamic memory allocated for the projection object created by mgNewProjection.

PARAMETERS

projection
the projection object to free

ACCESS LEVEL

Level 1
