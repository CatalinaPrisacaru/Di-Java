mgHasTextureTransparentValue

FUNCTION NAME

mgHasTextureTransparentValue - gets transparent value flag attribute from an image info object.

C SPECIFICATION

mgbool mgHasTextureTransparentValue (mgimageinfotextureInfo,
mgbool*hasTransparentValue );

PYTHON SPECIFICATION

mgbool, hasTransparentValue mgHasTextureTransparentValue (mgimageinfotextureInfo )

DESCRIPTION

mgHasTextureTransparentValue gets the transparent value flag attribute from the specified image info object textureInfo. The transparent value flag specifies whether or not the image has a value marked as transparent.

PARAMETERS

textureInfo
the image info object
hasTransparentValue
address of boolean to receive transparent value flag attribute

RETURNS

mgbool
MG_TRUE if value retrieved successfully, MG_FALSE otherwise.
hasTransparentValue
If function is successful, hasTransparentValue contains the retrieved value, otherwise hasTransparentValue is undefined.

ACCESS LEVEL

Level 1
