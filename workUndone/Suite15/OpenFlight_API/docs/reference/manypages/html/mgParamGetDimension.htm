mgParamGetDimension

FUNCTION NAME

mgParamGetDimension - gets the maximum dimension of a parameter in a parameter block.

SYNOPSIS       (C and Python)

unsigned int mgParamGetDimension (mgparamblockparamBlock,
const char*paramName );

DESCRIPTION

mgParamGetDimension gets the maximum dimension allowed for the parameter named paramValue in the parameter block paramBlock.

When a parameter is defined, it is assigned a maximum dimension. For most parameters, this value is 1, meaning that the parameter is one single value. For array type parameters, the maximum dimension is some number greater than 1 or is 0. If the maximum dimension is greater than 1, the value defines the finite number of values the array type parameter can contain. If the maximum dimension is 0, the parameter is an array type but does not define the maximum number of values it can contain.

Note that mgParamGetDimension returns the maximum number of values a parameter can contain and mgParamGetSize returns the number of values a parameter currently does contain.

PARAMETERS

paramBlock
the parameter block
paramName
the parameter name

RETURNS

Returns the maximum dimension of the parameter named paramValue. 0 indicates that the array type parameter can accept an unlimited number of values.

ACCESS LEVEL

Level 4
