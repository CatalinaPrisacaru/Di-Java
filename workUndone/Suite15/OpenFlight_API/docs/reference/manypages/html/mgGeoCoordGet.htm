mgGeoCoordGet

FUNCTION NAME

mgGeoCoordGet - gets the geospecific coordinates from a texture attribute record.

C SPECIFICATION

int mgGeoCoordGet (mgrec*imgRec,
mggeocoorddatageoCoordData[],
intarrayLen );

PYTHON SPECIFICATION

numGeoCoords, geoCoordData mgGeoCoordGet (mgrec*imgRec )

DESCRIPTION

mgGeoCoordGet gets the geospecific coordinates of the specified texture attribute record imgRec (of type fltImage ).

C NOTES

mgGeoCoordGet writes the data into the output parameter subTextureData. The size of the array is specified by arrayLen. The number of geospecific coordinates actually written into the array is returned. You can determine the actual number of geospecific coordinates contained in a texture attribute record by querying the attribute fltTGNumCoords .

PYTHON NOTES

mgGeoCoordGet returns the geospecific coordinate data. The number of geospecific coordinates included is also returned.

PARAMETERS

imgRec
the texture attribute record
geoCoordData[]
the array of geospecific coordinates to be filled in
arrayLen
the size of the array geoCoordData

RETURNS

numGeoCoords
The number of geospecific coordinate records being returned.
geoCoordData
The records returned.

C EXAMPLE

mgrec* imgRec = mgGetTextureAttributes (db, textureIndex);
int numCoords = mgGeoCoordCount (imgRec);
if (numCoords > 0) {
   int num;
   mggeocoorddata gcData[10];
   num = mgGeoCoordGet (imgRec, gcData, 10);
   // num will be minimum of 10 and numCoords
}

PYTHON EXAMPLE

imgRec = mgGetTextureAttributes (db, textureIndex)
num,geoCoords = mgGeoCoordGet (imgRec)
print "Number of Geo Coords:",num
for i in range (0, num):
   print "GeoCoord",i+1,":"
   geoCoord = geoCoords[i]
   print "   u,v,lat,lon:",geoCoord.u,geoCoord.v,geoCoord.lat,geoCoord.lon

Level 1
