mgrec

TYPE NAME

mgrec - Generic type used to access any type of record accessible by the OpenFlight API

DESCRIPTION

OpenFlight API users do not need to manipulate the internal fields of this structure directly; simply pass the structure into API functions instead. Variables of type mgrec are referred to as records.
