mgCoord2dRotateCCW

FUNCTION NAME

mgCoord2dRotateCCW - calculates 2D coordinates rotated counterclockwise by a specified angle.

SYNOPSIS       (C and Python)

mgcoord2d mgCoord2dRotateCCW (const mgcoord2d*coord,
doubledegrees );

DESCRIPTION

mgCoord2dRotateCCW calculates and returns 2D double precision floating point coordinates rotated counterclockwise by a specified angle.

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

coord
address of coordinates
degrees
angle in degrees

RETURNS

Returns a double precision 2D coordinate record.

ACCESS LEVEL

Level 1
