mgGetLicenseCustomerID

FUNCTION NAME

mgGetLicenseCustomerID - gets the customer id associated with a license.

C SPECIFICATION       (Not Available For Python)

char* mgGetLicenseCustomerID (mglicenselicense );

DESCRIPTION

mgGetLicenseCustomerID gets the customer id associated with a license that has been previously checked out using mgCheckoutLicense.

The character string returned is a dynamically allocated copy of the customer id. It is the responsibility of the caller to deallocate this string when it is no longer needed using the function mgFree.

PARAMETERS

license
the license to get the customer id for

RETURNS

Returns the customer id if the license is valid and is currently checked out, otherwise MG_NULL.

ACCESS LEVEL

Level 4

SEE ALSO

mgNewLicense, mgCheckoutLicense, mgCheckinLicense, mgGetLicenseVendorString
