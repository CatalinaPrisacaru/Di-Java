mgCoord3dToCoord3f

FUNCTION NAME

mgCoord3dToCoord3f - converts a 3D coordinate from double to single precision floating point.

C SPECIFICATION       (Not Available For Python)

mgcoord3f mgCoord3dToCoord3f (const mgcoord3d*coord );

DESCRIPTION

mgCoord3dToCoord3f converts the specified 3D double precision floating point coordinate to a 3D single precision floating point coordinate.

The original coordinate coord is not affected.

PARAMETERS

coord
address of double precision 3D coordinate to convert

RETURNS

Returns a single precision coordinate record.

ACCESS LEVEL

Level 1
