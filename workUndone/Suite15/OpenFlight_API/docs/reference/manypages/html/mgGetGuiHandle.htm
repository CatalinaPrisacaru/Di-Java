mgGetGuiHandle

FUNCTION NAME

mgGetGuiHandle - return a native GUI handle.

C SPECIFICATION       (Not Available For Python)

mgguihandle mgGetGuiHandle (mgguigui );

DESCRIPTION

mgGetGuiHandle returns a platform specific handle to the specified dialog or control gui.

PARAMETERS

gui
abstract dialog or control to get native handle for

RETURNS

On Windows, this function returns an object of type HWND.

ACCESS LEVEL

Level 4

SEE ALSO

mgFindGuiById, mgResourceGetDialog
