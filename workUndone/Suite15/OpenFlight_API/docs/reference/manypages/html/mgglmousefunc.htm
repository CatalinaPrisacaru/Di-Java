mgglmousefunc

CALLBACK NAME

mgglmousefunc - GL mouse function

SYNOPSIS

mgstatus (*mgglmousefunc) ( mggui control,
mgcontrolid controlId,
mgglmouseaction mouseAction,
void* userData,
void* callData );

DESCRIPTION

This is the signature for GL mouse functions. If a GL control is to accept mouse input from the user, you will provide a GL mouse function of this form that will be called to notify your GL control of the actions from the user via the mouse.

PARAMETERS

control
the gl control
controlId
the identifier of the control
mouseAction
the mouse action that triggered the function
userData
user defined data specified when function assigned to control
callData
callback specific data - you can always safely cast this pointer to a pointer to an object of type mgglmousedatarec to determine the type of call data this object really is

RETURNS

Currently, the value returned by mgglmousefunc is ignored and reserved for future enhancement. For now always return MSTAT_OK.
