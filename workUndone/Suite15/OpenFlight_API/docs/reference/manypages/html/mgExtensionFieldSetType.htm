mgExtensionFieldSetType

FUNCTION NAME

mgExtensionFieldSetType - sets the data type of an extension field.

SYNOPSIS       (C and Python)

mgbool mgExtensionFieldSetType (const char*fieldGuid,
mgtagtypetype );

DESCRIPTION

mgExtensionFieldSetType sets the data type of an extension field identified by fieldGuid. The data type of an extension field identifies the kind of data this extension stores. A data type of MTYPE_INT, for example, indicates that an extension stores an integer value.

Note that although the enumeration type mgtagtype contains many more enumerants, extension fields can be only the following types:
MTYPE_INT
MTYPE_FLOAT
MTYPE_DOUBLE
MTYPE_TEXT
MTYPE_XML
MTYPE_FLAG

PARAMETERS

fieldGuid
the GUID of the site
type
the data type to assign

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
