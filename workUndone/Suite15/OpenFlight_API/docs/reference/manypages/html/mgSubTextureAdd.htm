mgSubTextureAdd

FUNCTION NAME

mgSubTextureAdd - adds subtexture definitions to a texture attribute record.

SYNOPSIS       (C and Python)

int mgSubTextureAdd (mgrec*imgRec,
mgsubtexturedatasubTextureData[],
intn );

DESCRIPTION

mgSubTextureAdd adds n subtexture definitions to the specified texture attribute record imgRec (of type fltImage ). The subtexture definitions are passed via the array subTextureData. The number of subtextures actually added is returned.

PARAMETERS

imgRec
the texture attribute record
subTextureData[]
the array of subtexture definitions to be added
n
the number of subtextures in the array subTextureData

RETURNS

Returns the number of subtextures added.

C EXAMPLE

mgrec* imgRec = mgGetTextureAttributes (db, textureIndex);
mgsubtexturedata subtextures[2];

// set the data for each element of the sequence
strcpy (subtextures[0].name, "Subtexture 1");
subtextures[0].l = 0;
subtextures[0].b = 0;
subtextures[0].r = 31;
subtextures[0].t = 63;

strcpy (subtextures[1].name, "Subtexture 2");
subtextures[1].l = 32;
subtextures[1].b = 0;
subtextures[1].r = 63;
subtextures[1].t = 63;

// add these subtextures to the existing subtextures
mgSubTextureAdd (imgRec, subtextures, 2);

PYTHON EXAMPLE

imgRec = mgGetTextureAttributes (db, textureIndex)
# allocate 2 elements for the subtexture data
subtextures = mgsubtexturedata(2)

# set the data for each subtexture
subtextures[0].name = "Subtexture 1"
subtextures[0].l = 0
subtextures[0].b = 0
subtextures[0].r = 31
subtextures[0].t = 63

subtextures[1].name = "Subtexture 2"
subtextures[1].l = 32
subtextures[1].b = 0
subtextures[1].r = 63
subtextures[1].t = 63

# add these subtextures to the existing subtextures
mgSubTextureAdd (imgRec, subtextures, 2)
