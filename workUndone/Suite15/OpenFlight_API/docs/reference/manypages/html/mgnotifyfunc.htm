mgnotifyfunc

CALLBACK NAME

mgnotifyfunc - Notifier callback function

SYNOPSIS

void (*mgnotifyfunc) ( mgnotifier notifier,
mgnotifierevent event,
mgrec* db,
mgrec* rec,
void* userData );

PARAMETERS

notifier
the notifier
event
the notification event that triggered the callback
db
the database to which or in which the event occurred
rec
the specific node in the database to which the event occurred
userData
user defined data specified when notifier registered
