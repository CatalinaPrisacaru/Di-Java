mgGetCurrentLookAt

FUNCTION NAME

mgGetCurrentLookAt - returns the current eyepoint in "look at" form.

C SPECIFICATION

mgbool mgGetCurrentLookAt (mgrec*db,
double*eyex,
double*eyey,
double*eyez,
double*centerx,
double*centery,
double*centerz,
double*upx,
double*upy,
double*upz );

PYTHON SPECIFICATION

mgbool, eyex, eyey, eyez, centerx, centery, centerz, upx, upy, upz mgGetCurrentLookAt (mgrec*db )

DESCRIPTION

mgGetCurrentLookAt returns the current eyepoint using "look at" style parameters. The "look at" parameters include 3 sets of values. The first set, eyex, eyey, and eyez is the position of the eye point (the "look from"). The second set, centerx, centery, centerz is the position of the reference point (the "look at"). The third set, upx, upy, and upz is the direction of the up vector.

PARAMETERS

db
the database
eyex
address of value to receive x component of the eyepoint position
eyey
address of value to receive y component of the eyepoint position
eyez
address of value to receive z component of the eyepoint position
centerx
address of value to receive x component of the reference point
centery
address of value to receive y component of the reference point
centerz
address of value to receive z component of the reference point
upx
address of value to receive i component of the up vector
upy
address of value to receive j component of the up vector
upz
address of value to receive k component of the up vector

RETURNS

mgbool
MG_TRUE if current "look at" selected for db could be determined, MG_FALSE otherwise.
eyex
If function is successful, eyex contains the retrieved eyepoint x, otherwise eyex is undefined.
eyey
If function is successful, eyey contains the retrieved eyepoint y, otherwise eyey is undefined.
eyez
If function is successful, eyez contains the retrieved eyepoint z, otherwise eyez is undefined.
centerx
If function is successful, centerx contains the retrieved reference x, otherwise centerx is undefined.
centery
If function is successful, centery contains the retrieved reference y, otherwise centery is undefined.
centerz
If function is successful, centerz contains the retrieved reference z, otherwise centerz is undefined.
upx
If function is successful, upx contains the retrieved up vector x, otherwise upx is undefined.
upy
If function is successful, upy contains the retrieved up vector y, otherwise upy is undefined.
upz
If function is successful, upz contains the retrieved up vector z, otherwise upz is undefined.

ACCESS LEVEL

Level 4
