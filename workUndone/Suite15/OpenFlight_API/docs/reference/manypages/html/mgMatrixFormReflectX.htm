mgMatrixFormReflectX

FUNCTION NAME

mgMatrixFormReflectX - creates a matrix that will reflect about the X axis.

C SPECIFICATION

void mgMatrixFormReflectX (mgmatrix*matrix );

PYTHON SPECIFICATION

mgmatrix mgMatrixFormReflectX ( )

DESCRIPTION

mgMatrixFormReflectX gets a matrix that will reflect about the X axis or pitch 180 degrees.

C NOTES

This function writes the resulting matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the resulting matrix if successful.

PARAMETERS

matrix
address of matrix to receive result

RETURNS

mgmatrix
the matrix.

ACCESS LEVEL

Level 1
