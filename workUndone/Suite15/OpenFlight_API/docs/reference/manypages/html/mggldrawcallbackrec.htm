mggldrawcallbackrec

STRUCT NAME

mggldrawcallbackrec - callback structure for the draw control event sent to GL controls

SYNOPSIS

typedef struct  {
   mgguicalldatatype callDataType;
   int width;
   int height;
   mgglcontext glContext;
} mggldrawcallbackrec;

MEMBERS

callDataType
mgguicalldatatype - call data type that indicates the type of call data this object really is - will always be MGCD_GLDRAW
width
int - width, in pixels of the GL control
height
int - height, in pixels of the GL control
glContext
mgglcontext - rendering context associated with GL control

DESCRIPTION

When the MGCB_DRAW control event is sent to a GL control via its mgguifunc, the calldata parameter passed in will point to an object of this type.
