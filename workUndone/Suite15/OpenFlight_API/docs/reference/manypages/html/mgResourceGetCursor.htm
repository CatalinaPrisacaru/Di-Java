mgResourceGetCursor

FUNCTION NAME

mgResourceGetCursor - retrieves a cursor from a resource.

C SPECIFICATION       (Not Available For Python)

mgcursor mgResourceGetCursor (mgresourceresource,
mgcursoridcursorId );

DESCRIPTION

mgResourceGetCursor retrieves a cursor corresponding to a specified cursor identifier cursorId from the resource.

The OpenFlight API provides several predefined cursors. To use one of these predefined cursors, set resource to MG_NULL and set cursorId to one of the following values:

MCURS_ARROW
MCURS_WAIT
MCURS_CROSS
MCURS_NO
MCURS_SIZEALL
MCURS_SIZENESW
MCURS_SIZENS
MCURS_SIZENWSE
MCURS_SIZEWE
MCURS_UPARROW
MCURS_ZOOMIN
MCURS_ZOOMOUT
MCURS_SPIN
MCURS_PEN
MCURS_BUCKET
MCURS_EYEDROPPER
MCURS_SPLITH
MCURS_SPLITV

PARAMETERS

resource
the resource containing the cursor
cursorId
the cursor identifier

RETURNS

If the cursor is found in the resource, the abstract representation of the cursor is returned, MG_NULL otherwise.

ACCESS LEVEL

Level 4
