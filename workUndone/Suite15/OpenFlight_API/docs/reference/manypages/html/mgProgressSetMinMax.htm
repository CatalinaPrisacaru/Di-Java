mgProgressSetMinMax

FUNCTION NAME

mgProgressSetMinMax - sets the minimum and maximum logical values for a progress control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgProgressSetMinMax (mgguicontrol,
doublemin,
doublemax );

DESCRIPTION

mgProgressSetMinMax sets the range of logical values valid for the specified progress control. The minimum value is specified by min, the maximum by max.

The default minimum and maximum values for a progress control are 0 and 100 respectively.

If the current progress value is outside the new range, the progress value will be clamped to the new maximum or minimum value.

PARAMETERS

control
the progress control
min
the minimum value for the control
max
the maximum value for the control

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
