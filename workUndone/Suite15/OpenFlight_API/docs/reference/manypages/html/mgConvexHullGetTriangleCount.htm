mgConvexHullGetTriangleCount

FUNCTION NAME

mgConvexHullGetTriangleCount - gets the number of triangles comprising a convex hull.

SYNOPSIS       (C and Python)

int mgConvexHullGetTriangleCount (mgconvexhullhull );

DESCRIPTION

mgConvexHullGetTriangleCount returns the number of triangles that are contained in the specified convex hull.

PARAMETERS

hull
the convex hull

RETURNS

Returns the number of triangles in the convex hull.

EXAMPLE

See mgGetConvexHull for an example of how to build and examine a convex hull.

ACCESS LEVEL

Level 1
