mgGetColorRGBA

FUNCTION NAME

mgGetColorRGBA - gets values out of a fltColorRGBA record.

C SPECIFICATION

mgbool mgGetColorRGBA (mgrec*rec,
mgcodecolorRGBACode,
float*red,
float*green,
float*blue,
float*alpha );

PYTHON SPECIFICATION

mgbool, red, green, blue, alpha mgGetColorRGBA (mgrec*rec,
mgcodecolorRGBACode )

DESCRIPTION

Given a record, rec, mgGetColorRGBA gets the values of fltColorRGBA attribute record colorRGBACode from rec and stores them in red, green, blue and alpha.

PARAMETERS

rec
the record from which to get the fltColorRGBA attribute code
colorRGBACode
the fltColorRGBA attribute code
red
address of value to receive red component
green
address of value to receive green component
blue
address of value to receive blue component
alpha
address of value to receive alpha component

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
red
If function is successful, red contains the corresponding numeric value, otherwise red is undefined.
green
If function is successful, green contains the corresponding numeric value, otherwise green is undefined.
blue
If function is successful, blue contains the corresponding numeric value, otherwise blue is undefined.
alpha
If function is successful, alpha contains the corresponding numeric value, otherwise alpha is undefined.

C EXAMPLE

mgrec* db;
mgrec* ltsRec;
float r, g, b, a;
int index;

db = mgOpenDb ("file.flt");
ltsRec = mgGetFirstLightSource (db, &index);
mgGetColorRGBA (ltsRec, fltLtspAmbient, &r, &g, &b, &a);

ACCESS LEVEL

Level 1
