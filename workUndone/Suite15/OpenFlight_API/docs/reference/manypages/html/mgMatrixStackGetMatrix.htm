mgMatrixStackGetMatrix

FUNCTION NAME

mgMatrixStackGetMatrix - get the current top-of-stack matrix from a matrix stack.

C SPECIFICATION

mgbool mgMatrixStackGetMatrix (const mgmatrixstackmatrixStack,
mgmatrix*matrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixStackGetMatrix (mgmatrixstackmatrixStack )

DESCRIPTION

mgMatrixStackGetMatrix gets the current top-of-stack matrix from a specified matrix stack matrixStack. The top-of-stack matrix is returned in the ouput parameter matrix.

C NOTES

This function writes the top-of-stack matrix into the output parameter matrix if successful.

PYTHON NOTES

This function returns the top-of-stack matrix if successful.

PARAMETERS

matrixStack
matrix stack to get top matrix from
matrix
address of matrix to receive top matrix

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix.

ACCESS LEVEL

Level 1
