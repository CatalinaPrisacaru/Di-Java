ddNameToCode

FUNCTION NAME

ddNameToCode - gets the code of a record or data field with a given name.

SYNOPSIS       (C and Python)

mgcode ddNameToCode (const char*name );

DESCRIPTION

ddNameToCode gets the code associated with a record or data field with the specified name.

PARAMETERS

name
the record or data field name

RETURNS

Returns the code of the record or field if found, 0 otherwise.

ACCESS LEVEL

Level 1
