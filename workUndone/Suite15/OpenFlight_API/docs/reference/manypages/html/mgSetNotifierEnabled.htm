mgSetNotifierEnabled

FUNCTION NAME

mgSetNotifierEnabled - enables or disables a notifier.

C SPECIFICATION       (Not Available For Python)

void mgSetNotifierEnabled (mgnotifiernotifier,
mgboolenabled );

DESCRIPTION

Depending on the value of the parameter enabled, mgSetNotifierEnabled either enables (MG_TRUE) or disables (MG_FALSE) the specified notifier.

A notifier that is enabled reports events; one that is disabled does not.

PARAMETERS

notifier
the notifier to enable/disable
enabled
enum mgbool MG_TRUE to enable notifier, MG_FALSE to disable it

ACCESS LEVEL

Level 4
