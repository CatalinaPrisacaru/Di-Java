mgguicalldatatype

ENUM NAME

mgguicalldatatype - GUI callback data type

SYNOPSIS

typedef enum mgguicalldatatype {
   MGCD_COMMON,
   MGCD_TEXTACTIVATE,
   MGCD_SCALEACTIVATE,
   MGCD_GLDRAW,
   MGCD_DIALOGSIZE,
   MGCD_TABRULERACTIVATE,
   MGCD_LISTACTIVATE,
} mgguicalldatatype;

MEMBERS

MGCD_COMMON
Common Call Data. This indicates that the callData parameter to a control callback or dialog function is a pointer to an object of type mgguicallbackrec and has no additional data.
MGCD_TEXTACTIVATE
Text Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mgtextactivatecallbackrec.
MGCD_SCALEACTIVATE
Scale Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mgscaleactivatecallbackrec.
MGCD_GLDRAW
GL Draw Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mggldrawcallbackrec.
MGCD_DIALOGSIZE
Dialog Size Call Data. This indicates that the callData parameter to a dialog function is a pointer to an object of type mgdialogsizecallbackrec.
MGCD_TABRULERACTIVATE
Tab ruler Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mgtabruleractivatecallbackrec.
MGCD_LISTACTIVATE
List Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mglistactivatecallbackrec.

DESCRIPTION

This type is used to indicate the type of call data passed to control callback and dialog functions. When a control callback or dialog function is called, the callData parameter passed in will be a pointer to an object type that is dependent on the context of the callback function.

Each member of this type corresponds to a different callback data structure.
