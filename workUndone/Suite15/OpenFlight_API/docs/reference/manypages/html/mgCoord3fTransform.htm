mgCoord3fTransform

FUNCTION NAME

mgCoord3fTransform - transforms a single precision coordinate using a specified matrix.

SYNOPSIS       (C and Python)

mgcoord3d mgCoord3fTransform (const mgmatrixmatrix,
const mgcoord3f*coord );

DESCRIPTION

mgCoord3fTransform applies the specified matrix to the specified 3D single precision floating point coordinate coord and returns the resulting coordinate.

The original coordinate coord is not affected.

Note: For efficiency, matrix and coord are not verified before being dereferenced. You must not pass either of these as MG_NULL.

PARAMETERS

matrix
matrix to apply to coord
coord
coordinate to transform

RETURNS

Returns the transformed single precision 3D coordinate record.

ACCESS LEVEL

Level 1
