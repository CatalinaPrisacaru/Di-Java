mgListSelectItem

FUNCTION NAME

mgListSelectItem - selects an item in a list control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgListSelectItem (mgguicontrol,
const char*itemString );

DESCRIPTION

mgListSelectItem selects the first item in the specified list control that matches the specified text string itemString.

The MGCB_ACTIVATE control event is not sent when this function is called.

PARAMETERS

control
the list control
itemString
the text of the item to select

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4

SEE ALSO

mgListSelectItemAtPos, mgListSelectAllItems
