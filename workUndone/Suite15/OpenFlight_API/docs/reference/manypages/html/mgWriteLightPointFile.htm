mgWriteLightPointFile

FUNCTION NAME

mgWriteLightPointFile - writes a database’s light point appearance and animation palettes to a file.

SYNOPSIS       (C and Python)

mgbool mgWriteLightPointFile (mgrec*db,
const char*fileName );

DESCRIPTION

mgWriteLightPointFile writes the light point appearance and animation palettes for database db to a file named fileName.

The resulting file can be read into another database using mgReadLightPointFile or via the Load Palette command in the light point palette in Creator.

Note that even though the light point appearance and animation palettes are really separate, this function, as well as mgReadLightPointFile groups the entries in both into one single file.

PARAMETERS

db
the database that contains the light point appearance and animation palettes to write
fileName
the light point palette file name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
