mgPluginGetProperty

FUNCTION NAME

mgPluginGetProperty - retrieves user defined data associated with a plug-in module.

C SPECIFICATION       (Not Available For Python)

mgpropertyvalue mgPluginGetProperty (mgpluginplugin,
const mgpropertynamepropName );

DESCRIPTION

Use mgPluginGetProperty to retrieve the data named propName that was associated with the plug-in module plugin using mgPluginPutProperty.

PARAMETERS

plugin
the plug-in module
propName
the name of the property to get

RETURNS

Returns the user-defined data if found, MG_NULL otherwise.

ACCESS LEVEL

Level 3

SEE ALSO

mgPluginPutProperty, mgPluginDeleteProperty, mgPluginToolPutProperty, mgPluginToolGetProperty, mgPluginToolDeleteProperty
