mgGLColorIndex

FUNCTION NAME

mgGLColorIndex - specifies a color for GL rendering.

SYNOPSIS       (C and Python)

mgbool mgGLColorIndex (mgglcontextglContext,
mgrec*db,
unsigned intindex,
floatintensity );

DESCRIPTION

mgGLColorIndex sets the current color for GL rendering to that specified by palette index and intensity, in database db.

This function loads the red, green and blue color components associated with the specified color in preparation for rendering into GL controls. It is the equivalent of calling the OpenGL routine glColor3*().

PARAMETERS

glContext
Gl rendering context into which color is to be loaded
db
database whose color palette defines the color to load
index
index of color in database's palette to load
intensity
intensity of color to load

RETURNS

Returns MG_TRUE if color could be loaded successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
