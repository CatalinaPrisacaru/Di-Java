mgRegisterSite

FUNCTION NAME

mgRegisterSite - registers a data extension site.

C SPECIFICATION       (Not Available For Python)

mgpluginsite mgRegisterSite (mgpluginplugin,
char*siteId,
record_def*recordDefTbl,
mgcode*recordCodeMap,
intrecordOffset,
intrecordMax,
data_def*dataDefTbl,
mgcode*dataCodeMap,
intdataOffset,
intdataMax,
... );

DESCRIPTION

mgRegisterSite registers a data extension site for the specified plug-in module plugin with the specified extension site attributes.

The required extension site attributes include the name of the site siteId and several attributes that describe the contents of the data extension. The values provides for these attributes are contained in the auto-generated header files for your data extension.

recordDefTbl - &RecordDefTable[0]
recordCodeMap - [Prefix]RecordCodeMap
recordOffset - [PREFIX]OP_RECORD_OFFSET
recordMax - [PREFIX]_RECORD_MAX
dataDefTbl - &DataDefTable[0]
dataCodeMap - [Prefix]DataCodeMap
dataOffset - [PREFIX]OP_DATA_OFFSET
dataMax - [PREFIX]_DATA_MAX

[Prefix] is your site's prefix with the first letter capitalized, and [PREFIX] is your site's prefix with all letters capitalized.

Additional optional extension site attributes include MTA_VERSION and MTA_HELPCONTEXT.

All additional (required and optional) extension site attributes are passed using variable argument style and must be terminated with MG_NULL.

PARAMETERS

plugin
the plug-in module registering the data extension
siteId
the name of the extension site
recordDefTbl
the address of the record definition table for the site
recordCodeMap
the address of the record code map for the site
recordOffset
the index of the first record defined by the site
recordMax
the number of records defined by the site
dataDefTbl
the address of the data definition table for the site
dataCodeMap
the address of the data code map for the site
dataOffset
the index of the first data field defined by the site
dataMax
the number of data fields defined by the site
...
additional extension site attributes in variable argument style

RETURNS

Returns the data extension site registered if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 3
