mgNewDb

FUNCTION NAME

mgNewDb - creates a new database.

SYNOPSIS       (C and Python)

mgrec* mgNewDb (const char*fileName );

DESCRIPTION

mgNewDb creates a new database with the specified file name fileName. The new database consists of a database node and two default group nodes g1 and g2 as shown here:

The database node is returned by this function.

The behavior of mgNewDb is affected by the function: mgSetNewOverwriteFlag.

Note: If your plug-in or script opens a file in the Creator modeling environment with this function or mgOpenDb, the file is not opened in an active window on the Creator desktop. Your plug-in or script can still access the database but must explicitly save and close the file when done.

Note: If invoked from within the modeler environment, this function will fail if the named file is already open in an active window.

PARAMETERS

fileName
the name of the new database file

RETURNS

Returns a pointer to the new database node if successful, MG_NULL otherwise.

C EXAMPLE

The following example creates a new database.
mgrec* db;
db = mgNewDb ("newfile.flt");
// note that child of db is g1 and child of g1 is g2.

ACCESS LEVEL

Level 1
