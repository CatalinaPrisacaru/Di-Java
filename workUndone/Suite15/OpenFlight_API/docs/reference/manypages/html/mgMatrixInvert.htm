mgMatrixInvert

FUNCTION NAME

mgMatrixInvert - forms the inverse of a matrix.

C SPECIFICATION

mgbool mgMatrixInvert (const mgmatrixmatrix,
mgmatrix*resultMatrix );

PYTHON SPECIFICATION

mgbool, mgmatrix mgMatrixInvert (mgmatrixmatrix )

DESCRIPTION

mgMatrixInvert inverts the specified matrix.

C NOTES

This function writes the inverse matrix into the output parameter resultMatrix if successful.

PYTHON NOTES

This function returns the inverse matrix if successful.

PARAMETERS

matrix
matrix to invert
resultMatrix
address of matrix to receive result

RETURNS

mgbool
MG_TRUE if successful, MG_FALSE otherwise.
mgmatrix
the matrix inverse.

ACCESS LEVEL

Level 1
