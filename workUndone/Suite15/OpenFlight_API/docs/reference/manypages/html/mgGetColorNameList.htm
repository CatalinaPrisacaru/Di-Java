mgGetColorNameList

FUNCTION NAME

mgGetColorNameList - gets the list of names for a particular color index.

SYNOPSIS       (C and Python)

mgstringlist mgGetColorNameList (mgrec*db,
intindex );

DESCRIPTION

Each color palette entry can have a list of names associated with it. Given a database node, db and an index into the color palette, index, mgGetColorNameList returns index’s color name list.

C NOTES

The string list returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFreeStringList.

PYTHON NOTES

It is not necessary to free the returned string list in Python.

PARAMETERS

db
the database
index
the color palette index

RETURNS

Returns the color name list for the given index if successful, MG_NULL otherwise.

EXAMPLE

The following examples print all the names associated with color index 4 in a database.

C EXAMPLE

mgstringlist nameList;

// get the list of color names
nameList = mgGetColorNameList (db, 4);
if (nameList)
{
   mgstringlist cursor = nameList;
   int n = 1;
   while (*cursor != MG_NULL)
   {
      // at this point name holds the nth name associated with color index 4
      printf ("Name %d: %s\n", n, *cursor);

      // and go get the next one
      n++;
      cursor++;
   }
   // remember to free the list when you're done with it
   mgFreeStringList (nameList);
}

PYTHON EXAMPLE

# get the list of color names
nameList = mgGetColorNameList (db, 4)

n = 1
for colorName in nameList:
   print "Name",n,":",colorName
   n = n + 1

# in Python, you don't have to free the list

ACCESS LEVEL

Level 1
