mgRegisterSetMaxId

FUNCTION NAME

mgRegisterSetMaxId - sets the Set Max Id (extension node helper) function for a database extension.

C SPECIFICATION       (Not Available For Python)

mgstatus mgRegisterSetMaxId (mgpluginsitepluginSite,
mgsetmaxidfuncsetMaxIdFunc );

DESCRIPTION

mgRegisterSetMaxId assigns the specified Set Max Id function setMaxIdFunc to the specified data extension site pluginSite.

PARAMETERS

pluginSite
the data extension site
setMaxIdFunc
the Set Max Id function

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 3
