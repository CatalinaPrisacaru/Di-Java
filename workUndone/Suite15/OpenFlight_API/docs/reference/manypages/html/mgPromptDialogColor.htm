mgPromptDialogColor

FUNCTION NAME

mgPromptDialogColor - prompts the user to select a color using a standard color chooser.

C SPECIFICATION

mgstatus mgPromptDialogColor (mgguiparent,
shortinR,
shortinG,
shortinB,
short*outR,
short*outG,
short*outB );

PYTHON SPECIFICATION

mgstatus, outR, outG, outB mgPromptDialogColor (mgguiparent,
intinR,
intinG,
intinB )

DESCRIPTION

mgPromptDialogColor displays a modal color chooser dialog in which the user is prompted to select a color.

When the dialog is initially displayed, it will present the color specified by the input parameters inR, inG, and inB. The input and output color values are in the range 0 to 255.

C NOTES

The dialog is created as a child of the specified dialog parent. If you specify parent as MG_NULL, the dialog is created as a child of the Creator main window.

PYTHON NOTES

Currently the parent parameter is reserved for future enhancement and should be specified as None. For now the dialog is created as a child of the main Creator main window.

PARAMETERS

parent
the dialog parent to attach the color dialog
inR
the red component of the initial color
inG
the green component of the initial color
inB
the blue component of the initial color
outR
the red component of the selected color
outG
the green component of the selected color
outB
the blue component of the selected color

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success. Success indicates that the user selected and accepted a color.
outR
If function is successful, outR contains the corresponding color value, otherwise outR is undefined.
outG
If function is successful, outG contains the corresponding color value, otherwise outG is undefined.
outB
If function is successful, outB contains the corresponding color value, otherwise outB is undefined.

ACCESS LEVEL

Level 4
