ddIsDisplay

FUNCTION NAME

ddIsDisplay - checks if a record or data code is displayable in the modeler attribute page.

SYNOPSIS       (C and Python)

mgbool ddIsDisplay (mgcodecode );

DESCRIPTION

ddIsDisplay checks if the record or data identified by code is displayable in the attribute page in the modeler. The value for the keyword NODISP controls this attribute.

By default, record and data codes are displayable. Use the keyword NODISP to override the value for this attribute. This keyword is not valid for data of type MTYPE_FLAG if the ENUM attribute is set.

PARAMETERS

code
the record or data code

RETURNS

Returns MG_TRUE if record or data is displayable, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
