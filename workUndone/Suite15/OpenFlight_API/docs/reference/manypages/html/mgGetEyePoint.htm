mgGetEyePoint

FUNCTION NAME

mgGetEyePoint - gets an eyepoint record from the database

SYNOPSIS       (C and Python)

mgrec* mgGetEyePoint (mgrec*db,
intindex );

DESCRIPTION

mgGetEyePoint gets the eyepoint record defined by index from the database db, index must be between 1 and 9. Eyepoint 0 is reserved for use by MultiGen.

PARAMETERS

db
the database
index
the index of the eyepoint to get

RETURNS

Returns the eyepoint record if successful, MG_NULL otherwise.

ACCESS LEVEL

Level 1

SEE ALSO

mgSetEyePoint, mgSetAttList
