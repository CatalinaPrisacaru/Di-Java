mgPtrArraySearch

FUNCTION NAME

mgPtrArraySearch - searches a pointer array for a specific item using a linear search.

C SPECIFICATION       (Not Available For Python)

int mgPtrArraySearch (mgptrarrayptrArray,
void*key );

DESCRIPTION

mgPtrArrayBSearch uses a linear search to locate a specific item key in a pointer array ptrArray. The linear search starts at the item at index 1 and searches the list until the key is found.

PARAMETERS

ptrArray
the pointer array
key
the item to search for

RETURNS

If the key is found, returns the index of the item (which will be in the range 1..length of array). If the key is not found, returns 0.

ACCESS LEVEL

Level 1
