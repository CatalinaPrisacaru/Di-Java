mgrenderercalldatatype

ENUM NAME

mgrenderercalldatatype - Renderer callback data type

SYNOPSIS

typedef enum mgrenderercalldatatype {
   MRCD_COMMON,
   MRCD_TEXTACTIVATE,
   MRCD_SCALEACTIVATE,
   MRCD_GLDRAW,
   MRCD_DIALOGSIZE,
} mgrenderercalldatatype;

MEMBERS

MRCD_COMMON
Common Call Data. This indicates that the callData parameter to a renderer callback function is a pointer to an object of type mgguicallbackrec and has no additional data.
MRCD_TEXTACTIVATE
Text Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mgtextactivatecallbackrec.
MRCD_SCALEACTIVATE
Scale Activate Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mgscaleactivatecallbackrec.
MRCD_GLDRAW
GL Draw Call Data. This indicates that the callData parameter to a control callback is a pointer to an object of type mggldrawcallbackrec.
MRCD_DIALOGSIZE
Dialog Size Call Data. This indicates that the callData parameter to a dialog function is a pointer to an object of type mgdialogsizecallbackrec.

DESCRIPTION

This type is used to indicate the type of call data passed to a renderer callback functions. When a renderer callback function is called, the callData parameter passed in will be a pointer to an object type that is dependent on the context of the callback function.

Each member of this type corresponds to a different callback data structure.
