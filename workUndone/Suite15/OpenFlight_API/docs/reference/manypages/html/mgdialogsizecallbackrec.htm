mgdialogsizecallbackrec

STRUCT NAME

mgdialogsizecallbackrec - callback structure for the size dialog event sent to dialogs

SYNOPSIS

typedef struct  {
   mgguicalldatatype callDataType;
   mgbool isMinimized;
} mgdialogsizecallbackrec;

MEMBERS

callDataType
mgguicalldatatype - call data type that indicates the type of call data this object really is - will always be MGCD_DIALOGSIZE
isMinimized
mgbool - MG_TRUE if dialog is being minimized, MG_FALSE otherwise

DESCRIPTION

When the MGCB_SIZE dialog event is sent to a dialog via its mgdialogfunc, the calldata parameter passed in will point to an object of this type.
