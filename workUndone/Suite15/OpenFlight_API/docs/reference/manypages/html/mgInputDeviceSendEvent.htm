mgInputDeviceSendEvent

FUNCTION NAME

mgInputDeviceSendEvent - sends the current event to the tool.

SYNOPSIS       (C and Python)

mgstatus mgInputDeviceSendEvent (mginputdeviceinputDevice );

DESCRIPTION

mgInputDeviceSendEvent sends the events specified by any mgInputDeviceSetVertex, mgInputDeviceSetPoint, mgInputDeviceSetButtonStatus, mgInputDeviceSetVertex and mgInputDeviceSetDeviceData calls to the requesting tool. The device inputDevice must have been previously obtained by mgInputDeviceGetHandle

PARAMETERS

inputDevice
the input device obtained from mgInputDeviceGetHandle

RETURNS

Use MSTAT_ISOK to check return value for success

ACCESS LEVEL

Level 4
