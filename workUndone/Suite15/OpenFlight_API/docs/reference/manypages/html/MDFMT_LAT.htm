MDFMT_LAT

NAME

MDFMT_LAT - DMS (Degrees, Minutes, Seconds) format string for standard Latitude display

DESCRIPTION

This is the format string you will specify when calling mgTextGetDMS or mgTextSetDMS if the input or output string is formatted as a measurement of latitude. The standard format for latitude display is DD~MM'SS"H, where DD is the number of degrees, MM is the number of minutes, SS is the number of seconds and H is either 'N' or 'S' depending on whether the value lies in the northern or southern hemisphere, respectively.
