ddIsEnumSet

FUNCTION NAME

ddIsEnumSet - checks if the ENUM keyword is set for a specific data code.

SYNOPSIS       (C and Python)

mgbool ddIsEnumSet (mgcodedcode );

DESCRIPTION

ddIsEnumSet checks if the data associated with dcode is a special flag field of an ENUM group.

PARAMETERS

dcode
the data code

RETURNS

Returns MG_TRUE if this is an ENUM flag field, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
