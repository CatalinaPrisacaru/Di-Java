mgGraphicsViewSettingSetMatrix

FUNCTION NAME

mgGraphicsViewSettingSetMatrix - sets a matrix view setting on one or more graphics views.

SYNOPSIS       (C and Python)

mgbool mgGraphicsViewSettingSetMatrix (mgrec*db,
const mgsettingnamesettingName,
mgmatrixsettingValue,
unsigned intsetMask );

DESCRIPTION

mgGraphicsViewSettingSetMatrix sets the value of a named graphics view setting, settingName, to the value specified by settingValue. Depending on the value of setMask, this function will set the value on either the current graphics view or all the graphics views of db. To set the value on the current graphics view only, specify MGVSM_SETCURRENT for setMask. To set the value on all graphics views of db specify MGVSM_SETALL.

Currently only a small subset of the graphics view settings are accessible. For a complete list, see mgsettingname.

Note that the prefix of the setting name, either MGVD, MGVI, or MGVM indicates whether the corresponding setting value is double, integer or mgmatrix, respectively. If the value is double, use mgGraphicsViewSettingGetDouble and mgGraphicsViewSettingSetDouble to get and set the value, respectively. If the value is integer, use mgGraphicsViewSettingGetInteger and mgGraphicsViewSettingSetInteger. If the value is mgmatrix, use mgGraphicsViewSettingGetMatrix and mgGraphicsViewSettingSetMatrix.

PARAMETERS

db
the database
settingName
the graphics view setting name
settingValue
the value to set
setMask
the mask specifying which graphics views to set

RETURNS

Returns MG_TRUE if there is a graphics view visible in Creator, settingName represents a valid mgmatrix graphics view setting, and the value was set successfully, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
