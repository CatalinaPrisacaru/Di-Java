mgGetSoundCount

FUNCTION NAME

mgGetSoundCount - gets the number of entries in a database’s sound palette.

SYNOPSIS       (C and Python)

int mgGetSoundCount ( void )

DESCRIPTION

mgGetSoundCount gets the number of sound entries for a given database db.

RETURNS

Returns the number of sound entries.

ACCESS LEVEL

Level 1
