mgeditorcreatedialogfunc

CALLBACK NAME

mgeditorcreatedialogfunc - Editor create dialog function

SYNOPSIS

mggui (*mgeditorcreatedialogfunc) ( mgplugintool editorPluginTool,
void* editorToolData );

DESCRIPTION

This is the signature for editor create dialog functions. If your editor tool displays a dialog, assign a create dialog function to your editor tool of this form.

Create dialog functions are called to create an editor dialog instance when your editor tool is launched. This function must create and return a new dialog instance each time it is invoked.

PARAMETERS

editorPluginTool
the editor plug-in tool for which the dialog is to be created
editorToolData
user defined tool instance data set in toolData field of callData parameter (mgeditorcallbackrec) during editor tool start function mgtoolstartfunc

RETURNS

Return the dialog instance created for editor tool instance.
