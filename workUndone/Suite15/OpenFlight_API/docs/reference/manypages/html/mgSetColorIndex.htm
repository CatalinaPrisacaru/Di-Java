mgSetColorIndex

FUNCTION NAME

mgSetColorIndex - sets a color palette entry to the given red, green, and blue values.

SYNOPSIS       (C and Python)

mgbool mgSetColorIndex (mgrec*db,
intindex,
shortr,
shortg,
shortb );

DESCRIPTION

Given a database node db, a color palette index index, and red, green, and blue color values, mgSetColorIndex sets the color of the given index in the database’s color palette.

PARAMETERS

db
the database
index
the color palette index to change
r
the red value, which ranges from 0 to 255
g
the green value, which ranges from 0 to 255
b
the blue value, which ranges from 0 to 255

RETURNS

Returns MG_TRUE if db contains a valid database and index represents a valid palette index, otherwise returns MG_FALSE.

ACCESS LEVEL

Level 2
