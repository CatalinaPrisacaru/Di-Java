mgSetConstructVertexColor

FUNCTION NAME

mgSetConstructVertexColor - sets the color of a construction vertex node.

SYNOPSIS       (C and Python)

mgstatus mgSetConstructVertexColor (mgrec*constructVtx,
mgconstructcolorconstructColor );

DESCRIPTION

mgSetConstructVertexColor sets the color of a construction vertex node constructVtx to the specified pre-defined color constructColor.

See mgconstructcolor for the list of pre-defined colors. By default, construction vertex nodes are created using color MCCOLOR_RED.

PARAMETERS

constructVtx
the construction vertex to color
constructColor
the color

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
