mgReadImageHeader

FUNCTION NAME

mgReadImageHeader - reads the header information from an image file.

C SPECIFICATION

int mgReadImageHeader (const char*textureName,
int*type,
int*width,
int*height );

PYTHON SPECIFICATION

status, type, width, height mgReadImageHeader (stringtextureName )

DESCRIPTION

mgReadImageHeader reads the header information of an image from the specified file textureName. The header information is returned in type, width and height.

PARAMETERS

textureName
the name of the image file on disk
type
address of value to receive image type Possible values returned are MIMG_INT, MIMG_INTA, MIMG_RGB, and MIMG_RGBA
width
address of value to receive image width
height
address of value to receive image height

RETURNS

status
Returns MIMG_NO_ERROR if successful, otherwise applicable texture error code.
type
image type - possible values returned are MIMG_INT, MIMG_INTA, MIMG_RGB, and MIMG_RGBA.
width
image width.
height
image height.

ACCESS LEVEL

Level 1
