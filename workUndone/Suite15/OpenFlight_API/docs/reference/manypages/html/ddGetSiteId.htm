ddGetSiteId

FUNCTION NAME

ddGetSiteId - gets the Site ID of a record or data code.

SYNOPSIS       (C and Python)

const char* ddGetSiteId (mgcodecode );

DESCRIPTION

ddGetSiteId returns the Site ID associated with a record or data code, code. This is the Site ID that was specified when the data extension was registered via mgRegisterSite.

Note: Do not free or modify the returned string.

PARAMETERS

code
the data or record code to check

RETURNS

Returns the Site ID string if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
