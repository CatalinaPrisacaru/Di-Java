mgShowPalette

FUNCTION NAME

mgShowPalette - displays a database palette window.

SYNOPSIS       (C and Python)

mgstatus mgShowPalette (mgpaletteidpaletteId );

DESCRIPTION

mgShowPalette displays the database palette specified paletteId. If the palette is not already displayed, it will be. If the palette is already displayed but occluded by another window, it will be brought to the top of the window order on the desktop.

See mgpaletteid for a complete list of palettes you can display with this function.

PARAMETERS

paletteId
the palette to show

RETURNS

Use MSTAT_ISOK to check return value for success.

EXAMPLE

The following examples (in C and Python) show how to display the Creator texture palette.

C EXAMPLE

mgShowPalette (MPID_TEXTUREPALETTE);

PYTHON EXAMPLE

mgShowPalette (MPID_TEXTUREPALETTE)

ACCESS LEVEL

Level 4
