mgTextSetTextFormat

FUNCTION NAME

mgTextSetTextFormat - sets the text format string for a text control.

C SPECIFICATION       (Not Available For Python)

mgstatus mgTextSetTextFormat (mgguicontrol,
const char*fmtString );

DESCRIPTION

mgTextSetTextFormat sets the format string for the text control that has a spin buddy control. The format string, fmtString, is assumed to be a printf style format string and is used to automatically format the numeric value that is displayed in the text control when the associated spin buddy is activated.

The default format string for a text control is "%lf".

This function is only meaningful for a text control that has a spin buddy control.

PARAMETERS

control
the text control
fmtString
the format string for the text buddy

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
