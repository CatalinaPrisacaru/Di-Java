mgDeleteTextureMappingByName

FUNCTION NAME

mgDeleteTextureMappingByName - deletes a texture mapping entry from a database’s texture mapping palette.

SYNOPSIS       (C and Python)

mgbool mgDeleteTextureMappingByName (mgrec*db,
const char*name );

DESCRIPTION

mgDeleteTextureMappingByName deletes the texture mapping entry defined by name from the texture mapping palette associated with database node db. If no texture mapping entry is found matching name, the texture mapping palette remains unchanged.

PARAMETERS

db
the database
name
the name of the texture mapping to delete

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
