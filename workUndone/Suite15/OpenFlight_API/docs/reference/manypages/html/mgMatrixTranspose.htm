mgMatrixTranspose

FUNCTION NAME

mgMatrixTranspose - calculates the transpose of a matrix.

C SPECIFICATION

void mgMatrixTranspose (const mgmatrixmatrix,
mgmatrix*resultMatrix );

PYTHON SPECIFICATION

mgmatrix mgMatrixTranspose (mgmatrixmatrix )

DESCRIPTION

mgMatrixTranspose calculates the transpose of matrix.

The original matrix matrix is not affected.

C NOTES

This function writes the transpose matrix into the output parameter resultMatrix if successful.

PYTHON NOTES

This function returns the transpose matrix if successful.

PARAMETERS

matrix
matrix to transpose
resultMatrix
address of matrix to receive result

RETURNS

mgmatrix
the matrix inverse.

ACCESS LEVEL

Level 1
