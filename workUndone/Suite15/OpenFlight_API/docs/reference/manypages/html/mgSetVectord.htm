mgSetVectord

FUNCTION NAME

mgSetVectord - sets the attribute values in a fltVectord record.

SYNOPSIS       (C and Python)

mgbool mgSetVectord (mgrec*rec,
mgcodevectorCode,
doublei,
doublej,
doublek );

DESCRIPTION

Given a record rec, and the name of one of its fltVectord attribute records, vectorCode, mgSetVectord sets the i, j, k attribute values in the fltVectord attribute record.

PARAMETERS

rec
the record whose fltVectord record is to be set
vectorCode
the name of the fltVectord attribute record of rec
i
the i value to set in the fltVectord attribute record
j
the j value to set in the fltVectord attribute record
k
the k value to set in the fltVectord attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2

SEE ALSO

mgSetCoord3d, mgSetVector
