mgSetConstructEdgeColor

FUNCTION NAME

mgSetConstructEdgeColor - sets the color of a construction edge.

SYNOPSIS       (C and Python)

mgstatus mgSetConstructEdgeColor (mgrec*constructEdge,
mgconstructcolorconstructColor );

DESCRIPTION

mgSetConstructEdgeColor sets the color of a construction edge constructEdge to the specified pre-defined color constructColor.

See mgconstructcolor for the list of pre-defined colors. By default, construction edge nodes are created using color MCCOLOR_RED.

PARAMETERS

constructEdge
the construction edge to color
constructColor
the color

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
