mgDeclarePlugin

FUNCTION NAME

mgDeclarePlugin - Declares a plug-in module.

SYNOPSIS       (C and Python)

mgDeclarePlugin ( vendor,
name,
uuid );

DESCRIPTION

You must instantiate this macro at the outer most scope of your source code so that your plug-in is properly recognized.

PARAMETERS

vendor
a string representing the name of the vendor of the plug-in
name
a string representing the name of the plug-in
uuid
a Universally Unique Identifier to identify the plug-in (On Windows, use the uuidgen utility to generate this identifier)

C EXAMPLE

The following example shows how to declare a plug-in module.
mgDeclarePlugin(
   "My Company",
   "My Plugin",
   "453E73F2-6DD8-4164-A6F3-B59E4C57998A"
);

ACCESS LEVEL

Instantiated by Plug-in Module
