mgGetFocusVertex

FUNCTION NAME

mgGetFocusVertex - get the current focus vertex.

C SPECIFICATION       (Not Available For Python)

mgrec* mgGetFocusVertex (mgeditorcontexteditorContext );

DESCRIPTION

mgGetFocusVertex gets the current focus vertex associated to the specified editor context, editorContext.

PARAMETERS

editorContext
the editor context used to identify the database for which the focus vertex is to be returned

RETURNS

Returns current focus vertex node if set, MG_NULL otherwise.

ACCESS LEVEL

Level 4
