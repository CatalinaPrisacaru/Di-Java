mgPtrArrayAppend

FUNCTION NAME

mgPtrArrayAppend - append an item onto a pointer array.

C SPECIFICATION       (Not Available For Python)

int mgPtrArrayAppend (mgptrarrayptrArray,
void*item );

DESCRIPTION

mgPtrArrayAppend appends the specified item to the end of the pointer array ptrArray.

The pointer array is automatically "grown" to accommodate the new item.

PARAMETERS

ptrArray
the pointer array
item
the item to be appended to the pointer array

RETURNS

The length of the array after the item is appended.

ACCESS LEVEL

Level 1
