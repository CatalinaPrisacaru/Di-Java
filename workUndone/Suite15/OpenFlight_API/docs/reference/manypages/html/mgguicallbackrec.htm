mgguicallbackrec

STRUCT NAME

mgguicallbackrec - generic callback structure for callData parameter of mgguifunc and mgdialogfunc

SYNOPSIS

typedef struct  {
   mgguicalldatatype callDataType;
} mgguicallbackrec;

MEMBERS

callDataType
mgguicalldatatype - call data type that indicates the type of call data this object really is

DESCRIPTION

When a control callback or dialog function is called, the callData parameter passed in will be a pointer to an object type that is dependent on the context of the callback function. The first field of each data structure is the same as this structure. In this way, the callData parameter can always safely be cast to a pointer to an object of this type. Doing so, you can then examine the callDataType field to determine which callback record structure has actually been passed.
