mgSetCurrentGridPoints

FUNCTION NAME

mgSetCurrentGridPoints - sets the current modeling grid using three points.

SYNOPSIS       (C and Python)

mgbool mgSetCurrentGridPoints (mgrec*db,
const mgcoord3d*origin,
const mgcoord3d*xAxis,
const mgcoord3d*third );

DESCRIPTION

mgSetCurrentGridPoints sets the position and orientation of the grid in the current Creator graphics view using three points. These points are origin, xAxis, and third where origin is the origin of the grid, xAxis is a point on the x axis of the grid and third is a point that along with origin and xAxis define the plane in which the grid is located.

PARAMETERS

db
the database
origin
3D coordinate defining the grid origin
xAxis
3D coordinate defining the x axis point of the grid
third
3D coordinate defining the third point on the grid plane

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
