mgGetTexturePosition

FUNCTION NAME

mgGetTexturePosition - gets the position of a texture palette entry.

C SPECIFICATION

mgbool mgGetTexturePosition (mgrec*db,
intindex,
int*x,
int*y );

PYTHON SPECIFICATION

mgbool, x, y mgGetTexturePosition (mgrec*db,
intindex )

DESCRIPTION

mgGetTexturePosition gets the position of the texture palette entry at the specified index contained in the texture palette of database db.

The position of a texture palette entry is the lower left corner where it is displayed in the texture palette. This position is measured in pixels.

PARAMETERS

db
the database
index
the index of the texture palette entry
x
address to receive x coordinate of position
y
address to receive y coordinate of position

RETURNS

mgbool
MG_TRUE if position retrieved successfully, MG_FALSE otherwise.
x
If function is successful, x contains the retrieved x position, otherwise x is undefined.
y
If function is successful, y contains the retrieved y position, otherwise y is undefined.

ACCESS LEVEL

Level 1
