mgpointinputdata

STRUCT NAME

mgpointinputdata - 2D Point Mouse Input data

SYNOPSIS

typedef struct  {
   mgmousestate mouseEvent;
   unsigned int keyboardFlags;
   unsigned int buttonFlags;
   mgcoord2i* thisPoint;
   mgcoord2i* firstPoint;
} mgpointinputdata;

MEMBERS

mouseEvent
mgmousestate - the point input sequence state
keyboardFlags
unsigned int - the state of the Alt, Ctrl, and Shift keys - the value of this field will be a bitwise combination of MKB_ALTKEY, MKB_CTRLKEY and MKB_SHIFTKEY
buttonFlags
unsigned int - the state of the Left, Middle and Right mouse buttons - the value of this field will be a bitwise combination of MMB_LEFTMOUSE, MMB_MIDDLEMOUSE and MMB_RIGHTMOUSE
thisPoint
mgcoord2i* - address of 2D coordinate for the current point being reported in this sequence
firstPoint
mgcoord2i* - address of 2D coordinate for the first point reported in this sequence

DESCRIPTION

When 2D point mouse input is reported to an editor tool via the corresponding 2D point input function, a pointer to a record of this type is passed to the point function that describes the point input.

The coordinates of 2D point mouse input are reported relative to the lower left corner of the graphics view in which the input was received. That is, if the point reported is at the lower left corner of the graphics view, it will be reported as (0, 0).
