mgUnregisterTimer

FUNCTION NAME

mgUnregisterTimer - remove a timer request.

C SPECIFICATION       (Not Available For Python)

void mgUnregisterTimer (mgtimertimerId );

DESCRIPTION

mgUnregisterTimer removes a timer request identified by timerId if it has not yet expired. The timer object timerId was returned from mgRegisterTimer.

PARAMETERS

timerId
timer object to remove

ACCESS LEVEL

Level 4

SEE ALSO

mgRegisterTimer, mgtimerfunc
