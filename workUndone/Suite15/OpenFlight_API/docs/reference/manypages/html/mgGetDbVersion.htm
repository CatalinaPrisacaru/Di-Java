mgGetDbVersion

FUNCTION NAME

mgGetDbVersion - gets the OpenFlight version number of a file.

SYNOPSIS       (C and Python)

int mgGetDbVersion (const char*fileName );

DESCRIPTION

Given a file name, mgGetDbVersion returns the OpenFlight version number if the specified file has a valid format, 0 otherwise.

PARAMETERS

fileName
the name of the file to be queried

RETURNS

Returns the OpenFlight version number if the specified file has a valid format, 0 otherwise.

ACCESS LEVEL

Level 1
