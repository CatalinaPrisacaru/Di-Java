mgGetNextLightSource

FUNCTION NAME

mgGetNextLightSource - gets the next entry record from a database’s light source palette.

C SPECIFICATION

mgrec* mgGetNextLightSource (mgrec*lightSource,
int*index );

PYTHON SPECIFICATION

mgrec, index mgGetNextLightSource (mgrec*lightSource )

DESCRIPTION

Given a light source entry record, lightSource, mgGetNextLightSource returns the light source entry record following lightSource in the light source palette. The index of the next light source in the palette is returned in index, if there is one. If there is no next light source entry record, MG_NULL is returned.

The code of the record returned will be fltLightSourcePalette .

PARAMETERS

lightSource
the previous light source palette entry record
index
address of value to receive index of entry returned

RETURNS

mgrec
The next light source entry record, or MG_NULL if unsuccessful.
index
If function is successful, index contains the retrieved index, otherwise index is undefined.

C EXAMPLE

mgrec* lightSource;
mgrec* db;
int index;
db = mgOpenDb ("anyfile.flt");
lightSource = mgGetFirstLightSource (db, &index);
while (lightSource != MG_NULL) {
   // do something with lightSource
   lightSource = mgGetNextLightSource (lightSource, &index);
}

ACCESS LEVEL

Level 1
