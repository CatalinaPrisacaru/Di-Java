mgeditorclosedialogfunc

CALLBACK NAME

mgeditorclosedialogfunc - Editor close dialog function

SYNOPSIS

mgbool (*mgeditorclosedialogfunc) ( mgeditorcontext editorContext,
mgclosedialogcallbackrec* callData,
void* editorToolData );

DESCRIPTION

This is the signature for editor close dialog functions. If your editor tool displays a dialog, you can assign a close dialog function to your editor tool of this form.

Close dialog functions are called to notify your editor tool when the user has pressed the Close Box on your dialog.

PARAMETERS

editorContext
the editor context that identifies the editor instance being closed
callData
callback data for close dialog function
editorToolData
user defined tool instance data set in toolData field of callData parameter (mgeditorcallbackrec) during editor tool start function mgtoolstartfunc

RETURNS

Return MG_TRUE to close the dialog, MG_FALSE to ignore the action from the user.
