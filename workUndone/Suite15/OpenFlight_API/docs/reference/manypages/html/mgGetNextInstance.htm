mgGetNextInstance

FUNCTION NAME

mgGetNextInstance - gets successive instance nodes of a reference node.

SYNOPSIS       (C and Python)

mgrec* mgGetNextInstance (mgrec*inst );

DESCRIPTION

mgGetNextInstance returns the instance that follows the specified instance node inst. If node inst is an instance of a reference node, this function will return the next instance of that reference node.

PARAMETERS

inst
the instance node to get next instance for

RETURNS

Returns the instance node if found. If inst is not an instance node, MG_NULL is returned.

ACCESS LEVEL

Level 1
