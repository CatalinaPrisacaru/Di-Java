mgGetBoundsForRecList

FUNCTION NAME

mgGetBoundsForRecList - retrieves the bounding box for a list of nodes.

C SPECIFICATION

mgbool mgGetBoundsForRecList (mgreclistlist,
mgboxd*box );

PYTHON SPECIFICATION

mgbool, box mgGetBoundsForRecList (mgreclistlist )

DESCRIPTION

mgGetBoundsForRecList retrieves the axis aligned bounding box for the nodes in the record list list. The bounding box is in world coordinates.

PARAMETERS

list
a record list
box
address of box to receive bounds

RETURNS

mgbool
MG_TRUE if the box was retrieved successfully, MG_FALSE otherwise.
box
If function is successful, box contains the retrieved coordinates, otherwise box is undefined.

ACCESS LEVEL

Level 1
