mgPluginPrefDelete

FUNCTION NAME

mgPluginPrefDelete - deletes a preference value from the plug-in preferences.

C SPECIFICATION       (Not Available For Python)

mgbool mgPluginPrefDelete (mgpluginplugin,
const char*prefName );

DESCRIPTION

mgPluginPrefDelete deletes a named preference key, prefName from plugin.

PARAMETERS

plugin
the plug-in handle
prefName
the preference key name to delete

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 3
