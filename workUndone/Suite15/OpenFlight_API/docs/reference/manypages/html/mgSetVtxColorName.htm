mgSetVtxColorName

FUNCTION NAME

mgSetVtxColorName - sets the color name of a fltVertex record.

SYNOPSIS       (C and Python)

mgbool mgSetVtxColorName (mgrec*vtx,
const char*name );

DESCRIPTION

Given a fltVertex record, vtx, and a string, name, mgSetVtxColorName stores name as the color name for vtx. If the name given is not in the color palette, mgSetVtxColorName returns MG_FALSE.

To clear the color name for a fltVertex , set name to NULL or the empty string.

PARAMETERS

vtx
a fltVertex record
name
the color name

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise..

ACCESS LEVEL

Level 2
