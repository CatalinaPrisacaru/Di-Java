mgSetModelingModeEdge

FUNCTION NAME

mgSetModelingModeEdge - sets current modeling mode to edge.

SYNOPSIS       (C and Python)

mgbool mgSetModelingModeEdge (mgrec*db );

DESCRIPTION

mgSetModelingModeEdge sets the current modeling mode for the specified database db to edge. The edge modeling mode is a special case of vertex mode. Since there is no "edge" type in OpenFlight, you must use this function to set the modeling mode to edge. Use mgSetModelingMode to set all other modes.

PARAMETERS

db
the database to set modeling mode for

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

EXAMPLE

See mgSetModelingMode for an example of how you might set the modeling modes to different levels.

ACCESS LEVEL

Level 4
