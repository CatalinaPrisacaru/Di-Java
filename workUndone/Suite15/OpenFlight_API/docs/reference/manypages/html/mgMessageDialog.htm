mgMessageDialog

FUNCTION NAME

mgMessageDialog - displays modal message dialog.

SYNOPSIS       (C and Python)

int mgMessageDialog (mgguiparent,
const char*title,
const char*messageText,
unsigned intflags );

DESCRIPTION

mgMessageDialog displays a modal dialog that contains a text message, an icon and any combination of push button controls. You specify which icon and which combination of push buttons using the parameter flags. The value of flags is the bitwise combination of masks as described here.

Choose one mask from the following group to specify which icon to display.

MMBX_STATUS displays an informational icon (lower case i in a circle).
MMBX_WARNING displays a warning icon (exclamation point).
MMBX_ERROR displays a warning icon (stop sign).
MMBX_QUESTION displays a warning icon (question mark).

Choose one mask from the following group to specify which combination of push buttons to display.

MMBX_OK displays a single button "OK".
MMBX_OKCANCEL displays two buttons "OK" and "Cancel".
MMBX_YESNO displays two buttons "Yes" and "No".
MMBX_YESNOCANCEL displays three buttons "Yes", "No" and "Cancel".

You must select one and only one mask from each group listed above. If you do not select a mask from each group or select more than one from either group, the results are undefined.

C NOTES

The dialog is created as a child of the specified dialog parent. If you specify parent as MG_NULL, the dialog is created as a child of the Creator main window.

PYTHON NOTES

Currently the parent parameter is reserved for future enhancement and should be specified as None. For now the dialog is created as a child of the main Creator main window.

PARAMETERS

parent
the dialog parent to attach the prompt dialog
title
the title string for the dialog
messageText
the message text to display in the dialog
flags
flags to specify which icon and push button controls to display in the dialog

RETURNS

Returns integer value indicating which push button was pressed by the user to dismiss the dialog. The return value depends on the push button combination you specified in the flags parameter.
For MMBX_OK, the return value is always 1 (OK).
For MMBX_OKCANCEL, the return value is either 1 (OK) or 2 (Cancel).
For MMBX_YESNO, the return value is either 1 (Yes) or 2 (No).
For MMBX_YESNOCANCEL, the return value is either 1 (Yes), 2 (No), or 3 (Cancel).

ACCESS LEVEL

Level 4
