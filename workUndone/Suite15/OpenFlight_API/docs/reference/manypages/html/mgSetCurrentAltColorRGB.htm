mgSetCurrentAltColorRGB

FUNCTION NAME

mgSetCurrentAltColorRGB - sets the current alternate modeling color to an index and intensity that matches RGB values.

SYNOPSIS       (C and Python)

mgbool mgSetCurrentAltColorRGB (mgrec*db,
shortred,
shortgreen,
shortblue );

DESCRIPTION

mgSetCurrentAltColorRGB sets the current alternate modeling color palette index and intensity for database db to values that most closely match the specified red, green, and blue values.

The color palette for db is searched for the color that most closely matches the specified RGB values. The current alternate modeling color is set to the color palette entry found.

The input color values are expected to be in the range 0..255.

PARAMETERS

db
the database
red
red component to match
green
green component to match
blue
blue component to match

RETURNS

Returns MG_TRUE if current alternate modeling color for db could be set, MG_FALSE otherwise.

ACCESS LEVEL

Level 4
