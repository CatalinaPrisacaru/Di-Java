mgGetPolyNormal

FUNCTION NAME

mgGetPolyNormal - retrieves the normal vector from a polygon record.

C SPECIFICATION

mgbool mgGetPolyNormal (mgrec*rec,
double*i,
double*j,
double*k );

PYTHON SPECIFICATION

mgbool, i, j, k mgGetPolyNormal (mgrec*rec )

DESCRIPTION

mgGetPolyNormal retrieves the i, j, and k values of the polygon normal vector from a polygon record rec.

You do not have to provide addresses for all i, j, and k values. For example, if you only want to receive the i and j components (but not k), specify valid addresses for i and j and leave k NULL.

PARAMETERS

rec
a polygon record
i
address of value to receive i component of normal vector
j
address of value to receive j component of normal vector
k
address of value to receive k component of normal vector

RETURNS

mgbool
MG_TRUE if the polygon has a normal vector, MG_FALSE otherwise.
i
If function is successful, i contains the retrieved value, otherwise i is undefined.
j
If function is successful, j contains the retrieved value, otherwise j is undefined.
k
If function is successful, k contains the retrieved value, otherwise k is undefined.

ACCESS LEVEL

Level 1
