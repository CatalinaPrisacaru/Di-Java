mgtoolactivationtype

ENUM NAME

mgtoolactivationtype - Tool Activation Type

SYNOPSIS

typedef enum mgtoolactivationtype {
   MTAT_UNKNOWN,
   MTAT_NORMAL,
   MTAT_REPEAT,
   MTAT_SCRIPT,
   MTAT_PROCESS,
} mgtoolactivationtype;

MEMBERS

MTAT_UNKNOWN
Tool Activation Unknown. Unknown tool activation type. This typically indicates an error condition.
MTAT_NORMAL
Tool Activation Normal. This indicates that the tool was launched normally by the user from within the interactive Creator modeling environment.
MTAT_REPEAT
Tool Activation Repeat. This indicates that the tool was launched by the user from the interactive Creator modeling environment using the Repeat Last command.
MTAT_SCRIPT
Tool Activation Script. This indicates that the tool was launched from a Creator script. When this is the case, the tool should perform its actions "silently" without displaying a dialog.
MTAT_PROCESS
Tool Activation Process. This indicates that the tool was launched by an external process (normally Terra Vista). When this is the case, the tool should perform its actions accordingly. Internal use only by Presagis.

DESCRIPTION

This enumerated type is used to indicate the activation type for a tool activation object. A plug-in tool may use this activation type to alter its behavior depending on how it was launched.
