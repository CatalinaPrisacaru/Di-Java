MPRIM_TRI_STRIP

NAME

MPRIM_TRI_STRIP - Mesh Primitive Type Triangle Strip

DESCRIPTION

This is the mesh primitive type defining a triangle strip. A triangle strip represents a series of 3-sided polygons (triangles) defined by the vertices of the mesh primitive.

One triangle is defined for each vertex of the primitive after the first two. For n (starting at 0), the nth triangle is defined as follows. For even n (0, 2, 4, etc), the nth triangle is defined by vertices n, n+1 and n+2. For odd n (1, 3, 5, etc), the nth triangle is defined by vertices n+1, n and n+2.

If a triangle strip mesh primitive has N vertices, N-2 triangles are defined.
