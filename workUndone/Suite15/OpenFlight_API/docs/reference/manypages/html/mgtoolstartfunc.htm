mgtoolstartfunc

CALLBACK NAME

mgtoolstartfunc - Start function for Database Importer/Exporter, Viewer and Editor Tools

SYNOPSIS

mgstatus (*mgtoolstartfunc) ( mgplugintool pluginTool,
void* userData,
void* callData );

DESCRIPTION

This is the signature for plug-in tool start functions. When you register a plug-in tool, provide a start function of this form that is called when your plug-in tool is invoked in Creator.

PARAMETERS

pluginTool
the plug-in tool
userData
user defined data specified when tool registered
callData
pointer to tool specific callback record
For image importer plug-in tools, this points to an mgimportercallbackrec record
For image exporter plug-in tools, this points to an mgexportercallbackrec record
For viewer plug-in tools, this points to an mgviewercallbackrec record
For editor plug-in tools, this points to an mgeditorcallbackrec record

RETURNS

Currently the value returned by mgtoolstartfunc is ignored and reserved for future enhancement. For now always return MSTAT_OK.
