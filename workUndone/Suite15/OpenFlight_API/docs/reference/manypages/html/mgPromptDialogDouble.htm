mgPromptDialogDouble

FUNCTION NAME

mgPromptDialogDouble - prompts the user to enter double precision floating point value.

C SPECIFICATION

mgstatus mgPromptDialogDouble (mgguiparent,
const char*messageText,
double*value );

PYTHON SPECIFICATION

mgstatus, number mgPromptDialogDouble (mgguiparent,
stringmessageText,
doublevalue )

DESCRIPTION

mgPromptDialogDouble displays a modal dialog in which the user is asked to enter a double precision floating point value.

The dialog displays a text message and contain a text control as well as two push button controls - OK and Cancel. The text control is used to enter the numeric value. The initial value displayed in the text control is the numeric value contained in the parameter value. The OK and Cancel push button controls are used to dismiss the dialog.

C NOTES

The dialog is created as a child of the specified dialog parent. If you specify parent as MG_NULL, the dialog is created as a child of the Creator main window. If the user enters a valid numeric value and chooses the OK button to dismiss the dialog, the value is returned in the parameter value.

PYTHON NOTES

Currently the parent parameter is reserved for future enhancement and should be specified as None. For now the dialog is created as a child of the main Creator main window. If the user enters a valid numeric value and chooses the OK button to dismiss the dialog, the value is returned.

PARAMETERS

parent
the dialog parent to attach the prompt dialog
messageText
a text string that is displayed in the dialog as a message to the user
value
on input, the address of the initial double value to display in the text control, on output receives the double value entered

RETURNS

mgstatus
Use MSTAT_ISOK to check return value for success. Success indicates that the value entered was a valid double value and the user dismissed the dialog using the OK push button.
number
If function is successful, contains the number entered by the user.

EXAMPLE

The following examples (in C and Python) prompt the user to enter a double precision floating point number (default value 4.0).

C EXAMPLE

double number = 4.0;
mgstatus status = mgPromptDialogDouble (MG_NULL, "Enter Number", &number);
if (MSTAT_ISOK (status)) {
   printf ("Number: %f", number);
}

PYTHON EXAMPLE

status,number = mgPromptDialogDouble (None, "Enter Number", 4.0)
if (MSTAT_ISOK (status)):
   print "Number:", number

Level 4
