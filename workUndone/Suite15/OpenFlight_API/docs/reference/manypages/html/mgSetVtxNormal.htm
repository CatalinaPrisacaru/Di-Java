mgSetVtxNormal

FUNCTION NAME

mgSetVtxNormal - sets the normal vector of a vertex record.

SYNOPSIS       (C and Python)

mgbool mgSetVtxNormal (mgrec*rec,
floati,
floatj,
floatk );

DESCRIPTION

mgSetVtxNormal sets the i, j, and k values of the vertex normal vector of a vertex record rec.

PARAMETERS

rec
a vertex record
i
i component of the normal vector
j
j component of the normal vector
k
k component of the normal vector

RETURNS

Returns MG_TRUE if the vertex normal was set, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
