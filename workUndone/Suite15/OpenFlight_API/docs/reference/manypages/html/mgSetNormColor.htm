mgSetNormColor

FUNCTION NAME

mgSetNormColor - sets attribute values in a fltNormColor record.

SYNOPSIS       (C and Python)

mgbool mgSetNormColor (mgrec*rec,
mgcodenormColorCode,
floatred,
floatgreen,
floatblue );

DESCRIPTION

Given a record, rec, and the name of one of its fltNormColor attribute records, normColorCode, mgSetNormColor sets the red, green, and blue attribute values in the fltNormColor attribute record.

PARAMETERS

rec
the record whose fltNormColor record is to be set
normColorCode
the name of the fltNormColor attribute code of rec
red
the red value to set in the fltNormColor attribute record
green
the green value to set in the fltNormColor attribute record
blue
the blue value to set in the fltNormColor attribute record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
