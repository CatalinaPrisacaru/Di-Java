mgEditorGetPluginTool

FUNCTION NAME

mgEditorGetPluginTool - gets the plug-in tool associated with an editor context.

C SPECIFICATION       (Not Available For Python)

mgplugintool mgEditorGetPluginTool (mgeditorcontexteditorContext );

DESCRIPTION

mgEditorGetPluginTool returns the plug-in tool that is associated with the specified editor context, editorContext.

PARAMETERS

editorContext
the editor context to get associated plug-in tool for

RETURNS

Returns the plug-in tool associated with this tool instance (if valid), MG_NULL otherwise.

ACCESS LEVEL

Level 4
