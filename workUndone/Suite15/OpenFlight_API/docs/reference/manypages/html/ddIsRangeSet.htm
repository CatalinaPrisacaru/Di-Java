ddIsRangeSet

FUNCTION NAME

ddIsRangeSet - checks if a data code has a range defined.

SYNOPSIS       (C and Python)

mgbool ddIsRangeSet (mgcodedcode );

DESCRIPTION

ddIsRangeSet checks if the data associated with dcode has a minimum and/or maximum value defined.

PARAMETERS

dcode
the data code

RETURNS

Returns MG_TRUE if a minimum and/or a maximum value was found, MG_FALSE otherwise.

ACCESS LEVEL

Level 1
