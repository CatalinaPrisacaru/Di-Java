mggetheadercodefunc

CALLBACK NAME

mggetheadercodefunc - Get Header Code function

SYNOPSIS

mgbool (*mggetheadercodefunc) ( mgrec* db,
mgcode* code );

DESCRIPTION

This is the signature for get header code (extension node helper) functions. If your data extension declares one or more new node types, assign a get header code function to your data extension of this form.

Get header code functions are called when nodes of your new type are created and are responsible for returning the record code corresponding to your header extension record.

PARAMETERS

db
the database into which the new node was created
code
address of value to receive record code

RETURNS

Return MG_TRUE if successful, MG_FALSE otherwise. If successful, the function returns the record code in the output parameter, code.
