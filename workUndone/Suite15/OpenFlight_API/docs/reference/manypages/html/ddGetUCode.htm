ddGetUCode

FUNCTION NAME

ddGetUCode - composes a unique code from a local code and Site ID.

SYNOPSIS       (C and Python)

mgcode ddGetUCode (const char*siteId,
mgcodelocalCode );

DESCRIPTION

ddGetUCode returns a unique code from the specified local code localCode and Site ID siteId.

This function is the inverse of ddGetMyCode.

PARAMETERS

siteId
the extension site’s ID string
localCode
the extension site’s own code

RETURNS

Returns the unique code if valid, 0 otherwise.

ACCESS LEVEL

Level 1
