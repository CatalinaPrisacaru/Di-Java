mgMeshSetVtxColorRGBA

FUNCTION NAME

mgMeshSetVtxColorRGBA - sets the RGBA color values for a vertex in a mesh.

SYNOPSIS       (C and Python)

mgbool mgMeshSetVtxColorRGBA (mgrec*rec,
intvtxIndex,
shortred,
shortgreen,
shortblue,
shortalpha );

DESCRIPTION

This function sets the red, green, blue, and alpha color values of the vertex at index vtxIndex in the vertex pool of mesh node rec.

Note: The first vertex in the mesh vertex pool is at index 0.

PARAMETERS

rec
the fltMesh node
vtxIndex
the vertex number in the mesh starting at 0
red
the red component
green
the green component
blue
the blue component
alpha
the alpha component

RETURNS

Returns MG_TRUE if the vertex data was set successfully, MG_FALSE otherwise. This function will fail if rec is not a valid mesh node, if the vertex at vtxIndex does not exist in the mesh, or if the vertex mask for the mesh does not specify that the vertices have color values.

ACCESS LEVEL

Level 2
