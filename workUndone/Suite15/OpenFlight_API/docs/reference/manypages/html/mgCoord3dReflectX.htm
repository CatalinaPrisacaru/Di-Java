mgCoord3dReflectX

FUNCTION NAME

mgCoord3dReflectX - calculates 3D coordinates reflected about the X axis.

SYNOPSIS       (C and Python)

mgcoord3d mgCoord3dReflectX (const mgcoord3d*coord );

DESCRIPTION

mgCoord3dReflectX calculates and returns 3D double precision floating point coordinates reflected about the X axis or pitched 180 degrees.

The original coordinate coord is not affected.

Note: For efficiency, coord is not verified before being dereferenced. You must not pass as MG_NULL.

PARAMETERS

coord
address of coordinates

RETURNS

Returns a double precision 3D coordinate record.

ACCESS LEVEL

Level 1
