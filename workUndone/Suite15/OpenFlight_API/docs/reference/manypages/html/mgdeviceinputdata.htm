mgdeviceinputdata

STRUCT NAME

mgdeviceinputdata - Device specific Input data

SYNOPSIS

typedef struct  {
   mginputdevice inputDevice;
   void* deviceData;
   int button;
   mgmousestate mouseEvent;
   const char* toolName;
} mgdeviceinputdata;

MEMBERS

inputDevice
mginputdevice - the input device originating the event
deviceData
void* - pointer to device specific data
button
int - button on device that was pressed
mouseEvent
mgmousestate - state of the button
toolName
const char* - pointer to the name of the input device

DESCRIPTION

When device input is reported to an editor tool via the corresponding device input function, a pointer to a record of this type is passed to the device function that describes the device input.
