mgEditorSetPrompt

FUNCTION NAME

mgEditorSetPrompt - sets the prompt string within an editor tool dialog.

C SPECIFICATION       (Not Available For Python)

mgstatus mgEditorSetPrompt (mgeditorcontexteditorContext,
const char*promptString );

DESCRIPTION

mgEditorSetPrompt copies the prompt string, promptString, into the prompt text field of the dialog associated with the specified editor context, editorContext.

If an editor tool instance creates a dialog that contains a text control whose identifier is MGID_PROMPT, mgEditorSetPrompt can be used as a convenience function to load a string into that text control.

PARAMETERS

editorContext
the editor context whose associated dialog prompt is to be set
promptString
the prompt string

RETURNS

Use MSTAT_ISOK to check return value for success.

ACCESS LEVEL

Level 4
