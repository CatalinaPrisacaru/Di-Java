mgDeleteSound

FUNCTION NAME

mgDeleteSound - deletes an entry, defined by the sound’s index, from a database’s sound palette.

SYNOPSIS       (C and Python)

mgbool mgDeleteSound (mgrec*db,
intindex );

DESCRIPTION

mgDeleteSound deletes the entry defined by the sound’s index from the sound palette associated with database node db. If no sound entry is found matching index, the sound palette remains unchanged.

PARAMETERS

db
the database
index
the index of the sound record

RETURNS

Returns MG_TRUE if successful, MG_FALSE otherwise.

ACCESS LEVEL

Level 2
