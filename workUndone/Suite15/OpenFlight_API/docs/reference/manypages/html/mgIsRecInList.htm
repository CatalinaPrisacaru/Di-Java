mgIsRecInList

FUNCTION NAME

mgIsRecInList - determines if a node is in a record list.

SYNOPSIS       (C and Python)

mgbool mgIsRecInList (mgreclistrecList,
mgrec*node );

DESCRIPTION

mgIsRecInList determines whether or not the specified node is contained in the specified record list recList.

This function does not affect the traversal pointer of the record list.

PARAMETERS

recList
the record list to check
node
node to look for

RETURNS

Returns MG_TRUE if the item is found in the record list, MG_FALSE otherwise.

ACCESS LEVEL

Level 1

mgResetRecList, mgGetNthRecInList
