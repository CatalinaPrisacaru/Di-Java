mgGetTextureTexels

FUNCTION NAME

mgGetTextureTexels - gets the texels for a texture palette entry.

SYNOPSIS       (C and Python)

unsigned char* mgGetTextureTexels (mgrec*db,
intindex );

DESCRIPTION

mgGetTextureTexels gets the texels for a texture palette entry at the specified index contained in the texture palette of database db.

Note: The texels returned are the actual texels, not a copy. For that reason, you should not modify them, free them or assign them to another texture in the database using mgSetTextureTexels. Doing any of these will lead to unexpected results and may cause a crash.

See mgReadImage for a description of the memory layout for the texels returned.

PARAMETERS

db
the database
index
the index of the texture palette entry

RETURNS

Returns the pointer to the texels of the texture palette entry if found, MG_NULL otherwise. This pointer should not be freed by the caller.

ACCESS LEVEL

Level 1

SEE ALSO

mgSetTextureTexels, mgReadImage
