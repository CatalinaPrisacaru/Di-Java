mgNameOfLightPointAnimation

FUNCTION NAME

mgNameOfLightPointAnimation - gets the name of an indexed entry in the light point appearance palette.

SYNOPSIS       (C and Python)

char* mgNameOfLightPointAnimation (mgrec*db,
intindex );

DESCRIPTION

mgNameOfLightPointAnimation returns the name of the entry specified by index in the light point appearance palette for database db.

C NOTES

The string returned is dynamically allocated and must be deallocated by the caller when it is no longer needed using mgFree.

PYTHON NOTES

It is not necessary to free the returned string in Python.

PARAMETERS

db
the database
index
the index of the entry to look up

RETURNS

Returns the name of the entry if found, MG_NULL otherwise.

ACCESS LEVEL

Level 1
