mgTextureGetSavePathType

FUNCTION NAME

mgTextureGetSavePathType - gets the save path type for the textures in a database.

SYNOPSIS       (C and Python)

mgsavepathtype mgTextureGetSavePathType (mgrec*db );

DESCRIPTION

mgTextureGetSavePathType returns how the names of the textures in a database db will be saved when they are written to the database file on disk.

The save names of textures in a database are the actual names that are saved in the OpenFlight database file. The save names may or may not include the full path specifications of the texture files. The may have partial path information (relative) or may include no path information.

The possible return values are:
MSPT_ABSOLUTE - The save names will include the full path specification (if known).
MSPT_RELATIVE - The save names will be relative names and will begin with the character sequence "./".
MSPT_NOPATH - The save names will include no path information and will simply be the name part of the file name.
MSPT_UNKNOWN - The save name type could not be determined.

PARAMETERS

db
the database

RETURNS

Returns MSPT_ABSOLUTE, MSPT_RELATIVE or MSPT_NOPATH if successful, MSPT_UNKNOWN otherwise.

ACCESS LEVEL

Level 1
